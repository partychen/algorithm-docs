## 一、什么是矩阵树定理

**矩阵树定理（Matrix Tree Theorem）**，又称 **Kirchhoff 定理**，是图论与线性代数之间最优美的桥梁之一。它揭示了一个令人惊叹的事实：**一个图的生成树个数，等于其 Kirchhoff 矩阵任意一个余子式的行列式**。

### Kirchhoff 定理的直观理解

假设你有一个 $n$ 个节点的连通无向图，你想知道它有多少棵不同的生成树。暴力枚举的复杂度是指数级的，但 Kirchhoff 定理告诉我们：只需构造一个 $n \times n$ 的矩阵，删去任意一行一列，再求剩余矩阵的行列式，就能得到生成树的数量。

```
直觉理解：
  图的 Kirchhoff 矩阵 = 度数矩阵 - 邻接矩阵
  删去第 i 行第 i 列后，得到 (n-1) × (n-1) 的矩阵
  该矩阵的行列式 = 图的生成树个数

  例如完全图 K4（4 个点两两相连）：
  Kirchhoff 矩阵 L =
  [ 3  -1  -1  -1]
  [-1   3  -1  -1]
  [-1  -1   3  -1]
  [-1  -1  -1   3]

  删去第 1 行第 1 列后：
  [ 3  -1  -1]
  [-1   3  -1]
  [-1  -1   3]

  det = 3*(9-1) - (-1)*(-3-1) + (-1)*(1+3)
      = 24 - 4 - 4 = 16
  K4 恰好有 16 棵不同的生成树
```

### 生成树计数问题

生成树计数在组合数学、电路分析、网络可靠性等领域有广泛应用。经典例子：

| 图 | 生成树数量 | 公式 |
|---|-----------|------|
| 完全图 $K_n$ | $n^{n-2}$（Cayley 公式） | $K_4$ 有 $4^2 = 16$ 棵 |
| 环图 $C_n$ | $n$ | $C_5$ 有 $5$ 棵 |
| 完全二部图 $K_{m,n}$ | $m^{n-1} \cdot n^{m-1}$ | $K_{3,3}$ 有 $3^2 \cdot 3^2 = 81$ 棵 |

### 与线性代数的联系

矩阵树定理将图论中的组合计数问题，转化为线性代数中的**行列式计算**问题。这种转化的优势在于：

- 行列式可以用**高斯消元**在 $O(n^3)$ 时间内求解
- 在模质数下可以求逆元，在模合数下可以用辗转相除
- 通过**变元矩阵树定理**，还能处理边权乘积之和等更复杂的计数

## 二、核心思想与前置知识

### 2.1 Kirchhoff 矩阵（拉普拉斯矩阵）

**定义**：对于一个 $n$ 个节点的无向图 $G$，其 **Kirchhoff 矩阵**（也叫**拉普拉斯矩阵** $L$）定义为：

$$L = D - A$$

其中：
- $D$ 是**度数矩阵**（对角矩阵），$D_{ii} = \deg(i)$
- $A$ 是**邻接矩阵**，$A_{ij}$ 为 $i$ 和 $j$ 之间的边数

具体来说：

$$L_{ij} = \begin{cases} \deg(i) & \text{if } i = j \ -(\text{边 } (i,j) \text{ 的重数}) & \text{if } i \neq j \end{cases}$$

**性质**：
- $L$ 是半正定矩阵，所有特征值 $\ge 0$
- 每行每列之和为 $0$（因此 $\text{rank}(L) \le n - 1$）
- 最小特征值为 $0$，对应特征向量为全 $1$ 向量

```
构造例子（4 个点的图，边：1-2, 1-3, 2-3, 2-4, 3-4）：

度数矩阵 D:          邻接矩阵 A:          Kirchhoff 矩阵 L = D - A:
[2 0 0 0]            [0 1 1 0]            [ 2 -1 -1  0]
[0 3 0 0]            [1 0 1 1]            [-1  3 -1 -1]
[0 0 3 0]            [1 1 0 1]            [-1 -1  3 -1]
[0 0 0 2]            [0 1 1 0]            [ 0 -1 -1  2]
```

**重边处理**：若 $i, j$ 之间有 $k$ 条边，则 $A_{ij} = k$，$\deg(i)$ 也对应增加 $k$。

**自环处理**：自环对 Kirchhoff 矩阵没有影响（度数和邻接矩阵对应项抵消），也不影响生成树计数。

### 2.2 矩阵树定理的陈述

**定理（Kirchhoff's Matrix Tree Theorem）**：

对于 $n$ 个节点的无向图 $G$，设 $L$ 为其 Kirchhoff 矩阵。设 $L_{ii}$ 为删去第 $i$ 行第 $i$ 列后得到的 $(n-1) \times (n-1)$ 子矩阵（$L$ 的第 $i$ 个**主余子式**），则：

$$t(G) = \det(L_{ii})$$

其中 $t(G)$ 为图 $G$ 的生成树个数。

**关键点**：删去哪一行哪一列都可以，结果相同。这是因为 $L$ 的所有 $(n-1)$ 阶主余子式相等。

**等价表述（特征值形式）**：若 $L$ 的特征值为 $0 = \lambda_1 \le \lambda_2 \le \cdots \le \lambda_n$，则：

$$t(G) = \frac{1}{n} \lambda_2 \lambda_3 \cdots \lambda_n$$

### 2.3 高斯消元求行列式

矩阵树定理将生成树计数归结为求行列式，而**高斯消元**是求行列式的标准方法。

**原理**：通过行变换将矩阵化为上三角矩阵，对角线元素之积即为行列式（需注意行交换引入的符号变化）。

```
高斯消元求行列式流程：
1. 设 sign = 1（记录行交换次数的符号）
2. 对第 k = 0, 1, ..., n-2 列：
   a. 选取第 k 列中绝对值最大的元素作为主元（部分选主元）
   b. 若主元为 0，det = 0，返回
   c. 若主元不在第 k 行，交换行，sign = -sign
   d. 用第 k 行消去下方所有行的第 k 列
3. det = sign * 对角线元素之积

时间复杂度：O(n^3)
```

```cpp
// 高斯消元求行列式（实数版）
double gauss_det(vector<vector<double>>& a, int n) {
    double det = 1;
    for (int i = 0; i < n; i++) {
        int pivot = i;
        for (int j = i + 1; j < n; j++)
            if (fabs(a[j][i]) > fabs(a[pivot][i])) pivot = j;
        if (fabs(a[pivot][i]) < 1e-9) return 0;
        if (pivot != i) { swap(a[i], a[pivot]); det = -det; }
        det *= a[i][i];
        for (int j = i + 1; j < n; j++) {
            double f = a[j][i] / a[i][i];
            for (int k = i; k < n; k++)
                a[j][k] -= f * a[i][k];
        }
    }
    return det;
}
```

**模质数版本**：将除法替换为乘逆元即可。

```cpp
// 高斯消元求行列式（模质数版）
long long gauss_det_mod(vector<vector<long long>>& a, int n, long long mod) {
    long long det = 1;
    for (int i = 0; i < n; i++) {
        int pivot = -1;
        for (int j = i; j < n; j++)
            if (a[j][i] % mod != 0) { pivot = j; break; }
        if (pivot == -1) return 0;
        if (pivot != i) { swap(a[i], a[pivot]); det = mod - det; }
        det = det % mod * a[i][i] % mod;
        long long inv = qpow(a[i][i], mod - 2, mod); // 费马小定理求逆元
        for (int j = i + 1; j < n; j++) {
            long long f = a[j][i] % mod * inv % mod;
            for (int k = i; k < n; k++)
                a[j][k] = (a[j][k] - f * a[i][k] % mod + mod) % mod;
        }
    }
    return (det % mod + mod) % mod;
}
```

### 2.4 变元矩阵树定理

**变元矩阵树定理**是矩阵树定理的推广版本。它处理的不再是"有多少棵生成树"，而是"所有生成树的边权乘积之和"。

**定义**：对于一个 $n$ 个节点的无向图 $G$，每条边 $e$ 有权值 $w(e)$。定义**加权 Kirchhoff 矩阵** $L^w$：

$$L^w_{ij} = \begin{cases} \displaystyle\sum_{e \sim i} w(e) & \text{if } i = j \ -w(i, j) & \text{if } i \neq j \end{cases}$$

其中 $e \sim i$ 表示与节点 $i$ 关联的所有边。

**定理**：

$$\det(L^w_{ii}) = \sum_{T \text{ 是生成树}} \prod_{e \in T} w(e)$$

即：加权 Kirchhoff 矩阵任意余子式的行列式 = 所有生成树的边权乘积之和。

```
应用场景：
- 令所有边权 = 1 → 退化为普通矩阵树定理（生成树计数）
- 令边权 = 该边被选中的"概率" → 可计算特定概率问题
- 重建问题：已知每条边存在的概率 p_e，求恰好得到某棵生成树的概率之和
```

### 2.5 有向图版本

矩阵树定理对有向图同样成立，但需要区分**外向树**和**内向树**。

**定义**：
- **外向树**（根向树 / arborescence）：以某个点 $r$ 为根，所有边从父亲指向儿子，即所有边远离根
- **内向树**（叶向根树）：以某个点 $r$ 为根，所有边从儿子指向父亲，即所有边朝向根

**有向图的 Kirchhoff 矩阵**：

对于以 $r$ 为根的**外向树**计数，构造矩阵 $L^{\text{out}}$：

$$L^{\text{out}}_{ij} = \begin{cases} \text{in-deg}(i) & \text{if } i = j \ -(\text{边 } j \to i \text{ 的条数}) & \text{if } i \neq j \end{cases}$$

即度数矩阵用**入度**，邻接矩阵记录的是**入边**。

删去第 $r$ 行第 $r$ 列后，$\det(L^{\text{out}}_{rr})$ = 以 $r$ 为根的外向树个数。

对于以 $r$ 为根的**内向树**计数，构造矩阵 $L^{\text{in}}$：

$$L^{\text{in}}_{ij} = \begin{cases} \text{out-deg}(i) & \text{if } i = j \ -(\text{边 } i \to j \text{ 的条数}) & \text{if } i \neq j \end{cases}$$

即度数矩阵用**出度**。删去第 $r$ 行第 $r$ 列后的行列式 = 以 $r$ 为根的内向树个数。

```
有向图矩阵树定理关键区别：
  无向图：删去任意一行一列，结果相同
  有向图：删去第 r 行第 r 列，得到以 r 为根的树形图个数
         不同的 r 对应不同的结果！

  外向树 → 用入度构造 Kirchhoff 矩阵
  内向树 → 用出度构造 Kirchhoff 矩阵
```

## 三、分类与变体详解

### 3.1 无向图生成树计数

这是矩阵树定理最基本的应用形式。

**步骤**：
1. 构造 $n \times n$ 的 Kirchhoff 矩阵 $L = D - A$
2. 删去第 $1$ 行第 $1$ 列（或任意一行一列）
3. 高斯消元求 $(n-1) \times (n-1)$ 矩阵的行列式
4. 行列式即为生成树个数

```cpp
// 无向图生成树计数核心代码
void add_edge(int u, int v, vector<vector<double>>& L) {
    L[u][u]++; L[v][v]++;
    L[u][v]--; L[v][u]--;
}

double count_spanning_trees(vector<vector<double>>& L, int n) {
    // 删去第 0 行第 0 列，对 [1..n-1] x [1..n-1] 子矩阵求行列式
    vector<vector<double>> a(n - 1, vector<double>(n - 1));
    for (int i = 1; i < n; i++)
        for (int j = 1; j < n; j++)
            a[i-1][j-1] = L[i][j];
    return gauss_det(a, n - 1);
}
```

**时间复杂度**：$O(n^3)$

### 3.2 有向图外向树/内向树计数

**外向树计数**（以 $r$ 为根）：
1. 构造以入度为对角元的 Kirchhoff 矩阵
2. 删去第 $r$ 行第 $r$ 列
3. 求行列式

**内向树计数**（以 $r$ 为根）：
1. 构造以出度为对角元的 Kirchhoff 矩阵
2. 删去第 $r$ 行第 $r$ 列
3. 求行列式

```cpp
// 有向图外向树计数（以 root 为根）
void add_directed_edge(int u, int v, vector<vector<double>>& L) {
    // 边 u -> v，v 的入度 +1
    L[v][v]++;
    L[v][u]--; // L[被指向][来源] -= 1
}

double count_arborescences(vector<vector<double>>& L, int n, int root) {
    // 删去第 root 行第 root 列
    vector<vector<double>> a;
    for (int i = 0; i < n; i++) {
        if (i == root) continue;
        vector<double> row;
        for (int j = 0; j < n; j++)
            if (j != root) row.push_back(L[i][j]);
        a.push_back(row);
    }
    return gauss_det(a, n - 1);
}
```

### 3.3 变元矩阵树定理（边权乘积之和）

当每条边有一个权值 $w_e$ 时，变元矩阵树定理给出所有生成树的边权乘积之和。

**构造方法**：将无权版本中的 $\pm 1$ 替换为 $\pm w_e$：

```cpp
// 变元矩阵树定理：加权 Kirchhoff 矩阵
void add_weighted_edge(int u, int v, double w, vector<vector<double>>& L) {
    L[u][u] += w; L[v][v] += w;
    L[u][v] -= w; L[v][u] -= w;
}

// 结果：det(L 去掉一行一列) = Σ_{生成树 T} Π_{e ∈ T} w(e)
```

**典型应用**：
- 每条边有概率 $p_e$ 存在，求恰好形成某棵生成树的概率之和
- 边权为多项式变量时，可提取组合信息

### 3.4 模质数下的高斯消元

当答案对质数 $p$ 取模时，高斯消元中的除法替换为模逆元运算。

```cpp
long long qpow(long long a, long long b, long long mod) {
    long long res = 1; a %= mod;
    for (; b > 0; b >>= 1) {
        if (b & 1) res = res * a % mod;
        a = a * a % mod;
    }
    return res;
}

// 模质数 p 下的高斯消元求行列式
long long gauss_det_prime(vector<vector<long long>>& a, int n, long long p) {
    long long det = 1;
    for (int i = 0; i < n; i++) {
        int pivot = -1;
        for (int j = i; j < n; j++)
            if (a[j][i] % p != 0) { pivot = j; break; }
        if (pivot == -1) return 0;
        if (pivot != i) { swap(a[i], a[pivot]); det = p - det; }
        det = det % p * (a[i][i] % p) % p;
        long long inv = qpow(a[i][i], p - 2, p);
        for (int j = i + 1; j < n; j++) {
            long long f = a[j][i] % p * inv % p;
            for (int k = i; k < n; k++)
                a[j][k] = (a[j][k] - f % p * a[i][k] % p + p * p) % p;
        }
    }
    return (det % p + p) % p;
}
```

### 3.5 模合数下的处理

当模数不是质数时，不能直接求逆元。此时需要使用**辗转相除法**来完成高斯消元。

**核心思想**：仿照辗转相除求 GCD 的方式，用行变换消元，避免除法。

```cpp
// 模合数下的高斯消元求行列式
long long gauss_det_composite(vector<vector<long long>>& a, int n, long long mod) {
    long long det = 1;
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            while (a[i][i]) {
                long long t = a[j][i] / a[i][i];
                for (int k = i; k < n; k++)
                    a[j][k] = (a[j][k] - t * a[i][k] % mod + mod) % mod;
                swap(a[i], a[j]);
                det = mod - det; // 每次 swap 改变符号
            }
            swap(a[i], a[j]);
            det = mod - det;
        }
        det = det % mod * (a[i][i] % mod) % mod;
        if (!a[i][i]) return 0;
    }
    return (det % mod + mod) % mod;
}
```

**时间复杂度**：$O(n^3 \log V)$，其中 $V$ 是矩阵元素的值域。类似辗转相除，内层 while 循环的总迭代次数为 $O(\log V)$。

## 四、6 道经典例题详解

### 题目 1：P4111 [HEOI2015]小Z的房间

> 给定一个 $n \times m$ 的网格图，其中一些格子是墙壁（用 `*` 表示），其余格子是空地（用 `.` 表示）。相邻（上下左右）的空地之间有一条通道。求这张图的生成树个数，答案对 $10^9$ 取模。

**分析**：这是矩阵树定理的直接应用。将每个空地格子看作一个节点，相邻空地之间连边，构造 Kirchhoff 矩阵后求行列式即可。关键在于模数 $10^9$ 不是质数，需要使用辗转相除法版本的高斯消元。

```
算法选择：矩阵树定理 + 辗转相除高斯消元
时间复杂度：O(n^3 log V)，其中 n 为空地数量（≤ 100）
核心步骤：编号空地 → 建 Kirchhoff 矩阵 → 删一行一列 → 辗转消元求行列式
难度：🟡
```

**C++ 代码**：

```cpp
const long long MOD = 1e9;
int id[10][10], cnt;
long long L[105][105];
int dx[] = {0, 0, 1, -1}, dy[] = {1, -1, 0, 0};

void build(char g[][10], int n, int m) {
    cnt = 0;
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            if (g[i][j] == '.') id[i][j] = cnt++;
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++) if (g[i][j] == '.') {
            for (int d = 0; d < 4; d++) {
                int ni = i + dx[d], nj = j + dy[d];
                if (ni >= 0 && ni < n && nj >= 0 && nj < m && g[ni][nj] == '.') {
                    int u = id[i][j], v = id[ni][nj];
                    if (u < v) { L[u][u]++; L[v][v]++; L[u][v]--; L[v][u]--; }
                }
            }
        }
    // 删去第 0 行第 0 列后用辗转消元求 det，模 MOD
}
```

**核心启示**：
- 网格图的矩阵树定理应用：先编号、再建矩阵，模板化操作
- 模数不是质数时，辗转相除法是关键技巧
- 空地数量不超过 $100$，$O(n^3)$ 消元绰绰有余

### 题目 2：P2144 [FJOI2007]轮状病毒

> 轮状图 $W_n$：中心点 $0$ 连向 $n$ 个外圈点 $1, 2, \ldots, n$，外圈点 $i$ 与 $i+1$（$n$ 与 $1$）相连。求 $W_n$ 的生成树个数。$n \le 100$，答案无需取模（需要高精度）。

**分析**：这道题有两种做法。

**方法一（矩阵树定理 + 递推）**：直接构造 Kirchhoff 矩阵后，利用矩阵的特殊结构（循环矩阵 + 一行一列的修正），可以推导出递推关系。设 $f(n)$ 为答案，可以证明：

$$f(n) = f(n-1) \times 3 - f(n-2) + 2$$

初始值 $f(1) = 1$，$f(2) = 5$。

**方法二**：利用特征值公式，$t(W_n) = \frac{1}{n+1}\prod_{k=1}^{n}\left(3 - 2\cos\frac{2k\pi}{n}\right)$，但不如递推实用。

由于答案增长很快且不取模，需要**高精度运算**。

```
算法选择：递推 f(n) = 3*f(n-1) - f(n-2) + 2 + 高精度
时间复杂度：O(n * L)，L 为高精度数位长度
核心步骤：推导递推式 → 高精度实现加减乘
难度：🟠
```

**C++ 代码**：

```cpp
// 高精度版本核心递推
// f[1] = 1, f[2] = 5
// f[n] = 3*f[n-1] - f[n-2] + 2
BigInt f_prev2(1), f_prev1(5);
for (int i = 3; i <= n; i++) {
    BigInt cur = f_prev1 * 3 - f_prev2 + 2;
    f_prev2 = f_prev1;
    f_prev1 = cur;
}
// 答案为 f_prev1（当 n >= 2）
```

**核心启示**：
- 特殊结构的图（轮状图、循环图等），Kirchhoff 矩阵有特殊形式，常可推出递推公式
- 当答案不取模时，高精度运算不可避免
- 推导递推关系：先写出 Kirchhoff 矩阵，利用行列式的展开/递推性质得到规律

### 题目 3：P3317 [SDOI2014]重建

> 给定 $n$ 个节点的完全图，每条边 $(i, j)$ 有存在概率 $p_{ij}$。每条边独立地以概率 $p_{ij}$ 存在或以概率 $1 - p_{ij}$ 不存在。求最终图恰好形成一棵树（即恰好剩下 $n-1$ 条边且连通）的概率。

**分析**：设 $S$ 为所有可能的边集，$T$ 为一棵生成树。我们需要求：

$$\text{ans} = \sum_{T \text{ 是生成树}} \prod_{e \in T} p_e \cdot \prod_{e \notin T} (1 - p_e)$$

将 $\prod_{e \notin T}(1-p_e)$ 改写为 $\frac{\prod_{\text{all } e}(1-p_e)}{\prod_{e \in T}(1-p_e)}$，则：

$$\text{ans} = \prod_{\text{all } e}(1-p_e) \cdot \sum_T \prod_{e \in T} \frac{p_e}{1-p_e}$$

后半部分 $\sum_T \prod_{e \in T} \frac{p_e}{1-p_e}$ 正是**变元矩阵树定理**的形式！令边 $(i,j)$ 的权值 $w_{ij} = \frac{p_{ij}}{1 - p_{ij}}$，构造加权 Kirchhoff 矩阵求行列式，再乘以所有边 $(1-p_e)$ 的乘积即可。

特殊处理：当 $p_{ij} = 1$ 时 $w_{ij}$ 无穷大，需要将 $p_{ij}$ 替换为 $1 - \varepsilon$（如 $1 - 10^{-8}$）避免除零。

```
算法选择：变元矩阵树定理
时间复杂度：O(n^3)
核心步骤：计算 w = p/(1-p) → 建加权 Kirchhoff 矩阵 → 行列式 × Π(1-p)
难度：🔴
```

**C++ 代码**：

```cpp
double p[MAXN][MAXN], L[MAXN][MAXN];

double solve(int n) {
    double all = 1.0; // Π(1 - p_e)
    for (int i = 0; i < n; i++)
        for (int j = i + 1; j < n; j++) {
            if (p[i][j] > 1 - 1e-8) p[i][j] = 1 - 1e-8;
            double w = p[i][j] / (1.0 - p[i][j]);
            all *= (1.0 - p[i][j]);
            L[i][i] += w; L[j][j] += w;
            L[i][j] -= w; L[j][i] -= w;
        }
    // 删去第 0 行第 0 列，高斯消元求 (n-1)×(n-1) 行列式
    double det = gauss_det(/* L[1..n-1][1..n-1] */, n - 1);
    return all * det;
}
```

**核心启示**：
- 变元矩阵树定理是处理"边权乘积之和"类问题的利器
- "恰好形成一棵树的概率" = 全局因子 $\times$ 变元行列式
- 浮点精度问题：$p=1$ 时需特殊处理，用 $1-\varepsilon$ 替代

### 题目 4：P4208 [JSOI2008]最小生成树计数

> 给定一个 $n$ 个点 $m$ 条边的无向图，每条边有权值。求**最小生成树的个数**，答案对 $31011$ 取模。$n \le 100$，$m \le 1000$。

**分析**：最小生成树有一个重要性质：**对于所有 MST，每种边权的边被使用的条数相同**。也就是说，如果我们把边按权值分组，每组中被 MST 选中的边数是固定的。

因此，我们可以对每种边权独立计算方案数，最后乘起来。对于同一权值的边，我们需要从中选出恰好 $k$ 条（$k$ 固定），使得与已选的更小权值边一起不形成环。这等价于在某个缩合后的图上计数生成森林——本质上是矩阵树定理的应用。

```
算法选择：Kruskal 分组 + 矩阵树定理
时间复杂度：O(m log m + Σ c_i^3)，c_i 为每组边对应的连通分量数
核心步骤：
  1. 边按权值排序分组
  2. 对每组权值的边：
     a. 将之前权值更小的边对应的连通分量缩点
     b. 在缩点后的图上，用矩阵树定理计算该组边形成生成森林的方案数
  3. 所有组的方案数相乘
难度：🟣
```

**C++ 代码**：

```cpp
const int MOD = 31011;
// 将同权值边分组处理
long long solve(int n, vector<Edge>& edges) {
    sort(edges.begin(), edges.end()); // 按权排序
    long long ans = 1;
    // 按权值分组，对每组边：
    //   缩已确定的连通分量 → 建该组边的 Kirchhoff 子矩阵
    //   对每个连通分量内部用矩阵树定理计数 → 乘入 ans
    for (int l = 0, r; l < m; l = r) {
        r = l;
        while (r < m && edges[r].w == edges[l].w) r++;
        // edges[l..r-1] 是同权值的一组边
        // 缩点后在子图上建 Kirchhoff 矩阵，对每个连通块求行列式
        // ans = ans * Π(每个连通块的行列式) % MOD
    }
    return ans % MOD;
}
```

**核心启示**：
- MST 计数的核心性质：**同权值边选中数量固定** —— 这允许分组独立计数
- 每组边的计数本质是在缩点后的子图上应用矩阵树定理
- 模数 $31011 = 3 \times 10337$ 不是质数，但由于连通分量规模很小，也可先在实数下算再取模（或 CRT）

### 题目 5：SPOJ DETER3 行列式求值（补充经典题）

> 给定 $n \times n$ 的整数矩阵和模数 $M$（$M$ 不一定是质数），求矩阵行列式模 $M$ 的值。

**分析**：这是模合数下求行列式的模板题，也是矩阵树定理在非质数模下的基础设施。核心方法是辗转相除法高斯消元。

```
算法选择：辗转相除法高斯消元
时间复杂度：O(n^3 log V)
核心步骤：逐列消元，用辗转相除代替除法
难度：🟡
```

**C++ 代码**：

```cpp
long long det_mod(vector<vector<long long>>& a, int n, long long mod) {
    long long res = 1;
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            while (a[i][i]) {
                long long t = a[j][i] / a[i][i];
                for (int k = i; k < n; k++)
                    a[j][k] = (a[j][k] - t * a[i][k] % mod + mod) % mod;
                swap(a[i], a[j]); res = mod - res;
            }
            swap(a[i], a[j]); res = mod - res;
        }
        res = res % mod * (a[i][i] % mod) % mod;
    }
    return (res + mod) % mod;
}
```

**核心启示**：
- 辗转相除法是模合数下求行列式的通用方法
- 每次 swap 需翻转行列式符号
- 这是所有模合数矩阵树定理题目的底层工具

### 题目 6：有向图树形图计数（补充经典题）

> 给定一个 $n$ 个节点 $m$ 条边的有向图，求以节点 $1$ 为根的外向树（根向树形图）个数，答案对质数 $p$ 取模。

**分析**：有向图树形图计数是矩阵树定理在有向场景下的直接应用。构造以入度为对角元的 Kirchhoff 矩阵，删去第 $1$ 行第 $1$ 列（根所在的行列），求行列式。

```
算法选择：有向图矩阵树定理 + 模质数高斯消元
时间复杂度：O(n^3)
核心步骤：建入度 Kirchhoff 矩阵 → 删根行列 → 模质数消元
难度：🟠
```

**C++ 代码**：

```cpp
long long L[MAXN][MAXN]; // Kirchhoff 矩阵

void add_edge(int u, int v) { // 有向边 u -> v
    L[v][v]++; // v 的入度 +1
    L[v][u]--; // L[v][u] -= 1
}

long long solve(int n, long long mod) {
    // 删去第 0 行第 0 列（根为节点 0）
    // 对 L[1..n-1][1..n-1] 求行列式
    vector<vector<long long>> a(n - 1, vector<long long>(n - 1));
    for (int i = 1; i < n; i++)
        for (int j = 1; j < n; j++)
            a[i-1][j-1] = (L[i][j] % mod + mod) % mod;
    return gauss_det_prime(a, n - 1, mod);
}
```

**核心启示**：
- 有向图版本的关键：**外向树用入度，内向树用出度**
- 删去的行列对应的是**根节点**，不同根结果不同
- 在质数模下直接求逆元消元即可

## 五、常见技巧与思路总结

### 5.1 构造 Kirchhoff 矩阵的注意事项

```
1. 重边：两点间有 k 条边 → A[i][j] = k, deg[i] += k
2. 自环：不影响 Kirchhoff 矩阵（度数和邻接项抵消）
3. 连通性：若图不连通 → 行列式为 0（无生成树）
4. 删哪一行一列：无向图任意，有向图删根节点对应的行列
5. 编号：从 0 或从 1 开始都可以，但要统一
```

### 5.2 高斯消元的模数处理

| 模数类型 | 方法 | 时间复杂度 |
|---------|------|-----------|
| 无模（精确值） | 浮点高斯消元 / 高精度 | $O(n^3)$ / $O(n^3 L)$ |
| 质数 $p$ | 逆元高斯消元（费马小定理） | $O(n^3)$ |
| 合数 $M$ | 辗转相除法消元 | $O(n^3 \log V)$ |
| 合数 $M$（可分解） | CRT + 对每个质因子分别求 | $O(n^3 \cdot k)$ |

### 5.3 变元矩阵树定理的应用范式

```
当题目涉及"所有生成树的某种权值统计"时，考虑变元矩阵树定理：

模式 1：概率问题
  每条边存在概率 p_e → w_e = p_e / (1-p_e)
  答案 = det × Π(1-p_e)

模式 2：边权乘积之和
  直接令 w_e 为边权
  det 即为 Σ_T Π_{e∈T} w_e

模式 3：计数（退化情况）
  所有 w_e = 1 → 退化为普通矩阵树定理
```

### 5.4 MST 计数的核心思路

```
MST 计数三步走：
1. 先用 Kruskal 确定每种边权需要选多少条边
2. 对同权值边分组，每组内用矩阵树定理计算方案数
3. 各组方案数相乘

核心性质：
  所有 MST 中，权值为 w 的边恰好用了 k_w 条（对所有 MST 相同）
  → 乘法原理，独立计数再相乘
```

### 5.5 特殊图的递推公式

对于结构规则的图，Kirchhoff 矩阵常有特殊形式，可以推导出递推关系或封闭公式：

| 图类型 | 生成树数量 | 公式/递推 |
|-------|-----------|----------|
| 完全图 $K_n$ | $n^{n-2}$ | Cayley 公式 |
| 环图 $C_n$ | $n$ | 显然 |
| 轮状图 $W_n$ | $f(n) = 3f(n-1) - f(n-2) + 2$ | $f(1)=1, f(2)=5$ |
| 完全二部图 $K_{m,n}$ | $m^{n-1} \cdot n^{m-1}$ | 特征值乘积 |
| 梯子图 $L_n$ | $\frac{(2+\sqrt{3})^n - (2-\sqrt{3})^n}{2\sqrt{3}}$ | 递推 $f(n) = 4f(n-1) - f(n-2)$ |

## 七、终极总结与归纳

### 7.1 矩阵树定理的本质

**矩阵树定理的本质是：将图的组合结构编码进一个代数对象（Kirchhoff 矩阵），使得行列式这一代数运算恰好等于组合计数的结果。** 它是组合数学与线性代数深刻联系的典范——看似毫无关联的"图的生成树个数"和"矩阵的行列式"，被 Kirchhoff 矩阵完美地联系在一起。

### 7.2 核心要点

1. **Kirchhoff 矩阵 = 度数矩阵 - 邻接矩阵**：构造方法简单，但蕴含丰富信息
2. **生成树个数 = 任意余子式的行列式**：无向图删哪行哪列都一样，有向图删根所在行列
3. **高斯消元是核心工具**：质数模用逆元，合数模用辗转相除
4. **变元矩阵树定理处理带权问题**：权 $w_e$ 替换 $1$，行列式 = 边权乘积之和
5. **有向版本区分外向树和内向树**：外向树用入度，内向树用出度
6. **MST 计数分组独立**：同权值边选中数量固定，可分组用矩阵树定理

