## 一、什么是树上问题

**树** 是图论中最重要的结构之一。一棵有 $n$ 个节点的树恰好有 $n-1$ 条边，并且任意两个节点之间有且仅有一条简单路径。

### 树的核心性质

```
        1
       / \
      2   3
     / \   \
    4   5   6
```

- **连通**：任意两个节点之间存在路径
- **无环**：不存在环路，删掉任意一条边图就不连通
- **边数** = 节点数 - 1
- 任意两点间 **路径唯一**

### 与一般图的区别

| 维度       | 一般图              | 树                   |
|-----------|--------------------|-----------------------|
| 是否有环    | 可能有环            | 无环                  |
| 边数       | 任意               | 恰好 $n-1$ 条         |
| 路径唯一性  | 可能有多条路径       | 任意两点路径唯一        |
| 遍历方式    | BFS/DFS + 判重      | DFS 天然不回头          |

### 树的分类

- **无根树**：没有指定根节点，边没有方向
- **有根树**：指定某个节点为根，具有父子关系
- **二叉树**：每个节点最多有两个子节点（左子、右子）
- **一般树**：子节点数量不受限制

## 二、核心思想与前置知识

### 2.1 树的基本遍历（前序/中序/后序/层序）

对于 **二叉树**，有四种经典遍历方式：

```
前序遍历（Pre-order）：  根 → 左 → 右
中序遍历（In-order）：   左 → 根 → 右
后序遍历（Post-order）：  左 → 右 → 根
层序遍历（Level-order）： 逐层从左到右（BFS）
```

**关键定理**：
- 已知 **前序 + 中序** 可以 **唯一确定** 一棵二叉树
- 已知 **后序 + 中序** 可以 **唯一确定** 一棵二叉树
- 仅知 **前序 + 后序** **不能** 唯一确定（但可以计算可能的方案数）

```cpp
// 二叉树遍历框架
void preorder(int u) {
    if (!u) return;
    visit(u);             // 前序：先访问根
    preorder(lch[u]);
    preorder(rch[u]);
}
void inorder(int u) {
    if (!u) return;
    inorder(lch[u]);
    visit(u);             // 中序：中间访问根
    inorder(rch[u]);
}
void postorder(int u) {
    if (!u) return;
    postorder(lch[u]);
    postorder(rch[u]);
    visit(u);             // 后序：最后访问根
}
```

### 2.2 树的直径（两遍 BFS/DFS 法）

**树的直径** 是树上最远两点之间的距离。

**两遍 BFS/DFS 算法**：
1. 从任意节点 $s$ 出发，找到离 $s$ 最远的节点 $u$
2. 从 $u$ 出发，找到离 $u$ 最远的节点 $v$
3. $u \to v$ 的距离即为树的直径

**正确性证明**：反证法可以证明第一次 BFS/DFS 找到的最远点一定是直径的一个端点。

```cpp
// 两遍 BFS 求树的直径
int dist[MAXN], far_node;
void bfs(int s) {
    memset(dist, -1, sizeof dist);
    queue<int> q;
    q.push(s); dist[s] = 0;
    far_node = s;
    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (int v : adj[u]) {
            if (dist[v] == -1) {
                dist[v] = dist[u] + 1;
                if (dist[v] > dist[far_node]) far_node = v;
                q.push(v);
            }
        }
    }
}
// 调用: bfs(1); bfs(far_node); 答案 = dist[far_node]
```

**DFS 一遍求直径**（维护最长链 + 次长链）：

```cpp
int diameter = 0;
int dfs(int u, int fa) {
    int first = 0, second = 0;
    for (int v : adj[u]) {
        if (v == fa) continue;
        int d = dfs(v, u) + 1;
        if (d >= first) { second = first; first = d; }
        else if (d > second) second = d;
    }
    diameter = max(diameter, first + second);
    return first;
}
```

### 2.3 最近公共祖先（倍增法 / 树链剖分法）

**最近公共祖先 (LCA)**：两个节点 $u, v$ 的所有公共祖先中，深度最大的那个节点。

**倍增法** 是最常用的 LCA 算法，预处理 $O(n \log n)$，单次查询 $O(\log n)$。

核心思想：$fa[u][k]$ 表示节点 $u$ 的第 $2^k$ 级祖先。

$$fa[u][k] = fa[fa[u][k-1]][k-1]$$

```cpp
int dep[MAXN], fa[MAXN][20];

void dfs(int u, int f) {
    dep[u] = dep[f] + 1;
    fa[u][0] = f;
    for (int k = 1; k < 20; k++)
        fa[u][k] = fa[fa[u][k-1]][k-1];
    for (int v : adj[u])
        if (v != f) dfs(v, u);
}

int lca(int u, int v) {
    if (dep[u] < dep[v]) swap(u, v);
    for (int k = 19; k >= 0; k--)
        if (dep[fa[u][k]] >= dep[v]) u = fa[u][k];
    if (u == v) return u;
    for (int k = 19; k >= 0; k--)
        if (fa[u][k] != fa[v][k]) u = fa[u][k], v = fa[v][k];
    return fa[u][0];
}
```

**树上两点距离**：$\text{dist}(u, v) = \text{dep}[u] + \text{dep}[v] - 2 \times \text{dep}[\text{lca}(u,v)]$

### 2.4 通用框架代码

```cpp
// 一般树的 DFS 框架
const int MAXN = 100005;
vector<int> adj[MAXN];

void dfs(int u, int fa) {
    // 预处理当前节点
    for (int v : adj[u]) {
        if (v == fa) continue;
        dfs(v, u);
        // 合并子节点信息
    }
    // 后处理当前节点
}
```

## 三、分类与变体详解

### 3.1 二叉树的遍历与重建

给定两种遍历序列，重建二叉树是经典问题。

**从中序 + 前序重建**：前序的第一个元素是根，在中序中找到根的位置，左侧是左子树、右侧是右子树，递归处理。

**从中序 + 后序重建**：后序的最后一个元素是根，其余同理。

```cpp
// 从中序 + 后序 → 输出前序
// in[] 中序序列, post[] 后序序列
void solve(int inL, int inR, int postL, int postR) {
    if (inL > inR) return;
    char root = post[postR]; // 后序最后一个是根
    cout << root;            // 前序：先输出根
    int k = find(in + inL, in + inR + 1, root) - in;
    int leftSize = k - inL;
    solve(inL, k - 1, postL, postL + leftSize - 1);
    solve(k + 1, inR, postL + leftSize, postR - 1);
}
```

### 3.2 二叉树的性质判断

常见问题包括：
- **对称二叉树**：左子树和右子树镜像对称
- **完全二叉树**判定
- **二叉搜索树**的性质验证

判断对称性的核心：递归比较左子树的左子 与 右子树的右子，左子树的右子 与 右子树的左子。

```cpp
bool check(int a, int b) {
    if (!a && !b) return true;
    if (!a || !b) return false;
    if (val[a] != val[b]) return false;
    return check(lch[a], rch[b]) && check(rch[a], lch[b]);
}
```

### 3.3 树的直径算法

树的直径有两种求法：

1. **两遍 BFS/DFS**：适用于边权非负的情况。从任意点出发找最远点，再从最远点出发找最远点。
2. **一遍 DFS（DP 法）**：维护每个点的最长链和次长链，直径 = $\max(\text{最长链} + \text{次长链})$。适用于有负边权的情况。

**重要性质**：
- 树的直径端点可能不唯一，但直径长度唯一
- 所有直径必过同一个"中心区域"
- 在森林中，需要对每棵树分别求直径

### 3.4 树的中心 / 重心

**树的中心**：直径的中点（或中点所在的边）。树的中心到所有节点的最远距离最小。

**树网的核**：在直径上选择一条长度不超过 $s$ 的路径，使得树上所有节点到该路径的最远距离最小。

**树的重心**：删除该节点后，最大子树最小的节点。重心满足一个重要性质：所有节点到重心的距离之和最小。

### 3.5 LCA 倍增法

倍增法核心步骤：

1. **预处理**：DFS 计算每个节点的深度和 $fa[u][k]$
2. **查询 LCA(u, v)**：
   - 先让深度大的节点跳到与另一节点同层
   - 然后两个节点一起往上跳，直到汇合

**时间复杂度**：预处理 $O(n \log n)$，单次查询 $O(\log n)$。

### 3.6 LCA 的应用（树上距离、路径查询等）

LCA 的常见应用：

- **树上两点距离**：$\text{dist}(u, v) = \text{dep}[u] + \text{dep}[v] - 2 \cdot \text{dep}[\text{lca}(u,v)]$
- **树上路径查询**：任意路径 $u \to v$ 可以拆分为 $u \to \text{lca}$ 和 $\text{lca} \to v$ 两段
- **三点汇聚**：三个点 $a, b, c$ 的最优集合点是 $\text{lca}(a,b)$、$\text{lca}(b,c)$、$\text{lca}(a,c)$ 中深度最大的那个
- **判断祖先关系**：$u$ 是 $v$ 的祖先当且仅当 $\text{lca}(u, v) = u$

### 题目 1：FBI 树（洛谷 P1087）

> 给定一个长度为 $2^n$ 的 01 串，构建一棵 FBI 树：叶节点对应串中的每个字符，内部节点对应一个子串。若子串全为 0 则标记 B，全为 1 则标记 I，否则标记 F。输出这棵树的后序遍历。

**分析**：

```
这是一道递归建树 + 后序遍历的入门题。
将字符串不断二分，递归建立左右子树，最后按后序遍历输出节点标记。
关键：先递归左右子树，最后输出当前节点标记（后序遍历的定义）。
```

**C++ 代码**：

```cpp
char s[1050];
void solve(int l, int r) {
    if (l < r) {
        int mid = (l + r) / 2;
        solve(l, mid);
        solve(mid + 1, r);
    }
    bool has0 = false, has1 = false;
    for (int i = l; i <= r; i++) {
        if (s[i] == '0') has0 = true;
        else has1 = true;
    }
    if (has0 && has1) printf("F");
    else if (has0) printf("B");
    else printf("I");
}
```

**核心启示**：递归天然对应二叉树的结构，后序遍历 = 先递归处理子树再处理根。

### 题目 2：求先序排列（洛谷 P1030）

> 已知一棵二叉树的中序遍历和后序遍历，求其前序遍历。

**分析**：

```
经典的二叉树重建问题。
后序遍历的最后一个字符是根节点，在中序遍历中找到根的位置，
左侧是左子树的中序，右侧是右子树的中序。
递归处理，按前序顺序输出即可。
```

**C++ 代码**：

```cpp
string in_s, post_s;
void solve(int inL, int inR, int poL, int poR) {
    if (inL > inR) return;
    char root = post_s[poR];
    cout << root;
    int k = in_s.find(root);
    int leftLen = k - inL;
    solve(inL, k - 1, poL, poL + leftLen - 1);
    solve(k + 1, inR, poL + leftLen, poR - 1);
}
```

**核心启示**：中序 + 后序（或前序）可以唯一确定一棵二叉树，后序末尾 = 前序开头 = 根。

### 题目 3：新二叉树（洛谷 P1305）

> 给定 $n$ 个节点，每个节点给出其左子和右子的编号，输出这棵二叉树的前序遍历。

**分析**：

```
纯模拟题。用数组存储每个节点的左右子节点，然后从根开始递归前序遍历。
需要找到根节点：根节点是没有被任何节点指向的节点。
```

**C++ 代码**：

```cpp
char lch[128], rch[128];
bool has_parent[128];

void preorder(char u) {
    if (u == '*') return;
    putchar(u);
    preorder(lch[u]);
    preorder(rch[u]);
}
// 读入后找到 has_parent[x]==false 的 x 即为根
```

**核心启示**：存储二叉树可以用数组 `lch[]`/`rch[]`，找根节点通过入度判断。

### 题目 4：遍历问题（洛谷 P1229）

> 已知一棵二叉树的前序遍历和后序遍历，求该二叉树可能有多少种不同的中序遍历。

**分析**：

```
前序 + 后序不能唯一确定二叉树，原因在于只有一个子节点时无法区分它是左子还是右子。
每当出现一个"只有一个子节点"的内部节点时，中序遍历就有 2 种可能。
判断方法：在前序中若 pre[i+1] == post[j-1]（即某节点的下一个前序元素等于其后序的前一个元素），
说明该节点只有一个子节点。
统计这样的节点个数 cnt，答案 = 2^cnt。
```

**C++ 代码**：

```cpp
string pre_s, post_s;
// 统计只有一个孩子的节点数
int cnt = 0;
void solve() {
    for (int i = 0; i + 1 < (int)pre_s.size(); i++)
        for (int j = 1; j < (int)post_s.size(); j++)
            if (pre_s[i] == post_s[j] && pre_s[i+1] == post_s[j-1])
                cnt++;
    // 答案 = 2^cnt
    long long ans = 1;
    for (int i = 0; i < cnt; i++) ans *= 2;
    printf("%lld\n", ans);
}
```

**核心启示**：前序 + 后序的歧义来自"独子节点"，每个独子节点贡献一倍的方案乘数。

### 题目 5：对称二叉树（洛谷 P5018）

> 给定一棵有 $n$ 个节点的二叉树（每个节点有权值），求其最大的对称子树的节点数。对称指的是左右子树镜像对称且对应节点权值相等。

**分析**：

```
枚举每个节点作为子树的根，判断该子树是否对称。
对称判断：递归地比较 左子的左子 vs 右子的右子，左子的右子 vs 右子的左子。
若对称，则计算子树大小并更新答案。
时间复杂度：每个节点最多被检查 O(n) 次，总体 O(n^2)。
```

**C++ 代码**：

```cpp
int val[MAXN], lc[MAXN], rc[MAXN], sz[MAXN];
bool isSym(int a, int b) {
    if (!a && !b) return true;
    if (!a || !b) return false;
    if (val[a] != val[b]) return false;
    return isSym(lc[a], rc[b]) && isSym(rc[a], lc[b]);
}
void calcSize(int u) {
    if (!u) return;
    sz[u] = 1;
    calcSize(lc[u]); calcSize(rc[u]);
    sz[u] += sz[lc[u]] + sz[rc[u]];
}
// 枚举每个节点 u，若 isSym(lc[u], rc[u]) 则 ans = max(ans, sz[u])
```

**核心启示**：对称性的递归判断是"交叉比较"——左子的左 vs 右子的右，左子的右 vs 右子的左。

### 题目 6：【XR-4】复读（洛谷 P5597）

> 给定一个无穷完全二叉树（根为 1，节点 $u$ 的左子为 $2u$，右子为 $2u+1$），给出 $n$ 个节点编号，求这些节点构成的虚树（Auxiliary Tree）的边数。虚树包含这些节点和它们两两之间的 LCA。

**分析**：

```
利用完全二叉树的性质：节点 u 的父亲是 u/2，可以直接通过二进制求 LCA。
方法：将所有节点按 DFS 序排序，相邻节点的 LCA 就包含了所有需要的关键 LCA。
虚树边数 = 虚树节点数 - 1。
由于节点编号可以很大（需要高精度/大整数），用字符串模拟二进制操作。
核心：两个节点的 LCA = 它们二进制表示的最长公共前缀。
```

**C++ 代码**：

```cpp
// 节点用二进制字符串表示（从根到该节点的路径，0=左 1=右）
// LCA = 最长公共前缀
string getLCA(const string &a, const string &b) {
    int len = min(a.size(), b.size());
    int i = 0;
    while (i < len && a[i] == b[i]) i++;
    return a.substr(0, i);
}
// 将所有节点转为二进制路径串，排序后，
// 虚树节点 = 原节点集合 ∪ 相邻节点的 LCA（去重）
// 答案 = 虚树节点数 - 1
```

**核心启示**：完全二叉树中节点编号的二进制表示天然对应从根到该节点的路径，LCA = 公共前缀。

### 题目 7：HXY 造公园（洛谷 P2195）

> 给定一个 $n$ 个节点 $m$ 条边的森林，支持两种操作：(1) 查询某节点所在树的直径；(2) 合并两棵树（在两棵树各选一个点连边），要求合并后直径尽量小。

**分析**：

```
用并查集维护连通性，同时维护每棵树的直径。
查询操作：直接返回所在树的直径。
合并操作：设两棵树的直径分别为 d1、d2，合并后新直径 = max(d1, d2, ⌈d1/2⌉ + ⌈d2/2⌉ + 1)。
合并后的直径取三者最大值，因为新直径要么是原来某棵树的直径，
要么是跨越新边的最长路径（各取半径相加再加 1）。
初始时对每棵树用两遍 BFS 求直径。
```

**C++ 代码**：

```cpp
int fa_set[MAXN], diam[MAXN]; // 并查集 + 每棵树直径
int find(int x) { return fa_set[x] == x ? x : fa_set[x] = find(fa_set[x]); }
void merge(int u, int v) {
    u = find(u); v = find(v);
    if (u == v) return;
    int d1 = diam[u], d2 = diam[v];
    int nd = max({d1, d2, (d1 + 1) / 2 + (d2 + 1) / 2 + 1});
    fa_set[u] = v;
    diam[v] = nd;
}
// 初始化：对每棵树 BFS 两遍求直径
```

**核心启示**：合并两棵树使直径最小，策略是连接两棵树各自的"中心"，新直径 = $\max(d_1, d_2, \lceil d_1/2 \rceil + \lceil d_2/2 \rceil + 1)$。

### 题目 8：[APIO2010] 巡逻（洛谷 P3629）

> 一棵 $n$ 个节点的树，邮递员需要从节点 1 出发遍历每条边并返回，每条边需要走两次（往返）。现在可以添加 $K$ 条新边（$K=1$ 或 $K=2$），使总路程最短。求最短总路程。

**分析**：

```
不加边时，遍历所有边的总路程 = 2(n-1)（每条边走两次）。
加一条边后，形成一个环，环上的边只需走一次，节省 = 环上边数 - 1 = 直径长度。
所以 K=1 时答案 = 2(n-1) - 直径 + 1。

K=2 时，第二条边也形成一个环。但两个环若有重叠部分，重叠部分反而要走两次。
做法：先求直径 d1，将直径路径上的边权取反（变为 -1），再求"直径" d2。
答案 = 2(n-1) - d1 - d2 + 2。
注意第二次求直径时可能有负权边，必须用 DP 法而非两遍 BFS。
```

**C++ 代码**：

```cpp
int n, K, ans;
int d1, d2; // 两次直径
// 第一遍：两遍 BFS 求直径 d1，记录直径路径
// 将直径路径上的边权从 1 改为 -1
// 第二遍：DFS（DP法）求带负权的直径 d2
int dfs(int u, int fa) {
    int best = 0;
    for (auto [v, w] : adj[u]) {
        if (v == fa) continue;
        int d = dfs(v, u) + w;
        d2 = max(d2, best + d);
        best = max(best, d);
    }
    return best;
}
// K==1: ans = 2*(n-1) - d1 + 1
// K==2: ans = 2*(n-1) - d1 - d2 + 2
```

**核心启示**：添加一条边形成环可以节省路程，但多条边的环可能重叠，用"取反边权"的技巧处理重叠。

### 题目 9：【XR-3】核心城市（洛谷 P5536）

> 一棵 $n$ 个节点的树，选出 $k$ 个核心城市，使得非核心城市到最近核心城市的最大距离最小。

**分析**：

```
答案具有单调性，但贪心策略更直观：
1. 先求树的直径，找到树的中心
2. 以中心为根，计算每个节点子树中的最深深度 maxdep[u]
3. 按 maxdep[u] 从大到小排序，贪心地选前 k 个节点作为核心城市
4. 答案 = 第 k+1 大的 maxdep 值（即未被选中的节点中最深的那个到核心的距离）

但更精确的做法：以直径中点为根做 DFS，算出每个节点的子树最大深度，
然后按"子树最大深度"从大到小贪心选 k 个点。
最终答案 = 未选节点中到最近核心节点的最远距离。
```

**C++ 代码**：

```cpp
int maxdep[MAXN]; // 以 root 为根时，u 的子树最大深度
void dfs(int u, int fa) {
    maxdep[u] = 0;
    for (int v : adj[u]) {
        if (v == fa) continue;
        dfs(v, u);
        maxdep[u] = max(maxdep[u], maxdep[v] + 1);
    }
}
// 对所有节点按 maxdep 降序排序
// 选前 k 个标记为核心
// 答案 = 未被选且未被核心覆盖的节点的最大距离
```

**核心启示**：选核心城市 = 以树的中心为根，按"子树纵深"贪心，优先覆盖最深的分支。

### 题目 10：树网的核（洛谷 P1099）

> 给定一棵带边权的树和参数 $s$，在树的直径上选一条长度不超过 $s$ 的连续路径，使得树上所有节点到该路径的最远距离最小。求这个最小的最远距离。

**分析**：

```
1. 首先求出树的直径路径（两遍 BFS/DFS）
2. 在直径路径上使用双指针/尺取法，枚举长度 ≤ s 的所有区间
3. 对每个区间，计算树上所有节点到该路径段的最远距离
4. 优化：预处理直径路径上每个节点往"非直径方向"伸出的最远距离
5. 最远距离 = max(直径端点到路径段端点的距离, 路径段上节点往侧边伸出的最大值)

关键观察：最远点要么是直径的端点，要么是从直径侧边伸出的最远点。
```

**C++ 代码**：

```cpp
// path[]: 直径路径上的节点序列，dist[i]: path[0] 到 path[i] 的距离
// side[i]: 从 path[i] 出发、不经过直径上的相邻节点、能到达的最远距离
int ans = INT_MAX;
// 双指针：维护 dist[r] - dist[l] <= s
for (int l = 0, r = 0; r < pathLen; r++) {
    while (dist[r] - dist[l] > s) l++;
    int cur = max(dist[l], dist[pathLen-1] - dist[r]);
    // 还需要考虑 [l, r] 区间内 side[] 的最大值
    // 可以用单调队列维护
    ans = min(ans, max(cur, maxSide));
}
```

**核心启示**：树网的核 = 直径 + 尺取法 + 侧枝最深深度，经典的"直径上做决策"问题。

### 题目 11：[NOI2003] 逃学的小孩（洛谷 P4408）

> 一棵带边权的树，选三个不同的点 $A, B, C$，$C$ 在家（终点），$A, B$ 在外面。$A, B$ 分别出发前往 $C$，但 $A$ 要先到 $B$ 处会合再一起去 $C$（或者 $B$ 先到 $A$ 处），求最坏情况下（即选择使时间最长的 $A, B, C$）的总距离。

**分析**：

```
设 d(x,y) 为 x 到 y 的距离。答案 = max over all (A,B,C) of [d(A,B) + min(d(A,C), d(B,C))]。
可以证明 A, B 一定是直径的两个端点（使 d(A,B) 最大）。
然后枚举 C，答案 = d(A,B) + min(d(A,C), d(B,C)) 的最大值。
先求直径的两个端点 u, v，再分别 BFS 求 u 和 v 到其他所有点的距离。
```

**C++ 代码**：

```cpp
long long du[MAXN], dv[MAXN]; // u/v 到各点距离
// 两遍 BFS 求直径端点 u, v
// 分别 BFS 求 du[], dv[]
long long ans = 0;
long long diam = du[v]; // 直径长度
for (int c = 1; c <= n; c++) {
    long long cost = diam + min(du[c], dv[c]);
    ans = max(ans, cost);
}
```

**核心启示**：固定直径端点为 $A, B$，然后枚举 $C$ 取最优，利用了"直径两端点是最远点对"的性质。

### 题目 12：【模板】最近公共祖先 / LCA（洛谷 P3379）

> 给定一棵 $n$ 个节点的有根树，$m$ 次查询两个节点的最近公共祖先。

**分析**：

```
LCA 倍增法模板题。
预处理：DFS 一遍计算深度 dep[] 和倍增数组 fa[u][k]（u 的第 2^k 级祖先）。
查询：先将深度大的节点跳到与另一节点同深度，然后一起向上跳。
时间复杂度：预处理 O(n log n)，单次查询 O(log n)。
```

**C++ 代码**：

```cpp
int dep[MAXN], fa[MAXN][21];
void dfs(int u, int f) {
    dep[u] = dep[f] + 1;
    fa[u][0] = f;
    for (int k = 1; k <= 20; k++)
        fa[u][k] = fa[fa[u][k-1]][k-1];
    for (int v : adj[u])
        if (v != f) dfs(v, u);
}
int lca(int u, int v) {
    if (dep[u] < dep[v]) swap(u, v);
    for (int k = 20; k >= 0; k--)
        if (dep[fa[u][k]] >= dep[v]) u = fa[u][k];
    if (u == v) return u;
    for (int k = 20; k >= 0; k--)
        if (fa[u][k] != fa[v][k])
            u = fa[u][k], v = fa[v][k];
    return fa[u][0];
}
```

**核心启示**：倍增法是 LCA 的万能模板，$fa[u][k] = fa[fa[u][k-1]][k-1]$ 是核心递推式。

### 题目 13：斐波那契（洛谷 P3938）

> 有一棵无穷大的树，根据斐波那契数列构建：第 $i$ 天新增 $F_i$ 个节点，它们挂在之前的某些节点下面（按照一定规则）。给出两个节点编号，求它们的 LCA。

**分析**：

```
观察树的构造规则：节点编号按斐波那契数列分段。
节点 x 的父亲可以通过找到 x 所在的斐波那契区间来确定。
设 F[i-1] < x <= F[i]，则 x 的父亲 = x - F[i-1]。
求 LCA：不断让编号较大的节点跳到其父亲，直到两节点相同。
类似于链表版的 LCA（类比求两个链表交点）。
```

**C++ 代码**：

```cpp
long long fib[90]; // 预处理斐波那契数列
void initFib() {
    fib[1] = fib[2] = 1;
    for (int i = 3; i < 90; i++) fib[i] = fib[i-1] + fib[i-2];
}
long long parent(long long x) {
    if (x == 1) return 0;
    int i = (int)(lower_bound(fib+1, fib+90, x) - fib);
    if (fib[i] == x) return fib[i] - fib[i-1]; // = fib[i-2]
    return x - fib[i-1];
}
long long lca(long long u, long long v) {
    while (u != v) {
        if (u > v) u = parent(u);
        else v = parent(v);
    }
    return u;
}
```

**核心启示**：特殊构造的树可以通过数学规律直接计算父节点，无需建树，LCA 退化为"交替跳父亲"。

### 题目 14：[AHOI2008] 紧急集合 / 聚会（洛谷 P4281）

> 一棵 $n$ 个节点的树，每次给出三个节点 $a, b, c$，求一个点 $p$ 使得 $\text{dist}(a, p) + \text{dist}(b, p) + \text{dist}(c, p)$ 最小。输出 $p$ 和最小距离。

**分析**：

```
结论：三个点的最优聚会点一定是三个 LCA 中深度最大的那个。
设 L1 = lca(a,b), L2 = lca(b,c), L3 = lca(a,c)。
最优点 p = 三个 LCA 中深度最大的那个。
证明：画树可以发现，三个 LCA 中有恰好两个相同，
第三个（深度最大的）就是三条路径汇聚的交点。
最小距离 = dist(a,p) + dist(b,p) + dist(c,p)
         = (dep[a] + dep[b] + dep[c] - dep[L1] - dep[L2] - dep[L3]) / 2
更直接：dist = dep[a]+dep[b]+dep[c] - dep[L1] - dep[L2] - dep[L3]。
```

**C++ 代码**：

```cpp
// 使用倍增 LCA（同 P3379 模板）
void solve(int a, int b, int c) {
    int l1 = lca(a, b), l2 = lca(b, c), l3 = lca(a, c);
    int p;
    if (dep[l1] >= dep[l2] && dep[l1] >= dep[l3]) p = l1;
    else if (dep[l2] >= dep[l3]) p = l2;
    else p = l3;
    long long dist = (long long)dep[a] + dep[b] + dep[c]
                     - dep[l1] - dep[l2] - dep[l3];
    printf("%d %lld\n", p, dist);
}
```

**核心启示**：三点聚会问题的最优点 = 三个两两 LCA 中深度最大的那个，这是 LCA 最经典的应用之一。

## 五、常见技巧与思路总结

### 5.1 递归是树的灵魂

树的无环性保证了递归不会死循环。几乎所有树上问题都可以用 DFS 递归来解决。

### 5.2 "根"的选择

- 无根树可以任选一个点作为根（通常选 1）
- 某些问题对根的选择敏感（如树的中心），需要选择合适的根
- 有时需要**换根**技巧（换根 DP）

### 5.3 边权 vs 点权

- 边权可以"下放"到子节点（根节点权值设为 0），转化为点权问题
- 点权有时需要在 LCA 处特殊处理（避免重复计算）

### 5.4 直径的万能性

树的直径是树上问题的核心工具：
- 直径端点是距离任意点最远的候选点
- 直径路径是树的"骨架"
- 很多最优化问题（如树网的核、核心城市）都在直径上做决策

### 5.5 LCA 的本质

LCA 本质上是将树上路径拆解为两条"从下到上"的链：
$$u \to \text{lca}(u,v) \to v$$

通过这种拆解，树上的路径问题可以转化为链上问题。

### 5.6 常见结论速查

| 结论 | 说明 |
|------|------|
| 直径端点最远 | 从任意点出发最远的点一定是直径端点 |
| 合并两棵树 | 新直径 $= \max(d_1, d_2, \lceil d_1/2\rceil + \lceil d_2/2\rceil + 1)$ |
| 三点聚会 | 最优点 = 三个 LCA 中最深的 |
| 树上距离 | $\text{dist}(u,v) = \text{dep}[u] + \text{dep}[v] - 2\text{dep}[\text{lca}]$ |
| 遍历边两次 | 遍历树所有边并返回原点 $= 2(n-1)$ |
| 加一条边 | 形成环，环上边可少走一次，节省 = 环长 - 1 |

## 六、优化方法 / 解题框架

### 6.1 三步解题法

```
第一步：识别问题类型
├── 二叉树遍历/重建 → 递归模拟
├── 树的直径/中心   → 两遍 BFS 或 DFS(DP)
├── 两点距离/路径   → LCA
└── 选点/选路径     → 直径 + 贪心/DP

第二步：选择算法
├── 纯遍历问题 → DFS/BFS
├── 需要祖先信息 → 倍增 LCA
├── 需要直径信息 → 两遍 BFS/DFS
└── 需要全局最优 → 综合使用

第三步：注意细节
├── 有根 vs 无根
├── 边权 vs 点权 vs 无权
├── 特殊边界（单节点、链状树）
└── 数据范围决定算法选择
```

### 6.2 算法选择表

| 问题类型 | 推荐算法 | 时间复杂度 |
|---------|---------|-----------|
| 二叉树遍历 | 递归 | $O(n)$ |
| 二叉树重建 | 递归 + 哈希查找 | $O(n)$ ~ $O(n^2)$ |
| 树的直径 | 两遍 BFS 或 DFS(DP) | $O(n)$ |
| LCA 查询 | 倍增法 | 预处理 $O(n \log n)$，查询 $O(\log n)$ |
| LCA 查询（极致优化） | 树链剖分 / Euler Tour + ST表 | 预处理 $O(n \log n)$ / $O(n)$，查询 $O(\log n)$ / $O(1)$ |
| 树上路径查询 | LCA + 差分 | $O(n \log n)$ |
| 树的中心 | 直径 + 找中点 | $O(n)$ |

### 6.3 防错清单

- [ ] **无根树**的 DFS 一定要传 `fa` 参数防止回头
- [ ] **倍增数组**的大小：$\log_2(n)$ 向上取整，通常开 20 就够 $10^6$
- [ ] **深度和距离**不要混淆：深度从根算起，距离从起点算起
- [ ] **森林**要对每个连通分量分别处理
- [ ] **边权为负**时不能用两遍 BFS 求直径，要用 DFS(DP) 法
- [ ] LCA 查询前确保 **深度大的节点先跳**

