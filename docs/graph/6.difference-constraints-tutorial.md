## 一、什么是差分约束

**差分约束系统**（Difference Constraints System）是一类特殊的不等式组：给定 n 个未知数 $x_1, x_2, \ldots, x_n$，以及 m 条形如 $x_i - x_j \le c_{ij}$ 的约束条件，要求找到一组满足所有约束的解，或判定无解。

这类不等式之所以叫"差分约束"，是因为每条约束都只涉及两个变量的**差**。看起来是代数问题，但它和图论里的**最短路**有着深刻的联系——每条不等式都可以对应图上的一条有向边，而求解过程本质上就是跑一次最短路或最长路算法。

```
差分约束系统示例（3 个变量，4 条约束）：

x1 - x2 ≤ 3       →  边 (x2, x1, 3)
x2 - x3 ≤ -1      →  边 (x3, x2, -1)
x1 - x3 ≤ 5       →  边 (x3, x1, 5)
x3 - x1 ≤ 2       →  边 (x1, x3, 2)
```

| 特性         | 说明                                      |
|--------------|-------------------------------------------|
| 约束形式     | $x_i - x_j \le c$（两变量之差有上界）      |
| 建图方式     | 每条约束建一条有向带权边                    |
| 无解判定     | 图中存在负环（约束互相矛盾）                |
| 求最大解     | 从超级源点跑最短路                          |
| 求最小解     | 从超级源点跑最长路                          |
| 核心算法     | SPFA / Bellman-Ford                        |

## 二、核心思想与前置知识

### 2.1 不等式转建图

差分约束系统的核心观察是：

> 不等式 $x_i - x_j \le c$ 与最短路中的三角不等式 $dist[i] \le dist[j] + c$ 形式完全一致。

因此，对于每条约束 $x_i - x_j \le c$，我们从节点 $j$ 向节点 $i$ 连一条权值为 $c$ 的有向边。如果图中存在最短路解，那么 $dist[]$ 就是原不等式组的一组可行解。

**建图规则**：

```
约束：xi - xj ≤ c
建边：j → i，边权 = c

含义：从 j 走到 i 的边权为 c
最短路保证：dist[i] ≤ dist[j] + c
即：dist[i] - dist[j] ≤ c  ✓
```

为了保证所有节点可达，通常需要建一个**超级源点** $x_0$，并对每个变量 $x_i$ 添加约束 $x_i - x_0 \le 0$（即从 $x_0$ 向所有节点连边权为 0 的边）。

**不等式变换技巧**：

| 原始约束               | 等价变换                           | 建边方式          |
|------------------------|-----------------------------------|-------------------|
| $x_i - x_j \le c$     | 直接建边                          | $j \to i$，权 $c$ |
| $x_i - x_j \ge c$     | $x_j - x_i \le -c$               | $i \to j$，权 $-c$|
| $x_i - x_j = c$       | 拆为 $\le c$ 和 $\ge c$ 两条     | 双向建边           |
| $x_i = x_j$           | $x_i - x_j \le 0$ 且 $\ge 0$    | 双向建权 0 的边    |
| $x_i \le c$           | $x_i - x_0 \le c$（引入基准点）  | $x_0 \to x_i$     |

### 2.2 最短路求最大解 / 最长路求最小解

差分约束有一个非常重要的性质：

- **跑最短路**：得到的是所有可行解中**每个变量的最大值**（在满足约束前提下，值尽可能大）。
- **跑最长路**：得到的是所有可行解中**每个变量的最小值**（在满足约束前提下，值尽可能小）。

**直觉理解**：

```
约束 xi - xj ≤ c 含义：xi 最多比 xj 大 c

最短路松弛：dist[i] = min(dist[i], dist[j] + c)
  → 不断收紧 xi 的上界，最终得到满足所有约束的最大可能值

最长路松弛：dist[i] = max(dist[i], dist[j] + c)
  → 不断抬高 xi 的下界，最终得到满足所有约束的最小可能值
```

**总结口诀**：

- 约束全部化为 $x_i - x_j \le c$ → 跑最短路 → 得最大解
- 约束全部化为 $x_i - x_j \ge c$ → 跑最长路 → 得最小解

### 2.3 SPFA 判负环框架

差分约束系统无解，当且仅当对应的图中存在**负环**（最短路模型）或**正环**（最长路模型）。使用 SPFA 算法时，判定方法是：如果某个节点入队次数超过 $n$（节点总数），则存在负环。

**SPFA 模板（判负环 + 求最短路）**：

```cpp
const int MAXN = 5005, MAXM = 10005;
int head[MAXN], nxt[MAXM], to[MAXM], w[MAXM], cnt;
int dist[MAXN], in_cnt[MAXN];
bool inq[MAXN];

void addEdge(int u, int v, int c) {
    to[++cnt] = v; w[cnt] = c;
    nxt[cnt] = head[u]; head[u] = cnt;
}

// 返回 true 表示有可行解，false 表示存在负环
bool spfa(int n, int s) {
    memset(dist, 0x3f, sizeof(dist));
    dist[s] = 0; inq[s] = true; in_cnt[s] = 1;
    queue<int> q;
    q.push(s);
    while (!q.empty()) {
        int u = q.front(); q.pop();
        inq[u] = false;
        for (int i = head[u]; i; i = nxt[i]) {
            int v = to[i];
            if (dist[v] > dist[u] + w[i]) {
                dist[v] = dist[u] + w[i];
                if (!inq[v]) {
                    inq[v] = true;
                    if (++in_cnt[v] > n) return false;
                    q.push(v);
                }
            }
        }
    }
    return true;
}
```

## 三、分类与变体详解

### 3.1 基本差分约束（求可行解）

**适用场景**：给出若干形如 $x_i - x_j \le c$ 的不等式，判断是否有解；如果有解，输出任意一组可行解。

**做法**：

1. 建立超级源点 $x_0$，向所有节点连边权为 0 的边。
2. 对每条约束 $x_i - x_j \le c$，从 $j$ 向 $i$ 连边权为 $c$ 的边。
3. 从 $x_0$ 跑 SPFA 最短路。
4. 若检测到负环 → 无解；否则 $dist[1..n]$ 即为一组可行解。

```
建图流程：
  1. 添加超级源点 x0
  2. x0 → xi (权 0)，对所有 i
  3. 对每条约束 xi - xj ≤ c，建边 j → i (权 c)
  4. 从 x0 跑 SPFA
  5. 判负环 → 无解 / 输出 dist[]
```

### 3.2 求最大/最小值

有时题目不仅要求可行解，还要求某个变量（或所有变量）在满足约束的前提下取到最大值或最小值。

**求最大值**：将所有约束化为 $x_i - x_j \le c$ 的形式，跑**最短路**。$dist[i]$ 就是 $x_i$ 能取到的最大值。

**求最小值**：将所有约束化为 $x_i - x_j \ge c$ 的形式，跑**最长路**。$dist[i]$ 就是 $x_i$ 能取到的最小值。

```
示例：求 x3 - x1 的最大值

约束系统:
  x2 - x1 ≤ 3
  x3 - x2 ≤ 2
  x3 - x1 ≤ 4

从 x1 跑最短路（以 x1 为源点）:
  dist[1] = 0
  dist[2] = 3
  dist[3] = min(4, 0+3+2) = 4

答案: x3 - x1 的最大值 = dist[3] = 4
```

### 3.3 等式约束处理

当约束中出现等式 $x_i = x_j + c$（或 $x_i - x_j = c$），需要拆成两条不等式：

$$x_i - x_j \le c \quad \text{且} \quad x_j - x_i \le -c$$

等价于建两条边：$j \to i$（权 $c$）和 $i \to j$（权 $-c$）。

等值约束 $x_i = x_j$ 则建两条权为 0 的边（双向连通）。在实际题目中，等式约束常出现在"两个量必须相等"的场景。

### 3.4 对数变换处理乘除约束

有些题目中的约束不是加减，而是乘除形式：

$$\frac{x_i}{x_j} \le c \quad \text{或} \quad x_i \cdot k \le x_j$$

这类约束无法直接用差分约束模型处理，但通过**取对数**可以将乘除转化为加减：

$$\frac{x_i}{x_j} \le c \implies \log x_i - \log x_j \le \log c$$

令 $y_i = \log x_i$，则原约束变为标准差分约束 $y_i - y_j \le \log c$。

**技巧要点**：

- 所有变量取对数后，乘法变加法，除法变减法
- 比较大小关系不变（对数函数单调递增）
- 最终结果需要用 $e^{y_i}$ 还原回原变量
- 实际编程中用 `double` 型 SPFA 处理浮点权值边

```
原始约束：xi / xj ≤ c  （其中 xi, xj > 0）
取对数后：ln(xi) - ln(xj) ≤ ln(c)
令 yi = ln(xi)，则：yi - yj ≤ ln(c)
标准差分约束形式！

建边：j → i，权 = ln(c)
```

## 四、5 道经典例题详解

### 题目 1：P5960 【模板】差分约束算法

> 给定 n 个变量和 m 条差分约束 $x_i - x_j \le c_k$，求一组可行解或判定无解。

**分析**：这是差分约束的标准模板题。建超级源点 $x_0$，向所有节点连权 0 的边，然后对每条约束建边，跑 SPFA 判负环。

```
建图方式：
  超级源点 0 → i (权 0)，对所有 1 ≤ i ≤ n
  对每条约束 xi - xj ≤ c：建边 j → i (权 c)

无解判定：SPFA 中某节点入队超过 n+1 次
可行解：dist[1..n]
```

**C++ 代码**：

```cpp
int head[MAXN], nxt[MAXM], to[MAXM], w[MAXM], cnt;
int dist[MAXN], in_cnt[MAXN];
bool inq[MAXN];

void addEdge(int u, int v, int c) {
    to[++cnt] = v; w[cnt] = c;
    nxt[cnt] = head[u]; head[u] = cnt;
}

bool spfa(int n) {
    memset(dist, 0x3f, sizeof(dist));
    queue<int> q;
    dist[0] = 0; inq[0] = true; in_cnt[0] = 1;
    q.push(0);
    while (!q.empty()) {
        int u = q.front(); q.pop(); inq[u] = false;
        for (int i = head[u]; i; i = nxt[i]) {
            int v = to[i];
            if (dist[v] > dist[u] + w[i]) {
                dist[v] = dist[u] + w[i];
                if (!inq[v]) {
                    inq[v] = true;
                    if (++in_cnt[v] > n + 1) return false;
                    q.push(v);
                }
            }
        }
    }
    return true;
}
```

答案位置：若 `spfa(n)` 返回 `true`，则 `dist[1..n]` 为一组可行解；返回 `false` 则无解。

**核心启示**：
- **超级源点**：保证所有节点可达，是差分约束的标准处理手法
- **判负环**：SPFA 中入队次数 > 节点总数 → 负环存在 → 无解
- **基础模板**：后续所有差分约束题都以此为骨架

### 题目 2：P3275 [SCOI2011]糖果

> 有 n 个小朋友，给出 m 条形如"第 a 个小朋友的糖果数与第 b 个的关系"（大于、等于、小于等于等 5 种），每人至少 1 颗糖。求最少需要多少总糖数，若无法满足输出 -1。

**分析**：这道题要求**每个变量的最小值**，因此需要跑**最长路**。将五种关系分别转化为差分约束：

```
关系转化（设 xa, xb 分别为两人糖果数）：
  类型 1：xa = xb   → xa - xb ≥ 0 且 xb - xa ≥ 0
  类型 2：xa < xb   → xb - xa ≥ 1
  类型 3：xa ≥ xb   → xa - xb ≥ 0
  类型 4：xa > xb   → xa - xb ≥ 1
  类型 5：xa ≤ xb   → xb - xa ≥ 0

每人至少 1 颗：xi - x0 ≥ 1（x0 = 0 为超级源点）
```

所有约束化为 $x_i - x_j \ge c$ 的形式后，跑最长路。若存在正环则无解，否则答案为 $\sum dist[i]$。

**C++ 代码**：

```cpp
// 最长路 SPFA，dist 初始化为 0
bool spfa_longest(int n) {
    queue<int> q;
    dist[0] = 0; inq[0] = true; in_cnt[0] = 1;
    q.push(0);
    while (!q.empty()) {
        int u = q.front(); q.pop(); inq[u] = false;
        for (int i = head[u]; i; i = nxt[i]) {
            int v = to[i];
            if (dist[v] < dist[u] + w[i]) {
                dist[v] = dist[u] + w[i];
                if (!inq[v]) {
                    inq[v] = true;
                    if (++in_cnt[v] > n + 1) return false;
                    q.push(v);
                }
            }
        }
    }
    return true;
}
```

答案位置：若可行，`ans = dist[1] + dist[2] + ... + dist[n]`

**核心启示**：
- **求最小总量 → 最长路**：要让每个人糖果尽可能少，恰好是最长路给出的下界
- **多种关系统一转化**：不同类型的比较关系都可以写成差分不等式
- **注意整数约束**："严格大于"在整数域中等价于"大于等于 1"

### 题目 3：P2294 [HNOI2005]狡猾的商人

> 商人报告了若干段账目，每段声明从第 l 月到第 r 月的总收入为 v。判断这些报告是否自相矛盾。

**分析**：设 $S_i$ 为前 $i$ 个月的前缀和，则"第 l 到第 r 月总收入为 v"等价于 $S_r - S_{l-1} = v$。这是一个等式约束，拆成两条不等式：

```
Sr - S(l-1) = v  拆为：
  Sr - S(l-1) ≤ v    →  建边 (l-1) → r，权 v
  S(l-1) - Sr ≤ -v   →  建边 r → (l-1)，权 -v

判断是否无解 ↔ 是否存在负环
```

也可以用带权并查集更高效地解决（合并与查询前缀和的关系），但差分约束是通用方法。

**C++ 代码**：

```cpp
// 建图后跑 SPFA 判负环
// 对每条声明 (l, r, v)：
//   addEdge(l - 1, r, v);   // Sr - S(l-1) ≤ v
//   addEdge(r, l - 1, -v);  // S(l-1) - Sr ≤ -v
// 超级源点 n+1 向所有节点连权 0 的边
// 跑 SPFA，若存在负环输出 "false"，否则 "true"

bool check(int n) {
    int src = n + 1;
    for (int i = 0; i <= n; i++)
        addEdge(src, i, 0);
    return spfa(n + 2);  // 共 n+2 个节点
}
```

答案位置：`check(n)` 返回 `true` 表示不矛盾，`false` 表示矛盾。

**核心启示**：
- **前缀和 + 差分约束**：区间和约束 $S_r - S_{l-1} = v$ 天然是差分形式
- **等式拆成双向边**：$= v$ 等价于 $\le v$ 且 $\ge v$
- **判定一致性**：差分约束不止用来求值，也常用来判断约束是否矛盾

### 题目 4：P4926 [1007]倍杀测量者

> 选手之间存在"倍杀"关系，例如"A 的分数至少是 B 的 k 倍"。给定若干此类约束，判断是否存在一组满足所有约束的分数。约束涉及乘除关系。

**分析**：这是**对数变换**处理乘除约束的典型题。原始约束形如 $x_a \ge k \cdot x_b$，取对数后变为 $\ln x_a - \ln x_b \ge \ln k$。

```
原始约束：xa ≥ k · xb  （倍杀关系）
取对数：  ln(xa) - ln(xb) ≥ ln(k)

令 yi = ln(xi)，化为：
  yb - ya ≤ -ln(k)
  建边：a → b，权 -ln(k)

问题转化为：差分约束系统是否有解
  ↔ 图中是否存在负环
```

实际处理时，题目通常还会给出一个误差容忍度 $T$，需要**二分答案** $T$ 的值，再用差分约束判定可行性。

**C++ 代码**：

```cpp
// 二分 + 差分约束判负环
// 对每条约束 xa ≥ k·xb，取对数后建边
// 使用 double 型 SPFA

double dist_d[MAXN];
int in_cnt_d[MAXN];
bool inq_d[MAXN];

bool spfa_double(int n, int s) {
    for (int i = 0; i <= n; i++)
        dist_d[i] = 1e18, inq_d[i] = false, in_cnt_d[i] = 0;
    dist_d[s] = 0; inq_d[s] = true; in_cnt_d[s] = 1;
    queue<int> q;
    q.push(s);
    while (!q.empty()) {
        int u = q.front(); q.pop(); inq_d[u] = false;
        for (int i = head[u]; i; i = nxt[i]) {
            int v = to[i];
            if (dist_d[v] > dist_d[u] + wd[i]) {
                dist_d[v] = dist_d[u] + wd[i];
                if (!inq_d[v]) {
                    inq_d[v] = true;
                    if (++in_cnt_d[v] > n) return false;
                    q.push(v);
                }
            }
        }
    }
    return true;
}
```

答案位置：二分得到最大可行的 $T$ 值

**核心启示**：
- **对数变换**：乘除约束取对数后变为加减，纳入差分约束框架
- **二分 + 差分约束**：参数化问题常用二分答案 + 差分约束判可行性
- **浮点精度**：使用 `double` 型边权和距离数组时要注意精度处理

### 题目 5：P5590 赛车游戏

> 给定一张有向图，每条边可以分配一个 $[1, 9]$ 的权值。要求从起点 $s$ 到终点 $t$ 的所有路径权值和相等。构造一种合法的边权分配方案。

**分析**：设每条边的权值为 $w_e$，"所有 $s$ 到 $t$ 的路径权值和相等"意味着：对于图中任意一个环（可达的），环上边权之和必须为 0。

先通过 BFS/DFS 删除不在任何 $s \to t$ 路径上的点和边。然后设 $d_u$ 为节点 $u$ 的势函数（从 $s$ 到 $u$ 的路径权值和，由于路径权值和必须唯一确定，这是良定义的）。对于任意边 $(u, v)$，有 $w_{uv} = d_v - d_u$，且 $1 \le w_{uv} \le 9$，即：

```
约束建立：
  对每条有效边 (u, v)：
    dv - du ≥ 1   →  建边 u → v，权 1（最长路）
    dv - du ≤ 9   →  建边 v → u，权 -1（或等价处理）

固定源点：ds = 0

跑最长路求每个节点的 d 值
每条边的权值 = dv - du
```

**C++ 代码**：

```cpp
// 1. BFS 从 s 正向遍历、从 t 反向遍历，取交集为有效节点
// 2. 对每条有效边 (u,v) 建差分约束：
//    dv - du >= 1  → addEdge(u, v, 1)
//    dv - du <= 9  → addEdge(v, u, -1) (最短路)
//                  或 du - dv >= -9 (最长路)
// 3. 跑 SPFA 最长路
// 4. 边权 = dist[v] - dist[u]

void solve(int n, int s, int t) {
    // 建图并跑最长路后
    // 输出每条边的权值
    for (int e = 0; e < m; e++) {
        int u = E[e].u, v = E[e].v;
        int wt = dist[v] - dist[u];
        // wt 应在 [1, 9] 范围内
    }
}
```

答案位置：每条边的权值 `dist[v] - dist[u]`

**核心启示**：
- **路径一致性 → 势函数**：所有路径等权 ↔ 可以为每个节点定义唯一势值
- **边权范围约束**：$1 \le d_v - d_u \le 9$ 正是差分约束的经典形式
- **预处理有效子图**：先删去不可达节点，避免无用约束导致无解

## 五、常见技巧与思路总结

### 5.1 约束统一化

差分约束的第一步永远是**把所有约束统一化为同一种形式**：

- 求最大解 → 全部化为 $x_i - x_j \le c$ → 跑最短路
- 求最小解 → 全部化为 $x_i - x_j \ge c$ → 跑最长路

常见的等价变换：

```
xi - xj ≥ c  ↔  xj - xi ≤ -c
xi - xj = c  ↔  xi - xj ≤ c 且 xi - xj ≥ c
xi < xj      ↔  xj - xi ≥ 1  （整数域）
xi ≤ xj      ↔  xj - xi ≥ 0
```

### 5.2 超级源点的设置

超级源点 $x_0$ 的作用是提供一个"零势能参考点"，保证所有节点可达。

- **最短路模型**：$x_0$ 向所有节点连权 0 的边（即 $x_i - x_0 \le 0$，所有变量 $\le 0$）
- **最长路模型**：$x_0$ 向所有节点连权 0 的边（即 $x_i - x_0 \ge 0$，所有变量 $\ge 0$）

如果题目有额外限制（如 $x_i \ge 1$），则调整连边权值即可。

### 5.3 负环 / 正环的含义

```
最短路模型中的负环：
  x1 - x2 ≤ a
  x2 - x3 ≤ b
  x3 - x1 ≤ c
  累加得：0 ≤ a + b + c
  若 a + b + c < 0，则矛盾 → 无解 → 对应图中负环

最长路模型中的正环：
  同理，约束累加后矛盾
```

### 5.4 前缀和与差分约束

很多区间约束问题可以通过前缀和转化为差分约束：

- 区间 $[l, r]$ 的和 $\ge v$：$S_r - S_{l-1} \ge v$
- 区间 $[l, r]$ 的和 $\le v$：$S_r - S_{l-1} \le v$
- 区间 $[l, r]$ 的和 $= v$：拆为上面两条

这类问题中，变量 $S_0, S_1, \ldots, S_n$ 就是前缀和数组。

### 5.5 二分答案 + 差分约束

当题目的答案具有单调性时，常用**二分答案后用差分约束判定可行性**的框架：

```
二分答案 mid:
  1. 根据 mid 建立差分约束
  2. 跑 SPFA 判负环
  3. 有解 → 调整二分上界
     无解 → 调整二分下界
```

典型场景：P4926 倍杀测量者中的容忍度参数。

## 六、优化方法 / 解题框架

### 6.1 解题流程框架

遇到差分约束问题时，按以下步骤处理：

```
Step 1: 识别变量与约束
  ├── 变量是什么？（节点值、前缀和、对数值...）
  └── 约束形式？（≤、≥、=、乘除...）

Step 2: 统一约束形式
  ├── 求最大解 → 化为 xi - xj ≤ c
  ├── 求最小解 → 化为 xi - xj ≥ c
  └── 乘除约束 → 取对数转化

Step 3: 建图
  ├── 对每条约束建有向带权边
  ├── 建超级源点，连接所有节点
  └── 等式约束建双向边

Step 4: 跑 SPFA
  ├── 最短路（求最大解）或最长路（求最小解）
  └── 判负环/正环 → 无解

Step 5: 提取答案
  ├── dist[] 即为可行解
  └── 根据题意计算最终答案
```

### 6.2 SPFA 优化技巧

- **栈优化（DFS-SPFA）**：将队列换成栈，在某些情况下能更快检测负环，但最坏情况仍为 $O(nm)$。
- **随机化**：随机打乱初始入队顺序，避免被卡。
- **SLF 优化**：新入队元素若 $dist$ 值小于队首，则插入队首而非队尾，可以加速收敛。

```cpp
// SLF 优化的 SPFA
deque<int> q;
// 入队时：
if (!q.empty() && dist[v] < dist[q.front()])
    q.push_front(v);
else
    q.push_back(v);
```

### 6.3 Bellman-Ford 替代

在某些场景下，Bellman-Ford 比 SPFA 更稳定（不会被卡）：

```cpp
bool bellman_ford(int n) {
    dist[0] = 0;
    for (int k = 0; k < n; k++)
        for (int u = 0; u <= n; u++)
            for (int i = head[u]; i; i = nxt[i]) {
                int v = to[i];
                if (dist[v] > dist[u] + w[i])
                    dist[v] = dist[u] + w[i];
            }
    // 第 n+1 轮还能松弛 → 负环
    for (int u = 0; u <= n; u++)
        for (int i = head[u]; i; i = nxt[i])
            if (dist[to[i]] > dist[u] + w[i])
                return false;
    return true;
}
```

## 七、终极总结与归纳

### 7.1 差分约束的本质

**一句话**：差分约束系统将变量之间的差值约束转化为图上的边权约束，利用最短路/最长路算法同时满足所有不等式，或检测出矛盾（负环/正环）。

它的美妙之处在于：一个看似纯代数的不等式求解问题，通过 $x_i - x_j \le c \iff dist[i] \le dist[j] + c$（三角不等式）这一观察，被完美映射到了图论的最短路框架中。

### 7.2 核心要点

1. **建图规则**：约束 $x_i - x_j \le c$ → 建边 $j \to i$（权 $c$）
2. **最短路 = 最大解，最长路 = 最小解**
3. **超级源点**：保证所有节点可达的"零势能参考点"
4. **负环 = 无解**：约束之间存在不可调和的矛盾
5. **等式拆双向**：$x_i - x_j = c$ 拆为 $\le c$ 和 $\ge c$
6. **对数变换**：乘除约束取对数后纳入差分约束框架
