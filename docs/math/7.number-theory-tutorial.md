## 一、知识顺序与整体结构

本文按下面的顺序讨论：

1. $\gcd$ 与欧几里得算法；
2. 裴蜀定理与扩展欧几里得；
3. 逆元与线性同余；
4. 欧拉函数、欧拉定理、费马小定理；
5. 中国剩余定理。

这条顺序对应着一个自然的推进过程：先研究整除关系，再研究线性组合，再转到模运算，最后处理多个模条件的合并。

整体结构如下：

```text
gcd

  ↓
裴蜀定理
  ↓

  ↓

  ↓

  ↓
CRT / EXCRT
```

后文只保留推导、联系与必要代码。

## 二、从整除出发：gcd 到底在描述什么

### 2.1 整除、公因子、互质

若整数 $d$ 能整除整数 $a$，记作 $d | a$，意思是存在整数 $k$，使得 $a = dk$。

这是整数结构里最基本的关系。所谓“公因子”，就是同时整除两个数的整数；所谓“互质”，就是它们除了 $1$ 以外没有别的公共因子。

于是最大公约数 $\gcd(a,b)$ 的本质就是：

> 在所有能同时整除 $a$ 和 $b$ 的整数里，最大的那个正整数。

例如：

- $\gcd(12,18)=6$
- $\gcd(14,25)=1$，所以 $14$ 和 $25$ 互质。

最大公约数不是一个孤立的数值，它代表的是 **两个数共享的整除结构到底有多大**。

### 2.2 gcd 的几个基本事实

这些事实看起来简单，但后面几乎每一步都会用到：

1. $\gcd(a,b)=\gcd(b,a)$
2. $\gcd(a,0)=|a|$
3. 若 $d=\gcd(a,b)$，则 $d|a$ 且 $d|b$
4. 任何同时整除 $a,b$ 的整数，都一定整除 $d$
5. $\gcd(a,b)=1$ 表示 $a,b$ 互质

第 3、4 条尤其重要，因为它说明 gcd 不是“某个公因子”，而是“所有公因子的总代表”。

### 2.3 为什么 $\gcd(a,b)=\gcd(b,a \bmod b)$

这就是欧几里得算法背后的核心等式，也是整篇文章第一块真正的基石。

设

$a = qb + r, 0 \le r < b$

其中 $r = a \bmod b$。

现在我们比较两组数：$(a,b)$ 和 $(b,r)$。

先证明：如果 $d$ 同时整除 $a,b$，那么它也整除 $r$。

因为：

$r = a - qb$

既然 $d|a$ 且 $d|b$，那么 $d$ 也整除 $a-qb$，所以 $d|r$。

再证明反过来：如果 $d$ 同时整除 $b,r$，那么它也整除 $a$。

因为：

$a = qb + r$

既然 $d|b$ 且 $d|r$，自然 $d|a$。

于是，$(a,b)$ 和 $(b,r)$ 拥有完全相同的公因子集合，因此它们的最大公约数相同：

$\gcd(a,b)=\gcd(b,r)=\gcd(b,a \bmod b)$

这一步非常关键，因为它告诉我们：

> **求 gcd 时，可以把“大问题”变成一个更小的同类问题，而且答案不变。**

这就是递归与迭代的来源。

### 2.4 欧几里得算法为什么一定会停

因为每做一步：

$(a,b) \to (b, a \bmod b)$

第二个数都会严格变小，而且始终非负。非负整数不可能无限下降，所以总有一天余数会变成 $0$。

一旦到了：

$a = qb + 0$

也就是 $b | a$，那么 $a$ 和 $b$ 的最大公约数就是 $b$ 本身。

所以欧几里得算法的终点是：

> 当余数第一次变成 $0$ 时，前一个非零余数就是 gcd。

#### 核心代码：欧几里得算法

```cpp
long long gcd(long long a, long long b) {
    if (b == 0) return a >= 0 ? a : -a;
    return gcd(b, a % b);
}

long long lcm(long long a, long long b) {
    return a / gcd(a, b) * b;
}
```

### 2.5 一个完整例子：$\gcd(252, 198)$

我们来完整做一次：

- $252 = 1 * 198 + 54$
- $198 = 3 * 54 + 36$
- $54 = 1 * 36 + 18$
- $36 = 2 * 18 + 0$

所以：

$\gcd(252,198)=18$

注意这里不只是“算出来了 18”，更重要的是你应该意识到：

- 每一步都没有改变公因子集合；
- 每一步都缩小了规模；
- 最终 gcd 是被“逼出来”的，而不是猜出来的。

## 三、从 gcd 到裴蜀定理：为什么 gcd 能写成线性组合

前面我们知道，$\gcd(a,b)$ 代表所有公因子的总代表。现在进一步追问：

> 这个 gcd 能不能只用 $a$ 和 $b$ 本身拼出来？

答案是能。这就是裴蜀定理。

### 3.1 裴蜀定理的内容

对于任意整数 $a,b$，存在整数 $x,y$，使得

$ax + by = \gcd(a,b)$

注意这里最重要的不是“有两个整数 x,y”，而是：

> **gcd 不仅是一个整除意义下的最大公共结构，它还可以被表达成原数的整数线性组合。**

这就把“因子问题”变成了“方程问题”。

### 3.2 为什么它成立：最小正线性组合证明

考虑所有正的整数线性组合：

$S = { ax + by | ax + by > 0, x,y \in Z }$

这是一个非空集合。原因很简单：你总能找到一些整数 $x,y$ 让 $ax+by$ 为正，例如直接让一个系数取很大。

根据正整数集合的良序性，$S$ 一定有最小正元素。记这个最小值为 $d$，即存在整数 $x_{0},y_{0}$ 使得

$d = ax_{0} + by_{0}$

接下来要证明：这个最小正线性组合就是 $\gcd(a,b)$。

#### 第一步：证明 $d|a$

对 $a$ 用带余除法：

$a = qd + r, 0 \le r < d$

于是

$r = a - qd = a - q(ax_{0} + by_{0})$

整理得

$r = a(1-qx_{0}) + b(-qy_{0})$

所以 $r$ 也是 $a,b$ 的整数线性组合。

但 $d$ 是所有正线性组合里最小的正数，如果 $r > 0$，那么 $r$ 比 $d$ 更小，矛盾。

所以只能 $r = 0$，即 $d|a$。

#### 第二步：同理证明 $d|b$

同样地，对 $b$ 做带余除法，也能推出 $d|b$。

#### 第三步：证明任何公因子都整除 $d$

若某个整数 $c$ 同时整除 $a,b$，那么它也必然整除任何线性组合 $ax+by$，所以 $c|d$。

于是：

- $d$ 是 $a,b$ 的一个公因子；
- 所有公因子都整除 $d$。

因此 $d$ 就是最大公约数：

$d = \gcd(a,b)$

裴蜀定理得证。

### 3.3 为什么裴蜀定理很重要

因为它立刻给出一个更强的结论：

$ax + by = c$

有整数解，当且仅当

$\gcd(a,b) | c$

理由非常自然：

- 如果方程有解，那么左边是 $a,b$ 的一个线性组合，因此一定是 $\gcd(a,b)$ 的倍数；
- 如果 $c$ 是 $\gcd(a,b)$ 的倍数，而 $\gcd(a,b)$ 本身又能写成 $ax+by$，那把这个式子整体乘上 $c/\gcd(a,b)$ 就得到 $c$ 的表达。

所以从这一刻开始，gcd 不再只是“求一下约数”，它变成了：

> **判断一个整系数方程能不能有整数解的核心标准。**

这一步直接通向扩展欧几里得和线性同余。

## 四、扩展欧几里得：把“存在”变成“构造”

裴蜀定理告诉你解存在，但竞赛和算法里通常不满足于“存在”。我们往往需要真的求出一组 $x,y$。

这就是扩展欧几里得算法的意义：

> 在求 $\gcd(a,b)$ 的同时，求出 $ax+by=\gcd(a,b)$ 的一组整数解。

### 4.1 它为什么能从欧几里得算法里长出来

欧几里得算法每一步都在做：

$\gcd(a,b)=\gcd(b,a \bmod b)$

假设我们已经知道对子问题 $(b, a \bmod b)$，存在一组解：

$bx_{1} + (a \bmod b)y_{1} = g$

其中 $g = \gcd(a,b)$。

而

$a \bmod b = a - \lfloor a/b \rfloor * b$

把它代回去：

$bx_{1} + (a - \lfloor a/b \rfloor b)y_{1} = g$

整理：

$ay_{1} + b(x_{1} - \lfloor a/b \rfloor y_{1}) = g$

于是对于原问题 $(a,b)$，只要取：

- $x = y_{1}$
- $y = x_{1} - \lfloor a/b \rfloor * y_{1}$

就得到一组解。

这就是扩展欧几里得递推式的来源。它本质上就是把欧几里得算法中的每一步倒着展开。

### 4.2 具体回代一次：以 $99$ 和 $78$ 为例

先用普通欧几里得算法：

- $99 = 1 * 78 + 21$
- $78 = 3 * 21 + 15$
- $21 = 1 * 15 + 6$
- $15 = 2 * 6 + 3$
- $6 = 2 * 3 + 0$

所以 $\gcd(99,78)=3$。

现在倒着回代：

从

$3 = 15 - 2 * 6$

开始，而

$6 = 21 - 1 * 15$

代入得

$3 = 15 - 2(21 - 15) = 3 * 15 - 2 * 21$

再由

$15 = 78 - 3 * 21$

代入得

$3 = 3(78 - 3 * 21) - 2 * 21 = 3 * 78 - 11 * 21$

再由

$21 = 99 - 1 * 78$

代入得

$3 = 3 * 78 - 11(99 - 78) = -11 * 99 + 14 * 78$

所以得到一组裴蜀系数：

$99 * (-11) + 78 * 14 = 3$

这就是 exgcd 的核心：正向递归做带余除法，反向回代恢复线性组合。

### 4.3 扩展欧几里得的作用

扩展欧几里得直接连接了三类问题：

1. 求 $ax+by=\gcd(a,b)$ 的一组解；
2. 求 $ax+by=c$ 的整数解；
3. 求逆元、解线性同余方程。

#### 核心代码：扩展欧几里得

```cpp
long long exgcd(long long a, long long b, long long &x, long long &y) {
    if (b == 0) {
        x = 1;
        y = 0;
        return a;
    }
    long long g = exgcd(b, a % b, y, x);
    y -= (a / b) * x;
    return g;
}
```

这段代码里最容易看懵的，就是这两句：

```cpp
long long g = exgcd(b, a % b, y, x);
y -= (a / b) * x;
```

可以按下面三步理解。

**第一步：递归返回后**

```cpp
long long g = exgcd(b, a % b, y, x);
```

对子问题 $(b, a \bmod b)$，我们设它返回的一组解为：

$b * x_{1} + (a \bmod b) * y_{1} = g$

这里参数写成 $(y, x)$，意思就是：

- 当前的 $y$ 接住子问题里的 $x_{1}$
- 当前的 $x$ 接住子问题里的 $y_{1}$

也就是递归返回后：

- $y = x_{1}$
- $x = y_{1}$

**第二步：代入回推公式**

由

$a \bmod b = a - \lfloor a / b \rfloor * b$

可得：

$b * x_{1} + (a \bmod b) * y_{1} = g$

等价于

$a * y_{1} + b * (x_{1} - \lfloor a / b \rfloor * y_{1}) = g$

因此当前层的一组解应当是：

- $x = y_{1}$
- $y = x_{1} - \lfloor a / b \rfloor * y_{1}$

**第三步：对照变量名修正 $y$**

递归返回后我们已经有：

- 当前 $x = y_{1}$
- 当前 $y = x_{1}$

所以只需要再做一步：

```cpp
y -= (a / b) * x;
```

代进去就是：

$y = x_{1} - (a / b) * y_{1}$

这正好就是上面的回推公式。

#### 统一模板：如何解 $ax+by=c$

这类方程的标准流程可以固定成四步：

1. 先求 $g = \gcd(a,b)$。
2. 若 $g$ 不整除 $c$，则方程无整数解。
3. 用扩展欧几里得求出一组 $ax_{0} + by_{0} = g$ 的解。
4. 将该解整体乘上 $c/g$，得到 $ax + by = c$ 的一组特解。

也就是：

$x_{1} = x_{0} * (c/g), y_{1} = y_{0} * (c/g)$

这时 $(x_{1}, y_{1})$ 是一组特解。

进一步，所有整数解都可写成：

$x = x_{1} + (b/g)t$

$y = y_{1} - (a/g)t$

其中 $t \in Z$。

为什么通解一定是这个形式？因为若 $(x,y)$ 和 $(x_{1},y_{1})$ 都是方程 $ax+by=c$ 的解，那么两式相减得到：

$a(x-x_{1}) + b(y-y_{1}) = 0$

即

$a(x-x_{1}) = -b(y-y_{1})$

把 $a,b$ 同时除以 $g = \gcd(a,b)$ 后，剩下的两个数互质，于是可以推出：

- $b/g | (x-x_{1})$
- $a/g | (y-y_{1})$

于是只能写成上面的通解形式。

核心实现如下：

```cpp
bool solve_diophantine(long long a, long long b, long long c,
                       long long &x, long long &y, long long &g) {
    g = exgcd(a, b, x, y);
    if (c % g != 0) return false;
    long long k = c / g;
    x *= k;
    y *= k;
    return true;
}
```

#### 用这个模板看两个具体例子

**例 1：$18x + 23y = 300$**

先求 $\gcd(18,23)=1$，所以有解。

扩展欧几里得可以求出：

$18 * 9 + 23 * (-7) = 1$

两边乘 $300$，得到一组特解：

$x = 2700, y = -2100$

因此通解为：

$x = 2700 + 23t$

$y = -2100 - 18t$

取 $t = -117$，得到更小的一组解：

$x = 9, y = 6$

检验：

$18 * 9 + 23 * 6 = 300$

**例 2：$99x + 78y = 3$**

先求 $\gcd(99,78)=3$，而 $3 | 3$，所以有解。

扩展欧几里得回代可得：

$99 * (-11) + 78 * 14 = 3$

所以一组特解就是：

$x = -11, y = 14$

通解为：

$x = -11 + 26t$

$y = 14 - 33t$

这里步长来自：

- $b/g = 78/3 = 26$
- $a/g = 99/3 = 33$

## 五、从线性组合走到模方程：逆元和线性同余

### 5.1 为什么同余方程本质上还是方程

同余

$ax \equiv b \pmod{m}$

按定义等价于：

$m | (ax-b)$

也就是存在整数 $y$，使得

$ax + my = b$

所以一个模方程，其实就是一个整数方程。

这是一个极其重要的等价改写，因为它让我们立刻回到裴蜀定理和 exgcd 的世界。

### 5.2 $ax \equiv b \pmod{m}$ 什么时候有解

因为它等价于

$ax + my = b$

所以根据裴蜀定理，它有解当且仅当

$\gcd(a,m) | b$

这就是线性同余方程最根本的判解条件。

这一步你应该特别熟悉它与前文的关系：

- 裴蜀定理告诉你 $ax+my$ 能表示哪些整数；
- 线性同余方程只是把这个问题换了个写法。

### 5.3 逆元为什么要求互质

所谓 $a$ 在模 $m$ 下的逆元，是指存在整数 $x$ 使得

$ax \equiv 1 \pmod{m}$

按刚才的等价改写，这等价于

$ax + my = 1$

根据裴蜀定理，这有解当且仅当

$\gcd(a,m)=1$

所以逆元存在的充要条件就是：

> $a$ 与 $m$ 互质。

这说明逆元并不是一个“凭空定义出的新东西”，它只是裴蜀定理在模意义下的直接翻译。

### 5.4 为什么 exgcd 能求逆元

如果扩展欧几里得求出了：

$ax + my = 1$

那么把这个式子对 $m$ 取模，第二项 $my$ 会消失，于是得到：

$ax \equiv 1 \pmod{m}$

所以这里的 $x$，就是 $a$ 在模 $m$ 下的逆元。

这就是 exgcd 求逆元的原理。

#### 核心代码：逆元与线性同余

```cpp
long long mod_inverse(long long a, long long m) {
    long long x, y;
    long long g = exgcd(a, m, x, y);
    if (g != 1) return -1;           // 逆元不存在
    return (x % m + m) % m;
}

// 解 ax ≡ b (mod m)
// 返回最小非负解；若无解则返回 -1
long long solve_linear_congruence(long long a, long long b, long long m) {
    long long x, y;
    long long g = exgcd(a, m, x, y);
    if (b % g != 0) return -1;
    long long mod = m / g;
    return ((__int128)x * (b / g) % mod + mod) % mod;
}
```

线性同余与逆元，都是裴蜀定理在模意义下的直接应用。

## 六、欧拉函数：为什么它会成为“幂模运算的周期长度”

下面转向幂模运算。这里的核心问题是：为什么幂在模意义下会出现周期。这就需要欧拉函数 $\varphi(n)$。

### 6.1 $\varphi(n)$ 到底在数什么

$\varphi(n)$ 表示 $1$ 到 $n$ 中，与 $n$ 互质的正整数个数。

例如：

- $\varphi(8)=4$，因为 $1,3,5,7$ 与 $8$ 互质；
- $\varphi(12)=4$，因为 $1,5,7,11$ 与 $12$ 互质。

为什么这个函数这么重要？

因为在模 $n$ 的世界里，真正能参与“乘法逆元”和“乘法群结构”的，正是那些与 $n$ 互质的剩余类。$\varphi(n)$ 数的就是这批元素的个数。

### 6.2 为什么 $\varphi(p^k)=p^k-p^{k-1}$

设 $n=p^k$，其中 $p$ 是质数。

在 $1$ 到 $p^k$ 中，不与 $p^k$ 互质的数，恰好就是 $p$ 的倍数。

这样的数有多少个？

$p, 2p, 3p, \cdots , p^{k-1}p$

总共正好有 $p^{k-1}$ 个。

所以与 $p^k$ 互质的数个数就是：

$\varphi(p^k)=p^k-p^{k-1}=p^k(1-1/p)$

这一步说明：

> 对一个质数幂来说，互质的本质就是“不要被这个质数整除”。

### 6.3 为什么 $\varphi$ 对互质乘法具有乘法性

若 $\gcd(m,n)=1$，为什么

$\varphi(mn)=\varphi(m)\varphi(n)$？

直觉上，这是因为：

- 一个数与 $mn$ 互质，等价于它同时与 $m$、$n$ 互质；
- 当 $m,n$ 互质时，模 $mn$ 的剩余类与一对模 $m$、模 $n$ 的剩余类一一对应。

也就是说，模 $mn$ 的“互质剩余类”其实可以看成：

- 任选一个模 $m$ 的互质剩余类；
- 再任选一个模 $n$ 的互质剩余类；
- 把它们拼成一个模 $mn$ 的剩余类。

选择数自然就是乘法：

$\varphi(mn)=\varphi(m)\varphi(n)$

这里你可以提前感受到 CRT 的影子：

> “一个模 $mn$ 的对象能拆成模 $m$ 和模 $n$ 的两个对象”，这正是中国剩余定理的精神。

### 6.4 一般公式为什么是 $\varphi(n)=n\prod(1-1/p)$

如果

$n = p_1^{e_1} \cdot p_2^{e_2} \cdot \cdots \cdot p_k^{e_k}$

那么利用：

- 质数幂公式 $\varphi(p_i^{e_i})=p_i^{e_i}(1-1/p_i)$
- 乘法性 $\varphi(ab)=\varphi(a)\varphi(b)$（当 $a,b$ 互质）

就立刻得到：

$\varphi(n)=n \cdot \prod(1-1/p_i)$

其中乘积遍历的是 $n$ 的所有不同质因子。

注意这里最值得理解的不是公式本身，而是公式为什么只和“不同的质因子”有关：

- 是否互质，取决于有没有共享某个质因子；
- 共享了多少次，并不影响“互不互质”这件事的真假。

#### 核心代码：单点求欧拉函数

```cpp
long long phi(long long n) {
    long long res = n;
    for (long long p = 2; p * p <= n; p++) {
        if (n % p == 0) {
            res = res / p * (p - 1);
            while (n % p == 0) n /= p;
        }
    }
    if (n > 1) res = res / n * (n - 1);
    return res;
}
```

## 七、欧拉定理：为什么幂模运算会出现循环

### 7.1 欧拉定理的内容

若 $\gcd(a,n)=1$，则

$a^{\varphi(n)} \equiv 1 \pmod{n}$

这是数论里最重要的周期结论之一。

### 7.2 为什么它成立：简化剩余系的排列证明

把所有与 $n$ 互质的模 $n$ 剩余类记成：

$r_1, r_2, \cdots , r_{\varphi(n)}$

因为 $\gcd(a,n)=1$，所以把每个 $r_i$ 都乘上 $a$ 后：

$ar_1, ar_2, \cdots , ar_{\varphi(n)}$

仍然都与 $n$ 互质。

而且不会出现重复。为什么？

如果

$ar_i \equiv ar_j \pmod{n}$

那么

$a(r_i-r_j) \equiv 0 \pmod{n}$

由于 $a$ 与 $n$ 互质，可以约去 $a$，得到

$r_i \equiv r_j \pmod{n}$

所以它们只是原来那组互质剩余类的一个重新排列。

于是两组元素的乘积模 $n$ 相等：

$(ar_1)(ar_2)\cdots (ar_{\varphi(n)}) \equiv r_1 r_2 \cdots r_{\varphi(n)} \pmod{n}$

左边提取出 $a^{\varphi(n)}$：

$a^{\varphi(n)} \cdot r_1 r_2 \cdots r_{\varphi(n)} \equiv r_1 r_2 \cdots r_{\varphi(n)} \pmod{n}$

而 $r_1 r_2 \cdots r_{\varphi(n)}$ 也与 $n$ 互质，因此可以在模 $n$ 下约掉，得到：

$a^{\varphi(n)} \equiv 1 \pmod{n}$

欧拉定理得证。

欧拉定理说明：对任意与 $n$ 互质的 $a$，$\varphi(n)$ 给出了一个通用的幂周期长度。这正是降幂的基础。

#### 核心代码：快速幂与欧拉降幂

```cpp
long long qpow(long long a, long long b, long long mod) {
    long long res = 1 % mod;
    while (b > 0) {
        if (b & 1) res = (__int128)res * a % mod;
        a = (__int128)a * a % mod;
        b >>= 1;
    }
    return res;
}

long long euler_pow(long long a, long long b, long long n) {
    if (gcd(a, n) != 1) return qpow(a, b, n);
    return qpow(a, b % phi(n), n);
}
```

这里和前文的联系是：前面讨论“互质时逆元是否存在”，这里讨论“所有可逆元素整体的乘法周期”。

## 八、费马小定理：欧拉定理在质数模下的特例

### 8.1 费马小定理的内容

若 $p$ 是质数，且 $p$ 不整除 $a$，则

$a^{p-1} \equiv 1 \pmod{p}$

因为当 $p$ 是质数时，$1,2,\cdots ,p-1$ 都与 $p$ 互质，所以

$\varphi(p)=p-1$

把它代入欧拉定理，就得到费马小定理。

所以最本质的理解应该是：

> **费马小定理不是一条孤立的公式，而是欧拉定理在质数模下的直接特例。**

### 8.2 为什么质数模下更简单

模质数 $p$ 时，除了 $0$ 以外所有剩余类都与 $p$ 互质，因此每个非零元素都有逆元，结论也最简洁。

### 8.3 为什么逆元会变成 $a^{p-2}$

由费马小定理：

$a^{p-1} \equiv 1 \pmod{p}$

把左边拆成

$a \cdot a^{p-2}$

就得到

$a \cdot a^{p-2} \equiv 1 \pmod{p}$

所以 $a^{p-2}$ 正是 $a$ 的逆元。

这一步不是“背公式”，而是直接从“逆元的定义 + 费马小定理”推出的。

#### 核心代码：质数模下的逆元

```cpp
long long mod_inverse_prime(long long a, long long p) {
    if (a % p == 0) return -1;
    return qpow(a, p - 2, p);
}
```

### 8.4 费马、欧拉与 exgcd 的关系

求逆元有三种常见写法：

1. exgcd：最一般；
2. 欧拉：$a^{\varphi(m)-1}$；
3. 费马：质数模下特化成 $a^{p-2}$。

三者处理的是同一个问题，只是适用条件不同。

## 九、中国剩余定理：为什么一组模条件能拼成一个整数

前面我们讨论的一直是“一个模数下”的事情。现在把问题升级：

> 如果一个整数分别满足多个模条件，能不能把这些条件统一起来？

这就是中国剩余定理。

### 9.1 先看问题长什么样

给定方程组：

- $x \equiv a_{1} \pmod{m_{1}}$
- $x \equiv a_{2} \pmod{m_{2}}$
- ...
- $x \equiv a_k \pmod{m_k}$

当 $m_1, m_2, \ldots, m_k$ **两两互质** 时，CRT 说：

> 这个方程组有唯一的模 $M = m_1m_2 \cdots m_k$ 意义下的解。

这里“唯一”不是说只有一个整数解，而是说所有解都属于同一个模 $M$ 的同余类。

### 9.2 为什么解是唯一的

假设 $x$ 和 $y$ 都满足这组方程，那么对每个 $i$ 都有：

$x \equiv y \pmod{m_i}$

也就是 $m_i \mid (x-y)$。

因为所有 $m_i$ 两两互质，所以它们的乘积

$M = m_1m_2 \cdots m_k$

也整除 $x-y$。

于是

$x \equiv y \pmod{M}$

所以解在模 $M$ 意义下唯一。

这一步本质上仍然是在利用“互质”让多个局部条件合成一个整体条件。

### 9.3 为什么解一定存在：构造法的本质

令：

$M = m_1m_2 \cdots m_k$

再令

$M_i = M / m_i$

由于模数两两互质，$M_i$ 与 $m_i$ 一定互质，所以 $M_i$ 在模 $m_i$ 下有逆元。记这个逆元为 $t_i$，即

$M_i t_i \equiv 1 \pmod{m_i}$

于是构造

$e_i = M_i t_i$

这个 $e_i$ 有非常妙的性质：

- $e_i \equiv 1 \pmod{m_i}$
- $e_i \equiv 0 \pmod{m_j}$，其中 $j \ne i$

因为当 $j \ne i$ 时，$M_i$ 本身就含有因子 $m_j$。

所以 $e_i$ 像一个“开关”：

- 对第 $i$ 个方程，它保留 $1$；
- 对其他方程，它都变成 $0$。

于是只要令

$x = a_1e_1 + a_2e_2 + \cdots + a_ke_k$

就会得到：

- $x \equiv a_1e_1 + a_2e_2 + \cdots + a_ke_k \equiv a_1 \pmod{m_1}$
- $x \equiv a_1e_1 + a_2e_2 + \cdots + a_ke_k \equiv a_2 \pmod{m_2}$
- ...
- $x \equiv a_1e_1 + a_2e_2 + \cdots + a_ke_k \equiv a_k \pmod{m_k}$

所以它同时满足所有同余条件。

这就是 CRT 的构造性证明。

#### 核心代码：标准 CRT

```cpp
long long crt(const vector<long long> &a, const vector<long long> &m) {
    long long M = 1;
    for (long long mod : m) M *= mod;

    long long ans = 0;
    for (int i = 0; i < (int)m.size(); i++) {
        long long Mi = M / m[i];
        long long ti = mod_inverse(Mi, m[i]);   // 这里要求互质
        ans = (ans + (__int128)a[i] * Mi % M * ti) % M;
    }
    return (ans % M + M) % M;
}
```

### 9.4 一个具体例子帮助理解

经典例子：

- $x \equiv 2 \pmod{3}$
- $x \equiv 3 \pmod{5}$
- $x \equiv 2 \pmod{7}$

总模数：

$M = 3 \times 5 \times 7 = 105$

分别有：

- $M_{1} = 35$
- $M_{2} = 21$
- $M_{3} = 15$

接下来找逆元：

- $35 \equiv 2 \pmod{3}$，$2$ 在模 $3$ 下的逆元是 $2$，所以 $t_{1}=2$
- $21 \equiv 1 \pmod{5}$，逆元是 $1$，所以 $t_{2}=1$
- $15 \equiv 1 \pmod{7}$，逆元是 $1$，所以 $t_{3}=1$

于是：

$x \equiv 2 \times 35 \times 2 + 3 \times 21 \times 1 + 2 \times 15 \times 1$

即

$x \equiv 140 + 63 + 30 = 233 \equiv 23 \pmod{105}$

所以最小正解是 $23$。

这里的关键不在于算出 $23$，而在于看清楚 CRT 的构造方式：每一项只负责自己的那个模条件。

### 9.5 为什么模数不互质时要小心

如果模数不互质，事情就复杂了，因为：

- 唯一性未必成立；
- 存在性也未必成立；
- $M_i$ 与 $m_i$ 不一定互质，因此逆元不一定存在。

这时不能直接套标准 CRT，而要转向 **扩展 CRT（EXCRT）**。

它的思路仍然很自然：

假设已经合并出一个条件

$x \equiv r \pmod{M}$

现在要并入一个新条件

$x \equiv a \pmod{m}$

把 $x$ 写成 $x = r + Mk$，代入得到：

$Mk \equiv a-r \pmod{m}$

这一下又回到了我们熟悉的线性同余方程。它有解的条件是：

$\gcd(M,m) | (a-r)$

只有满足这个条件，两个同余式才彼此兼容，才能继续合并。

EXCRT 的本质，就是不断把“多模问题”压回到“线性同余问题”，再交给 exgcd 处理。

#### 核心代码：扩展 CRT

```cpp
long long excrt(const vector<long long> &a, const vector<long long> &m) {
    long long R = a[0], M = m[0];
    for (int i = 1; i < (int)a.size(); i++) {
        long long x, y;
        long long g = exgcd(M, m[i], x, y);
        long long c = a[i] - R;
        if (c % g != 0) return -1;   // 无解

        long long mod = m[i] / g;
        x = ((__int128)x * (c / g) % mod + mod) % mod;
        R += (__int128)x * M;
        M = M / g * m[i];
        R = (R % M + M) % M;
    }
    return R;
}
```

## 十、总结

整条主线可以压缩成下面几步：

1. $\gcd$ 描述两个整数共享的整除结构；
2. 欧几里得算法利用 $\gcd(a,b)=\gcd(b,a \bmod b)$ 递归求解；
3. 裴蜀定理说明 $\gcd(a,b)$ 可以写成 $ax+by$ 的形式；
4. 扩展欧几里得把这组系数真正求出来；
5. 线性同余与逆元，是裴蜀定理在模意义下的应用；
6. 欧拉函数与欧拉定理说明互质元素在模乘法下具有统一周期；
7. 费马小定理是欧拉定理在质数模下的特例；
8. CRT 把多个互质模条件合并成一个整数条件，EXCRT 则把非互质情形重新化回线性同余。

这条线里最常反复出现的只有两个关键词：

- **互质**：决定逆元是否存在、欧拉定理能否直接使用、CRT 是否能直接构造；
- **改写**：$\gcd(a,b)$ 改写成 $\gcd(b,a \bmod b)$，$ax \equiv b \pmod{m}$ 改写成 $ax+my=b$，多模条件再继续改写成可合并的形式。

把这两点抓住，整篇内容就会连成一体。
