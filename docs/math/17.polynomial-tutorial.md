## 一、什么是多项式算法

多项式算法是竞赛数学中最强大的工具集之一。以 FFT/NTT 为根基，将多项式乘法从 $O(n^2)$ 加速到 $O(n\log n)$，继而通过 Newton 迭代衍生出求逆、开根、$\ln$、$\exp$ 等一整套操作，广泛应用于组合计数、生成函数与卷积优化。

**核心操作速查表**

| 操作 | 时间复杂度 | 依赖 |
|------|-----------|------|
| 多项式乘法（FFT/NTT） | $O(n\log n)$ | 单位根/原根 |
| 多项式求逆 | $O(n\log n)$ | Newton 迭代 + NTT |
| 多项式 $\ln$ | $O(n\log n)$ | 求导 + 求逆 + 积分 |
| 多项式 $\exp$ | $O(n\log n)$ | Newton 迭代 + $\ln$ |
| 多项式开根 | $O(n\log n)$ | Newton 迭代 + 求逆 |
| 多项式除法/取模 | $O(n\log n)$ | 翻转 + 求逆 |
| 分治 FFT | $O(n\log^2 n)$ | CDQ 分治 + NTT |
| 多点求值 | $O(n\log^2 n)$ | 分治求余 |
| 快速插值 | $O(n\log^2 n)$ | 分治 + 多点求值 |
| FWT（沃尔什变换） | $O(n\log n)$ | 按位蝶形变换 |
| 拉格朗日插值 | $O(n^2)$ / $O(n\log^2 n)$ | 点值还原 |

## 二、核心思想与前置知识

### 2.1 FFT（快速傅里叶变换）

将多项式从系数表示转为点值表示（DFT），逐点相乘后逆变换（IDFT）回系数表示。利用 $n$ 次单位根 $\omega_n = e^{2\pi i/n}$ 的性质递归分治，总时间 $O(n\log n)$。

**关键公式**：$\omega_n^k = e^{2\pi ik/n}$，满足消去引理 $\omega_{2n}^{2k} = \omega_n^k$ 和折半引理。

### 2.2 NTT（数论变换）

将 FFT 中的复数单位根替换为模素数下的原根。常用模数 $998244353 = 119 \times 2^{23} + 1$，原根 $g = 3$。在模意义下精确运算，避免浮点误差。

### 2.3 Newton 迭代

在形式幂级数上，Newton 迭代每步将精度从 $\bmod x^n$ 提升到 $\bmod x^{2n}$，总迭代 $O(\log n)$ 步。是求逆、开根、$\exp$ 的统一框架。

**通用公式**：$g_{new} = g_{old} - \frac{h(g_{old})}{h'(g_{old})} \pmod{x^{2n}}$

### 2.4 多项式 ln 和 exp

- $\ln f(x) = \int \frac{f'(x)}{f(x)} \, dx$，需要 $f(0) = 1$
- $\exp f(x)$：Newton 迭代 $g \leftarrow g \cdot (1 - \ln g + f) \pmod{x^{2n}}$，需要 $f(0) = 0$

### 2.5 FWT（快速沃尔什变换）

处理按位 OR/AND/XOR 三种卷积，形式类似 FFT 的蝶形变换，但作用于二进制位。

- OR 卷积：$c[s] = \sum_{i \mathbin{|} j = s} a[i] \cdot b[j]$
- AND 卷积：$c[s] = \sum_{i \mathbin{\&} j = s} a[i] \cdot b[j]$
- XOR 卷积：$c[s] = \sum_{i \oplus j = s} a[i] \cdot b[j]$

### 2.6 拉格朗日插值

已知 $n+1$ 个点值 $(x_0, y_0), \ldots, (x_n, y_n)$，还原 $n$ 次多项式：

$$L(x) = \sum_{i=0}^{n} y_i \prod_{j \neq i} \frac{x - x_j}{x_i - x_j}$$

直接计算 $O(n^2)$；连续点 $x_i = i$ 时可利用前后缀积优化到 $O(n)$。

### 2.7 全家桶依赖关系

```
NTT（乘法）
 └─→ 求逆（Newton 迭代）
      ├─→ ln（求导 + 求逆 + 积分）
      │    └─→ exp（Newton 迭代 + ln）
      │         └─→ 快速幂 = exp(k * ln f)
      ├─→ 开根（Newton 迭代 + 求逆）
      └─→ 除法/取模（翻转 + 求逆）
           └─→ 多点求值（分治求余）
```

### 题目 1：多项式乘法 FFT（洛谷 P3803）🟢

> 给定两个多项式 $A(x)$ 和 $B(x)$，求它们的乘积 $C(x) = A(x) \cdot B(x)$，输出各项系数。

**分析**：多项式乘法等价于系数的卷积 $c_k = \sum_{i+j=k} a_i \cdot b_j$，直接计算 $O(n^2)$。利用 FFT，先将两个多项式 DFT 到点值，逐点相乘，再 IDFT 回系数，总时间 $O(n\log n)$。

```
核心观察：多项式乘法 = 系数卷积，DFT 将卷积转化为逐点乘法
构造策略：补齐长度到 2 的幂 → FFT(A) → FFT(B) → 逐点乘 → IFFT(C)
正确性：卷积定理保证 DFT(A*B) = DFT(A) · DFT(B)
```

```cpp
// FFT 求多项式乘积（已有 fft 函数和 Complex 类型）
void poly_mul(Complex A[], Complex B[], int n, int m) {
    int len = 1;
    while (len <= n + m) len <<= 1;
    fft(A, len, 1); fft(B, len, 1);
    for (int i = 0; i < len; i++)
        A[i] = A[i] * B[i];
    fft(A, len, -1);
    for (int i = 0; i <= n + m; i++)
        ans[i] = (long long)(A[i].r + 0.5);
}
```

**核心启示**：

- **FFT 的本质是利用单位根将 $O(n^2)$ 卷积加速到 $O(n\log n)$**
- **长度必须补齐到 $\geq n+m+1$ 的最小 2 的幂**
- **IFFT 后要四舍五入取整，注意浮点精度**

### 题目 2：多项式求逆（洛谷 P4238）🟡

> 给定多项式 $f(x)$，求 $g(x)$ 使得 $f(x) \cdot g(x) \equiv 1 \pmod{x^n}$，在模 $998244353$ 下输出系数。

**分析**：这是 Newton 迭代在多项式上的经典应用。设已求出 $g_n$ 满足 $f \cdot g_n \equiv 1 \pmod{x^n}$，则可倍增得到 $\bmod x^{2n}$ 的解。令 $h(g) = 1/g - f = 0$，Newton 迭代得 $g_{2n} = g_n(2 - f \cdot g_n)$。边界 $g_1 = f_0^{-1}$。

```
核心观察：Newton 迭代每步精度翻倍，O(log n) 步达到 mod x^n
构造策略：g ← g * (2 - f * g) mod x^{2n}，从 n=1 倍增
正确性：令 e = fg - 1，由 e ≡ 0 (mod x^n) 推出 e^2 ≡ 0 (mod x^{2n})
```

```cpp
// 多项式求逆，结果存入 g，已有 ntt 函数
void poly_inv(int f[], int g[], int n) {
    if (n == 1) { g[0] = qpow(f[0], MOD - 2); return; }
    poly_inv(f, g, (n + 1) / 2);
    int len = 1;
    while (len < 2 * n) len <<= 1;
    // tmp = f，做 NTT(tmp) 和 NTT(g)
    for (int i = 0; i < len; i++)
        tmp[i] = (i < n) ? f[i] : 0;
    ntt(tmp, len, 1); ntt(g, len, 1);
    for (int i = 0; i < len; i++)
        g[i] = 1LL * g[i] * (2 - 1LL * tmp[i] * g[i] % MOD + MOD) % MOD;
    ntt(g, len, -1);
    for (int i = n; i < len; i++) g[i] = 0;
}
```

**核心启示**：

- **Newton 迭代是多项式全家桶的基石，求逆是最底层操作**
- **每层复杂度 $T(n) = T(n/2) + O(n\log n)$，总和仍为 $O(n\log n)$**
- **迭代后必须将 $[n, len)$ 清零，防止高次项污染**

### 题目 3：任意模数 NTT（洛谷 P4245）🟡

> 给定两个多项式和任意模数 $p$（$p$ 不一定是 NTT 友好素数），求多项式乘积模 $p$。

**分析**：当 $p$ 不满足 NTT 要求时，不能直接用 NTT。经典做法有两种：(1) 三模数 NTT：分别在三个 NTT 友好模数下卷积，再用 CRT 合并。(2) 拆系数 FFT：将系数拆为高低 15 位，用 4 次 FFT 完成，避免精度问题。

```
核心观察：NTT 需要模数形如 a * 2^k + 1，任意模数需特殊处理
构造策略：选三个 NTT 模数 p1,p2,p3（乘积 > n * V^2），分别 NTT 后 CRT 合并
正确性：CRT 唯一恢复真实卷积值（不超过 n * V^2），再对 p 取模
```

```cpp
// 三模数 NTT 的 CRT 合并（三个模数下的结果 r1, r2, r3）
// p1=998244353, p2=985661441, p3=469762049
long long crt_merge(long long r1, long long r2, long long r3, int p) {
    long long a = 1LL * (r2 - r1 + p2) % p2 * inv_p1_mod_p2 % p2;
    long long val = r1 + a * p1;  // mod p1*p2 的值
    long long b = ((r3 - val % p3 + p3) % p3) * inv_p1p2_mod_p3 % p3;
    return ((val % p + b % p * (p1p2_mod_p) % p) % p + p) % p;
}
```

**核心启示**：

- **三模数 NTT 的关键是选三个乘积足够大的 NTT 友好模数**
- **CRT 合并时中间值可能溢出，需用 `__int128` 或分步取模**
- **拆系数 FFT 是另一条路线，用浮点数但只需 4 次 FFT**

### 题目 4：FWT 模板（洛谷 P4717）🟡

> 给定两个长度为 $2^n$ 的序列 $A, B$，分别求 OR 卷积、AND 卷积、XOR 卷积的结果序列 $C$。

**分析**：按位卷积不能用 FFT，需要 FWT（快速沃尔什变换）。FWT 类似 FFT 的蝶形操作，但针对不同的按位运算有不同的变换矩阵。OR 卷积对应子集和变换，AND 对应超集和变换，XOR 对应沃尔什 - 哈达玛变换。

```
核心观察：三种按位卷积各有对应的正/逆变换，蝶形结构统一处理
构造策略：OR: (a,b)→(a,a+b); AND: (a,b)→(a+b,b); XOR: (a,b)→(a+b,a-b)
正确性：变换后逐点乘再逆变换即得卷积结果
```

```cpp
// FWT OR 变换，inv=1 正变换，inv=-1 逆变换
void fwt_or(long long a[], int n, int inv) {
    for (int len = 2; len <= n; len <<= 1)
        for (int i = 0; i < n; i += len)
            for (int j = 0; j < len / 2; j++) {
                if (inv == 1) a[i + j + len / 2] += a[i + j];
                else a[i + j + len / 2] -= a[i + j];
                a[i + j + len / 2] = (a[i + j + len / 2] % MOD + MOD) % MOD;
            }
}
// XOR 变换类似，蝶形为 (a,b) → (a+b, a-b)，逆变换除以 2
```

**核心启示**：

- **FWT 的三种变换本质是不同的 Mobius 变换/逆变换**
- **XOR 卷积的逆变换需要额外除以 $n$（类似 IFFT）**
- **FWT 时间 $O(n\log n)$（$n = 2^k$），空间 $O(n)$**

### 题目 5：分治 FFT（洛谷 P4721）🟠

> 给定序列 $g[1], g[2], \ldots, g[n-1]$，已知 $f[0] = 1$，递推 $f[i] = \sum_{j=1}^{i} f[i-j] \cdot g[j]$，求 $f[0], f[1], \ldots, f[n-1]$。

**分析**：直接递推 $O(n^2)$。注意到 $f = f * g + [x^0]$ 的形式，但 $f$ 依赖自身无法直接 NTT。CDQ 分治：处理左半 $[l, mid]$ 对右半 $[mid+1, r]$ 的卷积贡献，递归计算。每层 NTT 开销 $O(n\log n)$，共 $O(\log n)$ 层，总 $O(n\log^2 n)$。

```
核心观察：f 的递推是自卷积形式，f[i] 依赖所有 f[j] (j < i) 和 g
构造策略：CDQ 分治——先递归求左半，再用 NTT 算左半对右半的贡献，再递归右半
正确性：分治保证计算 f[i] 时所有 j < i 的贡献已累加完毕
```

```cpp
// 分治 FFT（已有 ntt 函数）
void cdq(int l, int r) {
    if (l == r) return;
    int mid = (l + r) >> 1;
    cdq(l, mid);
    // 计算 f[l..mid] 对 f[mid+1..r] 的贡献
    int len = 1;
    while (len <= r - l) len <<= 1;
    // A[i] = f[l+i], B[i] = g[i]（长度为 r-l+1）
    ntt(A, len, 1); ntt(B, len, 1);
    for (int i = 0; i < len; i++) A[i] = 1LL * A[i] * B[i] % MOD;
    ntt(A, len, -1);
    for (int i = mid + 1; i <= r; i++)
        f[i] = (f[i] + A[i - l]) % MOD;
    cdq(mid + 1, r);
}
```

**核心启示**：

- **分治 FFT 适用于"当前项依赖前面所有项的卷积"的递推**
- **也可用多项式 $\exp$ 替代：$F = \exp(\int G)$ 当 $F' = F \cdot G$ 时**
- **时间 $O(n\log^2 n)$，常数取决于 NTT 实现**

### 题目 6：多项式 ln（洛谷 P4725）🟠

> 给定多项式 $f(x)$（$f_0 = 1$），求 $g(x) = \ln f(x) \pmod{x^n}$，在模 $998244353$ 下输出系数。

**分析**：对 $\ln f(x)$ 求导得 $(\ln f)' = f'/f$，因此 $\ln f = \int f'/f \, dx$。实现上先求导（$O(n)$），再求逆（$O(n\log n)$），再卷积，最后积分（$O(n)$）。整体 $O(n\log n)$。

```
核心观察：ln f = ∫ f'/f dx，将 ln 转化为求导、求逆、积分三步
构造策略：(1) 求 f'；(2) 求 f 的逆 f^{-1}；(3) 卷积 f' * f^{-1}；(4) 积分
正确性：形式幂级数的微积分运算法则
```

```cpp
// 多项式 ln，结果存入 g（已有 poly_inv, ntt, 求导/积分）
void poly_ln(int f[], int g[], int n) {
    // df = f 的导数
    for (int i = 0; i < n - 1; i++) df[i] = 1LL * (i + 1) * f[i + 1] % MOD;
    df[n - 1] = 0;
    poly_inv(f, inv_f, n);
    int len = 1;
    while (len < 2 * n) len <<= 1;
    ntt(df, len, 1); ntt(inv_f, len, 1);
    for (int i = 0; i < len; i++) df[i] = 1LL * df[i] * inv_f[i] % MOD;
    ntt(df, len, -1);
    // 积分：g[i] = df[i-1] / i
    g[0] = 0;
    for (int i = 1; i < n; i++) g[i] = 1LL * df[i - 1] * qpow(i, MOD - 2) % MOD;
}
```

**核心启示**：

- **$\ln$ 的前提是 $f_0 = 1$，否则无法求逆**
- **积分时 $g_0 = 0$（$\ln$ 的常数项为 0）**
- **$\ln$ 是 $\exp$ 的前置操作，两者互逆**

### 题目 7：多项式 exp（洛谷 P4726）🟠

> 给定多项式 $f(x)$（$f_0 = 0$），求 $g(x) = \exp(f(x)) \pmod{x^n}$，在模 $998244353$ 下输出系数。

**分析**：求解 $\ln g - f = 0$。令 $h(g) = \ln g - f$，Newton 迭代：$g_{new} = g_{old} - \frac{\ln g_{old} - f}{1/g_{old}} = g_{old}(1 - \ln g_{old} + f)$。边界 $g_0 = 1$（因为 $f_0 = 0$ 所以 $e^0 = 1$）。每步需要调用 $\ln$，但精度倍增保证总复杂度仍为 $O(n\log n)$。

```
核心观察：Newton 迭代求解 ln g = f，即 h(g) = ln g - f = 0
构造策略：g ← g * (1 - ln(g) + f) mod x^{2n}，从 g=1 倍增
正确性：Newton 迭代收敛，每步精度翻倍
```

```cpp
// 多项式 exp，结果存入 g（已有 poly_ln, ntt）
void poly_exp(int f[], int g[], int n) {
    if (n == 1) { g[0] = 1; return; }
    poly_exp(f, g, (n + 1) / 2);
    poly_ln(g, lng, n);
    int len = 1;
    while (len < 2 * n) len <<= 1;
    for (int i = 0; i < n; i++) lng[i] = (f[i] - lng[i] + MOD) % MOD;
    lng[0] = (lng[0] + 1) % MOD;
    ntt(g, len, 1); ntt(lng, len, 1);
    for (int i = 0; i < len; i++) g[i] = 1LL * g[i] * lng[i] % MOD;
    ntt(g, len, -1);
    for (int i = n; i < len; i++) g[i] = 0;
}
```

**核心启示**：

- **$\exp$ 的前提是 $f_0 = 0$，否则 $\exp(f_0)$ 无法在模意义下计算**
- **$\exp$ 和 $\ln$ 互逆：$\exp(\ln f) = f$**
- **$\exp$ 是多项式快速幂的核心：$f^k = \exp(k \cdot \ln f)$**
- **复杂度递推 $T(n) = T(n/2) + O(n\log n) = O(n\log n)$**

### 题目 8：多项式开根（洛谷 P5205）🟠

> 给定多项式 $f(x)$（$f_0 = 1$），求 $g(x)$ 使 $g(x)^2 \equiv f(x) \pmod{x^n}$，在模 $998244353$ 下输出系数。

**分析**：令 $h(g) = g^2 - f = 0$，Newton 迭代：$g_{new} = g_{old} - \frac{g_{old}^2 - f}{2 g_{old}} = \frac{g_{old} + f/g_{old}}{2} = \frac{g_{old} + f \cdot g_{old}^{-1}}{2}$。边界 $g_0 = 1$（因为 $f_0 = 1$ 的二次剩余为 1）。每步需要一次多项式求逆。

```
核心观察：开根等价于解 g^2 = f，Newton 迭代求解
构造策略：g ← (g + f * g^{-1}) / 2 mod x^{2n}
正确性：Newton 迭代，精度每步翻倍
```

```cpp
// 多项式开根（已有 poly_inv, ntt）
void poly_sqrt(int f[], int g[], int n) {
    if (n == 1) { g[0] = 1; return; }
    poly_sqrt(f, g, (n + 1) / 2);
    poly_inv(g, ig, n);
    int len = 1;
    while (len < 2 * n) len <<= 1;
    for (int i = 0; i < n; i++) tmp[i] = f[i];
    for (int i = n; i < len; i++) tmp[i] = 0;
    ntt(tmp, len, 1); ntt(ig, len, 1);
    for (int i = 0; i < len; i++) tmp[i] = 1LL * tmp[i] * ig[i] % MOD;
    ntt(tmp, len, -1);
    long long inv2 = qpow(2, MOD - 2);
    for (int i = 0; i < n; i++) g[i] = 1LL * (g[i] + tmp[i]) % MOD * inv2 % MOD;
    for (int i = n; i < len; i++) g[i] = 0;
}
```

**核心启示**：

- **$f_0 \neq 1$ 时需先求 $f_0$ 的二次剩余（Cipolla 算法）**
- **开根是 Newton 迭代家族的重要成员，依赖求逆**
- **除以 2 在模意义下变为乘以 $2^{-1}$**

### 题目 9：多项式快速幂（洛谷 P5245）🔴

> 给定多项式 $f(x)$ 和正整数 $k$，求 $f(x)^k \pmod{x^n}$，在模 $998244353$ 下输出系数。

**分析**：直接快速幂 $O(n\log n \cdot \log k)$，但利用 $\ln/\exp$ 可做到 $O(n\log n)$：$f^k = \exp(k \cdot \ln f)$。需要 $f_0 = 1$，若 $f_0 \neq 1$ 则需提取首项系数和前导零进行归一化处理。

```
核心观察：f^k = exp(k * ln f)，将幂运算转化为 ln + 乘常数 + exp
构造策略：先提取前导零和首项，归一化后 ln → 乘 k → exp → 还原
正确性：exp 和 ln 互逆，exp(k * ln f) = f^k
```

```cpp
// 多项式快速幂（已有 poly_ln, poly_exp）
void poly_pow(int f[], int g[], int n, long long k) {
    // 找第一个非零项 f[t]，提取 f[t] 的 k 次幂
    int t = 0;
    while (t < n && !f[t]) t++;
    if ((long long)t * k >= n) { /* g 全零 */ return; }
    long long c = f[t], ic = qpow(c, MOD - 2), ck = qpow(c, k % (MOD - 1));
    // 移位并归一化：h[i] = f[i+t] * ic
    for (int i = 0; i + t < n; i++) h[i] = 1LL * f[i + t] * ic % MOD;
    poly_ln(h, lnh, n - t);
    for (int i = 0; i < n; i++) lnh[i] = 1LL * lnh[i] * (k % MOD) % MOD;
    poly_exp(lnh, g_tmp, n - t * k);
    // 还原：g[i + t*k] = g_tmp[i] * ck
    for (int i = 0; i + t * k < n; i++) g[i + t * k] = 1LL * g_tmp[i] * ck % MOD;
}
```

**核心启示**：

- **$f^k = \exp(k \cdot \ln f)$ 是 $O(n\log n)$ 的优雅做法**
- **前导零个数乘以 $k$，可能超过 $n$ 导致结果全零**
- **$k$ 对 $MOD$ 和 $MOD-1$ 分别取模用于不同位置**

### 题目 10：拉格朗日插值（洛谷 P4781）🟢

> 给定 $n$ 个互不相同的点 $(x_1, y_1), \ldots, (x_n, y_n)$，求过这些点的 $n-1$ 次多项式在 $x = k$ 处的值，答案模 $998244353$。

**分析**：直接套用拉格朗日插值公式 $L(k) = \sum_{i=1}^{n} y_i \prod_{j \neq i} \frac{k - x_j}{x_i - x_j}$。对每个 $i$，分子是 $\prod_{j \neq i} (k - x_j)$，分母是 $\prod_{j \neq i} (x_i - x_j)$。$O(n^2)$ 计算。

```
核心观察：拉格朗日插值公式直接给出任意点处的值
构造策略：对每个 i，O(n) 计算分子分母乘积，O(n^2) 总计
正确性：n 个点唯一确定 n-1 次多项式
```

```cpp
// 拉格朗日插值求 f(k)，n 个点 (x[i], y[i])
long long lagrange(long long x[], long long y[], int n, long long k) {
    long long ans = 0;
    for (int i = 0; i < n; i++) {
        long long num = y[i], den = 1;
        for (int j = 0; j < n; j++) if (i != j) {
            num = num % MOD * ((k - x[j]) % MOD + MOD) % MOD;
            den = den % MOD * ((x[i] - x[j]) % MOD + MOD) % MOD;
        }
        ans = (ans + num % MOD * qpow(den, MOD - 2) % MOD) % MOD;
    }
    return (ans + MOD) % MOD;
}
```

**核心启示**：

- **拉格朗日插值公式是"点值 → 求值"的标准工具**
- **当 $x_i$ 连续（$x_i = i$）时可用前后缀积优化到 $O(n)$**
- **插值求多项式系数需要 $O(n^2)$，或用分治 $O(n\log^2 n)$**

### 题目 11：约定（洛谷 P5437）🔴

> 给定 $n$ 个互不相同的正整数 $a_1, \ldots, a_n$ 和正整数 $m$，求 $\sum_{S \subseteq \{1,\ldots,n\}, |S|=m} \prod_{i \in S} a_i \cdot \left(\sum_{i \in S} a_i\right)$，答案模 $998244353$。

**分析**：将式子拆开，$\sum_{|S|=m} \prod a_i \cdot \sum_{j \in S} a_j = \sum_{|S|=m} \sum_{j \in S} a_j \prod_{i \in S} a_i$。交换求和顺序，枚举 $j$，$= \sum_j a_j^2 \cdot e_{m-1}(\{a_k : k \neq j\})$，其中 $e_k$ 是初等对称多项式。构造生成函数 $\prod (1 + a_i x)$，用 NTT 加速求对称多项式，再逐个"除去"$a_j$ 的贡献。

```
核心观察：将求和拆分后与初等对称多项式关联，可用生成函数表达
构造策略：(1) 求 P(x) = ∏(1 + a_i * x) 的各系数；(2) 对每个 j，P(x)/(1 + a_j * x) 的第 m-1 项即为所需
正确性：∏(1 + a_i * x) 的 x^k 项系数恰为 e_k
```

```cpp
// 核心：已求出 P(x) = prod(1 + a[i]*x) 的系数 p[0..n]
// 对每个 j，需要 P(x)/(1 + a[j]*x) 的第 m-1 项
// 除法可 O(n) 递推：q[0]=1, q[k] = p[k] - a[j]*q[k-1]
long long solve(long long a[], int n, int m, long long p[]) {
    long long ans = 0;
    for (int j = 0; j < n; j++) {
        long long q_prev = 1, q_cur;
        for (int k = 1; k < m; k++) {
            q_cur = (p[k] - a[j] * q_prev % MOD + MOD) % MOD;
            q_prev = q_cur;
        }
        ans = (ans + a[j] % MOD * a[j] % MOD * q_prev) % MOD;
    }
    return ans;
}
```

**核心启示**：

- **初等对称多项式与生成函数 $\prod(1 + a_i x)$ 直接对应**
- **"去掉一个元素"对应多项式除以一个线性因子，可 $O(n)$ 递推**
- **组合恒等式的变换是将问题转化为多项式操作的关键**

### 题目 12：力（洛谷 P3338）🟢

> 给定 $n$ 个电荷 $q_0, q_1, \ldots, q_{n-1}$，对每个 $i$ 求 $E_i = \sum_{j < i} \frac{q_j}{(i-j)^2} - \sum_{j > i} \frac{q_j}{(j-i)^2}$。

**分析**：令 $f(k) = 1/k^2$（$k \geq 1$），则 $\sum_{j<i} q_j \cdot f(i-j)$ 是卷积形式 $(q * f)[i]$。减去的部分也是卷积，翻转 $q$ 后同理。两次 FFT 卷积即可。

```
核心观察：每个求和都是标准卷积形式 (q * f)[i]，f[k] = 1/k^2
构造策略：正向卷积 A = q * f；翻转 q 得 q_rev，反向卷积 B = q_rev * f；E[i] = A[i] - B[n-1-i]
正确性：将差值公式展开后恰好对应卷积定义
```

```cpp
// 求力场（浮点 FFT，已有 fft 和 Complex）
void solve(double q[], double E[], int n) {
    // f[k] = 1.0 / k / k (k >= 1), f[0] = 0
    // A = conv(q, f): A[i] = sum_{j<i} q[j]*f[i-j]
    // 翻转 q 得 qr, B = conv(qr, f)
    // E[i] = A[i] - B[n-1-i]
    for (int i = 0; i < n; i++) qr[i] = q[n - 1 - i];
    // FFT(q)*FFT(f) → IFFT → A
    // FFT(qr)*FFT(f) → IFFT → B
    for (int i = 0; i < n; i++)
        E[i] = A[i].r - B[n - 1 - i].r;
}
```

**核心启示**：

- **识别卷积结构是 FFT 应用题的核心能力**
- **"翻转后卷积"等价于相关（cross-correlation）**
- **浮点 FFT 在精度够用时比 NTT 更灵活**

### 题目 13：礼物（洛谷 P3723）🔴

> 两个长度为 $n$ 的环形序列 $a$ 和 $b$，可以对所有 $a_i$ 加同一个整数 $c$（$|c|$ 任意），可以旋转 $b$，使得差异度 $\sum_{i=0}^{n-1}(a_i - b_i)^2$ 最小。

**分析**：展开平方得 $\sum(a_i + c)^2 + \sum b_i^2 - 2\sum(a_i + c) b_i$。对 $c$ 求导令导数为零得最优 $c$。固定 $c$ 后，需最大化 $\sum a_i b_{(i+k) \bmod n}$（旋转后的内积），这是经典的环形卷积问题：翻转 $a$ 后求卷积取最大值。

```
核心观察：差异度展开后，最优 c 可解析求出；旋转优化等价于最大化环形内积
构造策略：(1) 解析求最优 c；(2) 翻转 a 后 FFT 卷积求所有旋转的内积；(3) 取最大值
正确性：∑ a[i]*b[i+k] = (a_rev * b)[n-1+k]，卷积对应所有旋转
```

```cpp
// 核心：翻转 a 后与 b 拼接两倍做卷积，取卷积最大值
// 最优 c = round((sum_b - sum_a) / n)
void solve(int a[], int b[], int n) {
    long long sa = 0, sb = 0;
    for (int i = 0; i < n; i++) sa += a[i], sb += b[i];
    int c = (int)round((double)(sb - sa) / n);
    for (int i = 0; i < n; i++) a[i] += c;
    // 翻转 a：ar[i] = a[n-1-i]
    // bb[0..2n-1] = b 拼接 b
    // conv = FFT(ar) * FFT(bb)，取 conv[n-1..2n-2] 的最大值
    long long base = 0, mx = 0;
    for (int i = 0; i < n; i++) base += 1LL*a[i]*a[i] + 1LL*b[i]*b[i];
    // mx = max of conv[n-1+k] for k=0..n-1
    printf("%lld\n", base - 2 * mx);
}
```

**核心启示**：

- **"环形匹配最小差异"问题先解析优化平移，再 FFT 优化旋转**
- **翻转 + 卷积 = 互相关，是求所有位移内积的标准手段**
- **$c$ 的最优值由求导为零确定，取附近整数检验**

### 题目 14：多项式除法（洛谷 P4512）🔴

> 给定 $n$ 次多项式 $f(x)$ 和 $m$ 次多项式 $g(x)$（$n \geq m$），求商 $q(x)$（$n-m$ 次）和余数 $r(x)$（$< m$ 次）使得 $f = g \cdot q + r$。

**分析**：定义翻转操作 $f^R(x) = x^n f(1/x)$。代入 $x = 1/x$ 后两边乘 $x^n$ 得 $f^R = g^R \cdot q^R + x^{n-m+1} \cdot r^R$。因此 $f^R \equiv g^R \cdot q^R \pmod{x^{n-m+1}}$，于是 $q^R = f^R \cdot (g^R)^{-1} \pmod{x^{n-m+1}}$，再翻转回得 $q$，最后 $r = f - g \cdot q$。

```
核心观察：翻转多项式后，余数的贡献被 x^{n-m+1} 消去，可单独求商
构造策略：(1) 翻转 f, g；(2) 求 g^R 的逆 mod x^{n-m+1}；(3) q^R = f^R * inv mod x^{n-m+1}；(4) 翻转回 q；(5) r = f - g*q
正确性：翻转变换 + 取模消去余数项
```

```cpp
// 多项式除法（已有 poly_inv, ntt）
void poly_div(int f[], int g[], int q[], int r[], int n, int m) {
    // fr = reverse(f, n), gr = reverse(g, m)
    // 求 igr = gr^{-1} mod x^{n-m+1}
    poly_inv(gr, igr, n - m + 1);
    // qr = fr * igr mod x^{n-m+1}
    ntt(fr, len, 1); ntt(igr, len, 1);
    for (int i = 0; i < len; i++) qr[i] = 1LL * fr[i] * igr[i] % MOD;
    ntt(qr, len, -1);
    // 翻转 qr 得 q
    for (int i = 0; i <= n - m; i++) q[i] = qr[n - m - i];
    // r = f - g * q（NTT 卷积后逐项减）
    for (int i = 0; i < m; i++) r[i] = (f[i] - gq[i] + MOD) % MOD;
}
```

**核心启示**：

- **翻转技巧是多项式除法的关键，将余数"移到高次项"后取模消去**
- **商的次数为 $n - m$，余数次数 $< m$，与整数除法完全类比**
- **多项式除法是多点求值等高级操作的前置**

### 题目 15：多点求值（洛谷 P5050）🟣

> 给定 $n$ 次多项式 $f(x)$ 和 $m$ 个求值点 $a_1, \ldots, a_m$，对每个 $i$ 求 $f(a_i) \bmod 998244353$。

**分析**：暴力 $O(nm)$。利用分治求余：$f(a_i) = f \bmod (x - a_i)$ 的常数项。构建线段树式的分治结构，每个节点维护 $M_{[l,r]}(x) = \prod_{i=l}^{r}(x - a_i)$。从根到叶逐层对 $M$ 取模，最终叶节点的常数项即为答案。总时间 $O(n\log^2 n)$。

```
核心观察：f(a) = f mod (x-a) 的常数项；分治构造模多项式树
构造策略：(1) 建树：M[l,r] = M[l,mid] * M[mid+1,r]；(2) 从根向下：f mod M[l,r] → 分别 mod M[l,mid] 和 M[mid+1,r]
正确性：多项式取模的传递性：(f mod M[l,r]) mod M[l,mid] = f mod M[l,mid]
```

```cpp
// 多点求值核心——分治求余（已有 poly_mod、poly_mul）
// tree[node] 存储 M_{[l,r]} 的系数，自底向上建树
void build(int node, int l, int r, long long a[]) {
    if (l == r) { tree[node] = {MOD - a[l], 1}; return; }
    int mid = (l + r) / 2;
    build(node*2, l, mid, a);
    build(node*2+1, mid+1, r, a);
    tree[node] = poly_mul(tree[node*2], tree[node*2+1]);
}
void evaluate(int node, int l, int r, vector<int>& f, long long ans[]) {
    if ((int)f.size() <= 20 || l == r) {
        /* 暴力求值 */ return;
    }
    int mid = (l + r) / 2;
    auto fl = poly_mod(f, tree[node*2]);
    auto fr = poly_mod(f, tree[node*2+1]);
    evaluate(node*2, l, mid, fl, ans);
    evaluate(node*2+1, mid+1, r, fr, ans);
}
```

**核心启示**：

- **分治求余的核心是"$f \bmod (x-a)$ 就是 $f(a)$"这一性质**
- **$O(n\log^2 n)$ 的常数较大，实际竞赛中阈值切暴力很重要**
- **建树自底向上，求值自顶向下，方向相反**

### 题目 18：斗主地（洛谷 P5472）🔥

> NOI2019 题目。给定一棵 $n$ 个节点的树，每条边权值为 1 或 2。定义一条 $k$ 边路径的权值为边权之和。对所有满足条件的路径求权值的 $m$ 次幂之和。$n \leq 10^5$。

```
核心观察：幂 → 下降幂展开 w^m = ∑ S(m,j) * C(w,j) * j!，C(w,j) 可树上 DP
构造策略：(1) 枚举下降幂阶数 j；(2) 树形 DP 统计选 j 条特殊边的路径数；(3) 乘上 Stirling 数求和

```

```cpp
// 核心 DP：f[u][j] 表示 u 子树中，当前路径选了 j 条权值 2 的边的方案数
// 合并子树时卷积 f[u] 和 f[v]
void dfs(int u, int fa) {
    f[u][0] = 1;
    for (int v : adj[u]) {
        if (v == fa) continue;
        dfs(v, u);
        int w = edge_weight(u, v);
        // 合并：new_f[u][j] += f[u][j1] * f[v][j2] (j1+j2+[w==2] = j)
        // 这是卷积形式，可 NTT 加速
        merge(f[u], f[v], w);
    }
    // 统计以 u 为端点/经过 u 的路径
}
```

**核心启示**：

- **$w^m$ 用 Stirling 展开为下降幂是处理"幂次求和"的经典技巧**
- **树上路径统计 + NTT 卷积合并子树是竞赛中的高频模式**
- **NOI 综合题考察的是多种工具的融合能力**

## 五、常见技巧与思路总结

### 5.1 NTT 模数选择

常用模数 $998244353 = 119 \cdot 2^{23} + 1$，原根 $g = 3$，最大支持长度 $2^{23} \approx 8 \times 10^6$。

其他常用 NTT 模数：$985661441 = 235 \cdot 2^{22} + 1$（原根 3），$469762049 = 7 \cdot 2^{26} + 1$（原根 3）。

### 5.2 Newton 迭代统一框架

| 操作 | $h(g) = 0$ | 迭代公式 |
|------|-----------|----------|
| 求逆 | $1/g - f$ | $g \leftarrow g(2 - fg)$ |
| 开根 | $g^2 - f$ | $g \leftarrow (g + f/g)/2$ |
| $\exp$ | $\ln g - f$ | $g \leftarrow g(1 - \ln g + f)$ |
| 对数 | 不需要 Newton | $\ln f = \int f'/f$ |

### 5.3 拆系数 FFT

当模数不是 NTT 友好素数，且不想使用三模数 NTT 时，可将系数拆为高低 $B = \lfloor\sqrt{p}\rfloor$ 位：$a_i = a_i^{hi} \cdot B + a_i^{lo}$，4 次 FFT 完成乘法，避免中间值溢出。

### 5.4 分治 FFT vs 多项式 exp

对递推 $f_n = \sum_{j=1}^{n} f_{n-j} g_j$，有两种方法：

- 分治 FFT：$O(n\log^2 n)$，实现简单
- 多项式 $\exp$：当 $f = \exp(\int g)$ 时可直接 $O(n\log n)$

### 5.5 卷积识别清单

| 形式 | 类型 | 方法 |
|------|------|------|
| $c_k = \sum_{i+j=k} a_i b_j$ | 加法卷积 | FFT/NTT |
| $c_s = \sum_{i \mathbin{|} j = s} a_i b_j$ | OR 卷积 | FWT |
| $c_s = \sum_{i \mathbin{\&} j = s} a_i b_j$ | AND 卷积 | FWT |
| $c_s = \sum_{i \oplus j = s} a_i b_j$ | XOR 卷积 | FWT |
| $c_k = \sum_{ij=k} a_i b_j$ | Dirichlet 卷积 | $O(n\log n)$ 枚举 |

## 七、终极总结与归纳

### 7.1 本质

多项式算法的核心是 **DFT 将卷积从时域转到频域**，使得 $O(n^2)$ 的卷积变为 $O(n\log n)$。在此基础上，**Newton 迭代** 以每步精度翻倍的方式构建求逆、开根、$\exp$ 等操作，形成完整的多项式工具链。

### 7.2 核心要点

1. **FFT/NTT** 是一切多项式算法的基石，$O(n\log n)$ 乘法
2. **Newton 迭代** 统一框架：求逆 → 开根 → ln → exp → 快速幂
3. **分治 FFT** 处理"递推 = 自卷积"型问题，$O(n\log^2 n)$
4. **FWT** 处理按位卷积（OR/AND/XOR），$O(n\log n)$
5. **拉格朗日插值** 连接点值和系数表示，$O(n^2)$ 或 $O(n)$（连续点）
6. **分治求余** 实现多点求值/快速插值，$O(n\log^2 n)$
7. **幂次展开为下降幂**（Stirling 数）是组合计数 + 多项式的桥梁
