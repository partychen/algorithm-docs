## 一、什么是高次同余方程

高次同余方程是指形如 $a^x \equiv b \pmod{m}$ 的方程，其中已知 $a, b, m$，求最小非负整数 $x$。这类问题本质上是**离散对数问题**。

在密码学中，离散对数的困难性是许多加密算法的基础；而在算法竞赛中，我们利用 BSGS（Baby-step Giant-step）算法在 $O(\sqrt{m})$ 时间内求解。

| 问题类型 | 方程形式 | 求解算法 | 复杂度 |
|---------|---------|---------|--------|
| 离散对数 | $a^x \equiv b \pmod{m}$ | BSGS / exBSGS | $O(\sqrt{m})$ |
| 二次剩余 | $x^2 \equiv a \pmod{p}$ | Cipolla 算法 | $O(\log p)$ |
| 高次剩余 | $x^k \equiv a \pmod{p}$ | 原根 + BSGS | $O(\sqrt{p})$ |
| 线性同余 | $ax \equiv b \pmod{m}$ | exgcd | $O(\log m)$ |

## 二、核心思想与前置知识

### 2.1 前置知识

- **快速幂**：$O(\log n)$ 计算 $a^n \bmod m$
- **模运算性质**：$(a \cdot b) \bmod m = ((a \bmod m) \cdot (b \bmod m)) \bmod m$
- **欧拉定理**：若 $\gcd(a, m) = 1$，则 $a^{\varphi(m)} \equiv 1 \pmod{m}$
- **原根**：若 $g$ 是模 $p$ 的原根，则 $g^0, g^1, \ldots, g^{p-2}$ 恰好是 $1, 2, \ldots, p-1$ 的一个排列

### 2.2 BSGS 核心思想

令 $t = \lceil \sqrt{m} \rceil$，将 $x$ 表示为 $x = it - j$（其中 $0 \le j < t$，$0 \le i \le t$）。

则 $a^{it-j} \equiv b \pmod{m}$，即 $a^{it} \equiv b \cdot a^{j} \pmod{m}$。

**Baby-step**：预计算所有 $b \cdot a^j \bmod m$（$0 \le j < t$），存入哈希表。

**Giant-step**：依次计算 $a^{it} \bmod m$（$1 \le i \le t$），在哈希表中查找匹配。

```
// BSGS 骨架（要求 gcd(a, m) = 1）
long long bsgs(long long a, long long b, long long m) {
    unordered_map<long long, int> ht;
    long long t = (long long)ceil(sqrt((double)m));
    long long cur = b % m;
    for (int j = 0; j < t; j++) {
        ht[cur] = j;
        cur = cur * a % m;
    }
    long long at = qpow(a, t, m); // a^t mod m
    cur = at;
    for (int i = 1; i <= t; i++) {
        if (ht.count(cur)) return (long long)i * t - ht[cur];
        cur = cur * at % m;
    }
    return -1; // 无解
}
```

### 2.3 扩展 BSGS（exBSGS）

当 $\gcd(a, m) \neq 1$ 时，基本 BSGS 不再适用。exBSGS 的思路是不断提取公因子：

每次令 $d = \gcd(a, m)$，若 $d \nmid b$ 则无解，否则将方程化为 $\frac{a}{d} \cdot a^{x-1} \equiv \frac{b}{d} \pmod{\frac{m}{d}}$。

重复此过程直到 $\gcd(a, m') = 1$，然后对化简后的方程使用基本 BSGS。

### 2.4 Cipolla 算法（二次剩余）

求解 $x^2 \equiv n \pmod{p}$（$p$ 为奇素数）：

1. 判断 $n$ 是否为二次剩余：$n^{\frac{p-1}{2}} \equiv 1 \pmod{p}$
2. 随机选取 $a$ 使得 $a^2 - n$ 为二次非剩余
3. 在扩域 $\mathbb{F}_{p^2}$ 中计算 $(a + \sqrt{a^2-n})^{\frac{p+1}{2}}$，其实部即为答案

## 三、分类与变体详解

### 3.1 基本 BSGS

```
适用条件：gcd(a, m) = 1
时间复杂度：O(√m)
空间复杂度：O(√m)
核心操作：哈希表 + 分块枚举
```

### 3.2 扩展 BSGS（exBSGS）

```
适用条件：a, m 任意正整数
预处理：不断除 gcd(a, m) 直到互素
额外记录：提取因子个数 cnt 和累积系数
最终答案：BSGS 结果 + cnt
```

### 3.3 二次剩余（Cipolla）

```
适用条件：p 为奇素数
判定条件：n^((p-1)/2) ≡ 1 (mod p)
核心技巧：扩域 F_{p^2} 上的快速幂
解的个数：0 或 2（x 和 p-x）
```

### 3.4 高次剩余

```
适用条件：x^k ≡ a (mod p)，p 为素数
核心转化：令 g 为原根，x = g^y，a = g^c
化归为：ky ≡ c (mod p-1)（线性同余方程）
求原根：枚举 + 验证 g^((p-1)/q_i) ≢ 1
```

### 3.5 变体对比总表

| 变体 | 方程形式 | 前提条件 | 核心算法 |
|------|---------|---------|---------|
| BSGS | $a^x \equiv b$ | $\gcd(a,m)=1$ | 分块枚举 |
| exBSGS | $a^x \equiv b$ | 任意 | 提因子 + BSGS |
| Cipolla | $x^2 \equiv n$ | $p$ 奇素数 | 扩域快速幂 |
| 高次剩余 | $x^k \equiv a$ | $p$ 素数 | 原根 + BSGS |

## 四、15 道经典例题详解

### 🟢 入门级 Level 1：基础离散对数

### 题目 1：BSGS 模板（洛谷 P3846）

> 给定 $a, b, p$（$p$ 为素数），求最小非负整数 $x$ 使得 $a^x \equiv b \pmod{p}$。

**分析**：BSGS 模板题，$p$ 为素数保证 $\gcd(a, p) = 1$（除非 $a \equiv 0$）。

```
核心观察：p 为素数 → gcd(a,p) = 1（a ≢ 0 时）→ 直接 BSGS
构造策略：令 t = ⌈√p⌉，x = it - j，枚举匹配
正确性：x ∈ [0, p-1]，而 i*t - j 覆盖了 [0, t²] ⊇ [0, p-1]
```

**C++ 代码**：

```cpp
long long bsgs(long long a, long long b, long long p) {
    if (b == 1) return 0;
    unordered_map<long long, int> ht;
    long long t = (long long)ceil(sqrt((double)p));
    long long cur = b % p;
    for (int j = 0; j < t; j++) {
        ht[cur] = j;
        cur = cur * a % p;
    }
    long long at = qpow(a, t, p);
    cur = at;
    for (int i = 1; i <= t; i++) {
        if (ht.count(cur)) return (long long)i * t - ht[cur];
        cur = cur * at % p;
    }
    return -1;
}
```

**核心启示**：
- **分块思想**：将 $[0, m)$ 分为 $\sqrt{m}$ 块，空间换时间
- **哈希表查找**是 BSGS 的效率关键

### 题目 2：快速幂求逆元验证（AcWing 876）

> 给定 $a, p$（$p$ 为素数），利用费马小定理求 $a$ 在模 $p$ 下的逆元。

**分析**：费马小定理给出 $a^{p-2} \equiv a^{-1} \pmod{p}$，是 BSGS 的前置技能。

```
核心观察：p 为素数 → a^(p-1) ≡ 1 (mod p) → a^(p-2) 即为逆元
构造策略：快速幂计算 a^(p-2) mod p
正确性：费马小定理保证正确性，前提 gcd(a,p) = 1
```

**C++ 代码**：

```cpp
long long qpow(long long a, long long b, long long mod) {
    long long res = 1;
    a %= mod;
    while (b > 0) {
        if (b & 1) res = res * a % mod;
        a = a * a % mod;
        b >>= 1;
    }
    return res;
}
// 求逆元：qpow(a, p - 2, p)
```

**核心启示**：
- **快速幂**是几乎所有高次同余问题的基石
- 费马小定理是 $p$ 为素数时的特化版本

### 题目 3：原根判定（洛谷 P6091）

> 给定 $n$，求模 $n$ 的最小原根。

**分析**：原根是高次剩余的核心前置。若 $g$ 为原根，$g$ 的阶为 $\varphi(n)$，只需验证对 $\varphi(n)$ 的每个素因子 $q$，$g^{\varphi(n)/q} \not\equiv 1$。

```
核心观察：只需检查 φ(n) 的素因子处的幂
构造策略：从小到大枚举 g，逐一验证
正确性：若所有 g^(φ(n)/q_i) ≢ 1，则 g 的阶为 φ(n)
```

**C++ 代码**：

```cpp
// primes[] 存 φ(n) 的所有素因子
int find_primitive_root(int n, int phi_n) {
    for (int g = 2; g < n; g++) {
        if (__gcd(g, n) != 1) continue;
        bool ok = true;
        for (int q : primes)
            if (qpow(g, phi_n / q, n) == 1) { ok = false; break; }
        if (ok) return g;
    }
    return -1;
}
```

**核心启示**：
- **原根存在条件**：$n = 1, 2, 4, p^k, 2p^k$（$p$ 为奇素数）
- 最小原根通常很小，暴力枚举即可

### 🟡 进阶级 Level 2：exBSGS 与基本应用

### 题目 4：exBSGS 模板（洛谷 P4195）

> 给定 $a, b, m$（不保证 $\gcd(a, m) = 1$），求最小非负整数 $x$ 使得 $a^x \equiv b \pmod{m}$。

**分析**：经典 exBSGS。反复提取 $\gcd(a, m)$ 直到互素，记录提取次数 $cnt$ 和累积系数。

```
核心观察：gcd(a, m) ≠ 1 时直接 BSGS 会漏解
构造策略：令 d = gcd(a, m)，若 d ∤ b 无解；否则 b/=d, m/=d，乘上 a/d 的逆
正确性：等价变换保持方程成立，最终 gcd(a, m') = 1
```

**C++ 代码**：

```cpp
long long exbsgs(long long a, long long b, long long m) {
    if (b == 1 || m == 1) return 0;
    long long cnt = 0, coef = 1;
    while (true) {
        long long d = __gcd(a, m);
        if (d == 1) break;
        if (b % d) return -1;
        b /= d; m /= d;
        coef = coef * (a / d) % m;
        cnt++;
        if (coef == b) return cnt;
    }
    // 此时 gcd(a, m) = 1，用 BSGS 求解
    long long res = bsgs_with_coef(a, b, m, coef);
    return res == -1 ? -1 : res + cnt;
}
```

**核心启示**：
- **提因子**是处理不互素情况的通用手段
- 提因子后别忘了检查 `coef == b` 的提前退出

### 题目 5：计算器三合一（洛谷 P2485）

> 三种操作：(1) 求 $a^b \bmod p$；(2) 求 $a x \equiv b \pmod{p}$ 的最小正整数解；(3) 求 $a^x \equiv b \pmod{p}$ 的最小非负整数解。

**分析**：综合快速幂、exgcd、BSGS 三大工具。

```
核心观察：三种操作分别对应三种不同算法
构造策略：操作 1 快速幂，操作 2 exgcd，操作 3 BSGS
正确性：p 为素数保证 BSGS 可用
```

**C++ 代码**：

```cpp
void solve(int op, long long a, long long b, long long p) {
    if (op == 1) printf("%lld\n", qpow(a, b, p));
    else if (op == 2) {
        // ax ≡ b (mod p) → x ≡ b * a^{-1} (mod p)
        long long inv = qpow(a % p, p - 2, p);
        printf("%lld\n", b % p * inv % p);
    } else {
        long long res = bsgs(a, b, p);
        if (res == -1) puts("no solution");
        else printf("%lld\n", res);
    }
}
```

**核心启示**：
- **三合一**题型考察对基础算法的综合运用
- 注意各操作的边界情况（如 $a = 0$）

### 题目 6：随机数生成器（洛谷 P3306）

> 随机数生成器 $x_{n+1} = ax_n + b \pmod{p}$，给定 $x_1$ 和目标值 $t$，求最小的 $n$ 使得 $x_n = t$。

**分析**：递推式展开后为 $x_n = a^{n-1}x_1 + b \cdot \frac{a^{n-1}-1}{a-1} \pmod{p}$，化简后变为 BSGS 问题。

```
核心观察：令 y = x + b/(a-1)，则 y_n = a^{n-1} * y_1
构造策略：变量替换消除常数项，化为 a^k ≡ c (mod p) 的标准形式
正确性：a ≠ 1 时替换合法；a = 1 时线性递推特判
```

**C++ 代码**：

```cpp
long long solve(long long a, long long b, long long x1, long long t, long long p) {
    if (x1 == t) return 1;
    if (a == 0) return b == t ? 2 : -1;
    if (a == 1) {
        if (b == 0) return -1;
        // x_n = x1 + (n-1)*b, 求 (n-1)*b ≡ t-x1
        return (t - x1 + p) % p * qpow(b, p - 2, p) % p + 1;
    }
    // y = x + b*inv(a-1), y_n = a^{n-1} * y1
    long long inv = qpow(a - 1, p - 2, p);
    long long y1 = (x1 + b % p * inv) % p;
    long long yt = (t + b % p * inv) % p;
    if (y1 == 0) return yt == 0 ? 1 : -1;
    long long c = yt * qpow(y1, p - 2, p) % p;
    long long res = bsgs(a, c, p);
    return res == -1 ? -1 : res + 1;
}
```

**核心启示**：
- **线性递推化简**：通过变量替换消除常数项是常见技巧
- 边界情况（$a = 0, 1$）需单独处理

### 🟠 提高级 Level 3：二次剩余与综合应用

### 题目 7：二次剩余模板（洛谷 P5491）

> 给定 $n$ 和奇素数 $p$，求 $x^2 \equiv n \pmod{p}$ 的解。

**分析**：Cipolla 算法模板题。

```
核心观察：先用欧拉准则判定是否有解：n^((p-1)/2) ≡ 1
构造策略：随机 a 使 a²-n 为非二次剩余，在 F_{p²} 上计算 (a+ω)^{(p+1)/2}
正确性：F_{p²} 上的快速幂，ω² = a²-n，结果实部即为解
```

**C++ 代码**：

```cpp
long long w2; // ω² = a² - n，全局变量

struct Complex { long long r, i; };
Complex mul(Complex a, Complex b, long long p) {
    return {(a.r * b.r + a.i * b.i % p * w2) % p,
            (a.r * b.i + a.i * b.r) % p};
}
long long cipolla(long long n, long long p) {
    if (n == 0) return 0;
    if (qpow(n, (p - 1) / 2, p) != 1) return -1; // 非二次剩余
    long long a;
    do { a = rand() % p; w2 = (a * a - n % p + p) % p; }
    while (qpow(w2, (p - 1) / 2, p) != p - 1);
    Complex res = {1, 0}, base = {a, 1};
    long long exp = (p + 1) / 2;
    while (exp) {
        if (exp & 1) res = mul(res, base, p);
        base = mul(base, base, p);
        exp >>= 1;
    }
    return res.r;
}
```

**核心启示**：
- **扩域**思想：在 $\mathbb{F}_{p^2}$ 上运算后取实部
- 随机化算法，期望 $O(\log p)$ 次即可找到合适的 $a$

### 题目 8：高次剩余（CF 1106F）

> 给定素数 $p$、正整数 $k$ 和 $a$，求 $x^k \equiv a \pmod{p}$ 的所有解。

**分析**：先求原根 $g$，令 $x = g^y$, $a = g^c$（用 BSGS 求 $c$），化为 $ky \equiv c \pmod{p-1}$。

```
核心观察：原根将乘法群映射到加法群，高次方程化为线性同余
构造策略：BSGS 求离散对数 c，然后 exgcd 解 ky ≡ c (mod p-1)
正确性：原根保证映射是双射
```

**C++ 代码**：

```cpp
vector<long long> solve_kth(long long k, long long a, long long p) {
    int g = find_primitive_root(p, p - 1);
    long long c = bsgs(g, a, p); // a = g^c
    // 解 ky ≡ c (mod p-1)
    long long d = __gcd(k, p - 1);
    if (c % d) return {}; // 无解
    long long y0 = c / d * qpow(k / d, euler_phi(( p - 1) / d) - 1, (p - 1) / d) % ((p - 1) / d);
    vector<long long> res;
    for (int i = 0; i < d; i++)
        res.push_back(qpow(g, y0 + i * (p - 1) / d, p));
    sort(res.begin(), res.end());
    return res;
}
```

**核心启示**：
- **原根是桥梁**：将乘法问题转化为加法问题
- 解的个数为 $\gcd(k, p-1)$（当有解时）

### 题目 9：矩阵快速幂 + BSGS（CF 954H）

> 给定矩阵 $A$ 和 $B$（模素数 $p$），求最小的 $x$ 使得 $A^x = B$。

**分析**：将 BSGS 推广到矩阵上，用矩阵哈希代替整数哈希。

```
核心观察：BSGS 的 meet-in-the-middle 思想可推广到任何群
构造策略：Baby-step 存 B * A^j 的哈希，Giant-step 匹配 A^{it}
正确性：群的封闭性保证分块策略有效
```

**C++ 代码**：

```cpp
// 矩阵 BSGS：将矩阵编码为哈希值
long long mat_bsgs(Matrix A, Matrix B, long long p, int n) {
    long long t = (long long)ceil(sqrt((double)p));
    map<vector<vector<long long>>, int> ht;
    Matrix cur = B;
    for (int j = 0; j < t; j++) {
        ht[cur.data] = j;
        cur = cur * A; // 矩阵乘法 mod p
    }
    Matrix At = mat_pow(A, t, p); // A^t
    cur = At;
    for (int i = 1; i <= t; i++) {
        if (ht.count(cur.data)) return (long long)i * t - ht[cur.data];
        cur = cur * At;
    }
    return -1;
}
```

**核心启示**：
- **BSGS 可推广到任意群**（矩阵群、置换群等）
- 矩阵哈希是关键实现细节

### 🔴 困难级 Level 4：复杂综合题

### 题目 10：模方程综合（洛谷 P4195 加强版）

> 多组询问，每组给定 $a, b, m$，求 $a^x \equiv b \pmod{m}$ 的最小非负整数解。$m$ 可达 $10^{9}$。

**分析**：exBSGS 模板的多组询问版本，需注意清空哈希表的效率。

```
核心观察：多组询问需要高效的 exBSGS 实现
构造策略：每次清空 unordered_map 或改用 map
正确性：exBSGS 对任意正整数 m 均适用
```

**C++ 代码**：

```cpp
long long exbsgs(long long a, long long b, long long m) {
    a %= m; b %= m;
    if (b == 1 || m == 1) return 0;
    long long cnt = 0, coef = 1;
    while (true) {
        long long d = __gcd(a, m);
        if (d == 1) break;
        if (b % d) return -1;
        b /= d; m /= d;
        coef = coef * (a / d) % m;
        cnt++;
        if (coef == b) return cnt;
    }
    // BSGS on a^x ≡ b/coef (mod m)
    long long inv_coef = qpow(coef, euler_phi(m) - 1, m);
    long long target = b * inv_coef % m;
    long long res = bsgs(a, target, m);
    return res == -1 ? -1 : res + cnt;
}
```

**核心启示**：
- **多组询问**时 `unordered_map` 的 `clear()` 开销不可忽视
- 可用手写哈希表优化常数

### 题目 11：离散对数打表（CF 1106F 扩展）

> 给定素数 $p$ 和原根 $g$，对 $1 \le i \le p-1$ 求出所有 $\log_g i \bmod (p-1)$。

**分析**：可以用 BSGS 逐一求解，也可以利用原根的性质直接幂次枚举。

```
核心观察：g^0, g^1, ..., g^{p-2} 恰好遍历 1~p-1
构造策略：直接计算 g^k mod p，记录 ind[g^k] = k
正确性：原根保证双射
```

**C++ 代码**：

```cpp
const int MAXP = 1000006;
int ind[MAXP]; // ind[v] = log_g(v)

void build_ind_table(int g, int p) {
    long long cur = 1;
    for (int k = 0; k < p - 1; k++) {
        ind[cur] = k;
        cur = cur * g % p;
    }
}
```

**核心启示**：
- **离散对数表**是批量求解的高效手段
- 适用于 $p$ 较小（$\le 10^6$）的场景

### 题目 12：模素数下的 N 次根（SPOJ CMPLS）

> 给定 $a, k, p$（$p$ 为素数），求 $x^k \equiv a \pmod{p}$ 的所有解并升序输出。

**分析**：高次剩余标准流程：求原根 → BSGS 求离散对数 → exgcd 解线性同余。

```
核心观察：解的个数为 gcd(k, p-1)（若有解）
构造策略：原根 + BSGS + exgcd 三步走
正确性：原根将乘法群同构到 Z_{p-1}
```

**C++ 代码**：

```cpp
vector<long long> nth_root(long long a, long long k, long long p) {
    if (a == 0) return {0};
    int g = find_primitive_root(p, p - 1);
    long long c = bsgs(g, a, p);
    long long d = __gcd(k, p - 1);
    if (c % d) return {};
    long long base = c / d * qpow(k / d, euler_phi((p-1)/d) - 1, (p-1)/d) % ((p-1)/d);
    long long step = (p - 1) / d;
    vector<long long> res;
    for (int i = 0; i < d; i++)
        res.push_back(qpow(g, (base + (long long)i * step) % (p-1), p));
    sort(res.begin(), res.end());
    return res;
}
```

**核心启示**：
- 高次剩余问题的**三步标准流程**要熟练掌握
- 注意解可能有多个，需枚举所有同余类

### 题目 13：屠龙勇士（洛谷 P4774）

> 给定 $n$ 条龙，每条龙有生命值 $a_i$、恢复值 $p_i$，勇士有一把攻击力为 $atk_i$ 的剑。求最小的 $x$ 使得对每条龙，$atk_i \cdot x \equiv a_i \pmod{p_i}$。

**分析**：每条龙给出一个线性同余方程，最终用 exCRT 合并所有方程。

```
核心观察：每个方程 atk_i * x ≡ a_i (mod p_i) 可化为 x ≡ c_i (mod m_i)
构造策略：exgcd 求单个方程的解，exCRT 合并
正确性：exCRT 保证合并正确性，无解时方程组不相容
```

**C++ 代码**：

```cpp
// 对每条龙：atk * x ≡ a (mod p)
// 化为 x ≡ a * inv(atk/d) * (1/d) (mod p/d)，其中 d = gcd(atk, p)
// 然后 exCRT 合并所有 x ≡ r_i (mod m_i)
long long solve_dragon(long long atk[], long long a[], long long p[], int n) {
    long long R = 0, M = 1; // 当前合并结果：x ≡ R (mod M)
    for (int i = 0; i < n; i++) {
        long long d = __gcd(atk[i], p[i]);
        if (a[i] % d) return -1;
        long long mi = p[i] / d;
        long long ri = a[i] / d * qpow(atk[i] / d, euler_phi(mi) - 1, mi) % mi;
        // 合并 x ≡ R (mod M) 和 x ≡ ri (mod mi)
        // ... exCRT 合并逻辑
    }
    return R;
}
```

**核心启示**：
- **NOI 级别**的综合题，融合 exgcd + exCRT + multiset
- 多方程合并时注意溢出，需用 __int128

### 题目 14：幽灵乐团的离散对数（洛谷 P5518 简化版）

> 给定 $a, b, p$，求 $\sum_{x=0}^{p-2} [a^x \equiv b \pmod{p}]$（本质是判定是否有解并给出解的个数概念）。

**分析**：离散对数若有解则恰好一个（在 $[0, p-2]$ 范围内），因此答案为 0 或 1。

```
核心观察：模素数下离散对数解唯一（若存在）
构造策略：BSGS 求解，有解返回 1，无解返回 0
正确性：a 为原根时 a^x 在 [0,p-2] 上是双射
```

**C++ 代码**：

```cpp
int count_solutions(long long a, long long b, long long p) {
    if (b == 0) return 0; // 0 不在乘法群中
    long long res = bsgs(a % p, b % p, p);
    return res == -1 ? 0 : 1;
}
```

**核心启示**：
- 模素数下**离散对数解的唯一性**是重要性质
- 注意 $b = 0$ 时不在乘法群中

### 题目 15：Pohlig-Hellman 算法（CSES 离散对数进阶）

> $p$ 为素数但 $p-1$ 有小素因子分解，利用 Pohlig-Hellman 加速离散对数。

**分析**：将阶 $p-1$ 分解为 $p-1 = q_1^{e_1} \cdots q_k^{e_k}$，分别在每个子群中用 BSGS 求解，最后 CRT 合并。

```
核心观察：群的分解 → 子问题独立求解 → CRT 合并
构造策略：对每个 q^e，在阶为 q^e 的子群中逐位确定离散对数
正确性：中国剩余定理保证合并正确
```

**C++ 代码**：

```cpp
// Pohlig-Hellman: 对 p-1 = Π q_i^{e_i}
long long pohlig_hellman(long long a, long long b, long long p) {
    long long n = p - 1;
    vector<pair<long long,int>> fac = factorize(n);
    vector<long long> rems, mods;
    for (auto [q, e] : fac) {
        long long qe = 1;
        for (int i = 0; i < e; i++) qe *= q;
        long long gamma = qpow(a, n / q, p);
        long long x_qe = 0, q_pow = 1;
        for (int k = 0; k < e; k++) {
            long long h = qpow(qpow(a, n / (q_pow * q), p), p - 1 - x_qe, p);
            h = h * qpow(b, n / (q_pow * q), p) % p;
            long long d_k = bsgs(gamma, h, p);
            x_qe += d_k * q_pow;
            q_pow *= q;
        }
        rems.push_back(x_qe % qe);
        mods.push_back(qe);
    }
    return crt(rems, mods); // CRT 合并
}
```

**核心启示**：
- **Pohlig-Hellman** 将 $O(\sqrt{p})$ 优化为 $O(\sum e_i \sqrt{q_i})$
- 当 $p-1$ 光滑（素因子都小）时效果极佳

## 五、常见技巧与思路总结

### 5.1 哈希表选型

BSGS 中哈希表是性能瓶颈。`unordered_map` 常数大，可改用手写开地址哈希表：

```cpp
const int MOD = 1 << 20;
int key[MOD + 5], val[MOD + 5];
bool used[MOD + 5];
void insert(int k, int v) {
    int h = k & (MOD - 1);
    while (used[h] && key[h] != k) h = (h + 1) & (MOD - 1);
    key[h] = k; val[h] = v; used[h] = true;
}
```

### 5.2 特殊情况处理

- $a = 0$：$0^x \equiv b$ 仅当 $x > 0$ 且 $b = 0$ 时有解
- $a = 1$：$1^x \equiv b$ 仅当 $b = 1$ 时有解（$x = 0$）
- $b = 0$：求最小的 $x$ 使得 $a^x \equiv 0$，即 $m \mid a^x$

### 5.3 __int128 防溢出

当 $m$ 接近 $10^{18}$ 时，乘法 $a \cdot b$ 会溢出 `long long`，使用 `__int128` 或龟速乘：

```cpp
long long mul(long long a, long long b, long long m) {
    return (__int128)a * b % m;
}
```

### 5.4 二次剩余的解

$x^2 \equiv n \pmod{p}$ 若有解 $x_0$，则 $p - x_0$ 也是解。输出时通常取较小的那个。

### 5.5 离散对数的周期性

$a^x \bmod m$ 具有周期性（最终进入循环），周期为 $\text{ord}_m(a)$（$a$ 在模 $m$ 下的阶）。所有解为 $x_0 + k \cdot \text{ord}$。

## 六、优化方法 / 解题思路框架

```
输入：a^x ≡ b (mod m)
  │
  ├── gcd(a, m) = 1？
  │     ├── 是 → 直接 BSGS → O(√m)
  │     └── 否 → exBSGS（提因子）→ O(√m + log m)
  │
输入：x^k ≡ a (mod p)，p 素数
  │
  ├── k = 2 → Cipolla 算法 → O(log p)
  └── k > 2 → 原根 + BSGS + exgcd → O(√p)
  │
输入：离散对数，p-1 高度可分
  │
  └── Pohlig-Hellman → O(Σ e_i √q_i)
```

| 信号词 | 对应方法 |
|--------|---------|
| $a^x \equiv b$ | BSGS / exBSGS |
| $x^2 \equiv n$ | Cipolla |
| $x^k \equiv a$, $p$ 素数 | 原根 + BSGS |
| 多组 $a^x \equiv b$ | exBSGS + 手写哈希 |
| $p-1$ 光滑 | Pohlig-Hellman |

## 七、终极总结与归纳

### 7.1 高次同余方程的本质（一句话）

高次同余方程的核心是**离散对数**——利用分块枚举（BSGS）或代数结构（原根、扩域）将指数方程化归为可解的形式。

### 7.2 核心要点

1. **BSGS 是分块搜索**：将 $[0, m)$ 分为 $\sqrt{m}$ 块，用哈希表实现 $O(\sqrt{m})$
2. **exBSGS 提因子**：处理 $\gcd(a,m) \neq 1$ 的情况，反复除公因子直到互素
3. **Cipolla 算法**：扩域 $\mathbb{F}_{p^2}$ 上快速幂求二次剩余
4. **原根是万能桥梁**：将乘法群同构到加法群，高次方程化为线性方程
5. **Pohlig-Hellman**：利用阶的素因子分解加速离散对数
6. **边界处理至关重要**：$a = 0, 1$、$b = 0$、$\gcd$ 不互素等特殊情况
