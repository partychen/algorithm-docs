## 一、什么是欧拉函数

欧拉函数 $\varphi(n)$ 表示 $1$ 到 $n$ 中与 $n$ 互素的正整数个数。例如 $\varphi(12) = 4$（1, 5, 7, 11 与 12 互素）。

欧拉函数是数论中最重要的**积性函数**之一，在模运算、密码学（RSA）、竞赛数论中都有核心地位。

| 相关概念 | 定义 | 与 φ 的关系 |
|---------|------|------------|
| 互素 | $\gcd(a, n) = 1$ | $\varphi(n)$ 统计互素个数 |
| 欧拉定理 | $a^{\varphi(n)} \equiv 1$ | 幂运算的周期 |
| 费马小定理 | $a^{p-1} \equiv 1$ | $\varphi(p) = p-1$ 的特化 |
| 降幂公式 | $a^b \equiv a^{b \bmod \varphi(n) + \varphi(n)}$ | 处理超大指数 |

## 二、核心思想与前置知识

### 2.1 欧拉函数公式

若 $n = p_1^{e_1} \cdot p_2^{e_2} \cdots p_k^{e_k}$，则：

$$\varphi(n) = n \cdot \prod_{i=1}^{k} \left(1 - \frac{1}{p_i}\right)$$

### 2.2 单个数求 φ(n)

对 $n$ 做质因数分解，直接套公式：

```cpp
long long euler_phi(long long n) {
    long long res = n;
    for (long long i = 2; i * i <= n; i++) {
        if (n % i == 0) {
            res = res / i * (i - 1);
            while (n % i == 0) n /= i;
        }
    }
    if (n > 1) res = res / n * (n - 1);
    return res;
}
```

### 2.3 线性筛求 φ

利用欧拉筛，在 $O(n)$ 时间内求出 $1 \sim n$ 所有 $\varphi$ 值：

```cpp
const int MAXN = 1000006;
int phi[MAXN], primes[MAXN], cnt;
bool vis[MAXN];

void euler_sieve(int n) {
    phi[1] = 1;
    for (int i = 2; i <= n; i++) {
        if (!vis[i]) { primes[cnt++] = i; phi[i] = i - 1; }
        for (int j = 0; j < cnt && (long long)i * primes[j] <= n; j++) {
            vis[i * primes[j]] = true;
            if (i % primes[j] == 0) {
                phi[i * primes[j]] = phi[i] * primes[j];
                break;
            }
            phi[i * primes[j]] = phi[i] * (primes[j] - 1);
        }
    }
}
```

### 2.4 欧拉定理与扩展欧拉定理

**欧拉定理**：若 $\gcd(a, n) = 1$，则 $a^{\varphi(n)} \equiv 1 \pmod{n}$。

**扩展欧拉定理（降幂公式）**：对任意 $a, b, n$：

$$a^b \equiv \begin{cases} a^{b \bmod \varphi(n)} & \gcd(a,n) = 1 \ a^b & b < \varphi(n) \ a^{b \bmod \varphi(n) + \varphi(n)} & b \ge \varphi(n) \end{cases} \pmod{n}$$

### 2.5 重要性质

- $\sum_{d \mid n} \varphi(d) = n$（莫比乌斯反演的经典结论）
- $\varphi$ 是积性函数：若 $\gcd(a, b) = 1$，则 $\varphi(ab) = \varphi(a)\varphi(b)$
- $\varphi(p^k) = p^{k-1}(p-1)$

## 三、分类与变体详解

### 3.1 单值计算

```
适用场景：给定单个 n，求 φ(n)
算法：试除法质因数分解 + 公式
时间复杂度：O(√n)
```

### 3.2 批量计算（线性筛）

```
适用场景：求 1~n 所有 φ 值
算法：欧拉筛
时间复杂度：O(n)
空间复杂度：O(n)
```

### 3.3 欧拉函数求和

```
适用场景：∑φ(i) for i=1..n，n 可达 10^10
算法：杜教筛（利用 ∑_{d|n} φ(d) = n）
时间复杂度：O(n^{2/3})
```

### 3.4 降幂公式应用

```
适用场景：求 a^{a^{a^...}} mod m（无穷幂塔）
算法：递归 + 扩展欧拉定理
终止条件：φ 值最终变为 1
```

### 3.5 变体对比

| 变体 | 输入规模 | 算法 | 时间复杂度 |
|------|---------|------|-----------|
| 单值 φ(n) | $n \le 10^{18}$ | 试除法分解 | $O(\sqrt{n})$ |
| 批量 φ(1..n) | $n \le 10^7$ | 欧拉筛 | $O(n)$ |
| 前缀和 ∑φ | $n \le 10^{10}$ | 杜教筛 | $O(n^{2/3})$ |
| 幂塔取模 | 层数不限 | 递归降幂 | $O(\sqrt{m} \cdot \log)$ |

## 四、15 道经典例题详解

### 🟢 入门级 Level 1：基础计算

### 题目 1：欧拉函数模板（AcWing 873）

> 给定 $n$ 个正整数 $a_i$，对每个 $a_i$ 求 $\varphi(a_i)$。

**分析**：单值欧拉函数的直接应用。

```
核心观察：对每个数做质因数分解即可
构造策略：试除法 O(√a_i) 求 φ
正确性：公式 φ(n) = n·∏(1 - 1/p_i)
```

**C++ 代码**：

```cpp
long long euler_phi(long long n) {
    long long res = n;
    for (long long i = 2; i * i <= n; i++) {
        if (n % i == 0) {
            res = res / i * (i - 1);
            while (n % i == 0) n /= i;
        }
    }
    if (n > 1) res = res / n * (n - 1);
    return res;
}
```

**核心启示**：
- **先除后乘**避免溢出：`res / i * (i - 1)` 而非 `res * (i - 1) / i`
- 时间复杂度 $O(\sqrt{n})$ 足以应对 $n \le 10^{12}$

### 题目 2：欧拉筛模板（AcWing 874）

> 求 $1$ 到 $n$ 所有 $\varphi(i)$ 的和。

**分析**：线性筛批量求 $\varphi$，然后求和。

```
核心观察：欧拉筛利用最小素因子转移 φ 值
构造策略：i % p == 0 时 φ(ip) = φ(i)·p；否则 φ(ip) = φ(i)·(p-1)
正确性：积性函数在线性筛中的性质
```

**C++ 代码**：

```cpp
const int MAXN = 1000006;
int phi_val[MAXN], primes[MAXN], cnt;
bool vis[MAXN];

long long sieve_phi_sum(int n) {
    phi_val[1] = 1;
    for (int i = 2; i <= n; i++) {
        if (!vis[i]) { primes[cnt++] = i; phi_val[i] = i - 1; }
        for (int j = 0; j < cnt && (long long)i * primes[j] <= n; j++) {
            vis[i * primes[j]] = true;
            if (i % primes[j] == 0) { phi_val[i * primes[j]] = phi_val[i] * primes[j]; break; }
            phi_val[i * primes[j]] = phi_val[i] * (primes[j] - 1);
        }
    }
    long long sum = 0;
    for (int i = 1; i <= n; i++) sum += phi_val[i];
    return sum;
}
```

**核心启示**：
- 线性筛的**两种转移**对应 $p \mid i$ 和 $p \nmid i$ 两种情况
- $\varphi(ip) = \varphi(i) \cdot p$ 当 $p \mid i$（因为 $p$ 已在分解式中）

### 题目 3：仪仗队（洛谷 P2158）

> $n \times n$ 方阵中，从 $(0,0)$ 能看到多少个点？能看到 $(x, y)$ 当且仅当 $\gcd(x, y) = 1$。

**分析**：等价于求 $1 \le x, y \le n-1$ 中 $\gcd(x, y) = 1$ 的对数，加上边界。

```
核心观察：能看到 (x,y) ⟺ gcd(x,y) = 1
构造策略：答案 = 2·∑_{i=1}^{n-1} φ(i) + 1（加上 (1,0) 和 (0,1) 以及自身调整）
正确性：gcd(x,y) = 1 的对数等于 φ 求和
```

**C++ 代码**：

```cpp
// 预处理 φ 后
long long solve(int n) {
    if (n == 1) return 0;
    long long ans = 1; // (1, 1)
    for (int i = 1; i < n; i++) ans += phi_val[i];
    return 2 * ans + 1; // 对称 + (1,0) 和 (0,1) 的额外处理
}
```

**核心启示**：
- **可见点计数**是欧拉函数的经典几何应用
- 注意边界上的点（坐标轴和对角线）

### 🟡 进阶级 Level 2：欧拉定理应用

### 题目 4：GCD 问题（洛谷 P2568）

> 给定 $n$，求 $1 \le x, y \le n$ 中 $\gcd(x, y) = p$（$p$ 为素数）的对数。

**分析**：$\gcd(x, y) = p$ 等价于 $\gcd(x/p, y/p) = 1$，即求 $1 \le a, b \le \lfloor n/p \rfloor$ 中互素对数。

```
核心观察：gcd(x,y) = p 等价于除以 p 后互素
构造策略：枚举素数 p，答案 = ∑_p (2·∑_{i=1}^{⌊n/p⌋} φ(i) - 1)
正确性：互素对数 = 2·φ 前缀和 - 1（去掉 (1,1) 的重复计数）
```

**C++ 代码**：

```cpp
long long pre[MAXN]; // φ 前缀和

long long solve(int n) {
    long long ans = 0;
    for (int j = 0; j < cnt && primes[j] <= n; j++) {
        int m = n / primes[j];
        ans += 2 * pre[m] - 1;
    }
    return ans;
}
```

**核心启示**：
- **除以公因子**是处理 $\gcd = k$ 问题的通用手段
- 预处理 $\varphi$ 前缀和后可 $O(1)$ 查询

### 题目 5：GCD SUM（洛谷 P2398）

> 求 $\sum_{i=1}^{n} \sum_{j=1}^{n} \gcd(i, j)$。

**分析**：经典反演。$\sum_{i,j} \gcd(i,j) = \sum_{d=1}^{n} d \cdot \sum_{i,j} [\gcd(i,j)=d] = \sum_{d=1}^{n} d \cdot (2 \sum_{k=1}^{\lfloor n/d \rfloor} \varphi(k) - 1)$。

```
核心观察：∑gcd(i,j) = ∑_d d·f(d)，其中 f(d) 为 gcd=d 的对数
构造策略：f(d) = 2·∑φ(k) for k=1..⌊n/d⌋ - 1
正确性：与 P2568 相同的推导
```

**C++ 代码**：

```cpp
long long solve(int n) {
    long long ans = 0;
    for (int d = 1; d <= n; d++) {
        int m = n / d;
        ans += (long long)d * (2 * pre[m] - 1);
    }
    return ans;
}
```

**核心启示**：
- $\sum \gcd$ 问题几乎都用**枚举 gcd 值 + φ 前缀和**解决
- 进阶可用整除分块优化到 $O(\sqrt{n})$

### 题目 6：降幂公式（洛谷 P4139）

> 求 $2^{2^{2^{\cdots}}} \bmod p$（无穷层幂塔）。

**分析**：扩展欧拉定理递归降幂。$a^b \equiv a^{b \bmod \varphi(m) + \varphi(m)} \pmod{m}$，递归计算指数部分，最终 $\varphi$ 降为 1 时终止。

```
核心观察：φ(φ(φ(...))) 最多 O(log m) 步降到 1
构造策略：递归：solve(m) = 2^{solve(φ(m)) + φ(m)} mod m
正确性：扩展欧拉定理保证 b ≥ φ(m) 时降幂正确
```

**C++ 代码**：

```cpp
long long solve(long long m) {
    if (m == 1) return 0;
    long long ph = euler_phi(m);
    long long exp = solve(ph) + ph; // 递归求指数
    return qpow(2, exp, m);
}
```

**核心启示**：
- **扩展欧拉定理**是处理超大指数（幂塔、递归幂）的核心工具
- $\varphi$ 值序列 $O(\log n)$ 步降到 1，递归层数很少

### 🟠 提高级 Level 3：反演与求和

### 题目 7：GCD 计数加强（CSES 1081）

> 求 $1 \le i < j \le n$ 中 $\gcd(i, j) = 1$ 的对数。

**分析**：答案 = $\frac{1}{2}(\sum_{i=1}^{n} \varphi(i) \cdot 2 - 1) = \sum_{i=1}^{n} \varphi(i) - 1$（去掉 $(1,1)$ 后除以 2 并加回 $i < j$）。

```
核心观察：互素对数 = φ 前缀和的简单变形
构造策略：线性筛 + 前缀和
正确性：φ(i) 统计 [1,i] 中与 i 互素的数个数
```

**C++ 代码**：

```cpp
long long solve(int n) {
    long long ans = 0;
    for (int i = 2; i <= n; i++) ans += phi_val[i];
    return ans; // ∑_{i=2}^{n} φ(i) = 互素对数（i < j）
}
```

**核心启示**：
- 互素对计数是 $\varphi$ 前缀和的最直接应用
- 注意有序/无序对的区别

### 题目 8：φ 前缀和 · 杜教筛（洛谷 P4213 部分）

> 求 $\sum_{i=1}^{n} \varphi(i)$，$n \le 2 \times 10^9$。

**分析**：利用 $\sum_{d \mid n} \varphi(d) = n$，是杜教筛的经典应用。

```
核心观察：1 * φ = id，即 ∑_{d|n} φ(d) = n
构造策略：S(n) = n(n+1)/2 - ∑_{d=2}^{n} S(⌊n/d⌋)，整除分块
正确性：杜教筛标准推导
```

**C++ 代码**：

```cpp
unordered_map<long long, long long> memo;
long long S_phi(long long n) {
    if (n <= SIEVE_LIMIT) return pre[n]; // 预筛部分
    if (memo.count(n)) return memo[n];
    long long res = n * (n + 1) / 2;
    for (long long l = 2, r; l <= n; l = r + 1) {
        r = n / (n / l);
        res -= (r - l + 1) * S_phi(n / l);
    }
    return memo[n] = res;
}
```

**核心启示**：
- **杜教筛**：利用 $f * g = h$ 中 $h$ 的前缀和易求来加速 $f$ 的前缀和
- 预筛 $n^{2/3}$ 范围内的值是效率关键

### 题目 9：可见点进阶（CF 776E）

> 求 $1 \le x, y \le n$ 中满足 $\gcd(x, y) = 1$ 且 $x + y \le n$ 的对数。

**分析**：在标准互素计数基础上加了 $x + y \le n$ 的限制。

```
核心观察：枚举 y，对每个 y 统计 x ≤ n-y 且 gcd(x,y) = 1 的个数
构造策略：这等价于 ∑_{y=1}^{n-1} ∑_{x=1}^{n-y} [gcd(x,y)=1]
正确性：利用 φ 的部分和性质或莫比乌斯反演
```

**C++ 代码**：

```cpp
long long solve(int n) {
    long long ans = 0;
    for (int y = 1; y < n; y++) {
        // [1, n-y] 中与 y 互素的数的个数
        int m = n - y;
        // 容斥：∑_{d|y} μ(d) * ⌊m/d⌋
        ans += count_coprime(m, y);
    }
    return ans;
}
```

**核心启示**：
- 带附加条件的互素计数需要**容斥**或**莫比乌斯函数**
- 性能关键在于枚举因子而非逐个检查

### 🔴 困难级 Level 4：综合应用

### 题目 10：φ 函数与 LCM（AcWing 221）

> 求 $\sum_{i=1}^{n} \sum_{j=1}^{n} \text{lcm}(i, j)$。

**分析**：$\text{lcm}(i,j) = \frac{ij}{\gcd(i,j)}$，枚举 $d = \gcd(i,j)$，化为 $\sum_d \frac{1}{d} \sum_{a,b \le n/d, \gcd(a,b)=1} a \cdot b \cdot d^2$。

```
核心观察：lcm = ij/gcd → 枚举 gcd 后化为互素对的加权和
构造策略：f(m) = ∑_{a,b≤m, gcd(a,b)=1} ab，预处理后 O(√n) 枚举
正确性：莫比乌斯反演 + 整除分块
```

**C++ 代码**：

```cpp
// g(m) = ∑_{a=1}^{m} a·[gcd(a,m)=1] 的前缀和式
// 利用 ∑_{a≤m,gcd(a,m)=1} a = m·φ(m)/2 + [m==1]
long long solve(int n) {
    long long ans = 0;
    for (int d = 1; d <= n; d++) {
        int m = n / d;
        // 计算互素对的加权和...
        ans += (long long)d * weighted_coprime_sum(m);
    }
    return ans;
}
```

**核心启示**：
- **LCM 求和**类问题通常先转化为 GCD，再用 φ/μ 反演
- "互素对的加权和" $\sum_{a \le m, \gcd(a,m)=1} a = \frac{m \cdot \varphi(m)}{2} + [m=1]$

### 题目 11：原根个数（CF 1514D）

> 给定素数 $p$，求模 $p$ 的所有原根。

**分析**：原根个数 = $\varphi(\varphi(p)) = \varphi(p-1)$。找到一个原根 $g$ 后，所有原根为 $g^k$（$\gcd(k, p-1) = 1$）。

```
核心观察：模 p 的原根个数恰好是 φ(p-1)
构造策略：找最小原根 g，枚举 k 使 gcd(k, p-1) = 1，g^k 即为原根
正确性：原根的性质 + 欧拉函数的定义
```

**C++ 代码**：

```cpp
vector<long long> find_all_roots(long long p) {
    int g = find_primitive_root(p, p - 1);
    vector<long long> roots;
    for (int k = 1; k < p; k++) {
        if (__gcd((long long)k, p - 1) == 1)
            roots.push_back(qpow(g, k, p));
    }
    sort(roots.begin(), roots.end());
    return roots;
}
```

**核心启示**：
- $\varphi(\varphi(p))$ 给出原根个数，**双层 φ 嵌套**是常见模式
- 找到一个原根后其余原根由指数互素性确定

### 题目 12：同余幂塔（CF 906D）

> 给定数组 $a_1, a_2, \ldots, a_n$，多组询问 $[l, r]$，求 $a_l^{a_{l+1}^{a_{l+2}^{\cdots^{a_r}}}} \bmod m$。

**分析**：扩展欧拉定理的区间版。递归时 $m \to \varphi(m)$，最多 $O(\log m)$ 层。

```
核心观察：φ 值序列 O(log m) 步降到 1
构造策略：预处理 φ 链：m, φ(m), φ(φ(m)), ...；递归计算
正确性：扩展欧拉定理保证降幂正确
```

**C++ 代码**：

```cpp
long long phi_chain[64]; // φ(m), φ(φ(m)), ... 至多 log m 个
int chain_len;

long long solve(int l, int r, int dep) {
    long long mod = phi_chain[dep];
    if (mod == 1) return 0;
    if (l == r) return a[l] % mod;
    long long exp = solve(l + 1, r, dep + 1);
    // 扩展欧拉定理
    if (exp >= mod) exp = exp % mod + mod; // 标记 ≥ φ
    return qpow(a[l], exp, mod);
}
```

**核心启示**：
- **幂塔问题**的核心是 φ 链快速收敛
- 需要特别注意 $b < \varphi(m)$ 和 $b \ge \varphi(m)$ 两种情况

### 🟣 挑战级 Level 5：极限难度

### 题目 13：上帝与集合的正确用法（洛谷 P4139）

> 给定 $p$，求 $2^{2^{2^{\cdots}}} \bmod p$（无穷层 2 的幂塔）。

**分析**：题目 6 的多组询问版。由于幂塔无穷高，指数部分必定 $\ge \varphi(m)$，可以省去大小判断。

```
核心观察：无穷幂塔 → 指数一定足够大 → 一定走 b ≥ φ(m) 分支
构造策略：f(m) = 2^{f(φ(m)) + φ(m)} mod m，m=1 时返回 0
正确性：无穷高保证指数 ≥ φ(m)
```

**C++ 代码**：

```cpp
long long solve(long long m) {
    if (m == 1) return 0;
    long long ph = euler_phi(m);
    return qpow(2, solve(ph) + ph, m);
}
```

**核心启示**：
- 无穷幂塔时不需要判断 $b$ 与 $\varphi(m)$ 的大小关系
- 递归深度 = φ 链长度 = $O(\log m)$

### 题目 14：φ 函数前缀和进阶（SPOJ DCEPCA06）

> 求 $\sum_{i=1}^{n} \sum_{j=1}^{n} \varphi(\gcd(i, j))$，$n \le 10^5$。

**分析**：$\sum_{i,j} \varphi(\gcd(i,j)) = \sum_{d=1}^{n} \varphi(d) \cdot (2 \lfloor n/d \rfloor \sum_{k=1}^{\lfloor n/d \rfloor} \varphi(k) - \lfloor n/d \rfloor^2)$... 更简洁地，利用 $\varphi * 1 = \text{id}$ 得 $\sum \varphi(\gcd) = \sum_{d} d \cdot |\{(i,j) : d \mid \gcd(i,j)\}| - \text{correction}$。

```
核心观察：φ(gcd(i,j)) = ∑_{d|gcd} μ(gcd/d)·d（利用 φ = μ * id）
构造策略：换序求和 → ∑_d d · ∑_k μ(k)·⌊n/(dk)⌋²
正确性：标准莫比乌斯反演
```

**C++ 代码**：

```cpp
long long solve(int n) {
    long long ans = 0;
    for (int T = 1; T <= n; T++) { // T = dk
        long long cnt = (long long)(n / T) * (n / T);
        // 累加 ∑_{d|T} d·μ(T/d) = φ(T)
        ans += phi_val[T] * cnt;
    }
    return ans;
}
// 利用 φ = μ * id，∑_{d|T} d·μ(T/d) = φ(T)
```

**核心启示**：
- $\varphi = \mu * \text{id}$ 是连接 $\varphi$ 和 $\mu$ 的核心卷积关系
- 此题展示了**狄利克雷卷积**在化简中的威力

### 题目 15：Farey 序列长度（POJ 2478）

> Farey 数列 $F_n$ 是 $[0, 1]$ 中分母不超过 $n$ 的最简分数，求 $|F_n|$。

**分析**：$|F_n| = 1 + \sum_{i=1}^{n} \varphi(i)$（每个分母 $i$ 贡献 $\varphi(i)$ 个最简分数，加上 $0/1$）。

```
核心观察：Farey 序列长度 = φ 前缀和 + 1
构造策略：线性筛 + 前缀和
正确性：分母为 i 的最简分数恰有 φ(i) 个
```

**C++ 代码**：

```cpp
long long farey_length(int n) {
    long long ans = 1; // 0/1
    for (int i = 1; i <= n; i++) ans += phi_val[i];
    return ans;
}
```

**核心启示**：
- **Farey 序列**是欧拉函数的经典数论应用
- Stern-Brocot 树与 Farey 序列有深刻联系

## 五、常见技巧与思路总结

### 5.1 先除后乘避免溢出

计算 $\varphi(n)$ 时用 `res = res / p * (p - 1)` 而非 `res * (p - 1) / p`。

### 5.2 φ 的卷积恒等式

$\varphi * 1 = \text{id}$，即 $\sum_{d \mid n} \varphi(d) = n$。这是杜教筛和反演的基石。

### 5.3 降幂公式中的判断

使用扩展欧拉定理时，必须判断 $b$ 与 $\varphi(m)$ 的大小关系。可以在读入时用字符串比较或在递归中传递标记。

### 5.4 φ 链的快速收敛

$m \to \varphi(m)$ 最多 $O(\log m)$ 步降到 1，因为偶数的 $\varphi$ 至少减半，奇数变偶数。

### 5.5 整除分块加速

当需要计算 $\sum_{d=1}^{n} f(d) \cdot g(\lfloor n/d \rfloor)$ 时，由于 $\lfloor n/d \rfloor$ 只有 $O(\sqrt{n})$ 种取值，可以整除分块。

## 六、优化方法 / 解题思路框架

```
问题类型判断：
  │
  ├── 单个 φ(n) → 试除法 O(√n)
  ├── 批量 φ(1..n), n ≤ 10^7 → 线性筛
  ├── ∑φ(i), n ≤ 10^{10} → 杜教筛 O(n^{2/3})
  ├── a^b mod m, b 超大 → 扩展欧拉定理（降幂）
  ├── ∑gcd(i,j) → 枚举 gcd + φ 前缀和
  └── 幂塔 mod m → 递归 + φ 链
```

## 七、终极总结与归纳

### 7.1 欧拉函数的本质（一句话）

欧拉函数 $\varphi(n)$ 是连接**互素性**与**乘法群结构**的桥梁——它既统计互素个数，又给出幂运算的周期。

### 7.2 核心要点

1. **公式**：$\varphi(n) = n \cdot \prod(1 - 1/p_i)$，积性函数
2. **线性筛**：$O(n)$ 批量求值，利用最小素因子转移
3. **欧拉定理**：$a^{\varphi(n)} \equiv 1$，幂运算周期性的基石
4. **扩展欧拉定理**：降幂公式，处理超大指数和幂塔
5. **卷积恒等式**：$\varphi * 1 = \text{id}$，杜教筛和反演的关键
6. **φ 链收敛**：$O(\log m)$ 步降到 1，保证递归效率
