## 一、什么是卡特兰数与斯特林数

卡特兰数和斯特林数是组合数学中两类重要的递推数列。**卡特兰数** $C_n = \frac{1}{n+1}\binom{2n}{n}$ 统计括号匹配、二叉树、不越对角线路径等结构。**第一类斯特林数** $[n \atop k]$ 统计圆排列数，**第二类斯特林数** $\{n \atop k\}$ 统计集合划分数。**贝尔数** $B_n = \sum_k \{n \atop k\}$ 是所有划分方案总数。

| 数列 | 递推 | 组合意义 |
|------|------|---------|
| 卡特兰 $C_n$ | $\sum C_i C_{n-1-i}$ | 括号/二叉树/路径 |
| $[n \atop k]$ | $(n\!-\!1)[n\!-\!1 \atop k] + [n\!-\!1 \atop k\!-\!1]$ | 圆排列 |
| $\{n \atop k\}$ | $k\{n\!-\!1 \atop k\} + \{n\!-\!1 \atop k\!-\!1\}$ | 集合划分 |

## 二、核心思想与前置知识

### 2.1 卡特兰数

通项：$C_n = \binom{2n}{n} - \binom{2n}{n+1}$。反射原理：越过对角线的路径与从 $(-1,1)$ 出发的路径一一对应。

### 2.2 第二类斯特林数

容斥：$\{n \atop k\} = \frac{1}{k!}\sum_{i=0}^{k}(-1)^i\binom{k}{i}(k-i)^n$。核心恒等式：$x^n = \sum_k \{n \atop k\} x^{\underline{k}}$。

```cpp
// S2 行：容斥 + NTT
void stirling2_row(int n, long long res[], long long mod) {
    vector<long long> f(n+1), g(n+1);
    for (int i = 0; i <= n; i++) {
        f[i] = (i%2 ? mod-1 : 1) * inv_fac[i] % mod;
        g[i] = qpow(i, n, mod) * inv_fac[i] % mod;
    }
    ntt_conv(f, g, n+1);
    for (int k = 0; k <= n; k++) res[k] = f[k];
}
```

### 2.3 第一类斯特林数

生成函数：$\sum_k [n \atop k] x^k = x(x+1)\cdots(x+n-1)$。分治 NTT 计算。

### 2.4 斯特林反演

$f(n) = \sum_k \{n \atop k\} g(k) \iff g(n) = \sum_k (-1)^{n-k}[n \atop k] f(k)$

## 三、分类与变体详解

### 3.1 卡特兰数模型

括号匹配($C_n$)、二叉树($C_n$)、不越线路径($C_n$)、出栈序列($C_n$)、三角剖分($C_n$)、有序树($C_n$)。

### 3.2 斯特林数计算

| 方法 | 适用 | 复杂度 |
|------|------|--------|
| DP | 小 n,k | $O(nk)$ |
| 容斥+NTT | S2 行 | $O(n\log n)$ |
| 分治NTT | S1 行 | $O(n\log^2 n)$ |
| EGF+多项式幂 | S1/S2 列 | $O(n\log n)$ |

### 3.3 核心公式

$x^n = \sum \{n \atop k\} x^{\underline{k}}$，$\sum i^k = \sum_j \{k \atop j\} j! \binom{m+1}{j+1}$

### 题目 1：树屋阶梯（洛谷 P2532）

> 求第 $n$ 个卡特兰数 $C_n$（高精度），$n \le 100$。

**分析**：递推 $C_n = C_{n-1} \cdot \frac{2(2n-1)}{n+1}$，先乘后除保证整除。

```
核心观察：C_n 增长极快，n=100 时有 56 位
构造策略：递推公式 + 高精度乘除
正确性：卡特兰递推
```

**C++ 代码**：

```cpp
// BigInt 高精度类省略
BigInt catalan(int n) {
    BigInt res(1);
    for (int i = 1; i <= n; i++)
        res = res * BigInt(4 * i - 2) / BigInt(i + 1);
    return res;
}
```

**核心启示**：
- **递推公式**避免大阶乘计算
- 先乘后除保证中间结果为整数
- $C_{100} \approx 8.96 \times 10^{56}$

### 题目 2：有趣的数列（洛谷 P3200）

> 合法序列数 $\equiv C_n \pmod{p}$（$p$ 可能不是素数）。

**分析**：$C_n = \binom{2n}{n} - \binom{2n}{n+1}$。若 $p$ 非素数，用质因子分解（Legendre 公式）计算。

```
核心观察：差值公式避免除以 n+1
构造策略：C(2n,n) - C(2n,n+1) mod p；非素数用质因子分解
正确性：反射原理
```

**C++ 代码**：

```cpp
long long catalan_mod(int n, long long p) {
    if (is_prime(p)) return (C(2*n, n, p) - C(2*n, n+1, p) + p) % p;
    // 非素数：分解 C_n 中每个素因子的幂次
    long long res = 1;
    for (each prime q dividing p) {
        int exp_q = legendre(2*n, q) - 2*legendre(n, q) - legendre(n+1, q);
        // ... CRT 合并
    }
    return res;
}
```

**核心启示**：
- **差值公式**是模运算下的最优选择
- 非素数模时用 **Legendre 公式 + CRT**

### 题目 3：概率论（洛谷 P3978）

> 随机 $n$ 节点二叉树的叶子数期望。

**分析**：设 $L(n) = \sum_{\text{trees}} \text{叶子数}$，$E = L(n)/C_n$。利用递推结构推导。

```
核心观察：对卡特兰递推加权求和得 L(n)
构造策略：L(n) 满足与 C_n 类似的递推，化简得闭式
正确性：二叉树的左右子树递推
```

**C++ 代码**：

```cpp
// 推导结论：E[叶子] = n·C_{n-1} / C_n = n(n+1) / (2(2n-1))（n≥2）
long long expected_leaves(int n, long long mod) {
    if (n <= 1) return n % mod;
    long long num = (long long)n % mod * ((n + 1) % mod) % mod;
    long long den = 2 * (2 * n - 1) % mod;
    return num * qpow(den, mod - 2, mod) % mod;
}
```

**核心启示**：
- 卡特兰结构上的**期望**通过递推推导
- $E \to n/4$ 当 $n$ 很大时（约四分之一节点是叶子）

### 题目 4：小朋友的球（洛谷 P1655）

long long S2[MAXN][MAXN];
void init(int n) {

### 题目 5：第二类斯特林数·行（洛谷 P5395）

> 给定 $n$，求 $\{n \atop 0\}, \ldots, \{n \atop n\}$。

**分析**：容斥写成卷积 → NTT。

```
核心观察：容斥公式 = f * g 的卷积形式
构造策略：f[i]=(-1)^i/i!, g[i]=i^n/i!, S2[n][k]=(f*g)[k]
正确性：容斥展开
```

**C++ 代码**：

```cpp
void solve(int n, long long mod) {
    vector<long long> f(n+1), g(n+1);
    for (int i = 0; i <= n; i++) {
        f[i] = (i%2 ? mod-1 : 1) * inv_fac[i] % mod;
        g[i] = qpow(i, n, mod) * inv_fac[i] % mod;
    }
    int len = 1; while (len <= 2*n) len <<= 1;
    ntt(f, len, 1); ntt(g, len, 1);
    for (int i = 0; i < len; i++) f[i] = f[i] * g[i] % mod;
    ntt(f, len, -1);
    // f[0..n] 即为 S2[n][0..n]
}
```

**核心启示**：
- **容斥 → 卷积 → NTT**：$O(n^2) \to O(n\log n)$
- 这是求一整行斯特林数的标准方法

### 题目 6：第二类斯特林数·列（洛谷 P5396）

> 给定 $k$，求 $\{0 \atop k\}, \{1 \atop k\}, \ldots, \{N \atop k\}$。

**分析**：EGF $= (e^x - 1)^k / k!$，多项式快速幂。

```
核心观察：S2 列的 EGF = (e^x-1)^k / k!
构造策略：f(x) = e^x-1 = ∑x^i/i!（i≥1），求 f^k
正确性：EGF 定义
```

**C++ 代码**：

```cpp
void solve(int N, int k, long long mod) {
    vector<long long> f(N+1, 0);
    for (int i = 1; i <= N; i++) f[i] = inv_fac[i];
    auto res = poly_pow(f, k, N+1, mod); // exp(k·ln(f))
    for (int n = 0; n <= N; n++)
        ans[n] = res[n] * fac[n] % mod * inv_fac[k] % mod;
}
```

**核心启示**：
- **EGF + 多项式快速幂**是列计算的标准方法
- `poly_pow` = `exp(k · ln(f))`，需要多项式全家桶

### 题目 7：第一类斯特林数·行（洛谷 P5408）

> 给定 $n$，求 $[n \atop 0], \ldots, [n \atop n]$。

**分析**：$\sum_k [n \atop k] x^k = x(x+1)\cdots(x+n-1)$，分治 NTT。

```
核心观察：上升阶乘 = n 个一次多项式的乘积
构造策略：分治 solve(l,r)，merge 两半多项式
正确性：分治乘法
```

**C++ 代码**：

```cpp
vector<long long> solve(int l, int r, long long mod) {
    if (l == r) return {l % mod, 1}; // x + l
    int mid = (l + r) / 2;
    auto L = solve(l, mid, mod);
    auto R = solve(mid+1, r, mod);
    return ntt_multiply(L, R, mod);
}
// 调用 solve(0, n-1, mod)
```

**核心启示**：
- **分治 NTT**：$O(n \log^2 n)$
- 可用 **倍增法** 优化到 $O(n \log n)$

### 题目 10：Crash 的文明世界（洛谷 P4827）

> 树上每个节点 $u$，求 $\sum_v \text{dist}(u,v)^k$。$k \le 150$。

**分析**：$d^k = \sum_j \{k \atop j\}\binom{d}{j}j!$。$\binom{d}{j}$ 在树上可递推：$\binom{d+1}{j} = \binom{d}{j} + \binom{d}{j-1}$。换根 DP。

```
核心观察：幂 → 下降幂展开，C(d,j) 树上可递推
构造策略：对 j=0..k 换根 DP 求 g[u][j]=∑C(dist(u,v),j)，ans[u]=∑S2[k][j]·j!·g[u][j]
正确性：斯特林展开 + 组合数递推
```

**C++ 代码**：

```cpp
long long g[MAXN][155]; // g[u][j] = ∑_v C(dist(u,v), j)

void dfs_down(int u, int fa) {
    g[u][0] = 1;
    for (int v : adj[u]) {
        if (v == fa) continue;
        dfs_down(v, u);
        for (int j = 150; j >= 1; j--)
            g[u][j] = (g[u][j] + g[v][j] + g[v][j-1]) % MOD;
        g[u][0] = (g[u][0] + g[v][0]) % MOD;
    }
}
// 换根 DP 处理父方向贡献
// ans[u] = ∑_j S2[k][j] * fac[j] * g[u][j]
```

**核心启示**：
- **斯特林展开 + 换根 DP** 是"树上幂和"的标准方法
- $C(d,j)$ 的递推使树上转移自然
- 时间 $O(nk)$，$k \le 150$ 足够

### 题目 12：斯特林反演（综合）

> 已知 $f(k) = \sum_j \{k \atop j\} g(j)$，求 $g(k)$。

**分析**：$g(n) = \sum_k (-1)^{n-k}[n \atop k] f(k)$。

```
核心观察：S2 和 S1 互为反演
构造策略：g[n] = ∑ (-1)^{n-k} · S1[n][k] · f[k]
正确性：斯特林反演定理
```

**C++ 代码**：

```cpp
void invert(long long f[], long long g[], int n, long long mod) {
    // 预处理 S1[n][0..n]（分治 NTT 或 DP）
    for (int i = 0; i <= n; i++) {
        g[i] = 0;
        for (int k = 0; k <= i; k++) {
            long long sign = ((i-k)%2==0 ? 1 : mod-1);
            g[i] = (g[i] + sign * S1[i][k] % mod * f[k]) % mod;
        }
    }
}
```

**核心启示**：
- 斯特林反演 = **普通幂与下降幂互转**的反面
- 与二项式反演并列为最重要的组合反演

### 题目 13：贝尔数

> 求 $B_n$（$n$ 个元素的所有划分方案数）。

**分析**：Bell 三角形 $O(n^2)$ 或 EGF $e^{e^x-1}$ + NTT。

```
核心观察：B_n = ∑S(n,k) = [x^n/n!] exp(exp(x)-1)
构造策略：Bell 三角递推 或 多项式 exp
正确性：定义
```

**C++ 代码**：

```cpp
long long bell(int n, long long mod) {
    vector<long long> prev(n+1, 0), cur(n+1, 0);
    prev[0] = 1;
    for (int i = 1; i <= n; i++) {
        cur[0] = prev[i-1];
        for (int j = 1; j <= i; j++)
            cur[j] = (cur[j-1] + prev[j-1]) % mod;
        swap(prev, cur);
    }
    return prev[0];
}
```

**核心启示**：
- **Bell 三角形**：$O(n^2)$ 简单实现
- EGF 方法 $O(n \log n)$ 但需要多项式 exp
- $B_{15} = 1382958545$

### 题目 14：幂和（斯特林展开法）

> 求 $\sum_{i=1}^{m} i^k$，$k$ 可达 $10^6$，$m$ 可达 $10^{18}$。

**分析**：$\sum i^k = \sum_j \{k \atop j\} j! \binom{m+1}{j+1}$。

```
核心观察：幂和 = ∑S(k,j)·j!·C(m+1,j+1)
构造策略：NTT 求 S2 行，O(k) 求和
正确性：Hockey Stick 恒等式：∑C(i,j) = C(m+1,j+1)
```

**C++ 代码**：

```cpp
long long power_sum(long long m, int k, long long mod) {
    vector<long long> S(k+1);
    stirling2_row(k, S.data(), mod);
    long long ans = 0, comb = (m+1) % mod;
    for (int j = 0; j <= k; j++) {
        if (j > 0) comb = comb * ((m+1-j) % mod + mod) % mod * qpow(j+1, mod-2, mod) % mod;
        ans = (ans + S[j] * fac[j] % mod * comb) % mod;
    }
    return ans;
}
```

**核心启示**：
- 斯特林展开使 $m$ 任意大都能 $O(k)$ 计算
- **拉格朗日插值**也能 $O(k)$ 求幂和（$S_k$ 是 $m$ 的 $k+1$ 次多项式）

### 题目 15：幂和（拉格朗日插值法）

> 同题 14，用插值方法。

**分析**：$S_k(m) = \sum_{i=1}^{m}i^k$ 是 $m$ 的 $k+1$ 次多项式。取 $k+2$ 个点插值。

```
核心观察：S_k(m) 是 m 的 k+1 次多项式
构造策略：取 x=0,1,...,k+1 共 k+2 个点，连续点插值 O(k)
正确性：多项式由 k+2 个点唯一确定
```

**C++ 代码**：

```cpp
long long power_sum_lagrange(long long m, int k, long long mod) {
    int n = k + 2;
    vector<long long> y(n);
    for (int i = 1; i < n; i++) y[i] = (y[i-1] + qpow(i, k, mod)) % mod;
    vector<long long> pre(n+1, 1), suf(n+1, 1);
    for (int i = 0; i < n; i++) pre[i+1] = pre[i] * ((m-i)%mod+mod) % mod;
    for (int i = n-1; i >= 0; i--) suf[i] = suf[i+1] * ((m-i)%mod+mod) % mod;
    long long ans = 0;
    for (int i = 0; i < n; i++) {
        long long num = pre[i] * suf[i+1] % mod;
        long long den = inv_fac[i] * inv_fac[n-1-i] % mod;
        if ((n-1-i) % 2) den = (mod - den) % mod;
        ans = (ans + y[i] * num % mod * den) % mod;
    }
    return ans;
}
```

**核心启示**：
- **连续整数点插值** $O(k)$：分母 = $i! \cdot (n-1-i)! \cdot (-1)^{n-1-i}$
- 斯特林法和插值法**殊途同归**，复杂度相同

## 五、常见技巧与思路总结

### 5.1 识别卡特兰数
不越线、括号匹配、出栈序列、二叉树 → $C_n$。

### 5.2 普通幂 ↔ 下降幂
$x^n = \sum \{n \atop k\} x^{\underline{k}}$。

### 5.3 容斥求 S2
$\{n \atop k\}$ = 卷积形式 → NTT $O(n\log n)$。

### 5.4 分治 NTT 求 S1
$x(x+1)\cdots(x+n-1)$ 分治乘 $O(n\log^2 n)$。

### 5.5 EGF + 多项式快速幂求列
S2: $(e^x-1)^k/k!$，S1: $(-\ln(1-x))^k/k!$。

## 七、终极总结与归纳

### 7.1 本质
卡特兰刻画"不越界"结构，斯特林数是**普通幂与下降幂**的桥梁。

### 7.2 核心要点
1. $C_n = \binom{2n}{n}/(n+1)$，反射原理
2. S2 递推 + 容斥 + NTT
3. S1 上升阶乘 + 分治 NTT
4. 斯特林反演：S1 与 S2 互逆
5. 幂和：$\sum i^k = \sum \{k \atop j\} j! \binom{m+1}{j+1}$
6. EGF 是列计算核心
