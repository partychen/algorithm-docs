## 一、什么是矩阵

矩阵是数学中用于表示线性变换的核心工具。在算法竞赛中，**矩阵快速幂**是最重要的应用——它能将线性递推的求解从 $O(n)$ 加速到 $O(k^3 \log n)$（$k$ 为矩阵大小）。

| 应用场景 | 方法 | 时间复杂度 |
|---------|------|-----------|
| 常系数线性递推 | 矩阵快速幂 | $O(k^3 \log n)$ |
| 图上路径计数 | 邻接矩阵幂 | $O(n^3 \log k)$ |
| 矩阵求逆 | 高斯消元 | $O(n^3)$ |
| 线性变换组合 | 矩阵乘法 | $O(n^3)$ |

## 二、核心思想与前置知识

### 2.1 矩阵乘法

$C = A \times B$，其中 $C_{ij} = \sum_k A_{ik} \cdot B_{kj}$。乘法满足结合律但不满足交换律。

```cpp
struct Matrix {
    long long a[MAXK][MAXK];
    int n;
    Matrix operator*(const Matrix& o) const {
        Matrix res; res.n = n;
        for (int i = 0; i < n; i++)
            for (int j = 0; j < n; j++) {
                res.a[i][j] = 0;
                for (int k = 0; k < n; k++)
                    res.a[i][j] = (res.a[i][j] + a[i][k] * o.a[k][j]) % MOD;
            }
        return res;
    }
};
```

### 2.2 矩阵快速幂

利用结合律，$A^n$ 可以用快速幂在 $O(k^3 \log n)$ 内计算：

```cpp
Matrix mat_pow(Matrix A, long long n) {
    Matrix res; res.n = A.n;
    // 初始化为单位矩阵
    for (int i = 0; i < res.n; i++) res.a[i][i] = 1;
    while (n) {
        if (n & 1) res = res * A;
        A = A * A;
        n >>= 1;
    }
    return res;
}
```

### 2.3 矩阵加速递推的一般方法

对于递推 $f(n) = c_1 f(n-1) + c_2 f(n-2) + \cdots + c_k f(n-k)$，构造 $k \times k$ 转移矩阵：

$$\begin{pmatrix} f(n) \ f(n-1) \ \vdots \ f(n-k+1) \end{pmatrix} = \begin{pmatrix} c_1 & c_2 & \cdots & c_k \ 1 & 0 & \cdots & 0 \ \vdots & \ddots & & \vdots \ 0 & \cdots & 1 & 0 \end{pmatrix}^{n-k} \begin{pmatrix} f(k) \ f(k-1) \ \vdots \ f(1) \end{pmatrix}$$

## 三、分类与变体详解

### 3.1 常系数线性递推加速

> 状态向量：$(f(n), f(n-1), \ldots, f(n-k+1))$
> 转移矩阵：第一行为系数，其余为移位
> 时间：$O(k^3 \log n)$

### 3.2 带常数项的递推

若递推中有常数项 $f(n) = af(n-1) + b$，将状态扩展为 $(f(n), 1)$。

### 3.3 图上路径计数

邻接矩阵 $A$ 的 $k$ 次幂 $A^k_{ij}$ = 从 $i$ 到 $j$ 恰好走 $k$ 步的路径数。

### 3.4 矩阵求逆

在增广矩阵 $[A | I]$ 上做高斯消元，化为 $[I | A^{-1}]$。

### 3.5 变体对比

| 变体 | 矩阵大小 | 关键技巧 |
|------|---------|---------|
| 斐波那契 | $2 \times 2$ | 基础矩阵快速幂 |
| k 阶递推 | $k \times k$ | 构造转移矩阵 |
| 图路径 | $n \times n$ | 邻接矩阵幂 |
| 状态转移 | 视状态数 | 自动机 + 矩阵 |

### 题目 1：矩阵快速幂模板（洛谷 P3390）

> 给定 $n \times n$ 矩阵 $A$，求 $A^k \bmod p$。

**分析**：矩阵快速幂模板题。

> **核心观察**：矩阵乘法满足结合律 → 快速幂适用
> **构造策略**：二进制拆分 $k$，逐位乘
> **正确性**：快速幂的正确性

**C++ 代码**：

```cpp
Matrix mat_pow(Matrix A, long long k) {
    Matrix res; res.n = A.n;
    for (int i = 0; i < res.n; i++) res.a[i][i] = 1;
    while (k) {
        if (k & 1) res = res * A;
        A = A * A; k >>= 1;
    }
    return res;
}
```

**核心启示**：
- 矩阵快速幂是**所有矩阵加速问题的基础**
- 注意初始化为单位矩阵

### 题目 2：斐波那契数列（洛谷 P1962）

> 求斐波那契数列第 $n$ 项，$n$ 可达 $10^{18}$。

**分析**：$f(n) = f(n-1) + f(n-2)$，转移矩阵 $\begin{pmatrix} 1 & 1 \ 1 & 0 \end{pmatrix}^{n-1}$。

> **核心观察**：二阶递推 → $2 \times 2$ 矩阵
> **构造策略**：$(f(n), f(n-1)) = M^{n-1} \cdot (f(1), f(0))$
> **正确性**：矩阵乘法等价于递推

**C++ 代码**：

```cpp
long long fib(long long n, long long mod) {
    Matrix M; M.n = 2;
    M.a[0][0] = M.a[0][1] = M.a[1][0] = 1; M.a[1][1] = 0;
    Matrix res = mat_pow(M, n - 1);
    return res.a[0][0] % mod; // f(1) = 1
}
```

**核心启示**：
- 斐波那契是矩阵加速的**最经典例子**
- $2 \times 2$ 矩阵乘法极快

### 题目 3：矩阵加速数列（洛谷 P1939）

> $f(n) = f(n-1) + f(n-3)$，$f(1)=f(2)=f(3)=1$，求 $f(n)$。

**分析**：三阶递推，$3 \times 3$ 转移矩阵。

> **核心观察**：三阶递推 → $3 \times 3$ 矩阵
> **构造策略**：第一行 $(1,0,1)$，其余为移位矩阵
> **正确性**：验证 $M \cdot (f(n-1),f(n-2),f(n-3))^T = (f(n),f(n-1),f(n-2))^T$

**C++ 代码**：

```cpp
long long solve(long long n, long long mod) {
    if (n <= 3) return 1;
    Matrix M; M.n = 3;
    M.a[0][0] = 1; M.a[0][2] = 1;
    M.a[1][0] = 1; M.a[2][1] = 1;
    Matrix res = mat_pow(M, n - 3);
    return (res.a[0][0] + res.a[0][1] + res.a[0][2]) % mod;
}
```

**核心启示**：
- **k 阶递推对应 k×k 转移矩阵**
- 第一行放系数，其余是移位

### 题目 4：广义斐波那契数列（洛谷 P1349）

> $f(n) = p \cdot f(n-1) + q \cdot f(n-2)$，给定 $f(1), f(2), p, q$，求 $f(n)$。

**分析**：转移矩阵 $\begin{pmatrix} p & q \ 1 & 0 \end{pmatrix}$。

> **核心观察**：系数不为 1 时转移矩阵对应修改
> **构造策略**：$M = \begin{pmatrix} p & q \\ 1 & 0 \end{pmatrix}$
> **正确性**：$M \cdot (f(n-1),f(n-2))^T = (p \cdot f(n-1)+q \cdot f(n-2), f(n-1))^T$

**C++ 代码**：

```cpp
long long solve(long long p, long long q, long long f1, long long f2, long long n, long long mod) {
    if (n == 1) return f1 % mod;
    if (n == 2) return f2 % mod;
    Matrix M; M.n = 2;
    M.a[0][0] = p; M.a[0][1] = q;
    M.a[1][0] = 1; M.a[1][1] = 0;
    Matrix res = mat_pow(M, n - 2);
    return (res.a[0][0] * f2 + res.a[0][1] * f1) % mod;
}
```

**核心启示**：
- 广义递推只需修改转移矩阵的**第一行**

### 题目 5：可乐（洛谷 P3758）

> $n$ 个城市的无向图，每秒可以留在原地、走到相邻城市、或自爆。求从城市 1 出发 $t$ 秒后的总方案数。

**分析**：邻接矩阵 $A$ 加上自环（对角线 +1）再加一个吸收态（自爆），矩阵快速幂。

> **核心观察**：图上游走 = 邻接矩阵幂
> **构造策略**：扩展 $(n+1) \times (n+1)$ 矩阵，第 $n+1$ 列全 1（自爆是吸收态）
> **正确性**：$A^t$ 的第 1 行之和 = 所有可达状态方案

**C++ 代码**：

```cpp
long long solve(int adj[][MAXN], int n, int t, long long mod) {
    Matrix M; M.n = n + 1;
    for (int i = 0; i < n; i++) {
        M.a[i][i] = 1; // 留在原地
        M.a[i][n] = 1; // 自爆
        for (int j = 0; j < n; j++)
            if (adj[i][j]) M.a[i][j] = 1;
    }
    M.a[n][n] = 1; // 自爆后不动
    Matrix res = mat_pow(M, t);
    long long ans = 0;
    for (int j = 0; j <= n; j++) ans = (ans + res.a[0][j]) % mod;
    return ans;
}
```

**核心启示**：
- **图上游走**问题用邻接矩阵幂解决
- 吸收态（自爆、消失）加一行一列

### 题目 6：矩阵求逆（洛谷 P4783）

> 给定 $n \times n$ 矩阵，求其逆矩阵（模素数）。

**分析**：增广矩阵 $[A | I]$ 高斯消元化为 $[I | A^{-1}]$。

> **核心观察**：高斯消元同时作用于 $A$ 和 $I$
> **构造策略**：对 $[A|I]$ 做列主元消元
> **正确性**：初等行变换等价于左乘初等矩阵

**C++ 代码**：

```cpp
bool mat_inv(long long a[][MAXN], long long inv[][MAXN], int n, long long p) {
    // 初始化 inv 为单位矩阵
    for (int i = 0; i < n; i++) inv[i][i] = 1;
    for (int col = 0; col < n; col++) {
        int pivot = -1;
        for (int i = col; i < n; i++)
            if (a[i][col]) { pivot = i; break; }
        if (pivot == -1) return false;
        swap(a[col], a[pivot]); swap(inv[col], inv[pivot]);
        long long iv = qpow(a[col][col], p - 2, p);
        for (int j = 0; j < n; j++) { a[col][j] = a[col][j] * iv % p; inv[col][j] = inv[col][j] * iv % p; }
        for (int i = 0; i < n; i++) {
            if (i == col || !a[i][col]) continue;
            long long f = a[i][col];
            for (int j = 0; j < n; j++) {
                a[i][j] = (a[i][j] - f * a[col][j] % p + p) % p;
                inv[i][j] = (inv[i][j] - f * inv[col][j] % p + p) % p;
            }
        }
    }
    return true;
}
```

**核心启示**：
- 矩阵求逆 = 高斯消元的自然扩展
- 不可逆时行列式为 0

> 长度为 $n$ 的字符串，不包含给定子串，求方案数。

### 题目 10：分块（洛谷 P5343）

> 长度为 $n$ 的序列分块方案数，每块长度在给定集合 $S$ 中。

**分析**：$f(n) = \sum_{s \in S} f(n-s)$，这是多阶递推，转移矩阵大小 = $\max(S)$。

> **核心观察**：分块 DP → 常系数递推 → 矩阵快速幂
> **构造策略**：矩阵大小 = $\max(S)$，第一行在 $S$ 对应的位置为 1
> **正确性**：等价于原始递推

**C++ 代码**：

```cpp
long long solve(vector<int>& S, long long n, long long mod) {
    int m = *max_element(S.begin(), S.end());
    Matrix M; M.n = m;
    for (int s : S) M.a[0][s - 1] = 1;
    for (int i = 1; i < m; i++) M.a[i][i - 1] = 1;
    Matrix res = mat_pow(M, n);
    return res.a[0][0]; // f(n) = M^n · (1, 0, ..., 0)^T
}
```

**核心启示**：
- 块长集合决定矩阵大小和结构
- 当 $\max(S)$ 较大时需要**多项式取模优化**（Berlekamp-Massey）

### 题目 11：斐波那契数列进阶（洛谷 P4000）

> 求第 $n$ 个斐波那契数的精确值（高精度），$n$ 可达 $10^{15}$。

**分析**：矩阵快速幂 + 高精度。或者用快速倍增公式：

$f(2n) = f(n) \cdot (2f(n+1) - f(n))$，$f(2n+1) = f(n)^2 + f(n+1)^2$。

> **核心观察**：倍增公式避免矩阵乘法的高精度开销
> **构造策略**：递归倍增，每次 $n$ 减半
> **正确性**：由矩阵幂关系推导的恒等式

**C++ 代码**：

```cpp
// 倍增公式（使用 Python 或高精度库）
pair<BigInt, BigInt> fib_pair(long long n) {
    if (n == 0) return {0, 1};
    auto [a, b] = fib_pair(n / 2);
    BigInt c = a * (2 * b - a);
    BigInt d = a * a + b * b;
    if (n % 2 == 0) return {c, d};
    else return {d, c + d};
}
```

**核心启示**：
- **倍增公式**比矩阵快速幂更适合高精度场景
- $O(\log n)$ 次高精度乘法

### 题目 12：图上 k 步可达（AcWing 矩阵路径）

> $n$ 个点的有向图，求从 $u$ 到 $v$ 恰好走 $k$ 步的路径数。

**分析**：邻接矩阵 $A$ 的 $k$ 次幂。

> **核心观察**：$A^k[u][v]$ = 从 $u$ 到 $v$ 走 $k$ 步的路径数
> **构造策略**：矩阵快速幂
> **正确性**：邻接矩阵幂的组合意义

**C++ 代码**：

```cpp
long long paths(int adj[][MAXN], int n, int u, int v, long long k, long long mod) {
    Matrix A; A.n = n;
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++) A.a[i][j] = adj[i][j];
    Matrix res = mat_pow(A, k);
    return res.a[u][v];
}
```

**核心启示**：
- 图上路径计数是矩阵幂的**经典应用**
- 可推广到带权图（最短路用 min-plus 矩阵）

### 题目 14：矩阵求和 $\sum_{i=0}^{n} A^i$

> 求 $S = I + A + A^2 + \cdots + A^n$。

**分析**：分治：$S(n) = S(n/2) \cdot (I + A^{n/2+1})$，或者构造 $(n+1) \times (n+1)$ 块矩阵。

> **核心观察**：分治或扩展矩阵可求矩阵幂和
> **构造策略**：扩展为 $\begin{pmatrix} A & I \\ 0 & I \end{pmatrix}^{n+1}$ 的右上角块
> **正确性**：通过归纳验证

**C++ 代码**：

```cpp
// 方法：构造 2k×2k 矩阵 M = ((A, I), (0, I))
// M^{n+1} = ((A^{n+1}, S(n)), (0, I))
Matrix sum_pow(Matrix A, long long n) {
    int k = A.n;
    Matrix M; M.n = 2 * k;
    // 左上：A，右上：I，左下：0，右下：I
    for (int i = 0; i < k; i++)
        for (int j = 0; j < k; j++)
            M.a[i][j] = A.a[i][j];
    for (int i = 0; i < k; i++) M.a[i][i + k] = M.a[i + k][i + k] = 1;
    M = mat_pow(M, n + 1);
    // 提取右上 k×k 块
    Matrix res; res.n = k;
    for (int i = 0; i < k; i++)
        for (int j = 0; j < k; j++)
            res.a[i][j] = M.a[i][j + k];
    return res;
}
```

**核心启示**：
- **矩阵幂和**通过扩展矩阵可在同一次快速幂中求出
- 这是一个非常优雅的技巧

### 题目 15：十进制快速幂（CF）

> $n$ 以十进制给出（位数可达 $10^6$），求 $A^n \bmod m$。

**分析**：从高位到低位逐位处理，每位做"平方再乘"。

> **核心观察**：十进制快速幂：从高位开始，每位 $res = res^{10} \cdot A^{digit}$
> **构造策略**：预处理 $A^0 \ldots A^9$，逐位处理
> **正确性**：十进制拆分

**C++ 代码**：

```cpp
Matrix decimal_pow(Matrix A, string& n_str, long long mod) {
    Matrix res; res.n = A.n;
    for (int i = 0; i < res.n; i++) res.a[i][i] = 1;
    Matrix pw[10]; pw[0] = res; // I
    for (int i = 1; i <= 9; i++) pw[i] = pw[i-1] * A;
    for (char c : n_str) {
        // res = res^10 * A^digit
        Matrix tmp = res;
        for (int i = 1; i < 10; i++) tmp = tmp * res; // res^10
        res = tmp * pw[c - '0'];
    }
    return res;
}
```

**核心启示**：
- 十进制快速幂处理**超大指数**（不能用二进制）
- 预处理 $A^{0..9}$ 避免重复计算

### 题目 16：矩阵快速幂优化有限自动机（综合）

> 给定 DFA 和 $n$，求长度为 $n$ 的被接受字符串数。

**分析**：DFA 转移矩阵 $M$，$M^n$ 的初始态行与所有接受态列之和。

> **核心观察**：DFA 转移 = 矩阵乘法
> **构造策略**：$M[i][j]$ = 从 $i$ 经一步到 $j$ 的转移数，$M^n$
> **正确性**：路径计数

**C++ 代码**：

```cpp
long long dfa_count(int trans[][SIGMA], int n_states, int start,
                    vector<int>& accept, long long n, long long mod) {
    Matrix M; M.n = n_states;
    for (int i = 0; i < n_states; i++)
        for (int c = 0; c < SIGMA; c++)
            M.a[i][trans[i][c]]++;
    Matrix res = mat_pow(M, n);
    long long ans = 0;
    for (int j : accept) ans = (ans + res.a[start][j]) % mod;
    return ans;
}
```

**核心启示**：
- **DFA + 矩阵快速幂**是字符串计数的通用框架
- 数位 DP 也可以用类似思路

## 五、常见技巧与思路总结

### 5.1 确定状态向量

矩阵加速的关键是找到**最小的状态向量**。所有需要的量都要放入向量。

### 5.2 常数项处理

递推有常数项时，扩展一维（值恒为 1）。

### 5.3 矩阵大小优化

矩阵乘法是 $O(k^3)$，$k$ 小至关重要。有时可以通过数学化简减小 $k$。

### 5.4 取模注意

矩阵乘法中间结果可能溢出 `long long`，当模数较大时用 `__int128` 或拆分。

### 5.5 特殊矩阵加速

上三角/下三角矩阵、循环矩阵等有更快的乘法。

## 六、优化方法 / 解题思路框架

> 问题类型判断：
> ├── 常系数线性递推 → 构造转移矩阵 + 快速幂

> ├── DFA/AC 自动机 + 长度 $n$ → 转移矩阵快速幂

## 七、终极总结与归纳

### 7.1 矩阵的本质（一句话）

矩阵快速幂将**线性递推**的求解从 $O(n)$ 加速到 $O(k^3 \log n)$——任何可以表示为矩阵乘法的状态转移都能用它加速。

### 7.2 核心要点

1. **矩阵乘法满足结合律**，快速幂适用
2. **状态向量**要包含递推所需的所有量
3. **图路径计数**= 邻接矩阵幂
4. **自动机 + 矩阵**是字符串计数的通用框架
5. **矩阵求逆**用高斯消元
6. **特征多项式**可以进一步优化
