## 一、什么是筛法

筛法是利用数论性质**批量计算**积性函数值或素数的方法。从埃拉托斯特尼的朴素筛法到现代的杜教筛、Min_25 筛，筛法家族覆盖了从 $O(n)$ 到 $O(n^{2/3})$ 甚至更快的各种需求。

| 筛法 | 时间复杂度 | 空间 | 适用场景 |
|------|-----------|------|---------|
| 埃氏筛 | $O(n \log\log n)$ | $O(n)$ | 素数判定/简单函数 |
| 线性筛（欧拉筛） | $O(n)$ | $O(n)$ | 批量积性函数 |
| 杜教筛 | $O(n^{2/3})$ | $O(n^{2/3})$ | 积性函数前缀和 |
| Min_25 筛 | $O(n^{2/3}/\log n)$ | $O(\sqrt{n})$ | 更一般的积性函数求和 |

## 二、核心思想与前置知识

### 2.1 埃拉托斯特尼筛

从 2 开始，将每个素数的所有倍数标记为合数。$O(n \log\log n)$。

```cpp
const int MAXN = 10000006;
bool vis[MAXN];

void eratosthenes(int n) {
    for (int i = 2; i * i <= n; i++)
        if (!vis[i])
            for (int j = i * i; j <= n; j += i)
                vis[j] = true;
}
```

### 2.2 线性筛（欧拉筛）

每个合数只被其**最小素因子**筛去，保证 $O(n)$。同时可附带计算任意积性函数。

```cpp
int primes[MAXN], cnt;
int phi_val[MAXN], mu_val[MAXN];

void euler_sieve(int n) {
    phi_val[1] = mu_val[1] = 1;
    for (int i = 2; i <= n; i++) {
        if (!vis[i]) {
            primes[cnt++] = i;
            phi_val[i] = i - 1;
            mu_val[i] = -1;
        }
        for (int j = 0; j < cnt && (long long)i * primes[j] <= n; j++) {
            int p = primes[j];
            vis[i * p] = true;
            if (i % p == 0) {
                phi_val[i * p] = phi_val[i] * p;
                mu_val[i * p] = 0;
                break;
            }
            phi_val[i * p] = phi_val[i] * (p - 1);
            mu_val[i * p] = -mu_val[i];
        }
    }
}
```

### 2.3 杜教筛

利用狄利克雷卷积 $f * g = h$，推导出：

$$S_f(n) = S_h(n) - \sum_{d=2}^{n} g(d) \cdot S_f\left(\lfloor n/d \rfloor\right)$$

预筛 $n^{2/3}$ 范围内的值 + 记忆化 + 整除分块，总时间 $O(n^{2/3})$。

### 2.4 Min_25 筛

分两阶段：
1. **素数贡献**：递推求 $\sum_{p \le n, p \text{ prime}} f(p)$
2. **合数贡献**：枚举最小素因子及其幂次，递归求解

适用于 $f(p)$ 是关于 $p$ 的多项式的情况。

## 三、分类与变体详解

### 3.1 线性筛的积性函数转移

```
关键分支：i % p == 0 时
  - φ(i·p) = φ(i) · p
  - μ(i·p) = 0
  - σ(i·p) 需要记录最小素因子的幂次
关键分支：i % p != 0 时
  - f(i·p) = f(i) · f(p)（积性）
```

### 3.2 杜教筛常用卷积

| $f$ | $g$ | $h = f * g$ | $S_h(n)$ |
|-----|-----|------------|----------|
| $\varphi$ | $\mathbf{1}$ | $\text{id}$ | $n(n+1)/2$ |
| $\mu$ | $\mathbf{1}$ | $\varepsilon$ | $1$ |
| $\text{id} \cdot \varphi$ | $\text{id}$ | $\text{id}^2$ | $n(n+1)(2n+1)/6$ |

### 3.3 Min_25 筛框架

```
阶段 1：g(n, j) = ∑_{i≤n, i是素数或min_factor(i)>p_j} f_poly(i)
阶段 2：S(n, j) = ∑_{i≤n, min_factor(i)>p_j} f(i)
       = g(n, |P|) - 素数前缀和(p_j) + ∑_{p>p_j} ∑_{e≥1} f(p^e)·(S(n/p^e, idx(p))+[e>1])
```

## 四、15 道经典例题详解

### 🟢 入门级 Level 1：基础筛法

### 题目 1：线性筛素数（洛谷 P3383）

> 给定 $n$（$\le 10^8$）和 $q$ 次询问，每次询问第 $k$ 个素数。

**分析**：线性筛预处理所有素数，$O(1)$ 回答。

```
核心观察：线性筛 O(n) 遍历得到所有素数
构造策略：欧拉筛 + 素数数组
正确性：每个合数恰好被最小素因子筛去
```

**C++ 代码**：

```cpp
const int MAXN = 100000006;
int primes[6000006], cnt;
bool vis[MAXN];

void sieve(int n) {
    for (int i = 2; i <= n; i++) {
        if (!vis[i]) primes[++cnt] = i;
        for (int j = 1; j <= cnt && (long long)i * primes[j] <= n; j++) {
            vis[i * primes[j]] = true;
            if (i % primes[j] == 0) break;
        }
    }
}
// 查询：primes[k]
```

**核心启示**：
- **线性筛**是竞赛中最常用的素数预处理方法
- `if (i % primes[j] == 0) break;` 是保证线性的关键
- $10^8$ 范围约有 $5.76 \times 10^6$ 个素数

### 题目 2：A % B Problem（洛谷 P1865）

> 求区间 $[a, b]$ 中素数个数。

**分析**：线性筛 + 前缀和。$\pi(b) - \pi(a-1)$。

```
核心观察：素数计数 = 前缀和差分
构造策略：cnt[i] = [1,i] 中素数个数，cnt[b] - cnt[a-1]
正确性：前缀和差分
```

**C++ 代码**：

```cpp
int cnt_arr[MAXN]; // cnt_arr[i] = [1,i] 中素数个数

void preprocess(int n) {
    sieve(n);
    for (int i = 2; i <= n; i++)
        cnt_arr[i] = cnt_arr[i - 1] + (!vis[i] ? 1 : 0);
}
long long query(int a, int b) {
    return cnt_arr[b] - cnt_arr[a - 1];
}
```

**核心启示**：
- **前缀和**是区间统计的基础技巧
- 素数密度约 $1/\ln n$（素数定理）

### 题目 3：集合（洛谷 P1621）

> 给定 $[a, b]$ 和素数 $p$，求 $[a, b]$ 中有公共 $\ge p$ 的素因子的数对数。

**分析**：枚举 $\ge p$ 的素数 $q$，用并查集合并 $[a, b]$ 中 $q$ 的倍数。

```
核心观察：共享大素因子 → 同一并查集连通分量
构造策略：对每个 ≥ p 的素数 q，合并 [a,b] 中 q 的所有倍数
正确性：共享素因子 ⟺ 在某个素数的倍数集中
```

**C++ 代码**：

```cpp
int fa[MAXN];
int find(int x) { return fa[x] == x ? x : fa[x] = find(fa[x]); }

long long solve(int a, int b, int p) {
    for (int i = a; i <= b; i++) fa[i] = i;
    // 枚举 >= p 的素数 q
    for (int j = 0; j < cnt && primes[j] <= b; j++) {
        int q = primes[j];
        if (q < p) continue;
        int first = ((a + q - 1) / q) * q; // [a,b] 中 q 的最小倍数
        for (int x = first + q; x <= b; x += q)
            if (find(first) != find(x)) fa[find(x)] = find(first);
    }
    // 统计连通分量大小
    long long ans = 0;
    map<int, int> comp;
    for (int i = a; i <= b; i++) comp[find(i)]++;
    for (auto [_, sz] : comp) ans += (long long)sz * (sz - 1) / 2;
    return ans;
}
```

**核心启示**：
- **并查集**处理"共享属性"的分组问题
- 区间筛思想：只枚举范围内的倍数

### 🟡 进阶级 Level 2：积性函数线性筛

### 题目 4：线性筛求积性函数（AcWing 综合）

> 用线性筛同时求出 $\varphi(i)$、$\mu(i)$、$d(i)$（约数个数）、$\sigma(i)$（约数和）。

**分析**：积性函数在线性筛中通过最小素因子转移。$d$ 和 $\sigma$ 需要额外记录最小素因子的幂次。

```
核心观察：积性 → p∤i 时 f(ip) = f(i)·f(p)；p|i 时需特殊处理
构造策略：记录 low[i] = i 的最小素因子的最高幂次，用于 d 和 σ 的转移
正确性：积性函数的定义 + 最小素因子分解
```

**C++ 代码**：

```cpp
int d[MAXN], low[MAXN]; // d = 约数个数，low[i] = min_prime^max_exp
long long sigma[MAXN]; // σ = 约数和

void sieve_all(int n) {
    d[1] = 1; sigma[1] = 1; low[1] = 1;
    for (int i = 2; i <= n; i++) {
        if (!vis[i]) {
            primes[cnt++] = i;
            d[i] = 2; sigma[i] = 1 + i; low[i] = i;
            phi_val[i] = i - 1; mu_val[i] = -1;
        }
        for (int j = 0; j < cnt && (long long)i * primes[j] <= n; j++) {
            int p = primes[j];
            vis[i * p] = true;
            if (i % p == 0) {
                phi_val[i*p] = phi_val[i] * p;
                mu_val[i*p] = 0;
                low[i*p] = low[i] * p;
                d[i*p] = d[i] / d[low[i]] * d[low[i*p]]; // 更新约数个数
                // sigma 类似更新
                break;
            }
            phi_val[i*p] = phi_val[i] * (p-1);
            mu_val[i*p] = -mu_val[i];
            low[i*p] = p;
            d[i*p] = d[i] * 2;
            sigma[i*p] = sigma[i] * (1 + p);
        }
    }
}
```

**核心启示**：
- **low[] 数组**记录最小素因子的最高幂，是处理非完全积性函数的关键
- $d(p^e) = e + 1$，$\sigma(p^e) = 1 + p + \cdots + p^e$

### 题目 5：记忆（洛谷 P5438）

> 给定积性函数 $f$，$f(p) = p^2 - p$，求 $\sum_{i=1}^{n} f(i)$。

**分析**：$f(p) = p(p-1) = p \cdot \varphi(p)$。线性筛求 $f$：$p \nmid i$ 时 $f(ip) = f(i) \cdot f(p)$；$p \mid i$ 时需要根据 $p^e$ 处的值推导。

```
核心观察：f(p) = p²-p，f 是积性函数
构造策略：线性筛，f(p^e) = p^{2e} - p^{2e-1}，用 low 数组辅助
正确性：积性分解 + 逐素因子计算
```

**C++ 代码**：

```cpp
long long f[MAXN];
long long low_val[MAXN]; // low_val[i] = f(min_prime^max_exp_in_i)
long long low_pe[MAXN];  // low_pe[i] = min_prime^max_exp_in_i

void sieve_f(int n, long long mod) {
    f[1] = 1; low_val[1] = 1; low_pe[1] = 1;
    for (int i = 2; i <= n; i++) {
        if (!vis[i]) {
            primes[cnt++] = i;
            f[i] = ((long long)i * i - i) % mod;
            low_val[i] = f[i]; low_pe[i] = i;
        }
        for (int j = 0; j < cnt && (long long)i * primes[j] <= n; j++) {
            int p = primes[j];
            vis[i * p] = true;
            if (i % p == 0) {
                low_pe[i * p] = low_pe[i] * p;
                long long pe = low_pe[i * p];
                low_val[i * p] = (pe % mod * (pe % mod) - pe % mod * (pe / p % mod) + mod) % mod;
                f[i * p] = f[i] / low_val[i] * low_val[i * p] % mod; // 注意取模
                break;
            }
            low_pe[i * p] = p; low_val[i * p] = f[p];
            f[i * p] = f[i] * f[p] % mod;
        }
    }
}
```

**核心启示**：
- **自定义积性函数**的线性筛核心是确定 $f(p^e)$ 的递推
- low_pe 和 low_val 是"最小素因子幂次"的完整记录

### 题目 6：区间素数筛（经典）

> 求 $[L, R]$ 中的素数，$R$ 可达 $10^{12}$，$R - L \le 10^6$。

**分析**：先用线性筛求出 $\sqrt{R}$ 以内的素数，再用这些素数筛 $[L, R]$ 区间。

```
核心观察：合数 n ≤ R 必有 ≤ √R 的素因子
构造策略：预筛 √R 以内素数，标记 [L,R] 中每个素数的倍数
正确性：每个合数被标记
```

**C++ 代码**：

```cpp
bool is_prime_interval[1000006]; // is_prime_interval[i] 对应 L + i

void interval_sieve(long long L, long long R) {
    int sqr = (int)sqrt((double)R);
    sieve(sqr); // 预筛
    fill(is_prime_interval, is_prime_interval + (R - L + 1), true);
    if (L == 1) is_prime_interval[0] = false;
    for (int j = 0; j < cnt; j++) {
        long long p = primes[j];
        long long start = max(p * p, ((L + p - 1) / p) * p);
        for (long long x = start; x <= R; x += p)
            is_prime_interval[x - L] = false;
    }
}
```

**核心启示**：
- **区间筛**将 $O(R)$ 降为 $O(\sqrt{R} + (R-L) \log\log R)$
- 是处理大范围素数问题的标准技巧

### 🟠 提高级 Level 3：杜教筛

### 题目 7：杜教筛模板（洛谷 P4213）

> 求 $\sum_{i=1}^{n} \varphi(i)$ 和 $\sum_{i=1}^{n} \mu(i)$，$n \le 2 \times 10^{31}$。

**分析**：杜教筛。$\varphi * \mathbf{1} = \text{id}$，$\mu * \mathbf{1} = \varepsilon$。

```
核心观察：φ*1 = id → S_φ(n) = n(n+1)/2 - ∑_{d=2} S_φ(⌊n/d⌋)
构造策略：预筛 n^{2/3}，记忆化 + 整除分块
正确性：杜教筛的标准推导
```

**C++ 代码**：

```cpp
const int LIMIT = 5000000; // 预筛范围 ≈ n^{2/3}
long long pre_phi[LIMIT + 1], pre_mu[LIMIT + 1];
unordered_map<long long, long long> memo_phi, memo_mu;

long long S_phi(long long n) {
    if (n <= LIMIT) return pre_phi[n];
    if (memo_phi.count(n)) return memo_phi[n];
    long long res = n * (n + 1) / 2;
    for (long long l = 2, r; l <= n; l = r + 1) {
        r = n / (n / l);
        res -= (r - l + 1) * S_phi(n / l);
    }
    return memo_phi[n] = res;
}

long long S_mu(long long n) {
    if (n <= LIMIT) return pre_mu[n];
    if (memo_mu.count(n)) return memo_mu[n];
    long long res = 1; // ε 的前缀和
    for (long long l = 2, r; l <= n; l = r + 1) {
        r = n / (n / l);
        res -= (r - l + 1) * S_mu(n / l);
    }
    return memo_mu[n] = res;
}
```

**核心启示**：
- **杜教筛的核心**：找到 $g$ 使 $f * g = h$，$h$ 前缀和易求
- 预筛 $n^{2/3}$ 是最优平衡点（时间和空间）
- 整除分块将每层递归降为 $O(\sqrt{n})$

### 题目 8：简单的数学题（洛谷 P3768）

> 求 $\sum_{i=1}^{n} \sum_{j=1}^{n} ij \cdot \gcd(i,j)$，$n \le 10^{10}$。

**分析**：莫比乌斯反演化简后变为 $\sum_{d=1}^{n} d^2 \varphi(d) \cdot (\lfloor n/d \rfloor)^2$... 再整除分块，需要 $\sum d^2 \varphi(d)$ 的前缀和，用杜教筛。

```
核心观察：反演后需要 f(d) = d²·φ(d) 的前缀和
构造策略：f * id² = id³（因为 ∑_{d|n} d²φ(d) = n² · φ(n) / ... ），用杜教筛
正确性：狄利克雷卷积验证
```

**C++ 代码**：

```cpp
// h = f * g 其中 f(n) = n²φ(n), g(n) = n², h(n) = n³
// S_h(n) = (n(n+1)/2)²
// S_f(n) = S_h(n) - ∑_{d=2}^{n} d² · S_f(⌊n/d⌋)
long long S_f(long long n) {
    if (n <= LIMIT) return pre_f[n]; // 预筛
    if (memo_f.count(n)) return memo_f[n];
    long long s = n % MOD * ((n + 1) % MOD) % MOD * inv2 % MOD;
    s = s * s % MOD; // S_h(n) = (n(n+1)/2)²
    for (long long l = 2, r; l <= n; l = r + 1) {
        r = n / (n / l);
        // ∑_{d=l}^{r} d² = (∑1^r d²) - (∑1^{l-1} d²)
        long long sum_sq = (sum_of_squares(r) - sum_of_squares(l - 1) + MOD) % MOD;
        s = (s - sum_sq * S_f(n / l)) % MOD;
    }
    return memo_f[n] = (s + MOD) % MOD;
}
```

**核心启示**：
- **莫比乌斯反演 + 杜教筛**是处理 $\sum \gcd$ 类问题的标准终极方法
- $f(n) = n^2 \varphi(n)$ 不是标准积性函数，但仍可用杜教筛
- 需要平方和公式 $\sum d^2 = n(n+1)(2n+1)/6$

### 题目 9：素数计数 π(n)（经典 Meissel-Lehmer）

> 求 $n$ 以内的素数个数 $\pi(n)$，$n$ 可达 $10^{11}$。

**分析**：Meissel-Lehmer 方法或 Min_25 筛的副产品。将 $[1, n]$ 中的数按最小素因子分类递归计数。

```
核心观察：π(n) = #{p ≤ n : p 是素数}
构造策略：Min_25 筛第一阶段即可得到 π(n)
正确性：筛去所有合数后剩素数
```

**C++ 代码**：

```cpp
// Min_25 筛求 π(n) 的简化版
long long w[MAXSQRT * 2]; // 存储所有 ⌊n/k⌋ 的值
long long g[MAXSQRT * 2]; // g[idx] = #{2 ≤ i ≤ w[idx] : min_factor(i) > p_j 或 i 是素数}
int idx_small[MAXSQRT], idx_large[MAXSQRT];

long long prime_count(long long n) {
    int sqr = (int)sqrt((double)n);
    int tot = 0;
    for (long long l = 1, r; l <= n; l = r + 1) {
        r = n / (n / l);
        w[++tot] = n / l;
        g[tot] = w[tot] - 1; // 初始：2~w[tot] 中所有数
        if (w[tot] <= sqr) idx_small[w[tot]] = tot;
        else idx_large[n / w[tot]] = tot;
    }
    for (int j = 0; j < cnt && (long long)primes[j] * primes[j] <= n; j++) {
        int p = primes[j];
        for (int i = 1; i <= tot && (long long)p * p <= w[i]; i++) {
            long long v = w[i] / p;
            int k = (v <= sqr) ? idx_small[v] : idx_large[n / v];
            g[i] -= g[k] - j; // 减去 min_factor = p 的合数
        }
    }
    return g[1]; // π(n)
}
```

**核心启示**：
- **Min_25 筛的第一阶段**直接给出 $\pi(n)$
- 索引技巧：小值用 `idx_small`，大值用 `idx_large`
- $O(\sqrt{n})$ 个不同的 $\lfloor n/k \rfloor$ 值

### 🔴 困难级 Level 4：Min_25 筛

### 题目 10：Min_25 筛模板（洛谷 P5325）

> 求 $\sum_{i=1}^{n} f(i)$，其中 $f$ 是积性函数，$f(p^k) = p^k(p^k - 1)$（即 $f(p) = p(p-1) = p^2 - p$）。

**分析**：Min_25 筛的完整两阶段实现。

```
核心观察：f(p) = p² - p 是关于 p 的多项式 → Min_25 筛适用
构造策略：
  阶段1：分别维护 g1(n,j) = ∑p² 和 g2(n,j) = ∑p 的递推
  阶段2：递归求合数贡献 S(n, j)
正确性：Min_25 筛理论
```

**C++ 代码**：

```cpp
long long g1[MAXSQRT * 2], g2[MAXSQRT * 2]; // ∑p², ∑p

// 阶段 1：筛素数处的值
void phase1(long long n, int sqr) {
    // 初始化 g1[i] = ∑_{k=2}^{w[i]} k² = 平方和 - 1
    // g2[i] = ∑_{k=2}^{w[i]} k = 自然数和 - 1
    for (int j = 0; j < cnt && (long long)primes[j] * primes[j] <= n; j++) {
        long long p = primes[j];
        for (int i = 1; i <= tot && p * p <= w[i]; i++) {
            int k = get_idx(w[i] / p);
            g1[i] -= p * p % MOD * (g1[k] - sp1[j]) % MOD;
            g2[i] -= p % MOD * (g2[k] - sp2[j]) % MOD;
        }
    }
}

// 阶段 2：递归求 S(n, j) = ∑_{i: min_factor(i)>p_j} f(i)
long long S(long long n, int j) {
    if (n <= 1) return 0;
    int idx = get_idx(n);
    long long res = (g1[idx] - g2[idx] - (sp1_all[j] - sp2_all[j]) + 2 * MOD) % MOD;
    for (int k = j; k < cnt && (long long)primes[k] * primes[k] <= n; k++) {
        long long p = primes[k], pe = p;
        for (int e = 1; pe <= n; e++, pe *= p) {
            long long fpe = pe % MOD * ((pe - 1) % MOD) % MOD;
            if (e > 1) res = (res + fpe) % MOD;
            res = (res + fpe % MOD * S(n / pe, k + 1)) % MOD;
        }
    }
    return res;
}
```

**核心启示**：
- **Min_25 筛**是目前竞赛中求积性函数前缀和的最强工具
- 第一阶段用 $f(p)$ 的多项式分量分别递推
- 第二阶段递归枚举最小素因子和幂次

### 题目 12：约数和前缀和 ∑σ(i)

> 求 $\sum_{i=1}^{n} \sigma(i)$（$\sigma$ = 约数和），$n \le 10^{10}$。

**分析**：$\sigma = \text{id} * \mathbf{1}$，所以 $\sum_{i=1}^{n} \sigma(i) = \sum_{d=1}^{n} d \cdot \lfloor n/d \rfloor$，整除分块 $O(\sqrt{n})$。

```
核心观察：∑σ(i) = ∑_{d=1}^{n} d · ⌊n/d⌋（交换求和序）
构造策略：整除分块，每块 ∑d = 等差数列求和
正确性：约数和的定义展开
```

**C++ 代码**：

```cpp
long long sum_sigma(long long n, long long mod) {
    long long ans = 0;
    for (long long l = 1, r; l <= n; l = r + 1) {
        r = n / (n / l);
        long long sum_d = (l + r) % mod * ((r - l + 1) % mod) % mod * inv2 % mod;
        ans = (ans + sum_d * (n / l % mod)) % mod;
    }
    return ans;
}
```

**核心启示**：
- **交换求和序**是数论求和的第一步：$\sum_{i} \sum_{d \mid i} f(d) = \sum_{d} f(d) \cdot \lfloor n/d \rfloor$
- 整除分块 $O(\sqrt{n})$，无需杜教筛

### 🟣 挑战级 Level 5：高级筛法

### 题目 13：Powerful Number 筛（经典）

> 给定积性函数 $f$，$f(p) = p + 1$，求 $\sum_{i=1}^{n} f(i)$。

**分析**：PN 筛：将 $f$ 写成 $f = g * h$，其中 $g$ 是"简单"函数（前缀和易求），$h$ 只在 Powerful Number（所有素因子指数 $\ge 2$）处非零。Powerful Number 只有 $O(\sqrt{n})$ 个。

```
核心观察：f = g * h，h 只在 PN 处有值，PN 个数 = O(√n)
构造策略：g(p) = f(p) 使 h(p) = 0；枚举 PN，对每个 PN 用杜教筛算 g 的前缀和
正确性：DirichLet 卷积分解
```

**C++ 代码**：

```cpp
// PN 筛骨架
long long solve(long long n) {
    // 1. 选 g 使 g(p) = f(p) = p+1 → g = σ（约数和）或自定义
    // 2. 预计算 h(p^e) for small p
    // 3. DFS 枚举 PN = ∏p_i^{e_i}, e_i ≥ 2
    long long ans = 0;
    function<void(int, long long, long long)> dfs = [&](int idx, long long cur, long long h_val) {
        // cur = 当前 PN，h_val = h(cur)
        ans += h_val * S_g(n / cur); // S_g 用杜教筛
        for (int j = idx; j < cnt && (long long)primes[j] * primes[j] <= n / cur; j++) {
            long long p = primes[j], pe = p * p;
            for (int e = 2; pe <= n / cur; e++, pe *= p) {
                long long h_pe = compute_h(p, e);
                if (h_pe) dfs(j + 1, cur * pe, h_val * h_pe);
            }
        }
    };
    dfs(0, 1, 1);
    return ans;
}
```

**核心启示**：
- **PN 筛**利用 Powerful Number 的稀疏性，将复杂函数分解为简单函数 + 稀疏修正
- 适用于 $f(p)$ 简单但 $f(p^e)$ 复杂的函数

### 题目 15：自定义积性函数前缀和（综合应用）

> 给定 $f(p^k)$ 的递推式，求 $\sum_{i=1}^{n} f(i)$，$n \le 10^{10}$。


## 五、常见技巧与思路总结

### 5.1 线性筛的 p|i 分支

$p \mid i$ 时 $i \cdot p$ 的最小素因子次数 +1，涉及 $d, \sigma$ 等非完全积性函数时需要 low 数组。

### 5.2 杜教筛的卷积选择

选 $g$ 使得：(1) $h = f * g$ 的前缀和好算；(2) $g$ 的单点值好算（通常是 $\mathbf{1}$ 或 $\text{id}$）。

### 5.3 预筛范围

杜教筛预筛到 $n^{2/3}$ 是最优。预筛范围越大，递归次数越少，但空间越大。

### 5.4 Min_25 的索引技巧

$\lfloor n/k \rfloor$ 只有 $O(\sqrt{n})$ 个不同值。用 `idx_small[]` 和 `idx_large[]` 分别索引。

### 5.5 整除分块

$\sum_{d=1}^{n} f(d) \cdot g(\lfloor n/d \rfloor)$ 可以整除分块，每块内 $g$ 值相同。

## 六、优化方法 / 解题思路框架

```
问题类型判断：
  │
  ├── n ≤ 10^7，求所有 f(i) → 线性筛
  ├── n ≤ 10^{10}，求 ∑f(i)
  │     ├── f·1 = 已知 → 杜教筛
  │     ├── f(p) 是多项式 → Min_25 筛
  │     └── f = g*h，g 简单 → PN 筛
  ├── 区间 [L,R] 素数 → 区间筛
  └── ∑f(d)·g(⌊n/d⌋) → 整除分块
```

| 信号词 | 对应方法 |
|--------|---------|
| "批量求 f(i), n ≤ 10^7" | 线性筛 |
| "∑φ 或 ∑μ, n > 10^7" | 杜教筛 |
| "∑f(i), f(p)为多项式" | Min_25 筛 |
| "区间素数" | 区间筛 |
| "∑f(d)·g(n/d)" | 整除分块 |

## 七、终极总结与归纳

### 7.1 筛法的本质（一句话）

筛法利用积性函数的**乘法结构**，通过**最小素因子分类**实现高效的批量计算。

### 7.2 核心要点

1. **线性筛** $O(n)$：每个合数被最小素因子筛去，可附带任意积性函数
2. **杜教筛** $O(n^{2/3})$：利用 $f * g = h$ 的卷积关系递推前缀和
3. **Min_25 筛**：两阶段——素数贡献递推 + 合数递归枚举
4. **PN 筛**：将 $f$ 分解为简单函数 + 稀疏修正
5. **预筛 $n^{2/3}$** 是杜教筛的最优平衡点
6. **整除分块** $O(\sqrt{n})$ 是所有亚线性筛法的基础
