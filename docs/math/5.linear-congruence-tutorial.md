## 一、什么是线性同余方程

线性同余方程是指形如 $ax \equiv b \pmod{m}$ 的方程，已知 $a, b, m$，求 $x$。它是数论中最基础的方程形式，是乘法逆元、中国剩余定理等高级工具的前置知识。

**裴蜀定理**：$ax + by = c$ 有整数解当且仅当 $\gcd(a, b) \mid c$。这是线性同余方程有解的理论基础。

| 概念 | 定义 | 求解方法 |
|------|------|---------|
| 线性同余方程 | $ax \equiv b \pmod{m}$ | exgcd |
| 乘法逆元 | $ax \equiv 1 \pmod{m}$ | exgcd / 费马 / 线性递推 |
| 裴蜀定理 | $ax + by = \gcd(a,b)$ | exgcd |
| 不定方程 | $ax + by = c$ | 特解 + 通解 |

## 二、核心思想与前置知识

### 2.1 扩展欧几里得算法

exgcd 在求 $\gcd(a, b)$ 的同时，求出 $ax + by = \gcd(a, b)$ 的一组特解 $(x_0, y_0)$。

```cpp
long long exgcd(long long a, long long b, long long &x, long long &y) {
    if (b == 0) { x = 1; y = 0; return a; }
    long long d = exgcd(b, a % b, y, x);
    y -= (a / b) * x;
    return d;
}
```

### 2.2 求解 ax ≡ b (mod m)

$ax \equiv b \pmod{m}$ 等价于 $ax + my = b$。令 $d = \gcd(a, m)$：
- 若 $d \nmid b$，无解
- 否则两边除以 $d$：$\frac{a}{d} x \equiv \frac{b}{d} \pmod{\frac{m}{d}}$
- 用 exgcd 求特解，通解为 $x = x_0 + k \cdot \frac{m}{d}$

### 2.3 乘法逆元

当 $b = 1$ 时，$ax \equiv 1 \pmod{m}$，$x$ 即为 $a$ 的乘法逆元。

**三种求法**：

1. **exgcd**：适用于任意 $\gcd(a, m) = 1$
2. **费马小定理**：$a^{-1} \equiv a^{p-2} \pmod{p}$（$p$ 为素数）
3. **线性递推**：$O(n)$ 求 $1 \sim n$ 所有逆元
   ```cpp
   inv[1] = 1;
   for (int i = 2; i <= n; i++)
       inv[i] = (long long)(p - p / i) * inv[p % i] % p;
   ```

### 2.4 裴蜀定理

对任意整数 $a, b$，存在整数 $x, y$ 使得 $ax + by = \gcd(a, b)$。推论：$ax + by = c$ 有解 $\Leftrightarrow$ $\gcd(a, b) \mid c$。

## 三、分类与变体详解

### 3.1 裴蜀定理（不定方程）

```
方程形式：ax + by = c
有解条件：gcd(a, b) | c
求解方法：exgcd 求 ax + by = gcd(a,b) 的特解，再乘 c/gcd
通解：x = x0 + k·(b/gcd), y = y0 - k·(a/gcd)
```

### 3.2 单个逆元

```
方程形式：ax ≡ 1 (mod m)
前提条件：gcd(a, m) = 1
方法 1：exgcd → O(log m)
方法 2：费马小定理 a^{m-2} mod m（m 为素数）→ O(log m)
方法 3：欧拉定理 a^{φ(m)-1} mod m → O(√m + log m)
```

### 3.3 批量逆元（线性递推）

```
目标：求 1~n 在模 p 下的逆元
核心公式：inv[i] = -(p/i) · inv[p%i] mod p
时间复杂度：O(n)
前提条件：p 为素数
```

### 3.4 任意数组批量逆元

```
目标：给定 a[1..n]，求每个 a[i] 的逆元
核心技巧：前缀积 + 一次快速幂 + 后缀还原
时间复杂度：O(n + log p)
关键公式：pre[i] = a[1]·a[2]·...·a[i]，inv(pre[n])，逆推
```

### 3.5 变体对比

| 变体 | 输入 | 方法 | 时间 |
|------|------|------|------|
| 单个逆元 | $a, m$ | exgcd / 快速幂 | $O(\log m)$ |
| $1 \sim n$ 逆元 | $n, p$ | 线性递推 | $O(n)$ |
| 任意数组逆元 | $a[1..n], p$ | 前缀积 | $O(n + \log p)$ |
| 不定方程 | $ax + by = c$ | exgcd | $O(\log \min(a,b))$ |

### 🟢 入门级 Level 1：基础同余方程

### 题目 1：裴蜀定理模板（洛谷 P4549）

> 给定 $n$ 个整数 $a_1, \ldots, a_n$，求 $\sum a_i x_i$ 能表示的最小正整数。

**分析**：由裴蜀定理，$a_1 x_1 + a_2 x_2 + \cdots + a_n x_n$ 能表示的最小正整数为 $\gcd(a_1, a_2, \ldots, a_n)$。

```
核心观察：多个数的线性组合的最小正值 = 它们的 GCD
构造策略：逐个求 gcd(a1, a2, ..., an)
正确性：裴蜀定理的推广
```

**C++ 代码**：

```cpp
long long solve(long long a[], int n) {
    long long g = 0;
    for (int i = 0; i < n; i++) g = __gcd(g, abs(a[i]));
    return g;
}
```

**核心启示**：
- **裴蜀定理**将线性组合问题转化为 GCD
- 多个数的 GCD 可以逐个累计

### 题目 2：同余方程（洛谷 P1082）

> 求 $ax \equiv 1 \pmod{b}$ 的最小正整数解。

**分析**：即求 $a$ 在模 $b$ 下的逆元，用 exgcd 直接求解。

```
核心观察：ax ≡ 1 (mod b) 等价于 ax + by = 1
构造策略：exgcd 求特解 x0，取 x = ((x0 % b) + b) % b
正确性：gcd(a, b) = 1 时有解（题目保证）
```

**C++ 代码**：

```cpp
long long solve(long long a, long long b) {
    long long x, y;
    exgcd(a, b, x, y);
    return ((x % b) + b) % b;
}
```

**核心启示**：
- exgcd 求出的 $x$ 可能为负，需要调整到 $[0, m)$ 范围
- 这是乘法逆元的最直接求法

### 题目 3：有理数取余（洛谷 P2613）

> 给定 $a/b$，求 $a \cdot b^{-1} \bmod p$（$p$ 为素数）。

**分析**：费马小定理求 $b$ 的逆元，然后乘以 $a$。

```
核心观察：p 为素数 → b^{p-2} 即为 b 的逆元
构造策略：快速幂求 b^{p-2} mod p，然后 a * inv(b) mod p
正确性：费马小定理保证 b·b^{p-2} ≡ 1 (mod p)
```

**C++ 代码**：

```cpp
long long solve(long long a, long long b, long long p) {
    if (b % p == 0) return -1; // 逆元不存在（题目可能保证不会）
    return a % p * qpow(b % p, p - 2, p) % p;
}
```

**核心启示**：
- 费马小定理是**模素数**下求逆元的最简单方法
- 注意特判 $b \equiv 0 \pmod{p}$

### 🟡 进阶级 Level 2：逆元求解

### 题目 4：乘法逆元（洛谷 P3811）

> 给定 $n, p$（$p$ 为素数），求 $1 \sim n$ 每个数在模 $p$ 下的逆元。

**分析**：线性递推法。$p = ki + r$（$k = \lfloor p/i \rfloor$, $r = p \bmod i$），则 $ki + r \equiv 0$，故 $i^{-1} \equiv -k \cdot r^{-1}$。

```
核心观察：i 的逆元可以用 p%i 的逆元递推
构造策略：inv[i] = -(p/i) * inv[p%i] mod p
正确性：由 p = (p/i)*i + (p%i) 推导
```

**C++ 代码**：

```cpp
const int MAXN = 3000006;
long long inv[MAXN];

void solve(int n, long long p) {
    inv[1] = 1;
    for (int i = 2; i <= n; i++)
        inv[i] = (p - p / i) * inv[p % i] % p;
}
```

**核心启示**：
- **线性递推**是批量求逆元的 $O(n)$ 方法
- `(p - p/i)` 等价于 `-(p/i) mod p`，避免负数

### 题目 5：乘法逆元 2（洛谷 P5431）

> 给定 $n$ 个数 $a_1, \ldots, a_n$ 和素数 $p$，求 $\sum_{i=1}^{n} \frac{k^i}{a_i} \bmod p$。

**分析**：需要快速求每个 $a_i$ 的逆元。用**前缀积**技巧：一次快速幂即可求出所有逆元。

```
核心观察：n 个逆元用前缀积只需 1 次快速幂
构造策略：pre[i] = a1·a2·...·ai，求 inv(pre[n])，逆推得每个 inv(a_i)
正确性：inv(a_i) = inv(pre[i]) · pre[i-1]
```

**C++ 代码**：

```cpp
long long pre[MAXN], suf_inv[MAXN];

long long solve(long long a[], int n, long long k, long long p) {
    pre[0] = 1;
    for (int i = 1; i <= n; i++) pre[i] = pre[i-1] * a[i] % p;
    suf_inv[n] = qpow(pre[n], p - 2, p);
    for (int i = n; i >= 1; i--) suf_inv[i-1] = suf_inv[i] * a[i] % p;
    // inv(a[i]) = suf_inv[i] * pre[i-1]
    long long ans = 0, kp = k;
    for (int i = 1; i <= n; i++) {
        long long inv_ai = suf_inv[i] * pre[i-1] % p;
        ans = (ans + kp * inv_ai) % p;
        kp = kp * k % p;
    }
    return ans;
}
```

**核心启示**：
- **前缀积逆元**将 $n$ 次快速幂优化为 1 次，$O(n + \log p)$
- 这是竞赛中极其常用的技巧

### 题目 6：小凯的疑惑（洛谷 P3951）

> 给定互素的正整数 $a, b$，求不能被 $ax + by$（$x, y \ge 0$）表示的最大整数。

**分析**：经典的 Chicken McNugget 定理（Frobenius 数）：$\gcd(a,b) = 1$ 时，不能表示的最大整数为 $ab - a - b$。

```
核心观察：Frobenius 数 = ab - a - b（两个互素数的情况）
构造策略：直接公式计算
正确性：由 gcd(a,b) = 1 的性质推导
```

**C++ 代码**：

```cpp
long long solve(long long a, long long b) {
    return a * b - a - b;
}
```

**核心启示**：
- **Frobenius 数**是裴蜀定理的经典延伸
- 仅对两个互素数有简洁公式，三个以上是 NP-hard

### 🟠 提高级 Level 3：综合应用

### 题目 7：青蛙的约会（洛谷 P1516）

> 两只青蛙在环形跑道上，起点分别为 $x, y$，每步分别跳 $m, n$ 格，跑道总长 $L$。求最少多少步能相遇。

**分析**：相遇即 $(m-n) \cdot t \equiv (y-x) \pmod{L}$，用 exgcd 求解。

```
核心观察：环形追及问题 → 线性同余方程
构造策略：(m-n)·t + L·k = y-x，exgcd 求最小非负 t
正确性：exgcd 求特解后取模得最小正解
```

**C++ 代码**：

```cpp
long long solve(long long x, long long y, long long m, long long n, long long L) {
    long long a = m - n, b = y - x;
    // 解 a·t ≡ b (mod L)
    long long tx, ty;
    long long d = exgcd(a, L, tx, ty);
    if (b % d) return -1;
    long long mod = L / d;
    long long t = ((__int128)tx * (b / d) % mod + mod) % mod;
    return t;
}
```

**核心启示**：
- **环形追及**是线性同余方程的经典几何应用
- `__int128` 防止乘法溢出

### 题目 8：阶乘逆元（AcWing 890）

> 预处理 $n!$ 和 $(n!)^{-1} \bmod p$，从而 $O(1)$ 求组合数 $\binom{n}{k}$。

**分析**：$\binom{n}{k} = n! \cdot (k!)^{-1} \cdot ((n-k)!)^{-1}$。预处理阶乘和阶乘逆元即可。

```
核心观察：组合数 = 阶乘 × 阶乘逆元
构造策略：正向求 fac[]，逆向递推 inv_fac[]
正确性：inv_fac[i] = inv_fac[i+1] * (i+1) mod p
```

**C++ 代码**：

```cpp
long long fac[MAXN], inv_fac[MAXN];

void init(int n, long long p) {
    fac[0] = 1;
    for (int i = 1; i <= n; i++) fac[i] = fac[i-1] * i % p;
    inv_fac[n] = qpow(fac[n], p - 2, p);
    for (int i = n - 1; i >= 0; i--) inv_fac[i] = inv_fac[i+1] * (i+1) % p;
}
long long C(int n, int k, long long p) {
    if (k < 0 || k > n) return 0;
    return fac[n] * inv_fac[k] % p * inv_fac[n-k] % p;
}
```

**核心启示**：
- 阶乘逆元的**逆序递推**只需 1 次快速幂
- 这是组合数计算的标准预处理

### 题目 9：线性同余方程组（CF 687B）

> 给定多个方程 $x \equiv r_i \pmod{m_i}$，求最小正整数解。

**分析**：逐个合并方程。合并 $x \equiv r_1 \pmod{m_1}$ 和 $x \equiv r_2 \pmod{m_2}$：$m_1 k_1 + r_1 = m_2 k_2 + r_2$，用 exgcd 求解。

```
核心观察：两个同余方程合并为一个
构造策略：exgcd 解 m1·k = r2-r1 (mod m2)，新模数 = lcm(m1, m2)
正确性：逐个合并保持等价性
```

**C++ 代码**：

```cpp
pair<long long, long long> merge(long long r1, long long m1, long long r2, long long m2) {
    long long x, y;
    long long d = exgcd(m1, m2, x, y);
    if ((r2 - r1) % d) return {-1, -1}; // 无解
    long long mod = m1 / d * m2; // lcm(m1, m2)
    long long t = (__int128)(r2 - r1) / d % (m2 / d) * x % (m2 / d);
    long long r = ((r1 + m1 * t) % mod + mod) % mod;
    return {r, mod};
}
```

**核心启示**：
- **逐个合并**是解同余方程组不需要模数互素的通用方法
- 即 exCRT 的核心

### 🔴 困难级 Level 4：高级技巧

### 题目 10：Wilson 定理应用（CF 1514C）

> 给定素数 $p$，求满足 $n! \bmod p = n$ 的所有 $n$。

**分析**：Wilson 定理：$(p-1)! \equiv -1 \pmod{p}$。当 $n \ge p$ 时 $n! \equiv 0$，只需检查 $n < p$。

```
核心观察：Wilson 定理 (p-1)! ≡ -1 (mod p)
构造策略：从 (p-1)! = -1 出发逆推：(p-2)! = (-1)·inv(p-1) = 1,...
正确性：n! mod p 可以从 (p-1)! 逆推得到
```

**C++ 代码**：

```cpp
vector<long long> solve(long long p) {
    vector<long long> res;
    if (p == 2) return {1}; // 特判
    long long cur = p - 1; // (p-1)! mod p = p-1
    for (long long n = p - 1; n >= 1; n--) {
        if (cur == n % p) res.push_back(n);
        // (n-1)! = n! / n = cur * inv(n)
        cur = cur * qpow(n, p - 2, p) % p;
    }
    res.push_back(p); // p! mod p = 0, n mod p = 0
    sort(res.begin(), res.end());
    return res;
}
```

**核心启示**：
- **Wilson 定理**是连接阶乘与素数的桥梁
- 从 $(p-1)!$ 逆推比正向计算更高效

### 题目 11：逆元与前缀和（AcWing 综合）

> 给定数组 $a[1..n]$，在线回答多组查询：区间乘积的逆元 $\bmod p$。

**分析**：维护前缀乘积和前缀乘积逆元，区间 $[l, r]$ 的乘积逆元 = $\text{inv\_pre}[r] \cdot \text{pre}[l-1]$... 但更简单地：区间乘积 = $\text{pre}[r] \cdot \text{inv\_pre}[l-1]$，其逆元 = $\text{pre}[l-1] \cdot \text{inv\_pre}[r]$... 不对，直接 $\text{inv\_pre}[r] / \text{inv\_pre}[l-1]$ 不太对。正确做法：区间乘积 $= \text{pre}[r] \cdot \text{inv\_pre}[l-1]$，其逆元 $= \text{pre}[l-1]^{-1} \cdot \text{pre}[r]^{-1} \cdot \text{pre}[r] \cdot \text{pre}[l-1]$... 更简洁地：

```
核心观察：inv(∏a[l..r]) = inv_pre[r] * pre[l-1]，其中 inv_pre[i] = inv(pre[i])
构造策略：pre[i] = a1·...·ai，inv_pre[i] = inv(pre[i])
正确性：∏a[l..r] = pre[r] · inv(pre[l-1])，其逆 = pre[l-1] · inv_pre[r]
```

**C++ 代码**：

```cpp
long long pre[MAXN], inv_pre[MAXN];

void init(long long a[], int n, long long p) {
    pre[0] = 1;
    for (int i = 1; i <= n; i++) pre[i] = pre[i-1] * a[i] % p;
    inv_pre[n] = qpow(pre[n], p - 2, p);
    for (int i = n; i >= 1; i--) inv_pre[i-1] = inv_pre[i] * a[i] % p;
}
// 区间 [l,r] 乘积逆元
long long query(int l, int r) {
    return pre[l-1] * inv_pre[r] % p;
}
```

**核心启示**：
- **前缀积 + 逆前缀积**支持 $O(1)$ 区间乘积及其逆元查询
- 与前缀和思想完全类似

### 题目 12：模方程最小正解（POJ 2115）

> 给定 $a, b, c, k$，求最小非负整数 $x$ 使得 $(a + c \cdot x) \bmod 2^k = b$。

**分析**：即 $cx \equiv (b - a) \pmod{2^k}$，用 exgcd 求解。

```
核心观察：C 语言无符号整数溢出模型 → 模 2^k
构造策略：exgcd 解 c·x ≡ b-a (mod 2^k)
正确性：exgcd 通用方法
```

**C++ 代码**：

```cpp
long long solve(long long a, long long b, long long c, int k) {
    long long m = 1LL << k;
    long long target = ((b - a) % m + m) % m;
    long long x, y;
    long long d = exgcd(c, m, x, y);
    if (target % d) return -1;
    long long mod = m / d;
    x = (__int128)x * (target / d) % mod;
    return (x % mod + mod) % mod;
}
```

**核心启示**：
- **C 语言整数溢出**模型等价于模 $2^k$ 运算
- exgcd 求解后注意取模到最小非负解

### 题目 14：同余方程组进阶（洛谷 P4774 简化）

> 多个方程 $a_i x \equiv b_i \pmod{m_i}$（$m_i$ 不互素），求最小正整数解。

**分析**：每个方程先化为 $x \equiv c_i \pmod{M_i}$ 形式，然后用 exCRT 逐个合并。

```
核心观察：先独立求解再合并，比直接联立更灵活
构造策略：exgcd 求单方程 → exCRT 合并
正确性：逐步合并保持等价性
```

**C++ 代码**：

```cpp
long long solve(long long a[], long long b[], long long m[], int n) {
    long long R = 0, M = 1;
    for (int i = 0; i < n; i++) {
        // a[i]·x ≡ b[i] (mod m[i]) → x ≡ c[i] (mod M[i])
        long long d = __gcd(a[i], m[i]);
        if (b[i] % d) return -1;
        long long ai = a[i] / d, bi = b[i] / d, mi = m[i] / d;
        long long xi, yi;
        exgcd(ai, mi, xi, yi);
        long long ci = (__int128)xi * bi % mi;
        ci = (ci + mi) % mi;
        // 合并 x ≡ R (mod M) 和 x ≡ ci (mod mi)
        auto [nr, nm] = merge(R, M, ci, mi);
        if (nr == -1) return -1;
        R = nr; M = nm;
    }
    return R == 0 ? M : R;
}
```

**核心启示**：
- **线性同余方程 → 标准同余 → exCRT 合并**是通用流程
- 每步都要处理 GCD 和无解判断

### 题目 15：逆元在多项式中的应用（CF 综合）

> 给定多项式 $f(x) = \sum a_i x^i$（系数模素数 $p$），求其逆多项式 $g(x)$ 使得 $f \cdot g \equiv 1 \pmod{x^n}$。

**分析**：多项式求逆的核心是**倍增 + 逆元**。$g_{2n}(x) = g_n(x) \cdot (2 - f(x) \cdot g_n(x))$。

```
核心观察：多项式求逆 = 数值逆元的推广
构造策略：Newton 迭代法：g ← g·(2 - f·g)，每次精度翻倍
正确性：Newton 迭代在形式幂级数上的收敛性
```

**C++ 代码**：

```cpp
// poly_inv: 求 f 的逆多项式 mod x^n
void poly_inv(long long f[], long long g[], int n, long long p) {
    if (n == 1) { g[0] = qpow(f[0], p - 2, p); return; }
    poly_inv(f, g, (n + 1) / 2, p);
    // g = g * (2 - f * g)  mod x^n
    // 需要 NTT 辅助乘法
    static long long tmp[MAXN];
    int len = 1;
    while (len < 2 * n) len <<= 1;
    copy(f, f + n, tmp); fill(tmp + n, tmp + len, 0);
    ntt(tmp, len, 1); ntt(g, len, 1);
    for (int i = 0; i < len; i++)
        g[i] = (2 - tmp[i] * g[i] % p + p) % p * g[i] % p;
    ntt(g, len, -1);
    fill(g + n, g + len, 0);
}
```

**核心启示**：
- **多项式求逆**是 NTT 全家桶的基础模块
- Newton 迭代思想贯穿多项式算法

## 五、常见技巧与思路总结

### 5.1 exgcd 的符号处理

exgcd 返回的 $x$ 可能为负，需要 `((x % m) + m) % m` 调整。

### 5.2 防溢出

$a \cdot b$ 可能溢出 `long long`，使用 `__int128` 或龟速乘。

### 5.3 逆元不存在的判断

$a^{-1} \bmod m$ 存在当且仅当 $\gcd(a, m) = 1$。在模合数时需特别注意。

### 5.4 线性递推逆元的适用条件

$\text{inv}[i] = -(p/i) \cdot \text{inv}[p \bmod i]$ 仅在 $p$ 为素数时成立。

### 5.5 前缀积的零元素处理

若数组中有 $a_i \equiv 0 \pmod{p}$，前缀积法需要特殊处理（跳过零元素或分段）。

## 六、优化方法 / 解题思路框架

```
问题类型判断：
  │
  ├── ax ≡ b (mod m) → exgcd
  ├── 求 a^{-1} mod p（p 素数）→ 费马小定理 / 快速幂
  ├── 批量 1~n 逆元 → 线性递推 O(n)
  ├── 批量任意数组逆元 → 前缀积 O(n + log p)
  ├── ax + by = c → exgcd + 特解
  ├── 同余方程组 → exCRT（逐步合并）
  └── 多项式逆元 → Newton 迭代 + NTT
```

## 七、终极总结与归纳

### 7.1 线性同余方程的本质（一句话）

线性同余方程的核心是 **exgcd**——它将模运算下的除法转化为整数域上的不定方程求解。

### 7.2 核心要点

1. **exgcd** 是一切的基础：求 $ax + by = \gcd(a, b)$ 的特解
2. **裴蜀定理**：$ax + by = c$ 有解 $\Leftrightarrow$ $\gcd(a,b) \mid c$
3. **三种逆元求法**：exgcd（通用）、费马小定理（$p$ 素数）、线性递推（批量）
4. **前缀积技巧**：$n$ 个逆元只需 1 次快速幂
5. **同余方程组**用 exCRT 逐步合并
6. **溢出处理**：`__int128` 是必备工具
