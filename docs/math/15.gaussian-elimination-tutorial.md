## 一、什么是高斯消元

高斯消元法是求解线性方程组的经典算法。通过**初等行变换**将增广矩阵化为行阶梯形或行最简形，从而求出方程组的解。

在算法竞赛中，高斯消元不仅用于求解方程组，还广泛应用于行列式计算、Matrix-Tree 定理、概率 DP 中的方程求解等场景。

| 变体 | 数域 | 时间复杂度 | 应用场景 |
|------|------|-----------|---------|
| 实数消元 | $\mathbb{R}$ | $O(n^3)$ | 一般线性方程组 |
| 模素数消元 | $\mathbb{F}_p$ | $O(n^3)$ | 行列式、组合计数 |
| 异或消元 | $\mathbb{F}_2$ | $O(n^3/64)$ | 开关问题、线性基 |
| 带状消元 | — | $O(n \cdot w^2)$ | 特殊结构矩阵 |

## 二、核心思想与前置知识

### 2.1 增广矩阵与初等行变换

将线性方程组 $Ax = b$ 写成增广矩阵 $[A | b]$，三种初等行变换不改变解集：
1. 交换两行
2. 某行乘以非零常数
3. 某行加上另一行的倍数

### 2.2 实数高斯消元骨架

```cpp
const int MAXN = 105;
double a[MAXN][MAXN]; // 增广矩阵 a[i][0..n]
int n;

bool gauss() {
    for (int col = 0; col < n; col++) {
        int pivot = col;
        for (int i = col + 1; i < n; i++)
            if (fabs(a[i][col]) > fabs(a[pivot][col])) pivot = i;
        swap(a[col], a[pivot]);
        if (fabs(a[col][col]) < 1e-9) return false; // 无唯一解
        for (int i = col + 1; i < n; i++) {
            double f = a[i][col] / a[col][col];
            for (int j = col; j <= n; j++) a[i][j] -= f * a[col][j];
        }
    }
    // 回代
    for (int i = n - 1; i >= 0; i--) {
        for (int j = i + 1; j < n; j++) a[i][n] -= a[i][j] * a[j][n];
        a[i][n] /= a[i][i];
    }
    return true;
}
```

### 2.3 模意义高斯消元

将除法改为乘逆元，避免浮点误差：

```cpp
long long a[MAXN][MAXN], p;

void gauss_mod(int n) {
    for (int col = 0; col < n; col++) {
        int pivot = -1;
        for (int i = col; i < n; i++)
            if (a[i][col] % p != 0) { pivot = i; break; }
        if (pivot == -1) continue;
        swap(a[col], a[pivot]);
        long long inv = qpow(a[col][col], p - 2, p);
        for (int j = col; j <= n; j++) a[col][j] = a[col][j] * inv % p;
        for (int i = 0; i < n; i++) {
            if (i == col || a[i][col] == 0) continue;
            long long f = a[i][col];
            for (int j = col; j <= n; j++)
                a[i][j] = (a[i][j] - f * a[col][j] % p + p) % p;
        }
    }
}
```

### 2.4 异或方程组（bitset 优化）

当方程在 $\mathbb{F}_2$ 上时（加法 = 异或），用 bitset 加速到 $O(n^3/64)$：

```cpp
bitset<MAXN> a[MAXN]; // a[i][n] 为常数项

void gauss_xor(int n) {
    for (int col = 0; col < n; col++) {
        int pivot = -1;
        for (int i = col; i < n; i++)
            if (a[i][col]) { pivot = i; break; }
        if (pivot == -1) continue;
        swap(a[col], a[pivot]);
        for (int i = 0; i < n; i++)
            if (i != col && a[i][col]) a[i] ^= a[col];
    }
}
```

## 三、分类与变体详解

### 3.1 实数高斯消元

```
应用：物理模型、几何计算
关键：选主元（部分选主元避免数值不稳定）
精度：eps ≈ 1e-9
```

### 3.2 模素数高斯消元

```
应用：组合计数、行列式
关键：除法用逆元代替
优势：精确计算，无浮点误差
```

### 3.3 异或方程组

```
应用：开关问题、线性基
关键：bitset 加速到 O(n³/64)
特点：只有 0/1 两个值，加法 = 异或
```

### 3.4 行列式计算

```
方法：消元变为上三角，对角线乘积即行列式
注意：交换行时行列式变号
应用：Matrix-Tree 定理（生成树计数）
```

### 3.5 自由变量与解空间

```
自由变量：没有主元的列对应的变量
解集维度：自由变量个数
无解判断：出现 0 = 非零 的矛盾行
```

### 🟢 入门级 Level 1：基础消元

### 题目 1：高斯消元模板（洛谷 P3389）

> 给定 $n$ 元线性方程组，求唯一解或判断无解/无穷解。

**分析**：实数高斯消元的直接应用。

```
核心观察：n 元方程组 → n×(n+1) 增广矩阵
构造策略：部分选主元 + 前消 + 回代
正确性：初等行变换不改变解集
```

**C++ 代码**：

```cpp
const int MAXN = 105;
double a[MAXN][MAXN];

int gauss(int n) { // 返回 0 唯一解, 1 无穷解, 2 无解
    for (int col = 0; col < n; col++) {
        int pivot = col;
        for (int i = col + 1; i < n; i++)
            if (fabs(a[i][col]) > fabs(a[pivot][col])) pivot = i;
        swap(a[col], a[pivot]);
        if (fabs(a[col][col]) < 1e-9) return 1;
        for (int i = col + 1; i < n; i++) {
            double f = a[i][col] / a[col][col];
            for (int j = col; j <= n; j++) a[i][j] -= f * a[col][j];
        }
    }
    for (int i = n - 1; i >= 0; i--) {
        for (int j = i + 1; j < n; j++) a[i][n] -= a[i][j] * a[j][n];
        a[i][n] /= a[i][i];
    }
    return 0;
}
```

**核心启示**：
- **部分选主元**：选绝对值最大的作为主元，减少数值误差
- 无穷解和无解的区分需要更细致的判断

### 题目 3：异或方程组模板（AcWing 884）

> 给定 $n$ 元异或方程组，求解或判断无解/无穷解。

**分析**：异或消元模板，bitset 加速。

```
核心观察：F_2 上的高斯消元，加法 = 异或
构造策略：bitset 存储每行，异或操作 O(n/64)
正确性：与实数消元逻辑相同，只是运算在 F_2 上
```

**C++ 代码**：

```cpp
int gauss_xor(int n, int m) { // n 方程 m 变量
    int rank = 0;
    for (int col = 0; col < m && rank < n; col++) {
        int pivot = -1;
        for (int i = rank; i < n; i++)
            if (a[i][col]) { pivot = i; break; }
        if (pivot == -1) continue;
        swap(a[rank], a[pivot]);
        for (int i = 0; i < n; i++)
            if (i != rank && a[i][col]) a[i] ^= a[rank];
        rank++;
    }
    return rank; // 秩
}
```

**核心启示**：
- **秩** = 主元个数 = 有效方程数
- 自由变量个数 = $m - \text{rank}$

### 题目 4：球形空间产生器（洛谷 P4035）

> 给定 $n+1$ 个 $n$ 维空间中的点，求过这些点的球心。

**分析**：设球心 $(c_1, \ldots, c_n)$，对相邻两点作差消去 $R^2$，得到 $n$ 个线性方程。

```
核心观察：球面方程相减消去 R²，得到线性方程组
构造策略：对 i=1..n，(p_{i+1} - p_i) · 2c = |p_{i+1}|² - |p_i|²
正确性：每对相邻点的等距条件给出一个线性方程
```

**C++ 代码**：

```cpp
// p[0..n] 为 n+1 个点，每个点 n 维
void solve(double p[][MAXN], int n) {
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++)
            a[i][j] = 2 * (p[i+1][j] - p[i][j]);
        a[i][n] = 0;
        for (int j = 0; j < n; j++)
            a[i][n] += p[i+1][j] * p[i+1][j] - p[i][j] * p[i][j];
    }
    gauss(n); // a[i][n] 即为球心第 i 个坐标
}
```

**核心启示**：
- **几何建模**转化为线性方程组是高斯消元的典型应用
- 相邻点作差是消去非线性项的常用技巧

### 题目 5：外星千足虫（洛谷 P2447）

> $n$ 只虫子，每只有奇数或偶数条腿。给 $m$ 次观察，每次选一些虫子，告知总腿数奇偶。求最少多少次观察可确定每只虫子的奇偶。

**分析**：异或方程组，逐行加入，每加一行做一次消元，当秩达到 $n$ 时所有变量确定。

```
核心观察：奇偶性 → F_2 上的方程，逐行加入判断何时秩为 n
构造策略：增量式消元，每加一行检查秩是否增加
正确性：秩 = n 时方程组有唯一解
```

**C++ 代码**：

```cpp
bitset<1005> basis[1005];
int pivot_col[1005], rank_cnt = 0;

int add_equation(bitset<1005> eq, int n) {
    for (int j = 0; j < n; j++) {
        if (!eq[j]) continue;
        if (pivot_col[j] != -1) { eq ^= basis[pivot_col[j]]; continue; }
        basis[j] = eq;
        pivot_col[j] = j;
        rank_cnt++;
        return rank_cnt;
    }
    return rank_cnt; // 该方程线性相关，秩不变
}
```

**核心启示**：
- **增量式消元**：不需要一次性给出所有方程
- 当秩等于变量数时，立即输出答案

### 题目 7：小Z的房间（洛谷 P4111）

> 求网格图中生成树个数（模素数）。

**分析**：Matrix-Tree 定理：生成树个数 = Kirchhoff 矩阵的任意 $n-1$ 阶余子式。用模意义高斯消元求行列式。

```
核心观察：Matrix-Tree 定理将生成树计数转化为行列式
构造策略：建 Kirchhoff 矩阵（度数矩阵 - 邻接矩阵），删一行一列，求行列式
正确性：Matrix-Tree 定理（Kirchhoff 定理）
```

**C++ 代码**：

```cpp
long long det(int n, long long p) { // 求 n 阶矩阵行列式 mod p
    long long res = 1;
    for (int col = 0; col < n; col++) {
        int pivot = -1;
        for (int i = col; i < n; i++)
            if (a[i][col]) { pivot = i; break; }
        if (pivot == -1) return 0;
        if (pivot != col) { swap(a[col], a[pivot]); res = p - res; }
        res = res * a[col][col] % p;
        long long inv = qpow(a[col][col], p - 2, p);
        for (int i = col + 1; i < n; i++) {
            long long f = a[i][col] * inv % p;
            for (int j = col; j < n; j++)
                a[i][j] = (a[i][j] - f * a[col][j] % p + p) % p;
        }
    }
    return res;
}
```

**核心启示**：
- **Matrix-Tree 定理**：生成树数 = Kirchhoff 矩阵任意余子式的行列式
- 行列式消元时交换行要变号

### 题目 13：生成树计数进阶（洛谷 P4208）

> 给定带权无向图，求所有生成树的边权积之和（模素数）。

**分析**：广义 Matrix-Tree 定理：Kirchhoff 矩阵中 $L_{ij} = -w_{ij}$，$L_{ii} = \sum w_{ij}$，余子式即为带权生成树计数。

```
核心观察：广义 Matrix-Tree 定理处理带权情况
构造策略：建带权 Kirchhoff 矩阵，删行列，求行列式
正确性：广义 Matrix-Tree 定理
```

**C++ 代码**：

```cpp
long long weighted_spanning_trees(int n, long long w[][MAXN], long long p) {
    // 建 Kirchhoff 矩阵
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            if (i != j) {
                a[i][j] = (p - w[i][j]) % p;
                a[i][i] = (a[i][i] + w[i][j]) % p;
            }
    // 删第 n-1 行和列，求 (n-1)×(n-1) 行列式
    return det(n - 1, p);
}
```

**核心启示**：
- 广义 Matrix-Tree 定理将**带权生成树计数**归为行列式
- 应用广泛：BEST 定理（欧拉回路计数）也基于此

### 题目 14：BEST 定理（有向图欧拉回路计数）

> 给定有向图，求欧拉回路个数。

**分析**：BEST 定理：$ec(G) = t_w(G) \cdot \prod_{v} (\deg^+(v) - 1)!$，其中 $t_w$ 为以任意点为根的内向生成树个数（用 Matrix-Tree 求解）。

```
核心观察：BEST 定理 = Matrix-Tree 定理 + 阶乘
构造策略：有向 Kirchhoff 矩阵求内向树个数，乘以各点 (出度-1)!
正确性：BEST 定理（van Aardenne-Ehrenfest & de Bruijn）
```

**C++ 代码**：

```cpp
long long euler_circuit_count(int n, long long p) {
    // 建有向 Kirchhoff 矩阵（入度 - 邻接矩阵）
    long long tree_cnt = det(n - 1, p); // 内向树个数
    long long prod = 1;
    for (int v = 0; v < n; v++)
        prod = prod * fac[out_deg[v] - 1] % p;
    return tree_cnt * prod % p;
}
```

**核心启示**：
- **BEST 定理**是 Matrix-Tree 的深层应用
- 有向图的 Kirchhoff 矩阵与无向图不同

### 题目 15：自由变量枚举与最优解（CF 1100F 变体）

> 给定 $n$ 个方程 $m$ 个未知数（$m > n$），在异或方程组有多解时，求使目标函数最大/最小的解。

**分析**：高斯消元后有 $m - r$ 个自由变量，贪心地从高位到低位枚举自由变量取值。

```
核心观察：自由变量可以任意取值，通过贪心选择最优
构造策略：消元确定主变量与自由变量的关系，贪心枚举
正确性：线性空间中的贪心在 F_2 上等价于线性基的高位贪心
```

**C++ 代码**：

```cpp
long long solve_max(int n, int m) {
    int r = gauss_xor(n, m);
    // free_vars = 不是主元列的变量
    long long ans = 0;
    for (int j = m - 1; j >= 0; j--) {
        if (is_pivot[j]) continue; // 主变量，值由自由变量决定
        // 尝试设 x[j] = 1，检查是否使 ans 增大
        long long trial = ans ^ (1LL << j);
        // 传播到所有主变量...
        if (trial > ans) ans = trial;
    }
    return ans;
}
```

**核心启示**：
- 异或方程组的**自由变量贪心**与线性基本质相同
- 解空间的维度 = 自由变量个数

## 五、常见技巧与思路总结

### 5.1 选主元策略

实数消元必须选主元（绝对值最大），否则数值误差会导致错误。模意义下只需找非零即可。

### 5.2 bitset 加速

异或方程组用 `bitset` 存储每一行，异或操作从 $O(n)$ 降为 $O(n/64)$。

### 5.3 三对角消元（Thomas 算法）

当系数矩阵是三对角形时，$O(n)$ 即可完成消元，常见于概率 DP。

### 5.4 增量式消元

可以逐行加入方程并维护行阶梯形，适用于在线判断秩的场景。

### 5.5 行列式符号

每次交换行，行列式变号；乘以常数，行列式乘以该常数。

## 七、终极总结与归纳

### 7.1 高斯消元的本质（一句话）

高斯消元是将线性方程组通过初等行变换化为阶梯形的过程——它是线性代数在算法竞赛中最核心的计算工具。

### 7.2 核心要点

1. **三种变体**：实数、模素数、异或（$\mathbb{F}_2$）
2. **选主元**：实数消元的数值稳定性关键
3. **行列式**：消元到上三角后对角线乘积
4. **Matrix-Tree**：生成树计数 = Kirchhoff 矩阵余子式
5. **bitset 加速**：异或方程组 $O(n^3/64)$
6. **自由变量**：$m - \text{rank}$ 个，可贪心枚举
