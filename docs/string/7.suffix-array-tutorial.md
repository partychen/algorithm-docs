## 一、什么是后缀数组

**后缀数组**（Suffix Array，简称 SA）是将一个字符串 S 的所有后缀按字典序排序后，记录排序结果的整数数组。它是字符串处理中最实用、最高效的数据结构之一，能够在 O(n log n) 时间内构建，并配合 **height 数组**（LCP 数组）在 O(1) 内回答任意两后缀的最长公共前缀查询。

具体来说，对于长度为 n 的字符串 S，后缀数组 `sa[i]` 表示所有后缀按字典序排列后，排名第 i 的后缀在原串中的起始位置；`rk[i]` 表示从位置 i 开始的后缀的排名。两者互为逆映射：`sa[rk[i]] = i`，`rk[sa[i]] = i`。

```text
示例：S = "banana"

后缀列表：             字典序排序后：
0: banana              5: a
1: anana               3: ana
2: nana                1: anana
3: ana                 0: banana
4: na                  4: na
5: a                   2: nana

sa = [5, 3, 1, 0, 4, 2]
rk = [3, 2, 5, 1, 4, 0]
```

| 特性 | 后缀数组 SA | 后缀自动机 SAM | 后缀树 ST |
|------|------------|---------------|----------|
| 构建时间 | O(n log n) | O(n) | O(n) |
| 空间复杂度 | O(n) | O(n\|Σ\|) | O(n\|Σ\|) |
| 实现难度 | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| LCP 查询 | O(1)（需 RMQ） | 需 parent tree | 天然支持 |
| 多串扩展 | 拼接 + 分隔符 | 广义 SAM | 广义后缀树 |
| 离线 / 在线 | 离线 | 在线 | 在线 |

## 二、核心思想与前置知识

### 2.1 SA 的定义与基本数组

- `sa[i]`：排名第 i 的后缀的起始位置（1-indexed）
- `rk[i]`：起始位置为 i 的后缀的排名
- 互逆关系：`sa[rk[i]] = i`，`rk[sa[i]] = i`

### 2.2 倍增 + 基数排序构建 SA（O(n log n)）

核心思想：每轮将每个后缀的前 2^k 个字符作为关键字排序。第一轮按首字符排序，之后每轮利用上一轮的 rank 值，将 `(rk[i], rk[i + 2^(k-1)])` 作为双关键字进行基数排序，直到所有后缀排名两两不同。

```cpp
const int MAXN = 1000005;
int sa[MAXN], rk[MAXN], oldrk[MAXN], cnt[MAXN], id[MAXN], px[MAXN];

void buildSA(const char s[], int n) {
    int m = 128;
    for (int i = 1; i <= n; i++) cnt[rk[i] = s[i]]++;
    for (int i = 1; i <= m; i++) cnt[i] += cnt[i - 1];
    for (int i = n; i >= 1; i--) sa[cnt[rk[i]]--] = i;
    for (int w = 1; ; w <<= 1) {
        int p = 0;
        for (int i = n - w + 1; i <= n; i++) id[++p] = i;
        for (int i = 1; i <= n; i++) if (sa[i] > w) id[++p] = sa[i] - w;
        memset(cnt, 0, sizeof(int) * (m + 1));
        for (int i = 1; i <= n; i++) cnt[px[i] = rk[id[i]]]++;
        for (int i = 1; i <= m; i++) cnt[i] += cnt[i - 1];
        for (int i = n; i >= 1; i--) sa[cnt[px[i]]--] = id[i];
        memcpy(oldrk, rk, sizeof(int) * (n + 1));
        m = 0;
        for (int i = 1; i <= n; i++)
            rk[sa[i]] = (oldrk[sa[i]] == oldrk[sa[i - 1]] &&
                          oldrk[sa[i] + w] == oldrk[sa[i - 1] + w]) ? m : ++m;
        if (m == n) break;
    }
}
```

### 2.3 height 数组（LCP 数组）的 O(n) 求法

`height[i]` 定义为 `sa[i]` 与 `sa[i-1]` 对应后缀的最长公共前缀长度，即排名相邻的两个后缀的 LCP。关键性质：**height[rk[i]] ≥ height[rk[i-1]] - 1**，利用此性质可 O(n) 求出 height 数组。

```cpp
int ht[MAXN];

void getHeight(const char s[], int n) {
    for (int i = 1, k = 0; i <= n; i++) {
        if (rk[i] == 1) { k = 0; continue; }
        if (k) k--;
        int j = sa[rk[i] - 1];
        while (i + k <= n && j + k <= n && s[i + k] == s[j + k]) k++;
        ht[rk[i]] = k;
    }
}
```

### 2.4 SA + Sparse Table 实现 O(1) LCP 查询

任意两个后缀 `suf(i)` 和 `suf(j)` 的 LCP 长度等于 `min(height[rk[i]+1], ..., height[rk[j]])`（假设 `rk[i] < rk[j]`）。这是经典的 RMQ 问题，用 Sparse Table 预处理后可 O(1) 查询。

```cpp
int lg[MAXN], st[MAXN][21];

void buildST(int n) {
    for (int i = 2; i <= n; i++) lg[i] = lg[i / 2] + 1;
    for (int i = 1; i <= n; i++) st[i][0] = ht[i];
    for (int j = 1; (1 << j) <= n; j++)
        for (int i = 1; i + (1 << j) - 1 <= n; i++)
            st[i][j] = min(st[i][j - 1], st[i + (1 << (j - 1))][j - 1]);
}

int queryLCP(int i, int j) {
    int l = rk[i], r = rk[j];
    if (l > r) swap(l, r);
    l++;
    int k = lg[r - l + 1];
    return min(st[l][k], st[r - (1 << k) + 1][k]);
}
```

## 三、分类与变体详解

### 3.1 基础后缀排序

最基本的 SA 应用：构建后缀数组后直接利用排序结果解题。

```
数据结构：后缀数组 SA + rank 数组
支持操作：所有后缀字典序排序、第 k 小后缀查询
关键性质：sa 与 rk 互逆；倍增每轮倍增比较长度
时间复杂度：O(n log n) 构建
```

典型应用：环形字符串排序（将 S 复制为 S+S 后建 SA）、字典序最小/最大后缀。

### 3.2 height 数组与 LCP

height 数组是 SA 的精髓，将 **子串问题** 转化为 **height 数组上的区间问题**。

```
数据结构：height 数组 + Sparse Table
支持操作：任意两后缀 LCP 查询 O(1)、子串计数、重复子串检测
关键性质：height[rk[i]] ≥ height[rk[i-1]] - 1；LCP(sa[i], sa[j]) = min(ht[i+1..j])
时间复杂度：O(n) 求 height，O(n log n) 建 ST，O(1) 查询
```

核心公式：
- 本质不同子串数 = `n*(n+1)/2 - Σ height[i]`
- 出现至少 K 次的最长子串 → height 数组滑动窗口

### 3.3 SA + 单调栈

当需要统计 **所有后缀对的 LCP 贡献之和** 时，暴力 O(n²) 不可接受。利用单调栈在 O(n) 内计算 height 数组中每个元素作为区间最小值的贡献范围。

```
数据结构：SA + height + 单调栈
支持操作：∑ LCP(i, j) 求和、height 区间最小值贡献统计
关键性质：height[k] 作为区间最小值的左右边界可由单调栈 O(n) 求出
时间复杂度：O(n) 统计
```

```cpp
// 求所有后缀对 LCP 之和
long long sumLCP(int n) {
    long long ans = 0, sum = 0;
    int top = 0;
    static int stk[MAXN]; static long long w[MAXN];
    for (int i = 2; i <= n; i++) {
        long long cnt = 1;
        while (top && ht[i] <= stk[top]) {
            sum -= (long long)stk[top] * w[top];
            cnt += w[top--];
        }
        stk[++top] = ht[i]; w[top] = cnt;
        sum += (long long)ht[i] * cnt;
        ans += sum;
    }
    return ans * 2;
}
```

### 3.4 多串后缀数组

处理多个字符串的问题时，将所有字符串用**互不相同的分隔符**拼接成一个长串，建 SA + height。分隔符应选择不在任何串中出现的字符（如 ASCII 1, 2, 3...）。

```
数据结构：多串拼接 + SA + height
支持操作：多串最长公共子串、多串子串匹配、跨串 LCP 查询
关键性质：height 值不跨越分隔符；需记录每个位置属于哪个原串
时间复杂度：O(N log N)，N 为总长度
```

关键技巧：
- 二分长度 L + 分组：在 SA 上按 `height ≥ L` 分组，检查组内是否包含所有字符串
- 滑动窗口：维护窗口内字符串来源种类数

| 变体 | 核心思路 | 典型问题 |
|------|---------|---------|
| 基础排序 | 倍增 + 基数排序 | 后缀排序、环形加密 |
| height + 公式 | 直接利用 height 求不同子串数等 | 不同子串个数、重复子串 |
| height + 单调栈 | 统计 LCP 贡献之和 | 差异、找相同字符 |
| height + 二分 + 分组 | 二分答案长度，按 height 分组检查 | Milk Patterns、多串 LCS |
| SA + Sparse Table | 任意两后缀 O(1) LCP 查询 | DNA 匹配、字符串比较 |
| 多串 SA | 拼接 + 分隔符，记录来源 | 多串公共子串、口头禅 |
| 树上 SA | 倍增排序推广到树路径 | 树上后缀排序 |

### 题目 1：后缀排序（洛谷 P3809）

> 给定一个字符串 S（长度 ≤ 10^6），输出其后缀数组，即所有后缀按字典序排序后各后缀的起始位置。

**分析**：纯模板题，直接使用倍增 + 基数排序构建后缀数组，输出 `sa[1..n]`。

```
数据结构：后缀数组（倍增法）
支持操作：所有后缀字典序排序
关键性质：每轮倍增比较长度，O(log n) 轮即可区分所有后缀
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
const int MAXN = 1000005;
int sa[MAXN], rk[MAXN], oldrk[MAXN], cnt[MAXN], id[MAXN], px[MAXN];

void buildSA(const char s[], int n) {
    int m = 128;
    for (int i = 1; i <= n; i++) cnt[rk[i] = s[i]]++;
    for (int i = 1; i <= m; i++) cnt[i] += cnt[i - 1];
    for (int i = n; i >= 1; i--) sa[cnt[rk[i]]--] = i;
    for (int w = 1; ; w <<= 1) {
        int p = 0;
        for (int i = n - w + 1; i <= n; i++) id[++p] = i;
        for (int i = 1; i <= n; i++) if (sa[i] > w) id[++p] = sa[i] - w;
        memset(cnt, 0, sizeof(int) * (m + 1));
        for (int i = 1; i <= n; i++) cnt[px[i] = rk[id[i]]]++;
        for (int i = 1; i <= m; i++) cnt[i] += cnt[i - 1];
        for (int i = n; i >= 1; i--) sa[cnt[px[i]]--] = id[i];
        memcpy(oldrk, rk, sizeof(int) * (n + 1));
        m = 0;
        for (int i = 1; i <= n; i++)
            rk[sa[i]] = (oldrk[sa[i]] == oldrk[sa[i - 1]] &&
                          oldrk[sa[i] + w] == oldrk[sa[i - 1] + w]) ? m : ++m;
        if (m == n) break;
    }
}
// 调用 buildSA(s, n)，输出 sa[1..n]
```

**核心启示**：
- **倍增思想** 是 SA 构建的核心，每轮将比较长度翻倍，O(log n) 轮后所有后缀排名唯一
- **基数排序** 保证每轮排序 O(n)，总复杂度 O(n log n)
- 此模板是所有 SA 题目的基础，务必熟练掌握

### 题目 2：字符加密（洛谷 P4051）

> 将长度为 n 的字符串 S 排成环，产生 n 个循环同构串，按字典序排序后，依次取每个串的末尾字符，组成加密串。

**分析**：将 S 复制一份拼接为 S+S（长度 2n），对 S+S 建后缀数组。后缀 `suf(i)`（i ≤ n）对应第 i 个循环同构串，其末尾字符为 `S[(i+n-2) mod n + 1]`。按 sa 顺序输出即可。

```
数据结构：后缀数组（环形字符串处理）
支持操作：所有循环同构串的字典序排序
关键性质：S+S 的前 n 个后缀恰好对应 S 的 n 个循环同构串
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
// s 为原串（1-indexed，长度 n），先拼接 s[n+1..2n] = s[1..n]
// 对长度 2n 的串建 SA
void solve(const char s[], int n, char ans[]) {
    // buildSA(s, 2 * n);
    int p = 0;
    for (int i = 1; i <= 2 * n; i++)
        if (sa[i] <= n)
            ans[++p] = s[(sa[i] + n - 2) % n + 1];
}
```

**核心启示**：
- **环形 → 倍长**：将环形字符串展开为两倍长度，是处理循环同构问题的经典技巧
- **筛选合法后缀**：SA 建在长串上，但只有起始位置 ≤ n 的后缀才对应有效的循环串

### 🟡 基础级 Level 2：height 数组直接应用

### 题目 3：不同子串个数（洛谷 P2408）

> 给定字符串 S（长度 ≤ 10^5），求本质不同的子串个数。

**分析**：所有子串总数为 `n*(n+1)/2`，其中重复计数的部分恰好等于 `Σ height[i]`。排名第 i 的后缀新增的不同子串数为 `(n - sa[i] + 1) - height[i]`。

```
数据结构：SA + height 数组
支持操作：本质不同子串计数
关键性质：第 i 个后缀贡献的新子串数 = (n - sa[i] + 1) - height[i]
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
long long countDistinct(int n) {
    long long ans = 0;
    for (int i = 1; i <= n; i++)
        ans += (n - sa[i] + 1) - ht[i];
    return ans;
}
```

**核心启示**：
- **height 核心公式**：本质不同子串数 = `n*(n+1)/2 - Σ height[i]`，是 SA 最基础的应用
- 本质是排名第 i 的后缀相比前一名，新增了去掉 LCP 部分后的前缀子串

### 题目 4：Milk Patterns（洛谷 P2852）

> 给定长度为 n 的整数序列和整数 K，求出现至少 K 次的最长子串长度。

**分析**：出现 ≥ K 次等价于 SA 上存在连续 K 个后缀使得它们的 LCP ≥ L。即在 height 数组上，找长度为 K-1 的滑动窗口的最小值的最大值。用单调队列 O(n) 完成。

```
数据结构：SA + height + 单调队列
支持操作：出现 ≥ K 次的最长子串长度
关键性质：连续 K 个后缀 LCP ≥ L ⟺ 该段 height 最小值 ≥ L
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
int solve(int n, int K) {
    int ans = 0;
    deque<int> dq;
    for (int i = 2; i <= n; i++) {
        while (!dq.empty() && ht[dq.back()] >= ht[i]) dq.pop_back();
        dq.push_back(i);
        if (i - dq.front() >= K - 1) dq.pop_front();
        if (i >= K) ans = max(ans, ht[dq.front()]);
    }
    return ans;
}
```

**核心启示**：
- **"出现 ≥ K 次" → height 滑窗**：SA 中最经典的模式之一
- **单调队列维护滑窗最小值**：O(n) 完成检验，无需额外二分

### 题目 5：Prefixes and Suffixes（CF 432D）

> 给定字符串 S（长度 ≤ 10^5），找出所有既是前缀又是后缀的子串（border），并输出每个 border 在 S 中的出现次数。

**分析**：用 KMP 的失配数组 `fail` 找出所有 border：从 `fail[n]` 沿链跳到 0，经过的长度都是 border。统计出现次数：`cnt[i] = 1` 后从后往前累加 `cnt[fail[i]] += cnt[i]`。也可用 SA + height 二分统计每个前缀的出现次数。

```
数据结构：KMP fail 数组（或 SA + height + 二分）
支持操作：枚举所有 border 及其出现次数
关键性质：border 集合由 fail 链给出；出现次数通过 fail 链 DP 传递
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
const int MAXN = 100005;
int fail[MAXN], cnt[MAXN];

void solve(const char s[], int n) {
    fail[1] = 0;
    for (int i = 2, j = 0; i <= n; i++) {
        while (j && s[j + 1] != s[i]) j = fail[j];
        if (s[j + 1] == s[i]) j++;
        fail[i] = j;
    }
    for (int i = 1; i <= n; i++) cnt[i] = 1;
    for (int i = n; i >= 1; i--) cnt[fail[i]] += cnt[i];
    // 从 fail[n] 沿链收集所有 border 长度，出现次数为 cnt[len]
}
```

**核心启示**：
- **border = 前缀 ∩ 后缀**：KMP fail 链自然枚举所有 border
- **fail 链 DP**：`cnt[fail[i]] += cnt[i]` 利用前缀嵌套关系统计出现次数
- SA 方法：对每个 border 长度 L，在 SA 上二分找 LCP ≥ L 的区间宽度即为出现次数

### 🟠 进阶级 Level 3：SA + 二分 / 多串 / 单调栈

### 题目 6：Musical Themes（洛谷 P2743）

> 给定长度为 n 的整数序列（音符），求最长的"主题"长度。主题定义为：存在至少两个不重叠的子序列，它们整体加减同一个常数后完全相同。

**分析**：对原序列取差分 `d[i] = a[i+1] - a[i]`，问题转化为在差分数组中找最长的**重复非重叠子串**。建 SA + height，二分答案长度 L，在 height 分组中检查是否存在 LCP ≥ L 且起始位置差 > L 的两个后缀。

```
数据结构：差分数组 + SA + height + 二分
支持操作：最长重复非重叠子串
关键性质：差分消除常数偏移；非重叠要求 max(sa) - min(sa) > L
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
// d[] 为差分数组，长度 n-1，已建好 SA 和 height
bool check(int L, int n) {
    int mx = sa[1], mn = sa[1];
    for (int i = 2; i <= n; i++) {
        if (ht[i] >= L) {
            mx = max(mx, sa[i]);
            mn = min(mn, sa[i]);
            if (mx - mn > L) return true;
        } else {
            mx = mn = sa[i];
        }
    }
    return false;
}
// 二分 L，答案为最大满足 check 的 L + 1（差分还原）
```

**核心启示**：
- **差分消除平移**：将"允许整体加减常数"转化为差分后的精确匹配
- **二分 + height 分组**：SA 中处理"满足某长度条件"的经典范式
- 非重叠约束通过检查组内 `max(sa) - min(sa) > L` 实现

### 题目 7：Sandy的卡片（洛谷 P2463）

> 给定 N 个整数序列，求所有序列的最长"公共子串"长度。两个子序列相同定义为差分后相等。

**分析**：先对每个序列取差分，然后用分隔符拼接所有差分序列为一个长串，建 SA + height。二分答案长度 L，在 SA 上按 `height ≥ L` 分组，检查组内是否包含来自所有 N 个原串的后缀。

```
数据结构：差分 + 多串拼接 + SA + height + 二分
支持操作：N 串差分后最长公共子串
关键性质：按 height 分组后，组内后缀两两 LCP ≥ L；需覆盖所有串
时间复杂度：O(N_total × log N_total)
```

**C++ 代码**：

```cpp
int belong[MAXN]; // belong[i] 表示位置 i 属于第几个串
int N; // 串的数量

bool check(int L, int n) {
    bool vis[105] = {};
    int cnt = 0;
    for (int i = 2; i <= n; i++) {
        if (ht[i] >= L) {
            if (!vis[belong[sa[i - 1]]]) vis[belong[sa[i - 1]]] = true, cnt++;
            if (!vis[belong[sa[i]]]) vis[belong[sa[i]]] = true, cnt++;
            if (cnt == N) return true;
        } else {
            memset(vis, 0, sizeof(vis));
            cnt = 0;
        }
    }
    return false;
}
// 二分 L，答案 +1（差分还原）
```

**核心启示**：
- **多串 SA 标准范式**：拼接 + 分隔符 + 记录归属 + 二分 + 分组检查
- **差分预处理**：当问题允许整体平移时，差分是标准化手段
- 分隔符应选择不在原序列中出现的值

### 题目 8：差异（洛谷 P4248）

> 给定字符串 S（长度 ≤ 5×10^5），求 `∑_{1≤i<j≤n} [len(T_i) + len(T_j) - 2 × lcp(T_i, T_j)]`，其中 T_i 为第 i 个后缀。

**分析**：公式拆为两部分：`∑(len(T_i) + len(T_j)) = (n-1) × n × (n+1) / 2`，直接计算；`∑ 2×lcp(T_i, T_j)` 用单调栈在 height 数组上 O(n) 计算每个 height 值作为区间最小值时的贡献。

```
数据结构：SA + height + 单调栈
支持操作：所有后缀对 LCP 之和
关键性质：lcp(sa[i], sa[j]) = min(ht[i+1..j])；单调栈统计最小值贡献
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
long long solve(int n) {
    long long total = (long long)(n - 1) * n * (n + 1) / 2;
    long long lcpSum = 0, sum = 0;
    int top = 0;
    static int stk[MAXN]; static long long w[MAXN];
    for (int i = 2; i <= n; i++) {
        long long cnt = 1;
        while (top && ht[i] <= stk[top]) {
            sum -= (long long)stk[top] * w[top];
            cnt += w[top--];
        }
        stk[++top] = ht[i]; w[top] = cnt;
        sum += (long long)ht[i] * cnt;
        lcpSum += sum;
    }
    return total - 2 * lcpSum;
}
```

**核心启示**：
- **单调栈统计 LCP 贡献**：SA 中最重要的进阶技巧之一
- **拆分公式**：先算常数部分，再用单调栈处理 LCP 部分
- 栈中维护 (height 值, 控制后缀数)，弹出时累积被弹元素的计数

### 题目 9：生成魔咒（洛谷 P4070）

```
数据结构：SA + Sparse Table + set（平衡 BST）
支持操作：在线维护本质不同子串数
关键性质：插入新后缀时，贡献 = 后缀长度 - max(与左邻 LCP, 与右邻 LCP)
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp

set<int> S;
long long ans = 0;

void addSuffix(int rkVal, int suffLen) {
    auto it = S.insert(rkVal).first;
    int lcpL = 0, lcpR = 0;
    if (it != S.begin()) lcpL = queryLCP(sa[*prev(it)], sa[rkVal]);
    if (next(it) != S.end()) lcpR = queryLCP(sa[*next(it)], sa[rkVal]);

}
// 倒序处理：从最后一个字符开始，依次加入后缀
```

**核心启示**：
- **在线 → 离线反转**：将在线末尾添加转化为离线倒序插入后缀
- **set 维护邻居关系**：插入时只需与前驱后继比较 LCP，增量更新答案


```cpp

    int cnt = 0;
    for (int i = 1; i + lenT - 1 <= lenS; i++) {

        }

    }

}
```

### 题目 12：找相同字符（洛谷 P3181）

> 给定两个字符串 A 和 B，求所有公共子串的对数之和（计重），即 `∑ [A[i..j] = B[k..l]]`。

**分析**：将 A + '#' + B 拼接建 SA + height，计算所有跨串后缀对的 LCP 之和。用单调栈分别统计左侧来自 A、右侧来自 B 的后缀对贡献，再反向统计一次。

```
数据结构：两串拼接 + SA + height + 单调栈
支持操作：跨串公共子串对数统计
关键性质：跨串 LCP 贡献需分别统计 A→B 和 B→A 方向
时间复杂度：O(n)，n 为总长度
```

**C++ 代码**：

```cpp
// belong[i] = 0 表示来自 A，= 1 表示来自 B
// 正向扫描单调栈：统计左侧 A 后缀数对右侧 B 后缀的贡献
long long solve(int n) {
    long long ans = 0;
    for (int dir = 0; dir < 2; dir++) { // 两个方向
        int top = 0;
        long long sum = 0;
        static int stk[MAXN]; static long long cntS[MAXN];
        for (int i = 2; i <= n; i++) {
            long long c = (belong[sa[i - 1]] == dir);
            while (top && stk[top] >= ht[i]) {
                sum -= (long long)stk[top] * cntS[top];
                c += cntS[top--];
            }
            stk[++top] = ht[i]; cntS[top] = c;
            sum += (long long)ht[i] * c;
            if (belong[sa[i]] != dir) ans += sum;
        }
    }
    return ans;
}
```

**核心启示**：
- **跨串统计**：单调栈需区分后缀来源，分别累积不同来源的计数

- 这是 SA + 单调栈处理双串问题的标准方法

### 题目 13：品酒大会（洛谷 P2178）

> 给定字符串 S（长度 ≤ 3×10^5），每个后缀有一个权值 w[i]。对于每个 r = 0, 1, ..., n-1，求 LCP ≥ r 的后缀对数，以及这些对中权值乘积的最大值。

**分析**：按 height 从大到小，用并查集逐步合并排名相邻的后缀。合并时维护集合大小（计对数）和最大/最小权值（最大乘积可能来自两个最大正数或两个最小负数）。答案具有后缀累加性。

```
数据结构：SA + height + 并查集
支持操作：按 LCP 阈值统计后缀对数和最大权值乘积
关键性质：height 从大到小合并，ans[r] = ans[r+1] + 新增对数
时间复杂度：O(n α(n))
```

**C++ 代码**：

```cpp
int fa[MAXN], sz[MAXN];
long long mx[MAXN], mn[MAXN];
long long pairCnt[MAXN], bestProd[MAXN];

int find(int x) { return fa[x] == x ? x : fa[x] = find(fa[x]); }

void solve(int n, long long w[]) {
    for (int i = 1; i <= n; i++)
        fa[i] = i, sz[i] = 1, mx[i] = mn[i] = w[sa[i]];
    static int ord[MAXN];
    iota(ord + 2, ord + n + 1, 2);
    sort(ord + 2, ord + n + 1, [](int a, int b) { return ht[a] > ht[b]; });
    long long curPair = 0, curBest = -1e18;
    for (int k = 2; k <= n; k++) {
        int i = ord[k];
        int u = find(i - 1), v = find(i);
        curBest = max(curBest, max(mx[u] * mx[v], mn[u] * mn[v]));
        curPair += (long long)sz[u] * sz[v];
        fa[u] = v; sz[v] += sz[u];
        mx[v] = max(mx[v], mx[u]); mn[v] = min(mn[v], mn[u]);
        pairCnt[ht[i]] = curPair; bestProd[ht[i]] = curBest;
    }
    for (int r = n - 2; r >= 0; r--) {
        pairCnt[r] += pairCnt[r + 1];
        if (pairCnt[r]) bestProd[r] = max(bestProd[r], bestProd[r + 1]);
    }
}
```

**核心启示**：
- **并查集按 height 递减合并**：SA 中处理"LCP ≥ r"分级统计的强力工具
- **负数乘积**：最大乘积可能来自两个最小负数，需同时维护 max 和 min
- **答案后缀累加**：`ans[r]` 包含 `ans[r+1]` 的全部贡献

```cpp
    while (L < R) {

        else L = mid + 1;
```

### 题目 16：优秀的拆分（洛谷 P1117）

> 给定字符串 S（长度 ≤ 3×10^4），求有多少种方式将 S 的某个子串拆为 AABB 形式（即两个相同串 AA 拼接两个相同串 BB）的方案总数。

**分析**：枚举 AA 和 BB 的分界点 p，答案 = `∑ f[p] × g[p+1]`，其中 `f[p]` 为以 p 结尾的 AA 串数，`g[p]` 为以 p 开头的 AA 串数。对于固定周期长度 L，枚举等间距关键点 `0, L, 2L, ...`，对相邻关键点用正串和反串的 LCP/LCS 判定并差分更新 f、g。

```
数据结构：SA + Sparse Table（正串 + 反串）+ 差分 + 枚举周期
支持操作：统计所有 AA 形式子串的位置
关键性质：固定周期 L 的等间距关键点 + LCP/LCS 判定
时间复杂度：O(n log n)（调和级数 ∑ n/L = O(n log n)）
```

**C++ 代码**：

```cpp
// 对正串和反串分别建 SA + ST
// f[i] = 以 i 结尾的 AA 串数，g[i] = 以 i 开头的 AA 串数
int f[MAXN], g[MAXN];

void solve(const char s[], int n) {
    for (int L = 1; L <= n / 2; L++) {
        for (int i = 1; i + L <= n; i += L) {
            int j = i + L;
            int lcp = min(queryFwd(i, j), L);  // 正向 LCP
            int lcs = min(queryRev(i, j), L - 1); // 反向 LCS
            if (lcp + lcs >= L) {
                int cnt = lcp + lcs - L + 1;
                // 差分更新 f: [j+lcp-L, j+lcp-L+cnt-1] 区间 +1
                // 差分更新 g: [i-lcs, i-lcs+cnt-1] 区间 +1
            }
        }
    }
    // 前缀和还原 f、g
    long long ans = 0;
    for (int i = 1; i < n; i++) ans += (long long)f[i] * g[i + 1];
}
```

**核心启示**：
- **等间距关键点**：枚举周期 L 后，仅在 L 的倍数位置设关键点，避免 O(n²) 枚举
- **正串 + 反串双向查询**：正向 LCP 确定向后匹配长度，反向 LCS 确定回退长度
- **差分更新**：将连续区间的 f/g 增量用差分 O(1) 处理
- 调和级数保证总复杂度 O(n log n)

### 题目 17：树上后缀排序（洛谷 P5353）

> 给定一棵 n 个节点的有根树，每个节点有一个字符。定义节点 u 的"后缀"为从 u 到根路径上的字符组成的字符串。求所有节点后缀按字典序排序后的排列。


```cpp
    for (int k = 1; (1 << k) <= n; k++) {
        // 对每个节点 u，取双关键字：
        //   第一关键字: rk[u]（上一轮排名）
        //   第二关键字: rk[anc[u][k-1]]（2^(k-1) 级祖先上一轮排名，不存在则 0）
        // 基数排序更新 sa 和 rk
        // 若所有 rk 互不相同则 break
    }
    // 输出 sa[1..n]
}
```

**核心启示**：
- **树上倍增排序**：SA 倍增的自然推广，核心在于用祖先的 rank 作为第二关键字
- **边界处理**：深度不足时祖先不存在，rank 设为 0（等价于空串最小）
- 这一思想可进一步结合 height 数组计算树上路径串的 LCP

## 五、常见技巧与思路总结

### 5.1 子串 → height 区间

后缀数组的核心转化：**子串问题 → height 数组上的区间问题**。

- 两后缀的 LCP = `min(height[rk[i]+1..rk[j]])`
- 出现次数 ≥ K 的子串 → height 数组中连续 ≥ K-1 个值 ≥ L 的段
- 不同子串数 → `n*(n+1)/2 - Σ height[i]`

### 5.2 二分答案 + height 分组

当题目问"最长的满足某条件的子串"时：

1. 二分答案长度 L
2. 在 SA 上按 `height ≥ L` 将后缀分组
3. 检查组内是否满足条件（覆盖所有串、存在非重叠位置等）

```cpp
// 典型分组框架
bool check(int L, int n) {
    // 初始化组信息
    for (int i = 2; i <= n; i++) {
        if (ht[i] >= L) { /* 同组，累积信息 */ }
        else { /* 新组，检查上一组并重置 */ }
    }
    return false;
}
```

### 5.3 多串拼接三要素

1. **分隔符**：选择不出现在任何串中的字符，且互不相同
2. **归属数组**：`belong[i]` 记录位置 i 属于第几个原串
3. **分隔符跨越**：height 值不应跨越分隔符（因分隔符唯一而自动满足）

### 5.4 单调栈统计 LCP 贡献

当需要 `∑ lcp(i, j)` 或按来源统计时，用单调栈将 O(n²) 降至 O(n)：
- 维护递增栈，栈中记录 (height 值, 控制后缀数)
- 弹出时将控制数累加到新元素
- 区分来源时，栈中额外维护各来源计数

### 5.5 并查集按 height 合并

适用于"LCP ≥ r"分级统计的问题：
1. 将 height 从大到小排序
2. 依次合并排名相邻的后缀
3. 合并时维护集合信息（大小、最值等）
4. 答案具有后缀累加性

### 5.6 正串 + 反串双向 LCP

需要同时查询 LCP（最长公共前缀）和 LCS（最长公共后缀）时，对正串和反串分别建 SA + Sparse Table，两组查询互补。

## 七、终极总结与归纳

### 7.1 本质

**后缀数组的本质是将字符串的子串问题，通过后缀排序和 LCP 信息，转化为整数数组上的区间统计问题。** SA 提供了全局后缀排序视角，height 数组将子串关系编码为相邻排名后缀的 LCP 值，Sparse Table 支持 O(1) 区间最值查询。这三者的组合，构成了处理字符串问题的完整框架。

### 7.2 核心要点

1. **SA + rk 互逆**：`sa[rk[i]] = i`，`rk[sa[i]] = i`，这是所有推导的基础
2. **height 核心性质**：`height[rk[i]] ≥ height[rk[i-1]] - 1`，保证 O(n) 求解
3. **LCP = 区间 min**：`lcp(i, j) = min(ht[rk[i]+1..rk[j]])`，ST 表 O(1) 查询
4. **子串计数公式**：本质不同子串数 = `n(n+1)/2 - Σ height[i]`
5. **单调栈/并查集**：分别适用于"求和"类和"分级"类 LCP 统计问题
6. **多串三要素**：分隔符 + 归属数组 + 分组/滑窗检查
