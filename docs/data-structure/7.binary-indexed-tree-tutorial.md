## 一、什么是树状数组

**树状数组**（Binary Indexed Tree，简称 BIT），又称 **Fenwick Tree**，是一种支持**单点修改**和**前缀查询**的高效数据结构。它的核心优势在于代码极短、常数极小，在很多场景下是线段树的轻量替代品。

**核心特性：**

| 特性 | 说明 |
|------|------|
| 时间复杂度 | 单点修改 $O(\log n)$，前缀查询 $O(\log n)$ |
| 空间复杂度 | $O(n)$ |
| 核心操作 | `lowbit(x) = x & (-x)`，提取最低位的 1 |
| 本质 | 利用二进制分组思想，将前缀和拆分为 $O(\log n)$ 个区间的和 |

**lowbit 的含义：**

对于正整数 $x$，$\text{lowbit}(x) = x \mathbin{\&} (-x)$ 提取的是 $x$ 的二进制表示中最低位的 $1$ 所对应的值。例如：

- $\text{lowbit}(6) = \text{lowbit}(110_2) = 2 = 10_2$
- $\text{lowbit}(12) = \text{lowbit}(1100_2) = 4 = 100_2$

**前缀和结构：**

树状数组 `c[x]` 管理的是原数组中 $[x - \text{lowbit}(x) + 1, \; x]$ 这段区间的信息。查询前缀和时，从 $x$ 出发不断减去 $\text{lowbit}(x)$，累加所有经过的 `c` 值；修改时，从 $x$ 出发不断加上 $\text{lowbit}(x)$，更新所有受影响的 `c` 值。

## 二、核心思想与前置知识

### 2.1 lowbit 运算

`lowbit` 是树状数组的基石，所有操作都依赖它来确定节点管辖的区间范围。

```cpp
int lowbit(int x) {
    return x & (-x);
}
```

**原理：** 计算机中负数使用补码表示，$-x$ 等价于 $\sim x + 1$（按位取反加一）。取反后原来最低位的 $1$ 变成 $0$，加一后又变回 $1$，而更低位全部归零，更高位与原数相反。因此 $x \mathbin{\&} (-x)$ 恰好只保留最低位的 $1$。

### 2.2 单点修改 + 区间查询（基础模板）

这是树状数组最基本的用法：维护一个数组，支持单点加值、查询前缀和。

```cpp
int n, c[MAXN];

void update(int x, int val) {
    for (; x <= n; x += lowbit(x))
        c[x] += val;
}

int query(int x) {
    int sum = 0;
    for (; x > 0; x -= lowbit(x))
        sum += c[x];
    return sum;
}

// 查询区间 [l, r] 的和
int range_query(int l, int r) {
    return query(r) - query(l - 1);
}
```

**单点修改：** 从位置 $x$ 出发，沿着 $x \to x + \text{lowbit}(x)$ 的路径向上更新，直到超出 $n$。

**前缀查询：** 从位置 $x$ 出发，沿着 $x \to x - \text{lowbit}(x)$ 的路径向下累加，直到 $x = 0$。

### 2.3 区间修改 + 单点查询（差分树状数组）

通过**差分**思想，树状数组可以支持区间修改、单点查询。定义差分数组 $d[i] = a[i] - a[i-1]$，则 $a[x] = \sum_{i=1}^{x} d[i]$，即 $d$ 的前缀和就是原数组的值。

- **区间 $[l, r]$ 加 $val$：** 在差分数组上执行 `update(l, val)` 和 `update(r + 1, -val)`
- **单点查询 $a[x]$：** 执行 `query(x)` 求差分数组的前缀和

```cpp
int n, c[MAXN];

void update(int x, int val) {
    for (; x <= n; x += lowbit(x))
        c[x] += val;
}

int query(int x) {
    int sum = 0;
    for (; x > 0; x -= lowbit(x))
        sum += c[x];
    return sum;
}

void range_update(int l, int r, int val) {
    update(l, val);
    update(r + 1, -val);
}

int point_query(int x) {
    return query(x);
}
```

### 2.4 区间修改 + 区间查询（双树状数组）

在差分数组的基础上，我们推导区间和的公式。设差分数组为 $d$，则：

$$
\sum_{i=1}^{x} a[i] = \sum_{i=1}^{x} \sum_{j=1}^{i} d[j] = \sum_{j=1}^{x} d[j] \cdot (x - j + 1) = (x+1)\sum_{j=1}^{x} d[j] - \sum_{j=1}^{x} d[j] \cdot j
$$

因此需要维护两个树状数组：一个维护 $d[i]$，一个维护 $d[i] \cdot i$。

```cpp
typedef long long ll;
int n;
ll c1[MAXN], c2[MAXN];

void update(ll *c, int x, ll val) {
    for (; x <= n; x += lowbit(x))
        c[x] += val;
}

ll query(ll *c, int x) {
    ll sum = 0;
    for (; x > 0; x -= lowbit(x))
        sum += c[x];
    return sum;
}

void range_update(int l, int r, ll val) {
    update(c1, l, val);
    update(c1, r + 1, -val);
    update(c2, l, val * l);
    update(c2, r + 1, -val * (r + 1));
}

ll prefix_sum(int x) {
    return (x + 1) * query(c1, x) - query(c2, x);
}

ll range_query(int l, int r) {
    return prefix_sum(r) - prefix_sum(l - 1);
}
```

### 2.5 二维树状数组

二维树状数组将一维的思想推广到矩阵上，支持**单点修改**和**二维前缀和查询**。修改和查询都在两层循环中进行。

```cpp
int n, m, c[MAXN][MAXM];

void update(int x, int y, int val) {
    for (int i = x; i <= n; i += i & (-i))
        for (int j = y; j <= m; j += j & (-j))
            c[i][j] += val;
}

int query(int x, int y) {
    int sum = 0;
    for (int i = x; i > 0; i -= i & (-i))
        for (int j = y; j > 0; j -= j & (-j))
            sum += c[i][j];
    return sum;
}

int range_query(int x1, int y1, int x2, int y2) {
    return query(x2, y2) - query(x1 - 1, y2)
         - query(x2, y1 - 1) + query(x1 - 1, y1 - 1);
}
```

**时间复杂度：** 修改和查询均为 $O(\log n \cdot \log m)$。

### 2.6 实用技巧

**$O(n)$ 建树：** 如果一开始有初始数组，不必对每个元素都调用 `update`（那样是 $O(n \log n)$），可以用如下方式 $O(n)$ 建树：

```cpp
for (int i = 1; i <= n; i++) {
    c[i] += a[i];
    int j = i + (i & -i);
    if (j <= n) c[j] += c[i];
}
```

**下标从 1 开始：** 树状数组的下标**必须从 $1$ 开始**，因为 `lowbit(0) = 0` 会导致死循环。如果原数据下标从 $0$ 开始，需要整体加 $1$。

**离散化：** 使用权值树状数组时，值域可能很大（如 $10^9$），需要先离散化到 $[1, n]$：

```cpp
vector<int> vals(a, a + n);
sort(vals.begin(), vals.end());
vals.erase(unique(vals.begin(), vals.end()), vals.end());
for (int i = 0; i < n; i++)
    a[i] = lower_bound(vals.begin(), vals.end(), a[i]) - vals.begin() + 1;
```

**BIT 上倍增求第 $k$ 小：** 在权值树状数组上，可以 $O(\log n)$ 求第 $k$ 小值：

```cpp
int kth(int k) {
    int pos = 0;
    for (int pw = 1 << __lg(n); pw; pw >>= 1) {
        if (pos + pw <= n && c[pos + pw] < k) {
            pos += pw;
            k -= c[pos];
        }
    }
    return pos + 1;
}
```

**时间戳优化：** 多组数据时，可以用时间戳避免每次清空数组：

```cpp
int c[MAXN], stamp[MAXN], cur_time = 0;

void reset() { cur_time++; }

void update(int x, int v) {
    for (; x <= n; x += x & -x) {
        if (stamp[x] != cur_time) { c[x] = 0; stamp[x] = cur_time; }
        c[x] += v;
    }
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) {
        if (stamp[x] == cur_time) s += c[x];
    }
    return s;
}
```

## 三、分类与变体详解

| 变体 | 维护内容 | 典型应用 |
|------|---------|---------|
| **基础树状数组** | 单点修改 + 前缀和查询 | 动态前缀和、逆序对计数 |
| **差分树状数组** | 区间修改 + 单点查询 | 区间加、差分维护 |
| **双树状数组** | 区间修改 + 区间查询 | 维护两个 BIT 实现完整区间操作 |
| **二维树状数组** | 矩阵单点修改 + 子矩阵和 | 二维前缀和动态维护 |
| **权值树状数组** | 值域上的计数 | 逆序对、第 $k$ 小、排名查询 |
| **时间戳树状数组** | 离线 + 排序后维护 | 区间不同元素个数（HH 的项链） |
| **树上树状数组** | DFS 序 + BIT | 子树修改与查询（Promotion Counting） |

**选型建议：**
- 如果只需要前缀和类的操作，优先用树状数组，代码短、常数小
- 如果需要区间最值、区间合并等复杂操作，则需要线段树
- 如果需要持久化，树状数组不适用，应使用可持久化线段树

**与线段树的对比：**

| 维度 | 树状数组 | 线段树 |
|------|---------|--------|
| 代码量 | 极短（10 行左右） | 较长（50+ 行） |
| 常数 | 非常小 | 较大（2~4 倍） |
| 功能 | 前缀和类操作 | 任意区间合并操作 |
| 区间最值 | 不直接支持 | 原生支持 |
| 持久化 | 不支持 | 支持 |
| 空间 | $O(n)$ | $O(4n)$ |

### 🟢 入门级 Level 1：基础模板与简单应用

### 题目 1：单点修改 + 区间求和模板（洛谷 P3374）

> **题意：** 给定一个长度为 $n$ 的数组，支持两种操作：单点加值、查询区间和。

```
数据结构：基础树状数组
支持操作：单点修改（update）、前缀查询（query）
关键性质：区间 [l,r] 的和 = query(r) - query(l-1)
时间复杂度：O(n + m log n)
```

**C++ 代码**：

```cpp
int c[500005], n;

void update(int x, int v) {
    for (; x <= n; x += x & -x) c[x] += v;
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

// 对于每个操作：
// 操作1: update(x, k)
// 操作2: printf("%d\n", query(r) - query(l - 1))
```

**核心启示：**

- **模板基石** — 树状数组的入门模板，`update` 向上跳（加 lowbit），`query` 向下跳（减 lowbit）
- **前缀和思想** — 区间和可以通过前缀和做差得到，这是所有 BIT 题目的基础

### 题目 2：区间修改 + 单点查询模板（洛谷 P3368）

> **题意：** 给定一个长度为 $n$ 的数组，支持两种操作：区间加值、单点查询。

```
数据结构：差分树状数组
支持操作：差分数组上的单点修改、前缀查询还原单点值
关键性质：差分数组 d[i] = a[i] - a[i-1]，前缀和还原原值
时间复杂度：O(n + m log n)
```

**C++ 代码**：

```cpp
int c[500005], n;

void update(int x, int v) {
    for (; x <= n; x += x & -x) c[x] += v;
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

// 初始化：对原数组 a[]，执行 update(i, a[i] - a[i-1])
// 区间 [l,r] 加 val: update(l, val); update(r+1, -val);
// 单点查询: query(x)
```

**核心启示：**

- **差分技巧** — 将"区间修改 + 单点查询"转化为"单点修改 + 前缀查询"
- **互为对偶** — P3374 和 P3368 是树状数组两种基本用法的对偶形式

### 题目 3：逆序对（洛谷 P1908）

> **题意：** 给定一个长度为 $n$ 的序列，求逆序对的数量。逆序对 $(i, j)$ 满足 $i < j$ 且 $a[i] > a[j]$。

```
数据结构：权值树状数组
支持操作：值域上单点加 1、前缀查询计数
关键性质：从右往左扫描，query(a[i]-1) 即为右边比 a[i] 小的数个数
时间复杂度：O(n log n)（含离散化）
```

**C++ 代码**：

```cpp
int c[500005], n;
long long ans = 0;

void update(int x) {
    for (; x <= n; x += x & -x) c[x]++;
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

// 离散化后，从右往左扫描
for (int i = n; i >= 1; i--) {
    ans += query(a[i] - 1);
    update(a[i]);
}
```

**核心启示：**

- **权值树状数组** — 在值域上建 BIT，统计"比某数大/小的元素个数"是经典技巧
- **扫描方向** — 从右往左扫描查 query(a[i]-1)，或从左往右扫描查 i - query(a[i])，两种方式等价

### 题目 4：星星（POJ 2352 / 洛谷变体）

> **题意：** 给定 $n$ 颗星星的坐标 $(x, y)$，按 $y$ 升序、$y$ 相同按 $x$ 升序给出。定义一颗星星的等级为左下方（含正左和正下）的星星个数。求每个等级的星星各有多少颗。

```
数据结构：基础树状数组
支持操作：单点修改（标记 x 坐标出现）、前缀查询（查 x 左边已有多少颗星）
关键性质：输入已按 y 升序排列，只需在 x 维度维护 BIT
时间复杂度：O(n log MAXX)
```

**C++ 代码**：

```cpp
int c[32005], level[15005];

void update(int x) {
    for (; x <= 32001; x += x & -x) c[x]++;
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

// 按输入顺序处理，x 坐标 +1 避免下标为 0
for (int i = 0; i < n; i++) {
    int x = star[i].x + 1;
    level[query(x)]++;   // 左下方星星个数即为等级
    update(x);
}
```

**核心启示：**

- **降维** — 输入已按 y 排序，只需在 x 维度上使用 BIT，将二维问题降为一维
- **下标偏移** — 坐标可能为 0，必须加 1 避免 lowbit(0) 死循环

### 题目 5：区间修改 + 区间查询（洛谷 P3372 变体）

> **题意：** 给定一个长度为 $n$ 的数组，支持两种操作：区间加值、查询区间和。

```
数据结构：双树状数组（差分 BIT）
支持操作：区间加值（同时更新 c1 和 c2）、前缀和查询
关键性质：prefix_sum(x) = (x+1)*query(c1,x) - query(c2,x)
时间复杂度：O(m log n)
```

**C++ 代码**：

```cpp
typedef long long ll;
int n;
ll c1[MAXN], c2[MAXN];

void upd(ll *c, int x, ll v) {
    for (; x <= n; x += x & -x) c[x] += v;
}

ll qry(ll *c, int x) {
    ll s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

void range_update(int l, int r, ll val) {
    upd(c1, l, val); upd(c1, r + 1, -val);
    upd(c2, l, val * l); upd(c2, r + 1, -val * (r + 1));
}

ll prefix_sum(int x) {
    return (ll)(x + 1) * qry(c1, x) - qry(c2, x);
}

ll range_query(int l, int r) {
    return prefix_sum(r) - prefix_sum(l - 1);
}
```

**核心启示：**

- **双 BIT 推导** — 通过交换求和顺序，将区间和分解为两个与差分数组相关的前缀和
- **功能等价** — 双树状数组可以完全替代带懒标记的线段树处理区间加 / 区间求和问题，且常数更小

### 题目 6：二维树状数组 — 子矩阵求和（洛谷 P4054）

> **题意：** 给定 $n \times m$ 的网格，每个格子有颜色。支持两种操作：修改某格颜色、查询子矩阵中某颜色出现次数。

```
数据结构：多个二维树状数组（每种颜色一个）
支持操作：二维单点修改、二维前缀和查询
关键性质：颜色种数有限（<=100），为每种颜色各开一个二维 BIT
时间复杂度：O(q * log n * log m)
```

**C++ 代码**：

```cpp
int c[105][305][305], n, m, col[305][305];

void update(int k, int x, int y, int v) {
    for (int i = x; i <= n; i += i & -i)
        for (int j = y; j <= m; j += j & -j)
            c[k][i][j] += v;
}

int query(int k, int x, int y) {
    int s = 0;
    for (int i = x; i > 0; i -= i & -i)
        for (int j = y; j > 0; j -= j & -j)
            s += c[k][i][j];
    return s;
}

// 修改: update(old_col, x, y, -1); update(new_col, x, y, 1);
// 查询: query(k,x2,y2) - query(k,x1-1,y2) - query(k,x2,y1-1) + query(k,x1-1,y1-1)
```

**核心启示：**

- **多 BIT 并行** — 当属性种类有限时，为每种属性各开一个 BIT，用空间换编码简洁性
- **二维容斥** — 子矩阵和 = 四个前缀和的容斥，与一维做差原理一致

### 题目 7：权值树状数组求第 k 小（洛谷 P1168 中位数）

> **题意：** 给定一个长度为 $n$ 的序列，对于每个奇数前缀 $a[1..2k-1]$，输出其中位数。

```
数据结构：权值树状数组 + BIT 上倍增
支持操作：值域上单点加 1、BIT 上倍增求第 k 小
关键性质：倍增求第 k 小时间为 O(log n)，优于二分的 O(log^2 n)
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
int c[MAXN], n, tot;

void update(int x, int v) {
    for (; x <= tot; x += x & -x) c[x] += v;
}

int kth(int k) {
    int pos = 0;
    for (int pw = 1 << __lg(tot); pw; pw >>= 1) {
        if (pos + pw <= tot && c[pos + pw] < k) {
            pos += pw;
            k -= c[pos];
        }
    }
    return pos + 1;
}

// 离散化后，依次插入元素
// 每插入奇数个元素后，输出 kth((i + 1) / 2) 对应的原始值
```

**核心启示：**

- **BIT 上倍增** — 从高位到低位逐步确定位置，$O(\log n)$ 找到第 $k$ 小，是权值 BIT 的核心技巧
- **动态中位数** — 在线维护中位数只需要在权值 BIT 上查第 $\lceil n/2 \rceil$ 小

### 题目 8：离散化 + 逆序对（洛谷 P1966 火柴排队）

> **题意：** 给定两个长度为 $n$ 的数组 $a$ 和 $b$，可以交换相邻元素，使得 $\sum (a_i - b_i)^2$ 最小，求最少交换次数（对 $10^8 - 3$ 取模）。

```
数据结构：权值树状数组
支持操作：值域上单点加 1、前缀查询计数
关键性质：排序不等式 => 排名对应 => 最少交换次数 = 逆序对数
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
int c[100005], n, p[100005];
long long ans = 0;
const int MOD = 1e8 - 3;

void update(int x) {
    for (; x <= n; x += x & -x) c[x]++;
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

// 构造 p[]：a 的第 k 小对应 b 的第 k 小的位置关系
// 从右往左求逆序对
for (int i = n; i >= 1; i--) {
    ans = (ans + query(p[i] - 1)) % MOD;
    update(p[i]);
}
```

**核心启示：**

- **问题转化** — "最少相邻交换次数"本质上就是排列的逆序对数
- **排序不等式** — 将最优化问题转化为排列问题，再用 BIT 求逆序对

### 题目 9：区间不同元素个数（洛谷 P1972 HH 的项链）

> **题意：** 给定长度为 $n$ 的序列和 $m$ 次查询，每次查询区间 $[l, r]$ 中有多少种不同的数。

```
数据结构：树状数组 + 离线排序
支持操作：单点修改（+1/-1）、前缀查询
关键性质：每种颜色只在"最后一次出现位置"标记 1，查询按右端点排序离线处理
时间复杂度：O((n + m) log n)
```

**C++ 代码**：

```cpp
int c[1000005], n, last[1000005];

void update(int x, int v) {
    for (; x <= n; x += x & -x) c[x] += v;
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

// 按右端点排序后扫描
for (int i = 1, j = 1; i <= n; i++) {
    if (last[a[i]]) update(last[a[i]], -1);
    update(i, 1);
    last[a[i]] = i;
    while (j <= m && qr[j].r == i)
        qr[j].ans = query(qr[j].r) - query(qr[j].l - 1), j++;
}
```

**核心启示：**

- **离线 + 扫描线** — 查询按右端点排序后，从左到右扫描序列，只保留每种颜色最后一次出现的位置
- **通用套路** — "区间不同元素个数"是树状数组离线处理的经典范式

### 题目 10：DFS 序 + 树状数组（USACO P3605 Promotion Counting）

> **题意：** 给定一棵有根树，每个节点有权值。对每个节点 $u$，求其子树中权值严格大于 $u$ 的权值的节点个数。

```
数据结构：权值树状数组 + DFS 序
支持操作：值域上单点加 1、前缀查询
关键性质：DFS 进出时的"差值"技巧 — 退出时的查询减去进入时的查询即为子树贡献
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
int c[100005], n, ans[100005], val[100005];

void update(int x) {
    for (; x <= n; x += x & -x) c[x]++;
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

void dfs(int u) {
    int before = query(n) - query(val[u]);
    for (int v : children[u]) dfs(v);
    ans[u] = query(n) - query(val[u]) - before;
    update(val[u]);
}
```

**核心启示：**

- **DFS 序降维** — 利用 DFS 的进出时间差，将子树统计问题转化为 BIT 上的前缀查询差值
- **无需显式展开** — 不需要显式计算 DFS 序区间，DFS 递归过程本身就天然提供了"进出差"

### 题目 11：三维偏序 CDQ 分治 + 树状数组（洛谷 P3810 陌上花开）

> **题意：** 给定 $n$ 个三元组 $(a_i, b_i, c_i)$，对每个元素求满足 $a_j \le a_i, b_j \le b_i, c_j \le c_i$ 的 $j$ 的个数（不含自身）。

```
数据结构：CDQ 分治 + 权值树状数组
支持操作：CDQ 消除第一维排序，BIT 处理第三维
关键性质：排序消第一维 => CDQ 消第二维 => BIT 消第三维
时间复杂度：O(n log^2 n)
```

**C++ 代码**：

```cpp
int c[200005], k; // k 为第三维值域上限

void update(int x, int v) {
    for (; x <= k; x += x & -x) c[x] += v;
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

// CDQ 分治核心
void cdq(int l, int r) {
    if (l == r) return;
    int mid = (l + r) >> 1;
    cdq(l, mid); cdq(mid + 1, r);
    // 左半按 b 排序，右半按 b 排序
    // 双指针：对右半每个元素，将左半 b <= 当前 b 的元素的 c 值 update 进 BIT
    // 查询 query(c_j) 累加到右半元素的答案
    // 清空 BIT（逆向 update 还原，避免 memset）
}
```

**核心启示：**

- **逐维消除** — 三维偏序的经典解法：排序消第一维、CDQ 分治消第二维、BIT 消第三维
- **BIT 清空技巧** — 用逆向 update 还原而非 memset，保证总复杂度 $O(n \log^2 n)$

### 题目 12：动态逆序对（洛谷 P3157 / BZOJ 3295）

> **题意：** 给定一个 $1$ 到 $n$ 的排列，依次删除 $m$ 个元素，每次删除前输出当前的逆序对数。

```
数据结构：CDQ 分治 + 树状数组（或分块 + BIT）
支持操作：时间维消除删除操作，BIT 处理值域统计
关键性质：倒序处理删除变为插入，三维偏序 (时间, 位置, 值)
时间复杂度：O(n log^2 n)
```

**C++ 代码**：

```cpp
int c[MAXN], n;
long long ans;

void update(int x, int v) {
    for (; x <= n; x += x & -x) c[x] += v;
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

// 思路：
// 1. 先求初始逆序对数 total
// 2. 倒序处理删除（即依次插入）
// 3. 每次插入元素 val 在位置 pos：
//    - 贡献的逆序对 = (pos左边比val大的) + (pos右边比val小的)
//    - 用两个 BIT 分别维护位置维度和值域维度
// 4. 或使用 CDQ 分治统一处理三维偏序
```

**核心启示：**

- **时间倒流** — 删除操作难以维护，倒序处理将删除变为插入，大幅简化问题
- **贡献拆分** — 每个元素的逆序对贡献可以拆分为"左大"和"右小"两部分独立统计

### 题目 13：树状数组上二分 — 列队（洛谷 P3960）

> **题意：** $n \times m$ 的方阵，每次操作让 $(x, y)$ 位置的人出列，其所在行后面的人左移补位，最后一列下面的人上移补位，出列的人补到最后一列末尾。求每次出列的人编号。

```
数据结构：树状数组 + BIT 上倍增定位第 k 个存在元素
支持操作：BIT 维护"存在性"、倍增求第 k 个 1 的位置
关键性质：每行前 m-1 列独立维护，最后一列单独维护，动态插入删除
时间复杂度：O(q log(n + q))
```

**C++ 代码**：

```cpp
int kth(int *c, int len, int k) {
    int pos = 0;
    for (int pw = 1 << 20; pw; pw >>= 1) {
        if (pos + pw <= len && c[pos + pw] < k) {
            pos += pw;
            k -= c[pos];
        }
    }
    return pos + 1;
}

void update(int *c, int x, int v, int len) {
    for (; x <= len; x += x & -x) c[x] += v;
}

// 初始化：每行 BIT 和最后一列 BIT 的位置全标 1
// 操作时：kth 定位 -> 记录编号 -> update 删除 -> update 插入末尾
```

**核心启示：**

- **BIT 上倍增** — $O(\log n)$ 找到第 $k$ 个存在元素，可替代平衡树处理部分动态排名问题
- **行列分离** — 将二维矩阵拆为多个一维 BIT 独立维护，避免二维 BIT 的空间爆炸

### 题目 14：区间出现次数大于等于 2 的颜色种数（洛谷 P4113 采花）

> **题意：** 给定长度为 $n$ 的序列，$m$ 次查询区间 $[l, r]$ 中出现次数 $\geq 2$ 的颜色种数。

```
数据结构：树状数组 + 离线排序
支持操作：单点修改（+1/-1）、前缀查询
关键性质：追踪每种颜色的"最后两次出现位置"，在倒数第二次出现位置标记
时间复杂度：O((n + m) log n)
```

**C++ 代码**：

```cpp
int c[MAXN], n, last1[MAXN], last2[MAXN];

void update(int x, int v) {
    for (; x <= n; x += x & -x) c[x] += v;
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

// 扫描到位置 i，颜色 col = a[i]
if (last2[col]) update(last2[col], -1);
if (last1[col]) update(last1[col], 1);
last2[col] = last1[col];
last1[col] = i;
// 处理右端点为 i 的查询: query(r) - query(l - 1)
```

**核心启示：**

- **推广套路** — 将"出现次数 >= k"转化为追踪"倒数第 k 次出现位置"，k=1 即 HH 的项链
- **标记转移** — 每次新出现时进行三步操作：移除旧标记、添加新标记、更新位置记录

### 题目 15：可重集合查询（洛谷 P3586 LOG）

> **题意：** 维护一个可重集合，支持三种操作：插入一个数、删除一个数、查询是否能选出 $c$ 个数使得每个数都不小于给定的 $s$。

```
数据结构：权值树状数组（双指标：个数 + 总和）
支持操作：值域上单点修改（个数和总和）、前缀查询
关键性质：先检查 >= s 的数个数是否够 c 个，不够则用 < s 的最大数补齐并检查总和
时间复杂度：O(n log^2 n) 或 O(n log n)（BIT 上二分）
```

**C++ 代码**：

```cpp
long long cc[MAXN], cs[MAXN]; // cnt 和 sum 两个 BIT
int n;

void upd(long long *t, int x, long long v) {
    for (; x <= n; x += x & -x) t[x] += v;
}

long long qry(long long *t, int x) {
    long long s = 0;
    for (; x; x -= x & -x) s += t[x];
    return s;
}

// 查询：能否选 c 个 >= s 的数
// cnt_ge = qry(cc, n) - qry(cc, s - 1)
// sum_ge = qry(cs, n) - qry(cs, s - 1)
// 若 cnt_ge >= c -> TAK
// 否则 need = c - cnt_ge, 判断 sum_ge + (前 need 大的 < s 的数之和) >= c*s
```

**核心启示：**

- **多指标 BIT** — 权值树状数组可以同时维护多个统计量（个数、和、平方和等）
- **贪心判定** — 优先选大数，不足时贪心选次大数补齐，用前缀和快速判定

### 题目 16：CDQ 分治 + 树状数组综合 — 多维偏序问题（洛谷 P3157 扩展 / HNOI）

> **题意：** 给定 $n$ 个元素，每个元素有多个属性维度。支持在线/离线的多维偏序统计，例如四维偏序或带修改的三维偏序。

```
数据结构：多层 CDQ 分治 + 树状数组
支持操作：逐维消除，BIT 处理最内层维度
关键性质：k 维偏序可用 k-2 层 CDQ + 排序 + BIT 解决，时间 O(n log^(k-1) n)
时间复杂度：O(n log^(k-1) n)（k 为维度数）
```

**C++ 代码**：

```cpp
int c[MAXN], K; // K 为最内层值域上限

void update(int x, int v) {
    for (; x <= K; x += x & -x) c[x] += v;
}

int query(int x) {
    int s = 0;
    for (; x; x -= x & -x) s += c[x];
    return s;
}

// 四维偏序框架：
// 1. 按第一维排序
// 2. CDQ 分治消第二维
//    2a. 在 CDQ 内部，再做一层 CDQ 消第三维
//        2a-i. 最内层用 BIT 消第四维
// 3. 每层 CDQ 结束后逆向清空 BIT

// 带修改三维偏序框架：
// 1. 将修改拆为"删除旧 + 插入新"，带时间戳
// 2. 按时间排序后，CDQ 消时间维
// 3. 内层排序消第一维，BIT 消第二维
```

**核心启示：**

- **分治嵌套** — CDQ 分治可以多层嵌套，每层消除一个维度，树状数组始终处理最内层
- **修改 = 删除 + 插入** — 带修改问题可以拆为删除旧值和插入新值，增加一个时间维度后变为更高维偏序
- **复杂度递推** — $k$ 维偏序复杂度为 $O(n \log^{k-1} n)$，实际中 $k \le 4$ 时可接受

## 五、代码框架

以下是一个通用的树状数组模板，支持单点修改和区间查询：

```cpp
template<typename T>
struct BIT {
    int n;
    vector<T> c;

    BIT() {}
    BIT(int _n) : n(_n), c(_n + 1, 0) {}

    void update(int x, T val) {
        for (; x <= n; x += x & (-x))
            c[x] += val;
    }

    T query(int x) {
        T sum = 0;
        for (; x > 0; x -= x & (-x))
            sum += c[x];
        return sum;
    }

    T query(int l, int r) {
        return query(r) - query(l - 1);
    }

    void build(vector<T>& a) {
        for (int i = 1; i <= n; i++) {
            c[i] += a[i];
            int j = i + (i & (-i));
            if (j <= n) c[j] += c[i];
        }
    }

    int kth(int k) {
        int pos = 0;
        for (int pw = 1 << __lg(n); pw; pw >>= 1) {
            if (pos + pw <= n && c[pos + pw] < k) {
                pos += pw;
                k -= c[pos];
            }
        }
        return pos + 1;
    }
};
```

**差分版本（区间修改 + 区间查询）框架：**

```cpp
template<typename T>
struct BIT_range {
    int n;
    vector<T> c1, c2;

    BIT_range(int _n) : n(_n), c1(_n + 2, 0), c2(_n + 2, 0) {}

    void _update(vector<T>& c, int x, T val) {
        for (; x <= n; x += x & (-x)) c[x] += val;
    }

    T _query(vector<T>& c, int x) {
        T s = 0;
        for (; x > 0; x -= x & (-x)) s += c[x];
        return s;
    }

    void update(int l, int r, T val) {
        _update(c1, l, val);
        _update(c1, r + 1, -val);
        _update(c2, l, val * l);
        _update(c2, r + 1, -val * (r + 1));
    }

    T query(int x) {
        return (x + 1) * _query(c1, x) - _query(c2, x);
    }

    T query(int l, int r) {
        return query(r) - query(l - 1);
    }
};
```

## 六、常见误区与调试技巧

| 误区 | 后果 | 正确做法 |
|------|------|---------|
| 下标从 0 开始 | `lowbit(0)=0` 导致死循环 | 下标统一从 1 开始 |
| 忘记离散化 | 值域过大导致 MLE/TLE | 权值 BIT 必须先离散化 |
| 区间修改忘记 `r+1` | 差分不完整，结果错误 | `update(l, val); update(r+1, -val)` |
| BIT 清空用 memset | CDQ 分治中导致复杂度退化 | 逆向 update 还原 |
| 二维 BIT 循环嵌套方向反 | 修改和查询不一致 | 修改和查询的两层循环方向必须对应 |
| kth 函数起始 pw 太小 | 找不到正确位置 | `pw = 1 << __lg(n)` 或 `pw = 1 << 20` |

## 七、总结

### 7.1 树状数组的本质

树状数组的核心在于 **lowbit 分组** — 利用二进制最低位将区间巧妙地划分和组织，使得修改和查询都只需要 $O(\log n)$ 次操作。它本质上是**前缀和的动态版本**，用最简洁的代码实现了最高效的前缀操作。

### 7.2 核心要点

1. **lowbit 是基石** — `x & (-x)` 提取最低位的 1，决定了节点管辖区间的长度
2. **方向相反** — update 向上跳（加 lowbit），query 向下跳（减 lowbit），方向永远相反
3. **差分是搭档** — 差分数组将"区间修改 + 单点查询"与"单点修改 + 前缀查询"互相转化
4. **权值 BIT 是利器** — 在值域上建 BIT，可处理逆序对、排名、第 k 小等一大类问题
5. **离线是钥匙** — 配合查询排序和扫描线，BIT 可以处理区间不同元素、出现次数等复杂统计
6. **CDQ 是盟友** — 树状数组作为 CDQ 分治的最内层，是多维偏序问题的标准解法
