## 一、什么是ST表

ST表（Sparse Table，稀疏表）是一种用于解决**静态区间查询问题**的数据结构。它能在 $O(n \log n)$ 时间预处理后，以 $O(1)$ 回答每次区间最值查询（RMQ）。

```
数组: [3, 1, 4, 1, 5, 9, 2, 6]

query(2, 5) = min(1, 4, 1, 5) = 1  ← O(1) 完成！

预处理 O(n log n)，查询 O(1)
```

**核心特点：**

| 操作 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 预处理 | $O(n \log n)$ | $O(n \log n)$ |
| 单次查询 | $O(1)$ | — |

**适用条件：**
1. **不支持修改**：数据是静态的
2. **可重复贡献**：运算满足 $\text{opt}(a, a) = a$，如 $\min$、$\max$、$\gcd$、按位或/与。**区间求和不适用**（重叠会重复计算）

## 二、核心思想与前置知识

### 2.1 倍增思想

用 $f[i][j]$ 表示从下标 $i$ 开始、长度为 $2^j$ 的区间的最值：

$$f[i][j] = \text{opt}(f[i][j-1],\ f[i + 2^{j-1}][j-1])$$

一个长度 $2^j$ 的区间 = 两个长度 $2^{j-1}$ 的子区间合并。

```
f[i][j] 覆盖区间 [i, i + 2^j - 1]

j=0: [i]          长度 1
j=1: [i, i+1]     长度 2
j=2: [i, i+3]     长度 4
j=3: [i, i+7]     长度 8

f[3][2] = opt(f[3][1], f[5][1])
        = opt(区间[3,4], 区间[5,6])
        = 区间[3,6] 的最值
```

### 2.2 预处理

```cpp
const int N = 100010, LOG = 17;
int f[N][LOG + 1], lg[N];

void initLog(int n) {
    lg[1] = 0;
    for (int i = 2; i <= n; i++) lg[i] = lg[i / 2] + 1;
}

void build(int a[], int n) {
    for (int i = 1; i <= n; i++) f[i][0] = a[i];
    for (int j = 1; j <= lg[n]; j++)
        for (int i = 1; i + (1 << j) - 1 <= n; i++)
            f[i][j] = max(f[i][j - 1], f[i + (1 << (j - 1))][j - 1]);
}
```

### 2.3 $O(1)$ 查询

$$\text{query}(l, r) = \text{opt}(f[l][k],\ f[r - 2^k + 1][k])$$

其中 $k = \lfloor \log_2(r - l + 1) \rfloor$。两个子区间可能重叠，但运算的可重复贡献性质保证结果正确。

```cpp
int query(int l, int r) {
    int k = lg[r - l + 1];
    return max(f[l][k], f[r - (1 << k) + 1][k]);
}
```

### 2.4 对数表预处理

用 `lg[]` 数组代替 `log2()` 函数，避免浮点误差和性能损耗：

```cpp
lg[1] = 0;
for (int i = 2; i <= n; i++) lg[i] = lg[i / 2] + 1;
// 或使用 __lg(x)（GCC 内置函数）
```

## 三、分类与变体详解

### 3.1 RMQ（区间最值查询）

ST 表最经典的应用。支持区间最大值和最小值。

```cpp
// 区间最小值：将 max 改为 min
f[i][j] = min(f[i][j-1], f[i + (1 << (j-1))][j-1]);
```

### 3.2 区间 GCD

$\gcd$ 满足可重复贡献性质：$\gcd(a, a) = a$。直接将 `max` 替换为 `__gcd`。

```cpp
f[i][j] = __gcd(f[i][j-1], f[i + (1 << (j-1))][j-1]);
```

### 3.3 区间按位或/与

按位或和按位与也满足可重复贡献，可以用 ST 表。

### 3.4 二维 ST 表

在二维矩阵上进行子矩阵最值查询。$f[i][j][k][l]$ 表示左上角 $(i,j)$、行方向长 $2^k$、列方向长 $2^l$ 的子矩阵最值。预处理 $O(nm \log n \log m)$，查询 $O(1)$。

### 3.5 ST 表 vs 其他数据结构

| 数据结构 | 预处理 | 单次查询 | 是否支持修改 |
|---------|--------|---------|------------|
| **ST 表** | $O(n \log n)$ | $O(1)$ | 否 |
| 线段树 | $O(n)$ | $O(\log n)$ | 是 |
| 树状数组 | $O(n)$ | $O(\log n)$ | 是（仅前缀） |
| 分块 | $O(n)$ | $O(\sqrt{n})$ | 是 |

> ST 表的优势是**查询 $O(1)$**，适合查询密集、数据不变的场景。

### 题目 1：ST 表模板（洛谷 P3865）

> 给定 $n$ 个数和 $m$ 次查询，每次查询区间 $[l, r]$ 的最大值。

**分析**：ST 表裸模板题。预处理 $O(n \log n)$，查询 $O(1)$。

```
数据结构：ST 表（区间最大值）
支持操作：O(1) 区间最大值查询
关键性质：可重复贡献 → 重叠区间不影响 max 结果
时间复杂度：预处理 O(n log n)，查询 O(1)
```

**C++ 代码**：

```cpp
int f[N][LOG + 1], lg[N];

void build(int a[], int n) {
    for (int i = 2; i <= n; i++) lg[i] = lg[i / 2] + 1;
    for (int i = 1; i <= n; i++) f[i][0] = a[i];
    for (int j = 1; (1 << j) <= n; j++)
        for (int i = 1; i + (1 << j) - 1 <= n; i++)
            f[i][j] = max(f[i][j-1], f[i + (1 << (j-1))][j-1]);
}

int query(int l, int r) {
    int k = lg[r - l + 1];
    return max(f[l][k], f[r - (1 << k) + 1][k]);
}
```

**核心启示**：
- **ST 表三步**：预处理对数表 → 建 ST 表 → $O(1)$ 查询
- **外层枚举 $j$（幂次）**：保证递推时依赖的 $j-1$ 层已计算完毕

### 题目 2：质量检测（洛谷 P2251）

> 给定 $n$ 个产品的质量值和窗口大小 $m$，输出每个窗口的最小值。

**分析**：滑动窗口最小值。虽然单调队列也能做，但 ST 表更简洁——每个窗口直接 $O(1)$ 查询。

```
数据结构：ST 表（区间最小值）
支持操作：O(1) 查询 [i, i+m-1] 的最小值
关键性质：数据不修改，查询密集 → ST 表比单调队列代码更简洁
时间复杂度：O(n log n) 预处理 + O(n) 查询
```

**C++ 代码**：

```cpp
int f[N][LOG + 1];

void build(int a[], int n) {
    for (int i = 1; i <= n; i++) f[i][0] = a[i];
    for (int j = 1; (1 << j) <= n; j++)
        for (int i = 1; i + (1 << j) - 1 <= n; i++)
            f[i][j] = min(f[i][j-1], f[i + (1 << (j-1))][j-1]);
}

int query(int l, int r) {
    int k = lg[r - l + 1];
    return min(f[l][k], f[r - (1 << k) + 1][k]);
}

void solve(int n, int m) {
    for (int i = 1; i + m - 1 <= n; i++)
        printf("%d\n", query(i, i + m - 1));
}
```

**核心启示**：
- **ST 表 vs 单调队列**：滑动窗口最值两者都能做，ST 表代码更短但多用 $O(n \log n)$ 空间
- **只改 `max` 为 `min`**：ST 表切换最大/最小只需改一个运算符

### 题目 4：Balanced Lineup（洛谷 P2880 / POJ 3264）

> $n$ 头牛身高，$q$ 次查询区间 $[l, r]$ 的最大值与最小值之差。

**分析**：同时建最大值和最小值两个 ST 表，查询时分别取 max 和 min 做差。

```
数据结构：双 ST 表（区间最大 + 区间最小）
支持操作：O(1) 查区间最大和区间最小
关键性质：max 和 min 都满足可重复贡献
时间复杂度：预处理 O(n log n)，查询 O(1)
```

**C++ 代码**：

```cpp
int fmax[N][LOG + 1], fmin[N][LOG + 1];

void build(int a[], int n) {
    for (int i = 1; i <= n; i++) fmax[i][0] = fmin[i][0] = a[i];
    for (int j = 1; (1 << j) <= n; j++)
        for (int i = 1; i + (1 << j) - 1 <= n; i++) {
            fmax[i][j] = max(fmax[i][j-1], fmax[i + (1 << (j-1))][j-1]);
            fmin[i][j] = min(fmin[i][j-1], fmin[i + (1 << (j-1))][j-1]);
        }
}

int query(int l, int r) {
    int k = lg[r - l + 1];
    int mx = max(fmax[l][k], fmax[r - (1 << k) + 1][k]);
    int mn = min(fmin[l][k], fmin[r - (1 << k) + 1][k]);
    return mx - mn;
}
```

**核心启示**：
- **双 ST 表**：需要同时查最大和最小就建两个，空间 $2n \log n$
- **区间极差 = max - min**：在 $O(1)$ 内完成

### 题目 5：区间 GCD（洛谷 P1890 类型）

> 给定 $n$ 个数和 $q$ 次查询，求区间 $[l, r]$ 的最大公约数。

**分析**：$\gcd$ 满足可重复贡献性质：$\gcd(a, a) = a$。将 ST 表的运算改为 `__gcd` 即可。

```
数据结构：ST 表（区间 GCD）
支持操作：O(1) 查询区间 GCD
关键性质：gcd(a, a) = a → 可重复贡献 → 适用 ST 表
时间复杂度：预处理 O(n log n · log V)，查询 O(log V)（V 为值域）
```

**C++ 代码**：

```cpp
int f[N][LOG + 1];

void build(int a[], int n) {
    for (int i = 1; i <= n; i++) f[i][0] = a[i];
    for (int j = 1; (1 << j) <= n; j++)
        for (int i = 1; i + (1 << j) - 1 <= n; i++)
            f[i][j] = __gcd(f[i][j-1], f[i + (1 << (j-1))][j-1]);
}

int query(int l, int r) {
    int k = lg[r - l + 1];
    return __gcd(f[l][k], f[r - (1 << k) + 1][k]);
}
```

**核心启示**：
- **可重复贡献运算替换**：ST 表不仅能求 max/min，所有可重复贡献运算都适用
- **GCD 的额外 log**：`__gcd` 本身是 $O(\log V)$ 的，总预处理为 $O(n \log n \cdot \log V)$

### 题目 6：最大数（洛谷 P1198）

> 一个初始为空的数列，支持两种操作：在末尾添加一个数；查询最后 $L$ 个数的最大值。强制在线。

**分析**：由于只在末尾添加（不修改），可以动态维护 ST 表——每次添加时更新新位置的 $f$ 值。

```
数据结构：动态 ST 表（在线构建）
支持操作：尾部追加 + O(1) 查询后缀最大值
关键性质：只追加不修改 → 新元素的 ST 表只需向上递推
时间复杂度：每次追加 O(log n)，查询 O(1)
```

**C++ 代码**：

```cpp
int f[N][LOG + 1], cnt = 0;

void append(int x) {
    cnt++;
    f[cnt][0] = x;
    for (int j = 1; (1 << j) <= cnt; j++) {
        int i = cnt - (1 << j) + 1;
        if (i >= 1) f[i][j] = max(f[i][j-1], f[i + (1 << (j-1))][j-1]);
    }
}

int query(int L) { // 查询最后 L 个数的最大值
    int l = cnt - L + 1, r = cnt;
    int k = lg[r - l + 1];
    return max(f[l][k], f[r - (1 << k) + 1][k]);
}
```

**核心启示**：
- **动态 ST 表**：只在尾部追加时，ST 表可以在线维护
- **每次追加更新 $O(\log n)$**：只需更新以新位置结尾的所有 $f$ 值

### 题目 7：LCA 的 ST 表解法（洛谷 P3379）

> 给定一棵 $n$ 个节点的树和 $m$ 次查询，每次求两个节点的最近公共祖先（LCA）。

**分析**：欧拉序 + ST 表。DFS 生成欧拉序（每个节点在进出时都记录），记录每个节点首次出现的位置。LCA = 两节点首次出现位置之间深度最小的节点。用 ST 表 $O(1)$ 查询区间最小深度。

```
数据结构：欧拉序 + ST 表（区间最小深度位置）
支持操作：O(1) 求 LCA
关键性质：LCA(u,v) = 欧拉序中 [first[u], first[v]] 深度最小的节点
时间复杂度：预处理 O(n log n)，查询 O(1)
```

**C++ 代码**：

```cpp
int euler[2 * N], dep[2 * N], first[N], tot;
int f[2 * N][LOG + 1]; // ST 表存欧拉序中深度最小的下标

void dfs(int u, int fa, int d) {
    euler[++tot] = u; dep[tot] = d; first[u] = tot;
    for (int i = head[u]; ~i; i = ne[i]) {
        int v = e[i];
        if (v == fa) continue;
        dfs(v, u, d + 1);
        euler[++tot] = u; dep[tot] = d;
    }
}

void buildLCA(int n) {
    for (int i = 1; i <= tot; i++) f[i][0] = i;
    for (int j = 1; (1 << j) <= tot; j++)
        for (int i = 1; i + (1 << j) - 1 <= tot; i++) {
            int a = f[i][j-1], b = f[i + (1 << (j-1))][j-1];
            f[i][j] = dep[a] < dep[b] ? a : b;
        }
}

int lca(int u, int v) {
    int l = first[u], r = first[v];
    if (l > r) swap(l, r);
    int k = lg[r - l + 1];
    int a = f[l][k], b = f[r - (1 << k) + 1][k];
    return euler[dep[a] < dep[b] ? a : b];
}
```

**核心启示**：
- **欧拉序将 LCA 转化为 RMQ**：树上 LCA 问题可以在 $O(1)$ 内回答
- **ST 表存下标**：比较时用深度 `dep[]`，返回时用欧拉序 `euler[]`

### 题目 8：最大子矩阵（二维 ST 表）

> 给定 $n \times m$ 矩阵和多次查询，每次查询给定边长 $a \times b$ 的子矩阵的最大值。

**分析**：二维 ST 表。$f[i][j][k][l]$ 表示左上角 $(i,j)$、行方向 $2^k$、列方向 $2^l$ 的子矩阵最值。查询用 4 个重叠子矩阵覆盖。

```
数据结构：二维 ST 表
支持操作：O(1) 查询子矩阵最值
关键性质：行和列方向各自可重复贡献
时间复杂度：预处理 O(nm log n · log m)，查询 O(1)
```

**C++ 代码**：

```cpp
int f[N][N][LOG][LOG]; // f[i][j][k][l]

void build2D(int a[][N], int n, int m) {
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++) f[i][j][0][0] = a[i][j];
    // 先沿列方向倍增
    for (int l = 1; (1 << l) <= m; l++)
        for (int i = 1; i <= n; i++)
            for (int j = 1; j + (1 << l) - 1 <= m; j++)
                f[i][j][0][l] = max(f[i][j][0][l-1], f[i][j + (1<<(l-1))][0][l-1]);
    // 再沿行方向倍增
    for (int k = 1; (1 << k) <= n; k++)
        for (int l = 0; (1 << l) <= m; l++)
            for (int i = 1; i + (1 << k) - 1 <= n; i++)
                for (int j = 1; j + (1 << l) - 1 <= m; j++)
                    f[i][j][k][l] = max(f[i][j][k-1][l], f[i + (1<<(k-1))][j][k-1][l]);
}

int query2D(int r1, int c1, int r2, int c2) {
    int k = lg[r2 - r1 + 1], l = lg[c2 - c1 + 1];
    return max({f[r1][c1][k][l], f[r2-(1<<k)+1][c1][k][l],
                f[r1][c2-(1<<l)+1][k][l], f[r2-(1<<k)+1][c2-(1<<l)+1][k][l]});
}
```

**核心启示**：
- **二维 ST 表**：行和列各自独立倍增，查询用 4 个重叠子矩阵
- **空间大**：$O(nm \log n \cdot \log m)$，仅适合矩阵较小的情况

### 🟠 提高级 Level 3：ST 表在综合问题中的应用

### 题目 9：超级钢琴（洛谷 P2048）

> 长度 $n$ 的序列，选 $k$ 个长度 $\in [L, R]$ 的子段，使子段和的总和最大。

**分析**：前缀和 + ST 表 + 堆。对每个起点 $i$，最优终点 = $\arg\max_{j \in [i+L-1, i+R-1]} \text{pre}[j]$，用 ST 表 $O(1)$ 查询。堆取全局最优后区间分裂。

```
数据结构：ST 表（前缀和最大值位置）+ 大根堆
支持操作：ST 表 O(1) 查区间前缀和最大位置，堆取全局最优
关键性质：堆+区间分裂实现"第 k 大子段和"
时间复杂度：O((n+k) log n)
```

**C++ 代码**：

```cpp
long long pre[N];
int st[N][LOG + 1]; // ST 表存 pre[] 最大值的位置

int qmax(int l, int r) {
    int k = lg[r - l + 1];
    return pre[st[l][k]] >= pre[st[r-(1<<k)+1][k]] ? st[l][k] : st[r-(1<<k)+1][k];
}

struct Item {
    int i, lo, hi, best;
    bool operator<(const Item& o) const { return pre[best] - pre[i-1] < pre[o.best] - pre[o.i-1]; }
};

long long solve(int n, int k, int L, int R) {
    // 建 ST 表
    for (int i = 1; i <= n; i++) st[i][0] = i;
    for (int j = 1; (1 << j) <= n; j++)
        for (int i = 1; i + (1 << j) - 1 <= n; i++) {
            int a = st[i][j-1], b = st[i+(1<<(j-1))][j-1];
            st[i][j] = pre[a] >= pre[b] ? a : b;
        }
    priority_queue<Item> pq;
    for (int i = 1; i + L - 1 <= n; i++) {
        int lo = i + L - 1, hi = min(n, i + R - 1);
        pq.push({i, lo, hi, qmax(lo, hi)});
    }
    long long ans = 0;
    while (k--) {
        auto [i, lo, hi, best] = pq.top(); pq.pop();
        ans += pre[best] - pre[i - 1];
        if (best > lo) pq.push({i, lo, best - 1, qmax(lo, best - 1)});
        if (best < hi) pq.push({i, best + 1, hi, qmax(best + 1, hi)});
    }
    return ans;
}
```

**核心启示**：
- **ST 表 + 堆 + 区间分裂**：经典组合，用于"第 k 大子段和"类问题
- **ST 表存位置而非值**：查询返回最大值的下标，方便进行区间分裂

### 题目 10：Frequent Values（POJ 1368 / UVA 11235）

> 非递减序列，多次查询区间 $[l, r]$ 中出现次数最多的元素的出现次数。

**分析**：非递减序列中相同元素连续。将序列分成若干段（相同元素为一段），每段记录长度。对段长数组建 ST 表。查询时将 $l$ 和 $r$ 所在段的边界部分特殊处理，中间完整段用 ST 表查最大值。

```
数据结构：ST 表（段长数组最大值）
支持操作：O(1) 查询中间完整段的最大长度
关键性质：非递减 → 相同值连续成段 → 段间 ST 表
时间复杂度：预处理 O(n log n)，查询 O(1)
```

**C++ 代码**：

```cpp
int belong[N], segLen[N], segL[N], segR[N]; // 每个位置所属段、段长、段左右端
int f[N][LOG + 1]; // ST 表建在段长上
int numSeg;

void preprocess(int a[], int n) {
    numSeg = 0;
    for (int i = 1; i <= n; ) {
        int j = i;
        while (j <= n && a[j] == a[i]) j++;
        numSeg++;
        segLen[numSeg] = j - i;
        for (int k = i; k < j; k++) { belong[k] = numSeg; segL[k] = i; segR[k] = j - 1; }
        i = j;
    }
    // 对 segLen 建 ST 表
    for (int i = 1; i <= numSeg; i++) f[i][0] = segLen[i];
    for (int j = 1; (1 << j) <= numSeg; j++)
        for (int i = 1; i + (1 << j) - 1 <= numSeg; i++)
            f[i][j] = max(f[i][j-1], f[i + (1 << (j-1))][j-1]);
}

int query(int l, int r) {
    int bl = belong[l], br = belong[r];
    if (bl == br) return r - l + 1;
    int ans = max(segR[l] - l + 1, r - segL[r] + 1); // 两端不完整段
    if (bl + 1 <= br - 1) { // 中间完整段
        int k = lg[br - 1 - (bl + 1) + 1];
        ans = max(ans, max(f[bl+1][k], f[br-1-(1<<k)+1][k]));
    }
    return ans;
}
```

**核心启示**：
- **分段 + ST 表**：将问题空间从元素级别降到段级别
- **边界特殊处理**：首尾不完整的段需要手动计算长度

### 题目 12：Cartesian Tree 构造中的 RMQ

> 给定序列 $a[1..n]$，构造笛卡尔树。笛卡尔树的根是区间最小值，左右子树递归构造。

**分析**：笛卡尔树可以用栈 $O(n)$ 构造，但也可以用 ST 表辅助分治构造：`root = query_min(l, r)`，然后递归左右子区间。ST 表保证每次找最小值 $O(1)$。

```
数据结构：ST 表（区间最小值位置）+ 递归分治
支持操作：O(1) 找区间最小值位置 → 作为子树根
关键性质：笛卡尔树根 = 区间最小值，递归二分
时间复杂度：O(n log n) 预处理 + O(n) 构造
```

**C++ 代码**：

```cpp
int st[N][LOG + 1]; // 存最小值位置

int qmin_pos(int l, int r) {
    int k = lg[r - l + 1];
    return a[st[l][k]] <= a[st[r-(1<<k)+1][k]] ? st[l][k] : st[r-(1<<k)+1][k];
}

int build(int l, int r) {
    if (l > r) return 0;
    int root = qmin_pos(l, r);
    // left_child[root] = build(l, root - 1);
    // right_child[root] = build(root + 1, r);
    return root;
}
```

**核心启示**：
- **RMQ 辅助树构造**：ST 表让分治式构造笛卡尔树变得简单
- **栈 vs ST 表**：栈构造 $O(n)$ 但不直观，ST 表分治更好理解

### 题目 13：±1 RMQ 与 LCA（Bender-Farach-Colton 算法思想）

> 欧拉序的相邻元素深度差恰好为 $\pm 1$。利用这个性质可以将 RMQ 进一步优化到 $O(n)$ 预处理。

**分析**：将长度 $2n$ 的欧拉序分成 $2n / \lfloor \frac{\log n}{2} \rfloor$ 个块。块间用 ST 表，块内由于 $\pm 1$ 性质，所有可能的块只有 $O(\sqrt{n})$ 种，可以预处理所有块类型的查询表。

```
数据结构：分块 + ST 表 + 预处理块查询表
支持操作：O(1) RMQ（预处理 O(n)）
关键性质：±1 RMQ 的块内查询可以枚举所有 2^(block_size) 种模式
时间复杂度：预处理 O(n)，查询 O(1)
```

**C++ 代码**：

```cpp
// 核心思想（伪代码）
// 1. 将欧拉序分块，块大小 B = log(n) / 2
// 2. 块间最小值用 ST 表
// 3. 块内：由于±1性质，用 bitmask 表示块的形状
//    共 2^B 种形状，每种预处理所有 [i,j] 查询
// 4. 查询：块间 ST 表 + 两端块内查表
```

**核心启示**：
- **$O(n)$ 预处理 RMQ**：理论最优的 RMQ 算法，基于 $\pm 1$ 性质
- **竞赛中不常用**：$O(n \log n)$ 的普通 ST 表已经足够快，此方法主要有理论价值

### 题目 14：逛森林（洛谷 P5344 类型）

**核心启示**：

- **路径拆分**：路径 $(u, v)$ = 路径 $(u, \text{lca})$ + 路径 $(\text{lca}, v)$

### 🟣 竞赛级 Level 5：极致应用

### 题目 16：Disjoint ST 表（不可重复贡献的扩展）

> 对于区间求和等不满足可重复贡献的运算，ST 表不能直接使用。Disjoint Sparse Table 通过修改查询方式，使得每个元素恰好被贡献一次。

**分析**：Disjoint ST 表的构建方式不同于普通 ST 表。对于第 $j$ 层，将序列按 $2^j$ 分块，每个块的中点向两侧分别计算前缀和/后缀和。查询时确定 $l$ 和 $r$ 所在的最高不同块层级，用该层的前后缀信息合并。

```
数据结构：Disjoint Sparse Table
支持操作：O(1) 查询区间和/区间积等不可重复贡献运算
关键性质：查询时 l 和 r 分属不同半块 → 不重复贡献
时间复杂度：预处理 O(n log n)，查询 O(1)
```

**C++ 代码**：

```cpp
long long f[LOG + 1][N]; // f[j][i]

void build(int a[], int n) {
    for (int j = 0; (1 << j) <= n; j++) {
        int len = 1 << j;
        for (int mid = len; mid <= n; mid += 2 * len) {
            // 从 mid 向左累积
            f[j][mid] = a[mid];
            for (int i = mid - 1; i >= max(1, mid - len + 1); i--)
                f[j][i] = f[j][i + 1] + a[i]; // 以 + 为例
            // 从 mid+1 向右累积
            if (mid + 1 <= n) f[j][mid + 1] = a[mid + 1];
            for (int i = mid + 2; i <= min(n, mid + len); i++)
                f[j][i] = f[j][i - 1] + a[i];
        }
    }
}

long long query(int l, int r) {
    if (l == r) return a[l];
    int k = lg[l ^ r]; // 最高不同位
    return f[k][l] + f[k][r];
}
```

**核心启示**：
- **Disjoint ST 表**：突破了"可重复贡献"的限制，支持区间和、区间积等运算
- **异或找层级**：`l ^ r` 的最高位确定了 `l` 和 `r` 分属不同半块的层级
- **竞赛冷门但优雅**：当需要 $O(1)$ 查询区间和且不带修改时，比前缀和更通用

## 五、常见技巧与思路总结

### 5.1 对数表预处理

```cpp
lg[1] = 0;
for (int i = 2; i <= n; i++) lg[i] = lg[i / 2] + 1;
```

避免使用 `log2()` 函数的浮点误差。也可用 GCC 内置 `__lg(x)`。

### 5.2 ST 表存下标 vs 存值

- 存**值**：适合简单最值查询
- 存**下标**：需要知道最值位置时（如超级钢琴、笛卡尔树）

### 5.3 可重复贡献性质判断

运算 $\text{opt}$ 满足：$\text{opt}(a, a) = a$。
- ✅ min, max, gcd, 按位或, 按位与
- ❌ 加法, 乘法, 异或

### 5.4 内存优化

$O(n \log n)$ 可能很大。若 $n$ 达到 $10^7$，考虑 `short` 或分块替代。

### 5.5 在线 vs 离线

ST 表预处理后可在线回答。但若需要修改，应使用线段树或树状数组。

## 七、终极总结与归纳

### 7.1 ST 表的本质

**ST 表 = 倍增预处理 + 可重复贡献查询，$O(n \log n)$ 预处理换 $O(1)$ 查询。**

它是静态 RMQ 的最优实用解法，也是 LCA、笛卡尔树、超级钢琴等高级问题的核心辅助工具。

### 7.2 核心要点

1. **倍增递推**：$f[i][j] = \text{opt}(f[i][j-1], f[i+2^{j-1}][j-1])$，外层枚举 $j$
2. **可重复贡献**：min/max/gcd 适用，sum/product 不适用（用 Disjoint ST 表）
3. **查询公式**：$k = \lfloor \log_2(r-l+1) \rfloor$，答案 = $\text{opt}(f[l][k], f[r-2^k+1][k])$
4. **对数表预处理**：`lg[i] = lg[i/2] + 1`，避免浮点误差
5. **欧拉序 + ST 表**：将 LCA 转化为 RMQ，$O(1)$ 在线查询
6. **ST 表存下标**："超级钢琴"类问题需要知道最优位置而非最优值
