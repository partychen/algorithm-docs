## 一、什么是分块

**分块**（Sqrt Decomposition）是一种将长度为 $n$ 的序列划分为 $\sqrt{n}$ 个大小约为 $\sqrt{n}$ 的连续块，从而在 **暴力** 与 **高级数据结构** 之间取得平衡的算法技巧。

核心理念：**优雅的暴力（Brute Force with Elegance）**。

| 特征 | 说明 |
|------|------|
| 块大小 | $B = \lfloor\sqrt{n}\rfloor$ |
| 块数量 | $\lceil n / B \rceil \approx \sqrt{n}$ |
| 单次操作复杂度 | 通常 $O(\sqrt{n})$ |
| 编码难度 | 远低于线段树/平衡树 |
| 适用场景 | 区间修改查询、值域统计、离线问题等 |

分块并不追求最优渐进复杂度，而是以 **易于实现、常数极小、思路灵活** 著称，是竞赛中不可或缺的"万金油"工具。

## 二、核心思想与前置知识

### 2.1 分块基本思想

将序列 $a[1..n]$ 分为若干块，每块长度 $B$。对于第 $i$ 个元素，其所属块编号为 $\text{bel}[i] = \lfloor (i-1)/B \rfloor + 1$。

**预处理**：

```cpp
int B, bel[MAXN];
int L[MAXB], R[MAXB], tot; // 每块的左右端点、块数

void init(int n) {
    B = sqrt(n);
    tot = (n + B - 1) / B;
    for (int i = 1; i <= n; i++) bel[i] = (i - 1) / B + 1;
    for (int i = 1; i <= tot; i++) {
        L[i] = (i - 1) * B + 1;
        R[i] = min(i * B, n);
    }
}
```

**查询/修改**思路：
- 如果操作区间 $[l, r]$ 在同一块内 → **暴力处理**，复杂度 $O(B) = O(\sqrt{n})$。
- 否则 → 左右两端的 **不完整块暴力**，中间的 **整块批量处理**，复杂度 $O(B + n/B) = O(\sqrt{n})$。

### 2.2 区间修改 + 区间查询

以"区间加、区间求和"为例：

- 对每个块维护一个 **懒标记** $\text{add}[b]$ 和 **块内和** $\text{sum}[b]$。
- **区间加 $[l,r]$ 加 $v$**：不完整块逐元素修改并更新块内和；整块直接修改懒标记。
- **区间求和 $[l,r]$**：不完整块逐元素累加（记得加上懒标记）；整块直接用 $\text{sum}[b] + \text{add}[b] \times \text{len}$。

```cpp
long long a[MAXN], sum[MAXB], add[MAXB];

void range_add(int l, int r, long long v) {
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) a[i] += v, sum[bel[i]] += v;
        return;
    }
    for (int i = l; i <= R[bel[l]]; i++) a[i] += v, sum[bel[i]] += v;
    for (int i = bel[l] + 1; i < bel[r]; i++) add[i] += v;
    for (int i = L[bel[r]]; i <= r; i++) a[i] += v, sum[bel[i]] += v;
}

long long range_sum(int l, int r) {
    long long res = 0;
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) res += a[i] + add[bel[i]];
        return res;
    }
    for (int i = l; i <= R[bel[l]]; i++) res += a[i] + add[bel[i]];
    for (int i = bel[l] + 1; i < bel[r]; i++) res += sum[i] + add[i] * (R[i] - L[i] + 1);
    for (int i = L[bel[r]]; i <= r; i++) res += a[i] + add[bel[i]];
    return res;
}
```

### 2.3 块内排序（值域分块）

有时需要对每个块内的元素排序，以支持 **块内二分** 查询（如区间内小于某个值的元素个数）。

```cpp
vector<int> blk[MAXB]; // 每块排序后的副本

void build(int b) {
    blk[b].clear();
    for (int i = L[b]; i <= R[b]; i++) blk[b].push_back(a[i]);
    sort(blk[b].begin(), blk[b].end());
}

// 查询 [l,r] 中小于 v 的元素个数
int query_less(int l, int r, int v) {
    int res = 0;
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++)
            if (a[i] + add[bel[i]] < v) res++;
        return res;
    }
    for (int i = l; i <= R[bel[l]]; i++)
        if (a[i] + add[bel[i]] < v) res++;
    for (int i = bel[l] + 1; i < bel[r]; i++)
        res += lower_bound(blk[i].begin(), blk[i].end(), v - add[i]) - blk[i].begin();
    for (int i = L[bel[r]]; i <= r; i++)
        if (a[i] + add[bel[i]] < v) res++;
    return res;
}
```

### 2.4 莫队算法简介

**莫队算法**（Mo's Algorithm）是一种基于分块思想的 **离线** 区间查询算法。将所有查询按 $(\lfloor l/B \rfloor, r)$ 排序后，利用双指针逐步扩展/收缩区间，均摊复杂度 $O(n\sqrt{n})$。

```cpp
struct Query {
    int l, r, id;
};

bool cmp(const Query &a, const Query &b) {
    if (bel[a.l] != bel[b.l]) return bel[a.l] < bel[b.l];
    return (bel[a.l] & 1) ? a.r < b.r : a.r > b.r; // 奇偶排序优化
}

int cnt[MAXV], cur_ans;

void mo_add(int pos) {
    cnt[a[pos]]++;
    // 根据题意更新 cur_ans
}

void mo_del(int pos) {
    cnt[a[pos]]--;
    // 根据题意更新 cur_ans
}

void solve(Query q[], int m, int ans[]) {
    sort(q, q + m, cmp);
    int cl = 1, cr = 0;
    for (int i = 0; i < m; i++) {
        while (cr < q[i].r) mo_add(++cr);
        while (cl > q[i].l) mo_add(--cl);
        while (cr > q[i].r) mo_del(cr--);
        while (cl < q[i].l) mo_del(cl++);
        ans[q[i].id] = cur_ans;
    }
}
```

## 三、分类与变体详解

| 类别 | 核心思想 | 典型复杂度 | 适用场景 |
|------|---------|-----------|---------|
| **基础分块** | 序列分块 + 懒标记 | $O(\sqrt{n})$ 每次操作 | 区间加减、区间求和、区间赋值等 |
| **值域分块** | 对值域而非下标分块 | $O(\sqrt{V})$ | 区间第 $k$ 大、区间众数等 |
| **块状链表** | 用链表连接各块，支持插入删除 | $O(\sqrt{n})$ | 带插入/删除的序列维护 |
| **莫队** | 离线 + 双指针 + 分块排序 | $O(n\sqrt{n})$ | 离线区间不带修查询 |
| **带修莫队** | 莫队 + 时间维扩展 | $O(n^{5/3})$ | 离线区间带修查询 |
| **树上莫队** | 欧拉序/括号序 + 莫队 | $O(n\sqrt{n})$ | 树上路径查询问题 |
| **回滚莫队** | 只增不减 + 撤销 | $O(n\sqrt{n})$ | 删除操作不好维护的场景 |

### 题目 1：分块模板 — 区间加 + 区间求和（洛谷 P3372 简化版）

> **题意：** 给定长度为 $n$ 的序列，支持两种操作：区间 $[l,r]$ 每个元素加 $v$；查询区间 $[l,r]$ 的元素之和。

**分析**

这是分块最核心的模板题。每个块维护一个懒标记 $\text{add}[b]$ 和块内和 $\text{sum}[b]$。修改和查询均按"散块暴力 + 整块批量"的方式处理。

```
- 数据结构：数组 a[]、块内和 sum[]、懒标记 add[]
- 支持操作：区间加、区间求和
- 关键性质：散块暴力 O(B)，整块 O(n/B)，取 B=sqrt(n) 均衡
- 时间复杂度：O(sqrt(n)) 每次操作
```

**C++ 代码**：

```cpp
long long a[MAXN], sum[MAXB], add[MAXB];

void range_add(int l, int r, long long v) {
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) a[i] += v, sum[bel[i]] += v;
        return;
    }
    for (int i = l; i <= R[bel[l]]; i++) a[i] += v, sum[bel[i]] += v;
    for (int i = bel[l] + 1; i < bel[r]; i++) add[i] += v;
    for (int i = L[bel[r]]; i <= r; i++) a[i] += v, sum[bel[i]] += v;
}

long long range_sum(int l, int r) {
    long long res = 0;
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) res += a[i] + add[bel[i]];
        return res;
    }
    for (int i = l; i <= R[bel[l]]; i++) res += a[i] + add[bel[i]];
    for (int i = bel[l] + 1; i < bel[r]; i++)
        res += sum[i] + add[i] * (R[i] - L[i] + 1);
    for (int i = L[bel[r]]; i <= r; i++) res += a[i] + add[bel[i]];
    return res;
}
```

**核心启示**：
- 分块的核心套路：不完整块暴力、整块打标记/批量处理
- 块大小 $B = \sqrt{n}$ 时修改和查询均为 $O(\sqrt{n})$
- 是所有分块题目的基石，务必熟练掌握

### 题目 2：区间加 + 单点查询（洛谷 P3368 分块做法）

> **题意：** 给定长度为 $n$ 的序列，支持区间 $[l,r]$ 加 $v$ 和单点查询 $a[x]$ 的当前值。

**分析**

相比题目 1 更简单，只需要单点查询，因此不需要维护块内和。每个块仅维护一个懒标记即可：整块直接加标记，散块逐元素修改。查询时返回 $a[x] + \text{add}[\text{bel}[x]]$。

```
- 数据结构：数组 a[]、懒标记 add[]
- 支持操作：区间加、单点查询
- 关键性质：整块打标记 O(1)，散块暴力 O(B)
- 时间复杂度：O(sqrt(n)) 每次操作
```

**C++ 代码**：

```cpp
long long a[MAXN], add[MAXB];

void range_add(int l, int r, long long v) {
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) a[i] += v;
        return;
    }
    for (int i = l; i <= R[bel[l]]; i++) a[i] += v;
    for (int i = bel[l] + 1; i < bel[r]; i++) add[i] += v;
    for (int i = L[bel[r]]; i <= r; i++) a[i] += v;
}

long long point_query(int x) {
    return a[x] + add[bel[x]];
}
```

**核心启示**：
- 单点查询只需在返回时加上所在块的懒标记
- 不需要维护块内和，代码更简洁
- 体现"按需维护"的分块思想

### 题目 3：区间求和（根号分治做法）（洛谷 P3396）

> **题意：** 给定序列 $a[1..n]$，多次查询：给定 $p$ 和 $x$，求所有满足 $i \bmod p = x$ 的 $a[i]$ 之和。支持单点修改。

**分析**

对于 $p \leq \sqrt{n}$，可以预处理所有 $(p, x)$ 对的答案；对于 $p > \sqrt{n}$，满足条件的 $i$ 不超过 $\sqrt{n}$ 个，暴力枚举即可。这就是"根号分治"的经典思想。

```
- 数据结构：预处理表 f[p][x]（p <= B）
- 支持操作：单点修改、按模查询
- 关键性质：小模数查表 O(1)，大模数暴力枚举 O(sqrt(n))
- 时间复杂度：O(sqrt(n)) 每次操作
```

**C++ 代码**：

```cpp
int f[MAXB][MAXB]; // f[p][x]: 模 p 余 x 的和

void build(int n) {
    B = sqrt(n);
    for (int p = 1; p <= B; p++)
        for (int i = 1; i <= n; i++)
            f[p][i % p] += a[i];
}

void modify(int pos, int v) {
    int delta = v - a[pos];
    a[pos] = v;
    for (int p = 1; p <= B; p++) f[p][pos % p] += delta;
}

long long query(int p, int x) {
    if (p <= B) return f[p][x];
    long long res = 0;
    for (int i = x; i <= n; i += p) res += a[i];
    return res;
}
```

**核心启示**：
- 根号分治的经典应用：小的预处理、大的暴力，平衡点恰好在 $\sqrt{n}$
- 这种"阈值划分"思想在很多看似和分块无关的题目中也能用到

### 题目 4：区间加 + 区间求和（完整版）（洛谷 P3372）

> **题意：** 给定长度为 $n$ 的序列，支持区间加和区间求和，$n, m \leq 10^5$。

**分析**

与题目 1 相同的模型，但本题数据规模更大，需注意常数优化。预处理块的左右端点避免重复计算，散块操作前先下推标记保证元素值正确。这是用分块替代线段树解决 P3372 的做法。

```
- 数据结构：数组 a[]、块内和 sum[]、懒标记 add[]
- 支持操作：区间加、区间求和
- 关键性质：散块操作前必须先下推懒标记
- 时间复杂度：O(sqrt(n)) 每次操作，总 O(m*sqrt(n))
```

**C++ 代码**：

```cpp
long long a[MAXN], sum[MAXB], tag[MAXB];

void push_down(int b) {
    if (!tag[b]) return;
    for (int i = L[b]; i <= R[b]; i++) a[i] += tag[b];
    tag[b] = 0;
}

void range_add(int l, int r, long long v) {
    if (bel[l] == bel[r]) {
        push_down(bel[l]);
        for (int i = l; i <= r; i++) a[i] += v, sum[bel[i]] += v;
        return;
    }
    push_down(bel[l]);
    for (int i = l; i <= R[bel[l]]; i++) a[i] += v, sum[bel[i]] += v;
    for (int i = bel[l] + 1; i < bel[r]; i++) tag[i] += v;
    push_down(bel[r]);
    for (int i = L[bel[r]]; i <= r; i++) a[i] += v, sum[bel[i]] += v;
}

long long range_sum(int l, int r) {
    long long res = 0;
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) res += a[i] + tag[bel[i]];
        return res;
    }
    for (int i = l; i <= R[bel[l]]; i++) res += a[i] + tag[bel[i]];
    for (int i = bel[l] + 1; i < bel[r]; i++)
        res += sum[i] + tag[i] * (R[i] - L[i] + 1);
    for (int i = L[bel[r]]; i <= r; i++) res += a[i] + tag[bel[i]];
    return res;
}
```

**核心启示**：
- 散块暴力之前 **必须先下推标记**，这是分块中最常见的 bug
- 分块做法虽然复杂度不如线段树，但常数小、代码短，适合比赛中快速编写

### 题目 5：区间修改 + 区间最值（洛谷 P3870 开关）

> **题意：** $n$ 个开关初始关闭，$m$ 次操作：区间取反 / 查询区间开着的开关数。

**分析**

经典的分块入门题。每个块维护一个"翻转标记"和"块内亮灯数"。对不完整块暴力翻转，对整块打翻转标记。查询时，若整块被翻转，则亮灯数 = 块长 - 原亮灯数。

```
- 数据结构：数组 a[]、块内亮灯数 cnt[]、翻转标记 flip[]
- 支持操作：区间取反、区间计数查询
- 关键性质：翻转用异或实现，整块 flip ^= 1
- 时间复杂度：O(sqrt(n)) 每次操作
```

**C++ 代码**：

```cpp
int a[MAXN], cnt[MAXB], flip[MAXB];

void toggle(int l, int r) {
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) {
            a[i] ^= 1;
            cnt[bel[i]] += a[i] ? 1 : -1;
        }
        return;
    }
    for (int i = l; i <= R[bel[l]]; i++) a[i] ^= 1, cnt[bel[i]] += a[i] ? 1 : -1;
    for (int i = bel[l]+1; i < bel[r]; i++) flip[i] ^= 1;
    for (int i = L[bel[r]]; i <= r; i++) a[i] ^= 1, cnt[bel[i]] += a[i] ? 1 : -1;
}

int query(int l, int r) {
    int res = 0;
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) res += a[i] ^ flip[bel[i]];
        return res;
    }
    for (int i = l; i <= R[bel[l]]; i++) res += a[i] ^ flip[bel[i]];
    for (int i = bel[l]+1; i < bel[r]; i++) {
        int len = R[i] - L[i] + 1;
        res += flip[i] ? len - cnt[i] : cnt[i];
    }
    for (int i = L[bel[r]]; i <= r; i++) res += a[i] ^ flip[bel[i]];
    return res;
}
```

**核心启示**：
- 翻转操作用异或处理非常优雅
- 分块可以灵活维护各种"非可合并"信息，如翻转计数
- 体现了分块最基本的套路：不完整块暴力、整块打标记

### 题目 6：莫队算法 — 区间不同颜色数（洛谷 P1972）

> **题意：** 给定长度为 $n$ 的序列和 $m$ 次查询，每次查询区间 $[l, r]$ 中有多少种不同的数。

**分析**

经典的莫队入门题。将所有查询离线，按 $(\lfloor l/B \rfloor, r)$ 排序。维护当前区间 $[cl, cr]$ 中每种颜色的出现次数 $cnt[]$ 和不同颜色总数 $cur\_ans$。通过双指针的伸缩来增减元素。

```
- 数据结构：计数数组 cnt[]、当前答案 cur_ans
- 支持操作：离线区间查询不同颜色数
- 关键性质：奇偶排序优化可减少指针移动约一半
- 时间复杂度：O(n*sqrt(n))
```

**C++ 代码**：

```cpp
int cnt[MAXV], cur_ans;

void mo_add(int pos) {
    if (++cnt[a[pos]] == 1) cur_ans++;
}

void mo_del(int pos) {
    if (--cnt[a[pos]] == 0) cur_ans--;
}

void solve(Query q[], int m, int ans[]) {
    sort(q, q + m, [](const Query &a, const Query &b) {
        if (bel[a.l] != bel[b.l]) return bel[a.l] < bel[b.l];
        return (bel[a.l] & 1) ? a.r < b.r : a.r > b.r;
    });
    int cl = 1, cr = 0;
    cur_ans = 0;
    for (int i = 0; i < m; i++) {
        while (cr < q[i].r) mo_add(++cr);
        while (cl > q[i].l) mo_add(--cl);
        while (cr > q[i].r) mo_del(cr--);
        while (cl < q[i].l) mo_del(cl++);
        ans[q[i].id] = cur_ans;
    }
}
```

**核心启示**：
- 莫队是离线算法，必须先读入所有查询再排序处理
- 奇偶排序优化让右指针的运动方向在相邻块间交替，显著减少总移动量
- 莫队的适用条件：加入/删除一个元素对答案的贡献可以 $O(1)$ 维护

### 题目 7：带修莫队（洛谷 P1903）

> **题意：** 给定长度为 $n$ 的颜色序列，支持两种操作：单点修改颜色；查询区间 $[l,r]$ 中不同颜色数。

**分析**

在普通莫队的基础上增加"时间维"。每个查询记录其之前发生了多少次修改，排序时三关键字 $(\lfloor l/B \rfloor, \lfloor r/B \rfloor, t)$。在处理查询时，先将时间推进或回退到该查询对应的修改版本。块大小取 $B = n^{2/3}$ 时总复杂度 $O(n^{5/3})$。

```
- 数据结构：cnt[]、修改记录数组
- 支持操作：单点修改 + 离线区间颜色数查询
- 关键性质：三维排序 (l块, r块, 时间戳)，块大小取 n^{2/3}
- 时间复杂度：O(n^{5/3})
```

**C++ 代码**：

```cpp
int cnt[MAXV], cur_ans;
struct Modify { int pos, old_col, new_col; } mdf[MAXN];
struct Query { int l, r, t, id; } qry[MAXN];

void mo_add(int pos) { if (++cnt[a[pos]] == 1) cur_ans++; }
void mo_del(int pos) { if (--cnt[a[pos]] == 0) cur_ans--; }

void mo_modify(int pos, int cl, int cr, int col) {
    if (cl <= pos && pos <= cr) {
        mo_del(pos);
        a[pos] = col;
        mo_add(pos);
    } else {
        a[pos] = col;
    }
}

void solve(int qn, int mn) {
    int B = max(1, (int)cbrt((double)n * n));
    sort(qry, qry + qn, [&](const Query &a, const Query &b) {
        if (a.l / B != b.l / B) return a.l / B < b.l / B;
        if (a.r / B != b.r / B) return a.r / B < b.r / B;
        return a.t < b.t;
    });
    int cl = 1, cr = 0, ct = 0;
    cur_ans = 0;
    for (int i = 0; i < qn; i++) {
        while (cr < qry[i].r) mo_add(++cr);
        while (cl > qry[i].l) mo_add(--cl);
        while (cr > qry[i].r) mo_del(cr--);
        while (cl < qry[i].l) mo_del(cl++);
        while (ct < qry[i].t)
            mo_modify(mdf[ct].pos, cl, cr, mdf[ct].new_col), ct++;
        while (ct > qry[i].t)
            ct--, mo_modify(mdf[ct].pos, cl, cr, mdf[ct].old_col);
        ans[qry[i].id] = cur_ans;
    }
}
```

**核心启示**：
- 带修莫队在时间维上增加一个指针，排序变为三关键字
- 块大小取 $n^{2/3}$ 才能使三维移动总量均衡
- 修改操作需要记录旧值以便回退

### 题目 8：值域分块 — 区间内小于 $v$ 的个数（洛谷 P2801）

> **题意：** 给定长度为 $n$ 的序列，支持区间加操作，查询区间内大于等于给定值 $v$ 的元素个数。

**分析**

对序列分块，每块内部维护一个排好序的副本。区间加时，整块只改懒标记，散块暴力修改后重新排序该块。查询时，整块用二分查找（加上懒标记偏移），散块暴力计数。

```
- 数据结构：原数组 a[]、排序副本 blk[]、懒标记 add[]
- 支持操作：区间加、区间内 >= v 的个数查询
- 关键性质：整块二分查找 O(log B)，散块暴力 O(B)
- 时间复杂度：O(sqrt(n) * log(sqrt(n))) 每次操作
```

**C++ 代码**：

```cpp
vector<int> blk[MAXB];
long long add[MAXB];

void rebuild(int b) {
    blk[b].clear();
    for (int i = L[b]; i <= R[b]; i++) blk[b].push_back(a[i]);
    sort(blk[b].begin(), blk[b].end());
}

void range_add(int l, int r, long long v) {
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) a[i] += v;
        rebuild(bel[l]);
        return;
    }
    for (int i = l; i <= R[bel[l]]; i++) a[i] += v;
    rebuild(bel[l]);
    for (int i = bel[l] + 1; i < bel[r]; i++) add[i] += v;
    for (int i = L[bel[r]]; i <= r; i++) a[i] += v;
    rebuild(bel[r]);
}

int query_ge(int l, int r, long long v) {
    int res = 0;
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++)
            if (a[i] + add[bel[i]] >= v) res++;
        return res;
    }
    for (int i = l; i <= R[bel[l]]; i++)
        if (a[i] + add[bel[i]] >= v) res++;
    for (int i = bel[l] + 1; i < bel[r]; i++) {
        long long target = v - add[i];
        res += blk[i].end() - lower_bound(blk[i].begin(), blk[i].end(), (int)target);
    }
    for (int i = L[bel[r]]; i <= r; i++)
        if (a[i] + add[bel[i]] >= v) res++;
    return res;
}
```

**核心启示**：
- 值域分块的核心：每块维护排序副本，利用二分加速查询
- 散块修改后必须重新排序该块的副本
- 分块 + 块内排序 + 二分是处理"区间内值域统计"类问题的经典组合

### 题目 9：区间开方（洛谷 P4145）

> **题意：** 给定长度为 $n$ 的非负整数序列，支持两种操作：将区间 $[l,r]$ 中每个元素取下取整平方根；查询区间和。

**分析**

开方操作不能使用懒标记（不满足可合并性），但有一个关键性质：每个数最多被开方约 $O(\log \log V)$ 次后就变为 0 或 1，此后不再改变。因此对每个块维护一个标志，标记该块内元素是否全为 0/1。操作时，若整块已全为 0/1 则跳过，否则暴力修改。

```
- 数据结构：数组 a[]、块内和 sum[]、全01标志 done[]
- 支持操作：区间开方、区间求和
- 关键性质：每个元素最多被开方 O(log log V) 次后不变
- 时间复杂度：均摊 O(n * log(log V) * sqrt(n) / n) = O(sqrt(n) * log(log V))
```

**C++ 代码**：

```cpp
long long a[MAXN], sum[MAXB];
bool done[MAXB]; // 块内元素是否全 <= 1

void range_sqrt(int l, int r) {
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) {
            sum[bel[i]] -= a[i];
            a[i] = (long long)sqrt((double)a[i]);
            sum[bel[i]] += a[i];
        }
        return;
    }
    for (int i = l; i <= R[bel[l]]; i++) {
        sum[bel[i]] -= a[i];
        a[i] = (long long)sqrt((double)a[i]);
        sum[bel[i]] += a[i];
    }
    for (int b = bel[l] + 1; b < bel[r]; b++) {
        if (done[b]) continue;
        done[b] = true;
        for (int i = L[b]; i <= R[b]; i++) {
            sum[b] -= a[i];
            a[i] = (long long)sqrt((double)a[i]);
            sum[b] += a[i];
            if (a[i] > 1) done[b] = false;
        }
    }
    for (int i = L[bel[r]]; i <= r; i++) {
        sum[bel[i]] -= a[i];
        a[i] = (long long)sqrt((double)a[i]);
        sum[bel[i]] += a[i];
    }
}

long long range_sum(int l, int r) {
    long long res = 0;
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) res += a[i];
        return res;
    }
    for (int i = l; i <= R[bel[l]]; i++) res += a[i];
    for (int b = bel[l] + 1; b < bel[r]; b++) res += sum[b];
    for (int i = L[bel[r]]; i <= r; i++) res += a[i];
    return res;
}
```

**核心启示**：
- 开方/取模等"不可懒标记化"的操作，用势能分析证明暴力总次数有限
- 分块给出了天然的"跳过已完成块"的框架
- 这是分块"暴力标记跳过"思想的代表题型

### 题目 10：区间众数（洛谷 P4168 蒲公英）

> **题意：** 给定序列，多次询问区间众数（出现次数最多的数，相同则取最小值），强制在线。

**分析**

经典的分块求区间众数。预处理任意两个整块之间的众数。对于查询 $[l,r]$，答案只可能是"整块部分的众数"或"出现在不完整块中的某个值"。用预处理的前缀出现次数数组快速查频次。

```
- 数据结构：f[i][j] 块i到块j的众数、pre[i][v] 前i块中v出现次数
- 支持操作：在线区间众数查询
- 关键性质：候选答案只来自散块元素 + 整块预处理众数
- 时间复杂度：预处理 O(n*sqrt(n))，查询 O(sqrt(n))
```

**C++ 代码**：

```cpp
int f[MAXB][MAXB];      // f[i][j]: 块i到块j的众数
int pre[MAXB][MAXV];    // pre[i][v]: 前i块中v出现次数

void preprocess() {
    // 预处理 pre 数组
    for (int i = 1; i <= tot; i++) {
        for (int v = 1; v <= V; v++) pre[i][v] = pre[i-1][v];
        for (int j = L[i]; j <= R[i]; j++) pre[i][a[j]]++;
    }
    // 预处理 f 数组
    for (int i = 1; i <= tot; i++) {
        int best = 0, best_cnt = 0;
        int tmp[MAXV] = {};
        for (int j = i; j <= tot; j++) {
            for (int k = L[j]; k <= R[j]; k++) {
                tmp[a[k]]++;
                if (tmp[a[k]] > best_cnt ||
                    (tmp[a[k]] == best_cnt && a[k] < best))
                    best = a[k], best_cnt = tmp[a[k]];
            }
            f[i][j] = best;
        }
    }
}

int get_freq(int l, int r, int v) {
    // [l,r] 中 v 的精确出现次数需要散块暴力 + 整块前缀差
    return pre[bel[r]-1][v] - pre[bel[l]][v]; // 整块部分
    // 加上散块中实际出现次数（在 query 中处理）
}

int query(int l, int r) {
    int lb = bel[l], rb = bel[r];
    int ans = 0, best = 0;
    // 整块众数作为初始候选
    if (lb + 1 <= rb - 1)
        ans = f[lb+1][rb-1];
    best = pre[rb-1][ans] - pre[lb][ans]; // 整块频次
    // 加上散块中 ans 的出现
    for (int i = l; i <= min(R[lb], r); i++)
        if (a[i] == ans) best++;
    if (lb != rb)
        for (int i = L[rb]; i <= r; i++)
            if (a[i] == ans) best++;
    // 枚举散块元素作为候选
    for (int i = l; i <= min(R[lb], r); i++) {
        int v = a[i];
        int freq = pre[rb-1][v] - pre[lb][v];
        for (int j = l; j <= min(R[lb], r); j++)
            if (a[j] == v) freq++;
        if (lb != rb)
            for (int j = L[rb]; j <= r; j++)
                if (a[j] == v) freq++;
        if (freq > best || (freq == best && v < ans))
            ans = v, best = freq;
    }
    if (lb != rb)
        for (int i = L[rb]; i <= r; i++) {
            int v = a[i];
            int freq = pre[rb-1][v] - pre[lb][v];
            for (int j = l; j <= min(R[lb], r); j++)
                if (a[j] == v) freq++;
            for (int j = L[rb]; j <= r; j++)
                if (a[j] == v) freq++;
            if (freq > best || (freq == best && v < ans))
                ans = v, best = freq;
        }
    return ans;
}
```

**核心启示**：
- 分块求众数的核心在于"候选答案只来自不完整块中的元素 + 整块预处理的众数"
- 预处理 $O(n\sqrt{n})$ 换取查询 $O(\sqrt{n})$，典型空间换时间
- 强制在线时分块优势明显，线段树无法直接合并众数信息

### 题目 11：块状链表（洛谷 P4008 文本编辑器）

> **题意：** 实现一个文本编辑器，支持在光标位置插入字符串、删除光标后 $k$ 个字符、光标移动、输出光标后 $k$ 个字符。

**分析**

块状链表是分块思想在"带插入/删除"场景下的核心应用。将序列存储在若干块中，块之间用链表连接。每块大小控制在 $[\frac{B}{2}, 2B]$ 之间。插入时找到对应块，将块从指定位置劈开，插入新块后检查是否需要合并或分裂。删除类似。

```
- 数据结构：块状链表，每块内用数组存储字符
- 支持操作：定位插入、定位删除、区间输出
- 关键性质：块大小维持 [B/2, 2B]，过大分裂过小合并
- 时间复杂度：O(sqrt(n)) 每次操作
```

**C++ 代码**：

```cpp
struct Block {
    char data[MAXB * 2];
    int len, nxt; // 块长度、下一块编号
} blk[MAXBLOCK];
int head, pool_cnt;

int new_block() { return ++pool_cnt; }

// 在全局位置 pos 处将块劈开
// 返回 pos 所在块编号和块内偏移
pair<int,int> locate(int pos) {
    int cur = head;
    while (cur && pos > blk[cur].len) {
        pos -= blk[cur].len;
        cur = blk[cur].nxt;
    }
    return {cur, pos};
}

// 将块 b 从位置 pos 处分裂为两块
void split(int b, int pos) {
    if (pos == blk[b].len) return; // 无需分裂
    int nb = new_block();
    memcpy(blk[nb].data, blk[b].data + pos, blk[b].len - pos);
    blk[nb].len = blk[b].len - pos;
    blk[nb].nxt = blk[b].nxt;
    blk[b].len = pos;
    blk[b].nxt = nb;
}

// 尝试将过小的块与后继合并
void merge_check(int b) {
    while (blk[b].nxt && blk[b].len + blk[blk[b].nxt].len <= MAXB * 2) {
        int nb = blk[b].nxt;
        memcpy(blk[b].data + blk[b].len, blk[nb].data, blk[nb].len);
        blk[b].len += blk[nb].len;
        blk[b].nxt = blk[nb].nxt;
    }
}
```

**核心启示**：
- 块状链表的核心操作是 split（分裂）和 merge（合并），保持每块大小在合理范围内
- 适合需要频繁在任意位置插入/删除的场景，是 rope/平衡树的简化替代
- 时间复杂度 $O(\sqrt{n})$，实现比平衡树简单很多

### 题目 12：树上莫队（SPOJ COT2）

> **题意：** 给定一棵 $n$ 个节点的树，每个节点有颜色。$m$ 次查询，每次查询路径 $(u, v)$ 上有多少种不同的颜色。

**分析**

将树用欧拉序（括号序）展开为序列，然后将树上路径查询转化为序列上的区间查询，再用莫队处理。对每个节点记录首次出现 $\text{in}[u]$ 和最后出现 $\text{out}[u]$ 的时间戳。路径 $(u,v)$（设 $\text{in}[u] \leq \text{in}[v]$）对应欧拉序上的某段区间，但需要用标记数组排除出现两次（进出都在区间内）的节点。

```
- 数据结构：欧拉序数组、cnt[]、vis[] 标记
- 支持操作：离线树上路径颜色数查询
- 关键性质：出现两次的节点不在路径上，LCA 需要特判
- 时间复杂度：O(n*sqrt(n))
```

**C++ 代码**：

```cpp
int euler[2 * MAXN], in_time[MAXN], out_time[MAXN], timer;
int cnt[MAXV], cur_ans;
bool vis[MAXN];

void dfs(int u, int fa) {
    euler[++timer] = u;
    in_time[u] = timer;
    for (int v : adj[u]) if (v != fa) dfs(v, u);
    euler[++timer] = u;
    out_time[u] = timer;
}

void flip(int u) {
    vis[u] ^= 1;
    if (vis[u]) { if (++cnt[col[u]] == 1) cur_ans++; }
    else { if (--cnt[col[u]] == 0) cur_ans--; }
}

void solve(int qn) {
    // 对查询 (u,v)，设 in[u] <= in[v]
    // 若 lca(u,v)=u，区间为 [in[u], in[v]]
    // 否则区间为 [out[u], in[v]]，且需额外加入 lca
    sort(qry, qry + qn, mo_cmp);
    int cl = 1, cr = 0;
    cur_ans = 0;
    for (int i = 0; i < qn; i++) {
        int ql = qry[i].l, qr = qry[i].r, lca = qry[i].lca;
        while (cr < qr) flip(euler[++cr]);
        while (cl > ql) flip(euler[--cl]);
        while (cr > qr) flip(euler[cr--]);
        while (cl < ql) flip(euler[cl++]);
        if (lca) flip(lca); // LCA 不在欧拉序区间内需额外加入
        ans[qry[i].id] = cur_ans;
        if (lca) flip(lca); // 还原 LCA
    }
}
```

**核心启示**：
- 树上莫队的关键是将路径查询通过欧拉序转化为序列区间查询
- 出现两次的节点表示"进出都在区间内"，说明不在路径上，需要排除
- LCA 需要特殊处理：它可能不在欧拉序选取的区间中

### 题目 13：回滚莫队（AT_joisc2014_c 历史研究）

> **题意：** 给定长度为 $n$ 的序列，$m$ 次查询区间 $[l,r]$ 中 $\max(a[i] \times \text{cnt}(a[i]))$，即每种值乘以其出现次数的最大值。

**分析**

普通莫队在删除元素时难以维护最大值（删除后可能需要遍历整个值域找新最大值）。回滚莫队的思想是：只做添加操作，不做删除。对于同一块内的查询暴力处理；对于跨块查询，每次将右指针只增不减，左指针从块右端暴力扩展后回滚到块右端 +1。

```
- 数据结构：cnt[]、备份数组
- 支持操作：离线区间最大加权频次查询
- 关键性质：右指针单调右移，左指针每次扩展后回滚
- 时间复杂度：O(n*sqrt(n))
```

**C++ 代码**：

```cpp
long long cnt[MAXV], cur_ans;

long long solve_block(int bl, Query qs[], int qcnt) {
    // 处理左端点在块 bl 内的所有查询
    int block_r = R[bl];
    memset(cnt, 0, sizeof(cnt));
    cur_ans = 0;
    int cr = block_r; // 右指针从块右端开始
    for (int i = 0; i < qcnt; i++) {
        int ql = qs[i].l, qr = qs[i].r;
        if (bel[qr] == bl) {
            // 同块内暴力
            long long tmp = 0;
            int tmp_cnt[MAXV] = {};
            for (int j = ql; j <= qr; j++) {
                tmp_cnt[a[j]]++;
                tmp = max(tmp, (long long)a[j] * tmp_cnt[a[j]]);
            }
            ans[qs[i].id] = tmp;
            continue;
        }
        // 右指针单调右移
        while (cr < qr) {
            cr++;
            cnt[a[cr]]++;
            cur_ans = max(cur_ans, (long long)a[cr] * cnt[a[cr]]);
        }
        // 保存当前状态
        long long save_ans = cur_ans;
        // 左指针从 block_r+1 向左扩展到 ql
        for (int j = block_r; j >= ql; j--) {
            cnt[a[j]]++;
            cur_ans = max(cur_ans, (long long)a[j] * cnt[a[j]]);
        }
        ans[qs[i].id] = cur_ans;
        // 回滚左指针
        for (int j = block_r; j >= ql; j--) cnt[a[j]]--;
        cur_ans = save_ans;
    }
}
```

**核心启示**：
- 回滚莫队适用于"添加容易、删除困难"的场景
- 核心技巧：右指针只增不减，左指针每次临时扩展后回滚
- 相比普通莫队，牺牲了一些通用性，换取了对更多类型答案的支持

---

### 题目 14：区间加 + 区间立方和（洛谷 P3710）

> **题意：** 维护序列，支持区间加、区间求立方和。

**分析**

区间加 $v$ 后，$(a_i + v)^3 = a_i^3 + 3a_i^2 v + 3a_i v^2 + v^3$。因此需要同时维护块内的一次方和、二次方和、三次方和，以及懒标记。整块加 $v$ 时利用公式快速更新三个和。

```
- 数据结构：s1（一次方和）、s2（二次方和）、s3（三次方和）、tag（懒标记）
- 支持操作：区间加、区间立方和查询
- 关键性质：利用二项式展开 O(1) 更新整块三个和
- 时间复杂度：O(sqrt(n)) 每次操作
```

**C++ 代码**：

```cpp
long long s1[MAXB], s2[MAXB], s3[MAXB], tag[MAXB];

void push_down(int b) {
    long long v = tag[b];
    if (!v) return;
    for (int i = L[b]; i <= R[b]; i++) a[i] += v;
    tag[b] = 0;
}

void block_add(int b, long long v) {
    int len = R[b] - L[b] + 1;
    s3[b] += 3*s2[b]*v + 3*s1[b]*v*v + (long long)len*v*v*v;
    s2[b] += 2*s1[b]*v + (long long)len*v*v;
    s1[b] += (long long)len * v;
    tag[b] += v;
}

void brute_add(int l, int r, long long v) {
    int b = bel[l];
    push_down(b);
    for (int i = l; i <= r; i++) a[i] += v;
    s1[b] = s2[b] = s3[b] = 0;
    for (int i = L[b]; i <= R[b]; i++) {
        s1[b] += a[i];
        s2[b] += a[i]*a[i];
        s3[b] += a[i]*a[i]*a[i];
    }
}
```

**核心启示**：
- 维护高次幂和的关键在于利用二项式展开公式，让整块更新可以 $O(1)$ 完成
- 不完整块则暴力下推标记后重构所有统计量
- 这一技巧可推广到任意 $k$ 次方和

### 题目 15：逆序对动态维护（洛谷 P1975 排队）

> **题意：** 给定排列，支持交换两个位置的元素，每次操作后询问逆序对数。

**分析**

用分块维护每个块内的有序序列。交换 $a[x]$ 和 $a[y]$ 时，逆序对的变化仅与 $[x+1, y-1]$ 中比 $a[x]$ 大/小、比 $a[y]$ 大/小的元素个数有关。利用块内排序后二分查找快速统计。

```
- 数据结构：排序副本 blk[]
- 支持操作：交换两元素、查询逆序对变化量
- 关键性质：交换 (x,y) 只影响 [x+1,y-1] 中与 a[x],a[y] 的大小关系
- 时间复杂度：O(sqrt(n) * log(sqrt(n))) 每次操作
```

**C++ 代码**：

```cpp
int count_greater(int l, int r, int v) {
    int res = 0;
    if (l > r) return 0;
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) if (a[i] > v) res++;
        return res;
    }
    for (int i = l; i <= R[bel[l]]; i++) if (a[i] > v) res++;
    for (int i = bel[l]+1; i < bel[r]; i++)
        res += blk[i].end() - upper_bound(blk[i].begin(), blk[i].end(), v);
    for (int i = L[bel[r]]; i <= r; i++) if (a[i] > v) res++;
    return res;
}

int count_less(int l, int r, int v) {
    int res = 0;
    if (l > r) return 0;
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) if (a[i] < v) res++;
        return res;
    }
    for (int i = l; i <= R[bel[l]]; i++) if (a[i] < v) res++;
    for (int i = bel[l]+1; i < bel[r]; i++)
        res += lower_bound(blk[i].begin(), blk[i].end(), v) - blk[i].begin();
    for (int i = L[bel[r]]; i <= r; i++) if (a[i] < v) res++;
    return res;
}

void update_block(int b, int old_val, int new_val) {
    blk[b].erase(lower_bound(blk[b].begin(), blk[b].end(), old_val));
    blk[b].insert(lower_bound(blk[b].begin(), blk[b].end(), new_val), new_val);
}
```

**核心启示**：
- 分块 + 块内排序 + 二分，是处理"区间内值域统计"类问题的经典组合
- 逆序对的增量更新只需关注被交换元素之间的区间
- 这种局部更新思想在很多动态维护问题中都能用到

### 题目 16：区间值替换 + 区间第 $k$ 小（洛谷 P4119 Ynoi2018 未来日记）

> **题意：** 维护序列，支持区间内将值 $x$ 全部替换为 $y$，查询区间第 $k$ 小。

**分析**

这是 Ynoi 系列的经典难题，需要值域分块 + 序列分块 + 并查集的综合运用。对序列和值域同时分块。区间替换用并查集合并值，整块替换 $O(1)$；区间第 $k$ 小利用值域分块逐块计数。

```
- 数据结构：序列分块 + 值域分块 + 并查集
- 支持操作：区间值替换 x→y、区间第 k 小
- 关键性质：并查集合并使整块替换 O(1)，值域分块使第k小查询 O(sqrt(V))
- 时间复杂度：O(sqrt(n)) 每次操作（均摊）
```

**C++ 代码**：

```cpp
int fa[MAXB][MAXV]; // 并查集：块 b 中值 v 的实际代表
int val_cnt[MAXB][MAXV]; // 块 b 中值 v 的出现次数
int rng_cnt[MAXB][MAXVB]; // 块 b 中值域块 vb 的总计数

int find(int b, int v) {
    return fa[b][v] == v ? v : fa[b][v] = find(b, fa[b][v]);
}

void block_replace(int b, int x, int y) {
    int rx = find(b, x), ry = find(b, y);
    if (rx == ry) return;
    // 将 rx 的计数合并到 ry
    int vbx = rx / VB, vby = ry / VB;
    rng_cnt[b][vby] += val_cnt[b][rx];
    rng_cnt[b][vbx] -= val_cnt[b][rx];
    val_cnt[b][ry] += val_cnt[b][rx];
    val_cnt[b][rx] = 0;
    fa[b][rx] = ry;
}

int kth(int l, int r, int k) {
    // 枚举值域块
    for (int vb = 0; vb < val_blocks; vb++) {
        int c = 0;
        // 统计 [l,r] 中值域块 vb 的总计数
        for (int b = bel[l]; b <= bel[r]; b++) {
            if (L[b] >= l && R[b] <= r)
                c += rng_cnt[b][vb]; // 整块
            else {
                int lo = max(l, L[b]), hi = min(r, R[b]);
                for (int i = lo; i <= hi; i++)
                    if (find(b, a[i]) / VB == vb) c++;
            }
        }
        if (c >= k) {
            // 在值域块 vb 内逐值查找
            for (int v = vb * VB; v < (vb + 1) * VB; v++) {
                int cv = 0;
                for (int b = bel[l]; b <= bel[r]; b++) {
                    if (L[b] >= l && R[b] <= r)
                        cv += val_cnt[b][v];
                    else {
                        int lo = max(l, L[b]), hi = min(r, R[b]);
                        for (int i = lo; i <= hi; i++)
                            if (find(b, a[i]) == v) cv++;
                    }
                }
                if (cv >= k) return v;
                k -= cv;
            }
        }
        k -= c;
    }
    return -1;
}
```

**核心启示**：
- Ynoi 题目将分块推向极致：序列分块与值域分块双重叠加，配合并查集实现高效合并
- 并查集使得整块内的值替换可以 $O(1)$ 完成，避免了逐元素扫描
- 理解此题有助于掌握分块的上限

## 五、实用技巧与调参

1. **块大小的选取**：不一定恰好是 $\sqrt{n}$。根据操作的不同，有时取 $n^{2/3}$（带修莫队）或手动调参（如取 $300 \sim 500$）效果更优。

2. **奇偶排序优化**（莫队专用）：按左端点所在块的奇偶性决定右端点排序方向，可以减少指针移动总量约一半。

3. **散块暴力时注意下推标记**：修改不完整块之前，务必先将该块的懒标记下推到元素，否则元素值不正确。

4. **常数优化**：
   - 预处理块的左右端点，避免重复计算。
   - 块内循环尽量连续访存。
   - 对于莫队，使用数组而非 `map`/`set` 维护计数。

5. **分块与其他数据结构结合**：
   - 分块 + BIT/线段树：块内用树状数组维护。
   - 分块 + 并查集：如 P4119 中的值域替换。
   - 分块 + 离线：时间轴分块处理在线转离线。

6. **空间注意**：预处理 $f[\sqrt{n}][\sqrt{n}]$ 的数组可能较大，注意内存限制。

## 六、分块通用框架

```cpp
// ==================== 分块通用框架 ====================
const int MAXN = 100005;
const int MAXB = 320; // sqrt(MAXN)

int a[MAXN], bel[MAXN];
int L[MAXB], R[MAXB], B, tot;

// 每块维护的信息（根据题意修改）
long long sum[MAXB], tag[MAXB];

void init(int n) {
    B = max(1, (int)sqrt(n));
    tot = (n + B - 1) / B;
    for (int i = 1; i <= n; i++) bel[i] = (i - 1) / B + 1;
    for (int i = 1; i <= tot; i++) {
        L[i] = (i - 1) * B + 1;
        R[i] = min(i * B, n);
    }
    // 预处理块信息
    for (int i = 1; i <= tot; i++) {
        sum[i] = tag[i] = 0;
        for (int j = L[i]; j <= R[i]; j++) sum[i] += a[j];
    }
}

// 下推懒标记
void push_down(int b) {
    if (!tag[b]) return;
    for (int i = L[b]; i <= R[b]; i++) a[i] += tag[b];
    tag[b] = 0;
}

// 区间修改（以区间加为例）
void range_update(int l, int r, long long v) {
    if (bel[l] == bel[r]) {
        push_down(bel[l]);
        for (int i = l; i <= r; i++) a[i] += v, sum[bel[i]] += v;
        return;
    }
    // 左端不完整块
    push_down(bel[l]);
    for (int i = l; i <= R[bel[l]]; i++) a[i] += v, sum[bel[i]] += v;
    // 中间整块
    for (int i = bel[l] + 1; i < bel[r]; i++) tag[i] += v;
    // 右端不完整块
    push_down(bel[r]);
    for (int i = L[bel[r]]; i <= r; i++) a[i] += v, sum[bel[i]] += v;
}

// 区间查询（以区间求和为例）
long long range_query(int l, int r) {
    long long res = 0;
    if (bel[l] == bel[r]) {
        for (int i = l; i <= r; i++) res += a[i] + tag[bel[i]];
        return res;
    }
    for (int i = l; i <= R[bel[l]]; i++) res += a[i] + tag[bel[i]];
    for (int i = bel[l] + 1; i < bel[r]; i++)
        res += sum[i] + tag[i] * (R[i] - L[i] + 1);
    for (int i = L[bel[r]]; i <= r; i++) res += a[i] + tag[bel[i]];
    return res;
}
```

## 七、总结

### 7.1 分块的本质

分块的本质是 **平衡**——在"全部暴力"和"复杂数据结构"之间找到一个折中点。块大小 $B$ 就是这个平衡的旋钮：块越大，整块操作越快但散块越慢；块越小则相反。取 $B = \sqrt{n}$ 时两者均衡，达到 $O(\sqrt{n})$。

### 7.2 核心要点

| 要点 | 说明 |
|------|------|
| 散块暴力 + 整块批量 | 分块一切操作的基本范式 |
| 懒标记下推 | 散块操作前必须先将标记下推到元素 |
| 块内排序 | 维护排序副本支持二分查询 |
| 莫队排序 | 离线查询按块号和端点排序 |
| 势能分析 | 暴力操作总次数有限时可用分块框架跳过 |
| 根号分治 | 按阈值 $\sqrt{n}$ 划分小/大两种情况分别处理 |

### 7.3 学习路线图

```
基础分块（区间加/求和模板）
    ↓
区间修改变体（取反、赋值、开方）
    ↓

    ↓
莫队算法（离线 + 分块排序）
    ↓
带修莫队 / 回滚莫队
    ↓
树上莫队
    ↓
块状链表
    ↓
Ynoi 系列（分块的极致应用）
```
