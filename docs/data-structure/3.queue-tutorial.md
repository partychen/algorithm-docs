## 一、什么是队列

**队列（Queue）** 是一种遵循 **先进先出（FIFO, First In First Out）** 原则的线性数据结构。就像现实中的排队——先来的人先被服务，后来的人排在队尾等待。

```
入队 → ┌───┬───┬───┬───┐ → 出队
       │ D │ C │ B │ A │
       └───┴───┴───┴───┘
       队尾              队头
```

**核心操作与复杂度：**

| 操作 | 说明 | 时间复杂度 |
|------|------|-----------|
| push(x) / enqueue | 元素从队尾进入 | $O(1)$ |
| pop() / dequeue | 元素从队头离开 | $O(1)$ |
| front() | 查看队头元素 | $O(1)$ |
| empty() | 检查是否为空 | $O(1)$ |

**队列家族一览：**

| 类型 | 特点 | 典型应用 |
|------|------|----------|
| 普通队列 | 严格 FIFO | BFS、缓冲区 |
| 双端队列（Deque） | 两端可操作 | 单调队列容器 |
| 循环队列 | 数组首尾相连 | OS 缓冲区 |
| 单调队列 | 内部保持单调 | 滑动窗口最值、DP 优化 |
| 优先队列 | 按优先级出队 | Dijkstra、贪心 |

## 二、核心思想与前置知识

### 2.1 数组模拟队列

竞赛中推荐使用数组模拟，速度快且可随机访问：

```cpp
int q[N], hh = 0, tt = -1;

void push(int x) { q[++tt] = x; }
void pop() { hh++; }
int front() { return q[hh]; }
bool empty() { return hh > tt; }
```

### 2.2 单调队列

单调队列是竞赛中最重要的队列变体。它维护一个**单调递增或递减**的双端队列，核心作用是在 $O(1)$ 时间内获取**滑动窗口中的最值**。

以维护**滑动窗口最大值**为例（单调递减队列）：

1. 新元素从队尾进入时，弹出队尾所有**小于**它的元素（它们不可能再成为最大值）
2. 队头元素就是当前窗口的最大值
3. 当队头元素滑出窗口范围时，从队头弹出

```
数组: [1, 3, -1, -3, 5, 3, 6, 7]，窗口 k=3

i=0: 入队 0          deque: [0(1)]
i=1: 1<3, 弹出0      deque: [1(3)]
i=2: -1<3, 入队       deque: [1(3), 2(-1)]     窗口[0,2] 最大=3
i=3: -3<-1, 入队      deque: [1(3), 2(-1), 3(-3)] 窗口[1,3] 最大=3
i=4: 5>所有, 全弹出   deque: [4(5)]             窗口[2,4] 最大=5
...
```

**模板代码（数组模拟）：**

```cpp
int q[N], hh = 0, tt = -1; // 双端队列存下标

void slidingMax(int a[], int n, int k, int ans[]) {
    hh = 0; tt = -1;
    for (int i = 0; i < n; i++) {
        if (hh <= tt && q[hh] <= i - k) hh++;     // 弹出过期队头
        while (hh <= tt && a[q[tt]] <= a[i]) tt--;  // 维护递减
        q[++tt] = i;
        if (i >= k - 1) ans[i - k + 1] = a[q[hh]];
    }
}
```

**关键性质：** 每个元素至多入队一次、出队一次，总时间复杂度 $O(n)$。

### 2.3 优先队列

优先队列按**优先级**出队，底层由**二叉堆**实现。C++ 中 `priority_queue` 默认为大根堆：

```cpp
// 小根堆
priority_queue<int, vector<int>, greater<int>> pq;

// 自定义优先级
struct Node {
    int val, id;
    bool operator>(const Node& o) const { return val > o.val; }
};
priority_queue<Node, vector<Node>, greater<Node>> pq;
```

### 2.4 BFS 与队列

BFS（广度优先搜索）是队列最经典的应用。FIFO 保证按层遍历：

```cpp
int q[N], hh = 0, tt = -1;
bool vis[N];

void bfs(int start) {
    q[++tt] = start; vis[start] = true;
    while (hh <= tt) {
        int u = q[hh++];
        for (int i = head[u]; ~i; i = ne[i]) {
            int v = e[i];
            if (!vis[v]) { vis[v] = true; q[++tt] = v; }
        }
    }
}
```

## 三、分类与变体详解

### 3.1 单调队列 vs 单调栈

| | 单调队列 | 单调栈 |
|---|---------|--------|
| **结构** | 双端队列 | 栈 |
| **弹出位置** | 队头 + 队尾 | 仅栈顶 |
| **窗口约束** | 有（队头过期弹出） | 无 |
| **典型问题** | 滑动窗口最值 | 下一个更大元素 |

### 3.2 优先队列 vs 单调队列

| | 优先队列 | 单调队列 |
|---|---------|---------|
| **底层结构** | 堆 | 双端队列 |
| **获取最值** | $O(1)$ | $O(1)$ |
| **插入** | $O(\log n)$ | 均摊 $O(1)$ |
| **适用范围** | 全局最值 | 固定窗口最值 |

### 3.3 单调队列的方向选择

| 目标 | 队列单调性 | 队尾弹出条件 |
|------|-----------|-------------|
| 窗口最大值 | 递减（大→小） | `a[q[tt]] <= a[i]` |
| 窗口最小值 | 递增（小→大） | `a[q[tt]] >= a[i]` |

> 记忆技巧：**想要最大值，淘汰小的**（递减）；**想要最小值，淘汰大的**（递增）。

### 题目 1：机器翻译（洛谷 P1540）

> 内存中有容量为 $M$ 的缓存。每个单词若不在缓存中则需查词典并放入缓存，缓存满时淘汰最早进入的。求查词典次数。

**分析**：FIFO 淘汰策略 = 队列。用队列维护缓存，标记数组判断命中。

```
数据结构：队列 + 标记数组
支持操作：入队（加入缓存）、出队（淘汰最早）、查询是否在缓存中
关键性质：FIFO 淘汰策略天然对应队列的出队操作
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
const int MAXW = 1005;
int q[MAXW], hh = 0, tt = -1;
bool inCache[MAXW];

int solve(int words[], int n, int m) {
    int ans = 0;
    for (int i = 0; i < n; i++) {
        int w = words[i];
        if (inCache[w]) continue;
        ans++;
        q[++tt] = w; inCache[w] = true;
        if (tt - hh + 1 > m) { inCache[q[hh]] = false; hh++; }
    }
    return ans;
}
```

**核心启示**：
- **FIFO 缓存 = 队列**：淘汰策略为"先进先出"时，队列是最自然的选择
- **标记数组加速查询**：O(1) 判断元素是否在缓存中，避免遍历队列

### 题目 2：BFS 走迷宫（洛谷 P1141 / 经典 BFS）

> 给定 $n \times m$ 的 01 网格，求从起点到终点的最短步数。

**分析**：BFS 求无权图最短路是队列最经典的应用。每个状态入队一次，按层扩展保证首次到达即为最短路。

```
数据结构：队列
支持操作：入队新状态、出队当前状态、四方向扩展
关键性质：BFS 的 FIFO 性质保证首次到达 = 最短路
时间复杂度：O(n * m)
```

**C++ 代码**：

```cpp
const int dx[] = {0, 0, 1, -1}, dy[] = {1, -1, 0, 0};
int dist[N][N]; // -1 表示未访问
int qx[N*N], qy[N*N], hh = 0, tt = -1;

int bfs(int sx, int sy, int ex, int ey, int n, int m, int grid[][N]) {
    memset(dist, -1, sizeof dist);
    dist[sx][sy] = 0;
    qx[++tt] = sx; qy[tt] = sy;
    while (hh <= tt) {
        int x = qx[hh], y = qy[hh]; hh++;
        if (x == ex && y == ey) return dist[x][y];
        for (int d = 0; d < 4; d++) {
            int nx = x + dx[d], ny = y + dy[d];
            if (nx >= 0 && nx < n && ny >= 0 && ny < m &&
                grid[nx][ny] == 0 && dist[nx][ny] == -1) {
                dist[nx][ny] = dist[x][y] + 1;
                qx[++tt] = nx; qy[tt] = ny;
            }
        }
    }
    return -1;
}
```

**核心启示**：
- **BFS 求最短路**：无权图最短路的标准做法，FIFO 保证层级正确
- **`dist` 兼做 `vis`**：用 `-1` 表示未访问，省去额外的 `vis` 数组

### 题目 3：约瑟夫问题 队列解法（洛谷 P1996）

> $n$ 个人围成一圈，每次报到 $m$ 的人出列。用队列模拟。

**分析**：用队列模拟报数过程。前 $m-1$ 个人出队后重新入队（相当于"跳过"），第 $m$ 个人出队后淘汰。

```
数据结构：队列（模拟循环）
支持操作：出队+入队（跳过）、出队（淘汰）
关键性质：队列的 FIFO 自然模拟"报数"过程，出队再入队 = 循环移动
时间复杂度：O(n * m)
```

**C++ 代码**：

```cpp
int q[N], hh = 0, tt = -1;

void solve(int n, int m) {
    for (int i = 1; i <= n; i++) q[++tt] = i;
    while (hh <= tt) {
        for (int j = 1; j < m; j++) { q[++tt] = q[hh]; hh++; } // 前 m-1 个人循环
        cout << q[hh++] << " "; // 第 m 个人出列
    }
}
```

**核心启示**：
- **队列模拟循环**："出队再入队"等价于循环移动，无需真正的循环链表
- **两种解法对比**：链表解法 O(nm) 但可优化，队列解法直观简洁

### 题目 4：双端队列（洛谷 P1070 / Deque 基础）

> 给定 $n$ 个数，依次将每个数放入双端队列的头部或尾部，使最终的序列字典序最小。

**分析**：贪心策略。每次比较当前数与结果序列的首尾，选择能使字典序更小的一端插入。

```
数据结构：双端队列
支持操作：头部插入、尾部插入
关键性质：贪心 — 小的放前面，大的放后面
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
int dq[2 * N], l = N, r = N - 1; // 用数组模拟双端队列

void solve(int a[], int n) {
    for (int i = 0; i < n; i++) {
        if (l > r || a[i] <= dq[l]) dq[--l] = a[i]; // 放头部
        else dq[++r] = a[i];                          // 放尾部
    }
    for (int i = l; i <= r; i++) cout << dq[i] << " ";
}
```

**核心启示**：
- **双端队列的灵活性**：两端都可以操作，比普通队列适用范围更广
- **数组模拟双端队列**：中间开始向两边扩展，`l` 向左、`r` 向右

### 题目 5：滑动窗口（洛谷 P1886）

> 给定长度为 $n$ 的数组和窗口大小 $k$，求每个窗口的最小值和最大值。

**分析**：单调队列模板题。分别维护单调递增队列（求最小值）和单调递减队列（求最大值）。

```
数据结构：单调队列（双端队列）
支持操作：队头弹出过期元素、队尾维护单调性、队头取最值
关键性质：每个元素至多入队出队各一次，总 O(n)
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
int a[N], q[N], hh, tt;

void slidingWindow(int n, int k) {
    // 求最小值（单调递增队列）
    hh = 0; tt = -1;
    for (int i = 0; i < n; i++) {
        if (hh <= tt && q[hh] <= i - k) hh++;
        while (hh <= tt && a[q[tt]] >= a[i]) tt--;
        q[++tt] = i;
        if (i >= k - 1) printf("%d ", a[q[hh]]);
    }
    puts("");
    // 求最大值（单调递减队列）
    hh = 0; tt = -1;
    for (int i = 0; i < n; i++) {
        if (hh <= tt && q[hh] <= i - k) hh++;
        while (hh <= tt && a[q[tt]] <= a[i]) tt--;
        q[++tt] = i;
        if (i >= k - 1) printf("%d ", a[q[hh]]);
    }
    puts("");
}
```

**核心启示**：
- **单调队列模板**：三步操作——弹过期、维护单调、入队——是固定范式
- **最小/最大只差一个符号**：`>=` 求最小值，`<=` 求最大值

### 题目 6：切蛋糕（洛谷 P1714）

> 给定长度为 $n$ 的整数数组，求长度**不超过** $m$ 的连续子数组的最大和。

**分析**：前缀和 + 单调队列。`sum[r] - sum[l]` 是区间 `[l+1, r]` 的和。固定 `r`，在 `[r-m, r-1]` 范围内找最小的 `sum[l]`——滑动窗口求最小值。

```
数据结构：单调队列（递增）+ 前缀和
支持操作：滑动窗口内维护前缀和的最小值
关键性质：max(sum[r] - sum[l]) = sum[r] - min(sum[l])，窗口大小 m
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
long long pre[N]; // 前缀和
int q[N], hh = 0, tt = -1;

long long solve(int a[], int n, int m) {
    pre[0] = 0;
    for (int i = 1; i <= n; i++) pre[i] = pre[i - 1] + a[i];
    long long ans = -1e18;
    q[++tt] = 0; // 起始点
    for (int r = 1; r <= n; r++) {
        while (hh <= tt && q[hh] < r - m) hh++; // 弹出过期
        ans = max(ans, pre[r] - pre[q[hh]]);
        while (hh <= tt && pre[q[tt]] >= pre[r]) tt--; // 维护递增
        q[++tt] = r;
    }
    return ans;
}
```

**核心启示**：
- **前缀和 + 单调队列**：将"限长最大子段和"转化为"滑动窗口内前缀和最小值"
- **思维转化**：`max(sum[r] - sum[l])` = `sum[r] - min(sum[l])`，将最优化问题拆解

### 题目 8：滑动窗口中位数（LeetCode 480）

> 给定数组和窗口大小 $k$，求每个窗口的中位数。

**分析**：用两个优先队列（大根堆 + 小根堆）维护窗口内的元素，大根堆存较小的一半，小根堆存较大的一半。堆顶即为中位数。延迟删除处理窗口滑动。

```
数据结构：对顶堆（大根堆 + 小根堆）+ 延迟删除
支持操作：插入元素、延迟删除过期元素、查询中位数
关键性质：两个堆的大小之差 ≤ 1，堆顶即为中位数
时间复杂度：O(n log k)
```

**C++ 代码**：

```cpp
priority_queue<int> maxH;                                   // 较小的一半
priority_queue<int, vector<int>, greater<int>> minH;        // 较大的一半
unordered_map<int, int> delayed;                            // 延迟删除计数
int szMax = 0, szMin = 0;                                   // 有效元素数

void prune(priority_queue<int>& h) {
    while (!h.empty() && delayed[h.top()]) { delayed[h.top()]--; h.pop(); }
}
void prune(priority_queue<int, vector<int>, greater<int>>& h) {
    while (!h.empty() && delayed[h.top()]) { delayed[h.top()]--; h.pop(); }
}

void balance() {
    if (szMax > szMin + 1) { minH.push(maxH.top()); maxH.pop(); szMax--; szMin++; prune(maxH); }
    else if (szMin > szMax) { maxH.push(minH.top()); minH.pop(); szMin--; szMax++; prune(minH); }
}

void insert(int x) {
    if (maxH.empty() || x <= maxH.top()) { maxH.push(x); szMax++; }
    else { minH.push(x); szMin++; }
    balance();
}

void erase(int x) {
    delayed[x]++;
    if (x <= maxH.top()) { szMax--; prune(maxH); }
    else { szMin--; prune(minH); }
    balance();
}
// 中位数：k 为奇数取 maxH.top()，偶数取 (maxH.top() + minH.top()) / 2.0
```

**核心启示**：
- **对顶堆维护中位数**：大根堆存小的一半、小根堆存大的一半，堆顶之一即为中位数
- **延迟删除**：优先队列不支持删除任意元素，用延迟删除标记 + prune 来处理

### 🟠 提高级 Level 3：单调队列优化 DP

### 题目 9：跳跃游戏（洛谷 P1725）

> 给定长度为 $n$ 的数组，从位置 0 出发，每次可以向右跳 $[L, R]$ 步。经过的位置的值求和，求到达 $\geq n$ 时的最大和。

**分析**：DP + 单调队列优化。`f[i] = max(f[j]) + a[i]`，其中 `j` 的范围是 `[i-R, i-L]`——一个滑动窗口。

```
数据结构：单调队列（递减）优化 DP
支持操作：窗口 [i-R, i-L] 内维护 f[j] 的最大值
关键性质：DP 转移的决策区间是滑动窗口 → 单调队列优化
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
long long f[N]; // f[i] = 到达 i 的最大得分
int q[N], hh = 0, tt = -1;

void solve(int a[], int n, int L, int R) {
    memset(f, -0x3f, sizeof f);
    f[0] = a[0];
    for (int i = L; i <= n; i++) {
        // j = i-R..i-L 范围内的 f[j] 入队
        int j = i - L;
        if (j >= 0 && f[j] > -1e18) {
            while (hh <= tt && f[q[tt]] <= f[j]) tt--;
            q[++tt] = j;
        }
        while (hh <= tt && q[hh] < i - R) hh++;
        if (hh <= tt) f[i] = f[q[hh]] + a[i];
    }
    long long ans = -1e18;
    for (int i = n - R + 1; i <= n; i++) // 可以跳出 n
        if (i >= 0) ans = max(ans, f[i]);
}
```

**核心启示**：
- **DP + 单调队列**：当转移方程 `f[i] = opt(f[j]) + g(i)` 且 `j` 的范围是滑动窗口时，单调队列优化
- **从 O(n²) 到 O(n)**：暴力枚举决策点 → 单调队列维护窗口最优决策

### 题目 10：修剪草坪（洛谷 P2627）

> 有 $n$ 头牛排成一排，每头有效率值 $e_i$。最多选连续 $k$ 头（不能超过 $k$ 头连续选中），求最大效率总和。

**分析**：DP 状态：`f[i]` = 前 $i$ 头牛的最大效率。转移：要么不选第 $i$ 头 `f[i] = f[i-1]`，要么选以 $i$ 结尾的一段连续（长度 $\leq k$），用单调队列优化。

```
数据结构：单调队列优化 DP
支持操作：在窗口 [i-k, i-1] 内维护 f[j] - pre[j] 的最大值
关键性质：f[i] = pre[i] + max(f[j] - pre[j])，j ∈ [i-k, i-1] 且 j 处必须不选
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
long long f[N], pre[N]; // f[i]: 前 i 头最大效率, pre: 前缀和
int q[N], hh = 0, tt = -1;

long long val(int j) { return f[j] - pre[j]; }

void solve(int n, int k) {
    // f[i] = max(f[i-1], pre[i] + max{f[j]-pre[j]})，j ∈ [i-k, i-1]
    q[++tt] = 0;
    for (int i = 1; i <= n; i++) {
        while (hh <= tt && q[hh] < i - k) hh++;
        f[i] = max(f[i - 1], pre[i] + val(q[hh]));
        while (hh <= tt && val(q[tt]) <= val(i)) tt--;
        q[++tt] = i;
    }
}
// 答案位置：f[n]
```

**核心启示**：
- **变量分离**：将 `f[j] + (pre[i] - pre[j])` 拆为 `pre[i] + (f[j] - pre[j])`，后半部分只与 `j` 有关
- **单调队列优化 DP 的关键**：识别出"只与决策点相关"的部分，作为队列中维护的值

### 题目 11：绿色通道（洛谷 P2034）

> $n$ 道题，每道需时间 $t_i$。可以空出一些题不做，但任意连续空题的段长度 $\leq k$。最小化总用时。二分 $k$ + DP 验证。

**分析**：二分答案 $k$（最长可空段），对于固定 $k$，DP 验证：`f[i] = min(f[j]) + t[i]`，其中 `j ∈ [i-k-1, i-1]`。用单调队列优化 DP。

```
数据结构：二分答案 + 单调队列优化 DP
支持操作：窗口 [i-k-1, i-1] 内维护 f[j] 最小值
关键性质：先二分空题段长度 k，再用单调队列 O(n) 检验
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
long long f[N];
int q[N], hh, tt;

bool check(int t[], int n, int k, long long limit) {
    hh = 0; tt = -1;
    q[++tt] = 0; f[0] = 0;
    for (int i = 1; i <= n; i++) {
        f[i] = 1e18;
        while (hh <= tt && q[hh] < i - k - 1) hh++;
        if (hh <= tt) f[i] = f[q[hh]] + t[i];
        while (hh <= tt && f[q[tt]] >= f[i]) tt--;
        q[++tt] = i;
    }
    // 检查最后 k+1 个位置中的最小 f 值
    long long mn = 1e18;
    for (int i = max(0, n - k); i <= n; i++) mn = min(mn, f[i]);
    return mn <= limit;
}
```

**核心启示**：
- **二分 + 单调队列 DP**：当问题涉及"最小化最大空隙"，通常二分空隙长度，用 DP 验证可行性
- **外层二分 + 内层贪心/DP**：是竞赛中非常常见的组合模式

### 题目 12：旅行问题（洛谷 P1856 / 经典环形加油站）

> 环形路线上有 $n$ 个加油站，每站可以加 $p_i$ 升油，到下一站需要 $d_i$ 升油。求从哪些站点出发能走完一圈。

**分析**：将环展开为 $2n$ 长度，用前缀和表示"从起点出发后的油量变化"。对每个起点 $i$，检查 $[i, i+n-1]$ 范围内前缀和的最小值是否 $\geq 0$——即滑动窗口最小值。

```
数据结构：单调队列（递增）+ 环形展开
支持操作：滑动窗口长度 n，维护前缀和最小值
关键性质：净油量前缀和的最小值 ≥ 0 ↔ 能走完一圈
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
long long s[2 * N]; // 环展开后的前缀和
int q[2 * N], hh, tt;

void solve(int p[], int d[], int n) {
    // s[i] = sum(p[j]-d[j]), j=0..i-1 （展开为 2n）
    for (int i = 0; i < 2 * n; i++)
        s[i + 1] = s[i] + (p[i % n] - d[i % n]);
    // 对每个起点 i，检查 min(s[i+1..i+n]) - s[i] >= 0
    hh = 0; tt = -1;
    for (int i = 2 * n; i >= 1; i--) {
        while (hh <= tt && s[q[tt]] >= s[i]) tt--;
        q[++tt] = i;
        if (q[hh] > i + n) hh++;
        if (i <= n && s[q[hh]] - s[i - 1] >= 0)
            printf("%d can start\n", i);
    }
}
```

**核心启示**：
- **环形问题展开**：环形数组展开为 $2n$，配合滑动窗口处理
- **前缀和 + 窗口最小值**：判断区间内是否"一直非负"→ 前缀和始终 $\geq$ 起点值

### 题目 13：多重背包（单调队列优化）

> $n$ 种物品，第 $i$ 种体积 $v_i$、价值 $w_i$、数量 $s_i$。背包容量 $V$。求最大价值。

**分析**：多重背包的经典优化。按余数 $r = j \bmod v$ 分组，每组内转移形如 `f[r + k*v] = max(f[r + t*v] - t*w) + k*w`，`t` 的范围是 $[k-s, k-1]$——滑动窗口。

```
数据结构：单调队列优化 DP
支持操作：按余数分组，每组内用滑动窗口维护最优决策
关键性质：同余类内的转移可分离变量，决策范围是定长窗口
时间复杂度：O(nV)
```

**C++ 代码**：

```cpp
long long f[MAXV];
int q[MAXV], hh, tt;

void solve(int v, int w, int s, int V) {
    // 按余数 r 分组
    for (int r = 0; r < v; r++) {
        hh = 0; tt = -1;
        for (int k = 0; r + k * v <= V; k++) {
            long long val = f[r + k * v] - (long long)k * w;
            while (hh <= tt && q[hh] < k - s) hh++;
            while (hh <= tt && (f[r + q[tt] * v] - (long long)q[tt] * w) <= val) tt--;
            q[++tt] = k;
            f[r + k * v] = f[r + q[hh] * v] - (long long)q[hh] * w + (long long)k * w;
        }
    }
}
```

**核心启示**：
- **多重背包单调队列优化**：按余数分组是关键洞察，将混在一起的转移分离到各自的同余类
- **变量分离**：`f[r+k*v] = max{g(t)} + h(k)`，其中 `g(t)` 只与决策 `t` 有关

### 题目 14：01 BFS（双端队列 BFS）

> 给定 $n$ 个节点 $m$ 条边的图，边权为 0 或 1。求单源最短路。

**分析**：Dijkstra 对 01 权边太重，BFS 又不能处理不等权。01 BFS 用双端队列：遇到权 0 的边放队头，权 1 的边放队尾。保证队列中距离单调。

```
数据结构：双端队列（deque）
支持操作：权 0 放队头、权 1 放队尾
关键性质：队列中元素的距离至多差 1，保证队头总是当前最短
时间复杂度：O(n + m)
```

**C++ 代码**：

```cpp
int dist[N], q[2 * M], hh = 0, tt = -1; // 数组模拟双端队列

void bfs01(int s, int n) {
    memset(dist, 0x3f, sizeof dist);
    dist[s] = 0;
    // 用数组两端模拟 deque，或直接用 deque<int>
    deque<int> dq;
    dq.push_back(s);
    while (!dq.empty()) {
        int u = dq.front(); dq.pop_front();
        for (int i = head[u]; ~i; i = ne[i]) {
            int v = e[i], w = we[i]; // w 为 0 或 1
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                if (w == 0) dq.push_front(v);
                else dq.push_back(v);
            }
        }
    }
}
```

**核心启示**：
- **01 BFS 替代 Dijkstra**：边权只有 0 和 1 时，双端队列 BFS 是 $O(n+m)$ 的最优解
- **队列扩展**：权 0 放队头（等价于不花代价的扩展），权 1 放队尾（花 1 代价）

### 题目 15：最大子序和（洛谷 P1440）

> 给定长度为 $n$ 的数列和整数 $m$，对每个位置 $i$ 求 $\max(a[i-m], \ldots, a[i-1])$。

**分析**：滑动窗口大小为 $m$，求窗口内的最大值。注意窗口是"前 $m$ 个元素"，不包含当前位置。

```
数据结构：单调队列（递减）
支持操作：维护前 m 个元素中的最大值
关键性质：窗口不含当前元素 → 先取答案再入队
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
int a[N], q[N], hh = 0, tt = -1;

void solve(int n, int m) {
    for (int i = 1; i <= n; i++) {
        while (hh <= tt && q[hh] < i - m) hh++;
        // 先输出答案（窗口内是 [i-m, i-1]）
        if (i > 1) printf("%d\n", (hh <= tt) ? a[q[hh]] : 0);
        // 再将 a[i] 入队
        while (hh <= tt && a[q[tt]] <= a[i]) tt--;
        q[++tt] = i;
    }
}
```

**核心启示**：
- **先查后入**：当窗口不包含当前元素时，必须在入队前取答案
- **细节决定正确性**：入队和取值的顺序是单调队列题中最常犯的错误

### 题目 16：「NOI2005」瑰丽华尔兹（洛谷 P2254）

> $n \times m$ 的网格上有一架钢琴。给出 $K$ 个时段，每个时段钢琴向指定方向滑动（可选择停下）。求最长滑动距离。障碍物不可穿越。

**分析**：DP：`f[t][i][j]` = 前 $t$ 个时段到达 $(i,j)$ 的最长距离。每个时段内，同方向滑动 $[0, len]$ 步——沿一条线的滑动窗口最大值。用单调队列优化。

```
数据结构：单调队列优化 DP
支持操作：沿某方向的一维滑动窗口，维护 DP 值最大
关键性质：每个时段沿固定方向，等价于一维滑动窗口 DP
时间复杂度：O(K * n * m)
```

**C++ 代码**：

```cpp
int f[N][N]; // 当前时段的 DP 值
int q[N], hh, tt;

// 沿方向 (dx,dy) 处理，滑动窗口大小 len
void processLine(int sx, int sy, int dx, int dy, int len, int rows, int cols, int grid[][N]) {
    hh = 0; tt = -1;
    int x = sx, y = sy, cnt = 0;
    while (x >= 0 && x < rows && y >= 0 && y < cols) {
        if (grid[x][y]) { hh = 0; tt = -1; } // 遇障碍清空队列
        else {
            while (hh <= tt && q[hh] < cnt - len) hh++;
            while (hh <= tt && f[x - dx * (cnt - q[tt])][y - dy * (cnt - q[tt])] <= f[x][y]) tt--;
            // 简化：将坐标映射到一维处理
            q[++tt] = cnt;
            if (hh <= tt) f[x][y] = max(f[x][y], /* 队头决策的 f 值 */ + (cnt - q[hh]));
        }
        x += dx; y += dy; cnt++;
    }
}
```

**核心启示**：
- **网格 DP + 单调队列**：当 DP 的转移方向固定时，沿该方向展开为一维，用单调队列优化
- **障碍物处理**：遇到障碍时清空队列重新开始，相当于"分段"处理

## 五、常见技巧与思路总结

### 5.1 队列存下标而非值

单调队列中几乎总是**存储下标**，因为：
- 方便判断是否过期（`q[hh] <= i - k`）
- 通过下标可随时获取对应的值

### 5.2 数组模拟双端队列

```cpp
int q[N], hh = 0, tt = -1;
// push_back: q[++tt] = x;   pop_back: tt--;
// push_front: q[--hh] = x;  pop_front: hh++;
// front: q[hh];  back: q[tt];  empty: hh > tt;
```

### 5.3 单调队列优化 DP 的识别

当 DP 转移满足以下条件时可用单调队列：
1. `f[i] = opt{f[j] + g(i, j)}`
2. `j` 的范围是 `[i-k, i-1]`（滑动窗口）
3. `g(i, j)` 可以**分离**为 `h(i) + q(j)`

### 5.4 BFS 队列的变种

| 变种 | 队列类型 | 适用场景 |
|------|---------|---------|
| 普通 BFS | 队列 | 无权图最短路 |
| 01 BFS | 双端队列 | 01 权图最短路 |
| 优先队列 BFS (Dijkstra) | 小根堆 | 非负权图最短路 |
| 双向 BFS | 两个队列 | 起终点已知的搜索 |

### 5.5 常见坑点

1. **弹出前判空**：每次 `hh++` 或 `tt--` 前确保 `hh <= tt`
2. **先查后入 vs 先入后查**：窗口是否包含当前元素决定顺序
3. **`<=` vs `<`**：影响队列中是否允许重复值

## 七、终极总结与归纳

### 7.1 队列的本质

**队列 = 先进先出的有序等待线。** 单调队列在此基础上维护"有序性"，用 $O(1)$ 均摊代价获取窗口最值，是将 $O(nk)$ 优化为 $O(n)$ 的核心武器。

### 7.2 核心要点

2. **单调队列三步法**：弹过期 → 维护单调 → 入队，是固定范式
3. **存下标不存值**：方便判过期、反查值、计算距离
4. **前缀和 + 单调队列**：处理"限长最大子段和"的黄金组合
5. **单调队列优化 DP**：当决策区间是滑动窗口且可变量分离时适用
6. **01 BFS**：边权只有 0/1 时用双端队列代替 Dijkstra，$O(n+m)$
