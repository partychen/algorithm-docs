## 一、什么是二叉堆

二叉堆是一棵**完全二叉树**，满足**堆序性**：对于大根堆，每个节点的值都 $\geq$ 其子节点的值；对于小根堆，每个节点的值都 $\leq$ 其子节点的值。

二叉堆是**优先队列**最常见的底层实现，用**数组**紧凑存储：

```
数组: [_, 50, 30, 40, 10, 20]  (下标从 1 开始)

对应的完全二叉树（大根堆）:
        50          节点 i 的左孩子: 2i
       /  \         节点 i 的右孩子: 2i+1
     30    40       节点 i 的父亲:   i/2
    / \
  10   20
```

**核心操作与复杂度：**

| 操作 | 时间复杂度 |
|------|-----------|
| 插入元素（push） | $O(\log n)$ |
| 查询堆顶（top） | $O(1)$ |
| 删除堆顶（pop） | $O(\log n)$ |
| 建堆（heapify） | $O(n)$ |

## 二、核心思想与前置知识

### 2.1 堆的性质

1. **结构性**：完全二叉树，可用数组紧凑存储
2. **堆序性**：父 $\geq$ 子（大根堆）或父 $\leq$ 子（小根堆）。**只约束父子，不约束兄弟**

### 2.2 上浮与下沉

**上浮（Sift Up）**：新元素放到末尾后，不断与父节点比较并交换。用于**插入**。

**下沉（Sift Down）**：用末尾元素替换堆顶后，不断与较大/较小的子节点交换。用于**删除堆顶**。

```cpp
const int N = 100010;
int h[N], sz; // h[1..sz] 为堆

void up(int p) {
    while (p > 1 && h[p] < h[p / 2]) { // 小根堆
        swap(h[p], h[p / 2]); p /= 2;
    }
}

void down(int p) {
    while (p * 2 <= sz) {
        int s = p * 2;
        if (s + 1 <= sz && h[s + 1] < h[s]) s++;
        if (h[p] <= h[s]) break;
        swap(h[p], h[s]); p = s;
    }
}

void push(int x) { h[++sz] = x; up(sz); }
void pop() { h[1] = h[sz--]; down(1); }
int top() { return h[1]; }
```

### 2.3 建堆 $O(n)$

自底向上对每个非叶节点执行下沉。为什么是 $O(n)$？

$$\sum_{h=0}^{\lfloor \log n \rfloor} \frac{n}{2^{h+1}} \cdot O(h) = O\left(n \sum_{h=0}^{\infty} \frac{h}{2^h}\right) = O(n)$$

大部分节点在底层，下沉距离短。

```cpp
// 将 h[1..n] 建成小根堆
void build(int n) { sz = n; for (int i = n / 2; i >= 1; i--) down(i); }
```

### 2.4 STL `priority_queue`

```cpp
// 大根堆（默认）
priority_queue<int> maxH;

// 小根堆
priority_queue<int, vector<int>, greater<int>> minH;

// 自定义比较
struct Node {
    int val, id;
    bool operator>(const Node& o) const { return val > o.val; }
};
priority_queue<Node, vector<Node>, greater<Node>> pq;
```

**技巧**：
- 小根堆懒写法：`push(-x)`，取出时 `-top()`
- `priority_queue` **不支持** 删除任意元素和修改优先级 → 用懒删除或手写堆

## 三、分类与变体详解

### 3.1 大根堆 vs 小根堆

| 类型 | 堆顶 | 典型应用 |
|------|------|----------|
| 大根堆 | 最大值 | 取最大、TopK 小 |
| 小根堆 | 最小值 | Dijkstra、哈夫曼、多路归并 |

### 3.2 对顶堆

大根堆存较小一半 + 小根堆存较大一半，维护大小平衡。堆顶即为中位数。

```
大根堆（较小半）    小根堆（较大半）
     [5]                [6]
    /   \              /   \
  [3]   [4]          [8]   [9]

中位数 = 大根堆堆顶 5（大根堆多一个元素时）
```

### 3.3 懒删除堆

`priority_queue` 不支持删除任意元素。解决办法：维护一个"待删除计数"，取堆顶时跳过。

```cpp
priority_queue<int> heap;
unordered_map<int, int> del_cnt;

void lazy_erase(int x) { del_cnt[x]++; }
int lazy_top() {
    while (del_cnt[heap.top()]) { del_cnt[heap.top()]--; heap.pop(); }
    return heap.top();
}
```

### 3.4 变体对比

| 变体 | 核心思想 | 适用场景 |
|------|---------|---------|
| 基础堆 | 取极值 | 贪心、排序 |
| 对顶堆 | 分半维护 | 中位数、第 k 大 |
| 懒删除堆 | 延迟删除 | 需要删除任意元素 |
| 反悔堆 | 贪心撤回 | 反悔型贪心 |

### 题目 1：堆模板（洛谷 P3378）

> 实现一个小根堆，支持插入、查询最小值、删除最小值。

**分析**：堆的裸模板题，直接用 STL 小根堆或手写堆。

```
数据结构：小根堆
支持操作：push / top / pop
关键性质：堆顶始终是最小值
时间复杂度：每次操作 O(log n)
```

**C++ 代码**：

```cpp
int h[N], sz;
void up(int p) { while (p > 1 && h[p] < h[p/2]) { swap(h[p], h[p/2]); p /= 2; } }
void down(int p) {
    while (p * 2 <= sz) {
        int s = p * 2;
        if (s + 1 <= sz && h[s+1] < h[s]) s++;
        if (h[p] <= h[s]) break;
        swap(h[p], h[s]); p = s;
    }
}
void push(int x) { h[++sz] = x; up(sz); }
void pop() { h[1] = h[sz--]; down(1); }
int top() { return h[1]; }
```

**核心启示**：
- **手写堆**：竞赛中手写堆比 STL 更灵活（支持修改、删除任意位置）
- **下标从 1 开始**：简化父子关系（`2i`、`2i+1`、`i/2`）

### 题目 2：合并果子（洛谷 P1090）

> $n$ 堆果子，每次选两堆合并，代价为两者之和。求把所有果子合成一堆的最小总代价。

**分析**：哈夫曼编码思想。每次合并最小的两堆，总代价最小。用小根堆维护。

```
数据结构：小根堆
支持操作：取两个最小值合并后放回
关键性质：贪心——小的堆被合并次数多，放在底层
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
priority_queue<int, vector<int>, greater<int>> pq;

int solve(int a[], int n) {
    for (int i = 0; i < n; i++) pq.push(a[i]);
    int ans = 0;
    while (pq.size() > 1) {
        int x = pq.top(); pq.pop();
        int y = pq.top(); pq.pop();
        ans += x + y;
        pq.push(x + y);
    }
    return ans;
}
```

**核心启示**：
- **贪心 + 堆**：经典搭配。"每次选最优"的问题首先想堆
- **哈夫曼树**：合并果子 = 构造哈夫曼树，贪心正确性已被证明

### 题目 3：数据流中的第 K 大元素（LeetCode 703）

> 设计一个类，初始化时给定 $k$ 和初始数组。每次调用 `add(val)` 后返回当前数据流中第 $k$ 大的元素。

**分析**：维护一个大小为 $k$ 的小根堆。堆顶就是第 $k$ 大。新元素如果 $>$ 堆顶则替换。

```
数据结构：大小为 k 的小根堆
支持操作：插入时保持堆大小 ≤ k
关键性质：堆顶 = 第 k 大（堆中存最大的 k 个元素的最小值）
时间复杂度：每次 add O(log k)
```

**C++ 代码**：

```cpp
priority_queue<int, vector<int>, greater<int>> pq;
int K;

void init(int k, vector<int>& nums) {
    K = k;
    for (int x : nums) add(x);
}

int add(int val) {
    pq.push(val);
    while ((int)pq.size() > K) pq.pop();
    return pq.top();
}
```

**核心启示**：
- **固定大小堆**：维护 TopK 的标准方法。小根堆存最大的 K 个，堆顶即第 K 大
- **空间高效**：只需 $O(k)$ 空间，不需要存所有数据

### 题目 4：堆排序

> 给定 $n$ 个数，用堆排序对其进行排序。

**分析**：建大根堆后，每次取堆顶（最大值）放到末尾，堆大小减 1。原地排序，$O(n \log n)$。

```
数据结构：大根堆（原地）
支持操作：建堆 + 反复取堆顶放末尾
关键性质：原地排序，不需要额外空间
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
void down(int a[], int p, int sz) {
    while (p * 2 <= sz) {
        int s = p * 2;
        if (s + 1 <= sz && a[s + 1] > a[s]) s++;
        if (a[p] >= a[s]) break;
        swap(a[p], a[s]); p = s;
    }
}

void heapSort(int a[], int n) { // a[1..n]
    for (int i = n / 2; i >= 1; i--) down(a, i, n); // 建大根堆
    for (int i = n; i > 1; i--) {
        swap(a[1], a[i]); // 最大值放末尾
        down(a, 1, i - 1); // 堆大小减 1
    }
}
```

**核心启示**：
- **原地排序**：堆排序是唯一既 $O(n \log n)$ 又 $O(1)$ 额外空间的比较排序
- **不稳定**：堆排序不保持相等元素的相对顺序

### 题目 5：中位数（洛谷 P1168）

> 给定长度为 $n$ 的序列，对每个奇数位 $i$ 输出前 $i$ 个数的中位数。

**分析**：对顶堆。大根堆存较小半，小根堆存较大半。保持大根堆比小根堆多一个，堆顶即中位数。

```
数据结构：对顶堆（大根堆 + 小根堆）
支持操作：插入后平衡两堆大小
关键性质：大根堆恰好多 1 个元素 → 堆顶 = 中位数
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
priority_queue<int> maxH;
priority_queue<int, vector<int>, greater<int>> minH;

void insert(int x) {
    if (maxH.empty() || x <= maxH.top()) maxH.push(x);
    else minH.push(x);
    // 平衡
    while ((int)maxH.size() > (int)minH.size() + 1) { minH.push(maxH.top()); maxH.pop(); }
    while ((int)minH.size() > (int)maxH.size()) { maxH.push(minH.top()); minH.pop(); }
}
// 中位数：maxH.top()
```

**核心启示**：
- **对顶堆**：动态中位数的标准方法
- **平衡是关键**：每次插入后必须检查并调整两堆大小

### 题目 6：最小函数值（洛谷 P2085）

> $n$ 个二次函数 $f_i(x) = A_i x^2 + B_i x + C_i$（$A_i > 0$），求所有函数在正整数处取值的前 $m$ 小值。

**分析**：每个函数在正整数上递增。$n$ 个有序序列取前 $m$ 小 → **多路归并**。初始入堆 $f_i(1)$，每次弹出最小值后推入 $f_i(x+1)$。

```
数据结构：小根堆（多路归并）
支持操作：弹出最小 → 推入后继
关键性质：每个序列有序，堆维护各序列当前最小值
时间复杂度：O(m log n)
```

**C++ 代码**：

```cpp
int A[N], B[N], C[N];
long long f(int i, int x) { return (long long)A[i]*x*x + B[i]*x + C[i]; }

void solve(int n, int m) {
    priority_queue<tuple<long long,int,int>, vector<tuple<long long,int,int>>, greater<>> pq;
    for (int i = 1; i <= n; i++) pq.push({f(i, 1), i, 1});
    for (int k = 0; k < m; k++) {
        auto [val, i, x] = pq.top(); pq.pop();
        printf("%lld ", val);
        pq.push({f(i, x + 1), i, x + 1});
    }
}
```

**核心启示**：
- **多路归并**：$k$ 个有序序列取前 $m$ 小的标准方法
- **推入后继**：弹出 $(f_i, x)$ 后推入 $(f_i, x+1)$，保证不遗漏

### 题目 7：Dijkstra 最短路（洛谷 P4779）

> 给定 $n$ 个点 $m$ 条边的有向图（非负权），求单源最短路。

**分析**：Dijkstra 算法用小根堆维护"当前距离最小的未确定节点"。每次取堆顶扩展邻居。

```
数据结构：小根堆（Dijkstra）
支持操作：取距离最小的节点、松弛后入堆
关键性质：贪心——距离最小的节点不会再被更新
时间复杂度：O((n+m) log n)
```

**C++ 代码**：

```cpp
int dist[N];
bool vis[N];
typedef pair<int,int> pii;

void dijkstra(int s, int n) {
    memset(dist, 0x3f, sizeof dist);
    dist[s] = 0;
    priority_queue<pii, vector<pii>, greater<pii>> pq;
    pq.push({0, s});
    while (!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if (vis[u]) continue;
        vis[u] = true;
        for (int i = head[u]; ~i; i = ne[i]) {
            int v = e[i], w = we[i];
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                pq.push({dist[v], v});
            }
        }
    }
}
```

**核心启示**：
- **Dijkstra = 贪心 + 堆**：堆保证每次取出的是全局距离最小的节点
- **重复入堆**：同一节点可能多次入堆，用 `vis` 跳过已确定的

### 题目 8：超级丑数（LeetCode 313）

> 给定质因子集合 `primes`，求第 $n$ 个超级丑数（只含给定质因子的正整数）。

**分析**：多路归并。每个质因子 $p$ 维护一个指针，指向当前的超级丑数。每次从堆中取最小值，然后将该质因子的下一个候选入堆。注意去重。

```
数据结构：小根堆（多路归并 + 去重）
支持操作：弹最小 → 推入所有质因子的倍数
关键性质：超级丑数集合有序，用指针 + 堆避免重复生成
时间复杂度：O(n·k·log k)，k 为质因子个数
```

**C++ 代码**：

```cpp
int nthSuperUglyNumber(int n, vector<int>& primes) {
    vector<long long> ugly(n);
    ugly[0] = 1;
    // pq: {当前值, 质因子, 指针位置}
    priority_queue<tuple<long long,int,int>, vector<tuple<long long,int,int>>, greater<>> pq;
    for (int p : primes) pq.push({(long long)p, p, 0});
    for (int i = 1; i < n; ) {
        auto [val, p, idx] = pq.top(); pq.pop();
        if (val != ugly[i - 1]) { ugly[i] = val; i++; } // 去重
        pq.push({(long long)p * ugly[idx + 1], p, idx + 1});
    }
    return ugly[n - 1];
}
```

**核心启示**：
- **丑数生成 = 多路归并**：每个质因子对应一路有序序列
- **去重**：多路归并可能产生重复值，需要在递增时跳过

### 题目 9：蚯蚓（洛谷 P2827）

> $n$ 条蚯蚓，每秒选最长的切成两段，同时其他蚯蚓长度增加 $q$。输出每秒被切的蚯蚓长度和最终长度。

**分析**：全局加 $q$ 不可能逐一操作。维护**全局偏移量 $\delta$**：入堆时减 $\delta$，出堆时加 $\delta$。

```
数据结构：大根堆 + 全局偏移量（或三队列 O(n) 优化）
支持操作：取最大（加偏移）、放回（减偏移）
关键性质：全局加操作用偏移量代替，入堆减、出堆加
时间复杂度：O(m log n)（堆）/ O(n log n + m)（三队列）
```

**C++ 代码**：

```cpp
priority_queue<long long> pq;
long long delta = 0;

void solve(int n, int m, int q, int u, int v) {
    // 初始入堆
    for (int i = 1; i <= m; i++) {
        long long len = pq.top() + delta; pq.pop();
        long long a = len * u / v, b = len - a;
        delta += q;
        pq.push(a - delta);
        pq.push(b - delta);
    }
}
// 取出时真实值 = pq.top() + delta
```

**核心启示**：
- **全局偏移量**：入堆减、出堆加，避免逐一修改所有元素
- **三队列优化**：利用单调性可以 $O(1)$ 取最大值，将 $O(m \log n)$ 降为 $O(m)$

### 题目 10：Cow Coupons（洛谷 P3045）

> $n$ 头牛有原价 $c_i$ 和优惠价 $d_i$，$k$ 张优惠券，$m$ 元钱。求最多买几头。

**分析**：反悔贪心。先给优惠价最低的 $k$ 头用券。之后每头牛选"原价买"或"抢前面的券（补差价 $c_j-d_j$）+ 自己用券"中较小的。用小根堆维护反悔代价。

```
数据结构：小根堆（反悔堆）
支持操作：维护已用券的牛的"补差价"，贪心选最小反悔代价
关键性质：反悔贪心——允许撤回之前的选择，用堆维护最小反悔成本
时间复杂度：O(n log n)
```

**C++ 代码**：

```cpp
struct Cow { long long c, d; } cow[N];
priority_queue<long long, vector<long long>, greater<long long>> refund;

int solve(int n, int k, long long m) {
    sort(cow, cow + n, [](auto& a, auto& b) { return a.d < b.d; });
    long long cost = 0; int ans = 0;
    for (int i = 0; i < min(k, n); i++) {
        cost += cow[i].d;
        if (cost > m) return ans;
        ans++;
        refund.push(cow[i].c - cow[i].d);
    }
    for (int i = min(k, n); i < n; i++) {
        long long opt1 = cow[i].c;
        long long opt2 = cow[i].d + refund.top();
        long long best = min(opt1, opt2);
        cost += best;
        if (cost > m) return ans;
        ans++;
        if (opt2 < opt1) { refund.pop(); refund.push(cow[i].c - cow[i].d); }
    }
    return ans;
}
```

**核心启示**：
- **反悔贪心**：用堆维护"撤回之前选择的代价"，允许贪心纠错
- **堆中是"反悔的成本"**：弹出最小反悔代价，比较两种方案取更优

### 题目 11：丑数 II（LeetCode 264）

> 求第 $n$ 个丑数（只包含质因子 2、3、5 的正整数）。

**分析**：三指针法（比堆更优）。维护三个指针分别乘以 2、3、5，每次取最小值推进对应指针。

```
数据结构：三指针（等价于三路归并，无需堆）
支持操作：取三者最小值、推进对应指针
关键性质：dp[i] = min(dp[p2]*2, dp[p3]*3, dp[p5]*5)
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
int nthUglyNumber(int n) {
    vector<int> dp(n);
    dp[0] = 1;
    int p2 = 0, p3 = 0, p5 = 0;
    for (int i = 1; i < n; i++) {
        int v2 = dp[p2] * 2, v3 = dp[p3] * 3, v5 = dp[p5] * 5;
        dp[i] = min({v2, v3, v5});
        if (dp[i] == v2) p2++;
        if (dp[i] == v3) p3++;
        if (dp[i] == v5) p5++;
    }
    return dp[n - 1];
}
```

**核心启示**：
- **三指针替代堆**：当归并路数固定且很小时，指针法比堆更高效
- **去重技巧**：`if` 而非 `else if`——相等时所有指针都推进

### 题目 12：数据流的中位数（LeetCode 295）

> 设计一个类，支持 `addNum(num)` 和 `findMedian()`，在线维护中位数。

**分析**：对顶堆的在线版本。与题目 5 相同思路，但需要处理偶数个元素时取两个堆顶的平均值。

```
数据结构：对顶堆
支持操作：在线插入 + 查询中位数
关键性质：偶数个元素时中位数 = (maxH.top() + minH.top()) / 2.0
时间复杂度：插入 O(log n)，查询 O(1)
```

**C++ 代码**：

```cpp
priority_queue<int> maxH;
priority_queue<int, vector<int>, greater<int>> minH;

void addNum(int num) {
    if (maxH.empty() || num <= maxH.top()) maxH.push(num);
    else minH.push(num);
    if ((int)maxH.size() > (int)minH.size() + 1) { minH.push(maxH.top()); maxH.pop(); }
    if ((int)minH.size() > (int)maxH.size()) { maxH.push(minH.top()); minH.pop(); }
}

double findMedian() {
    if (maxH.size() > minH.size()) return maxH.top();
    return (maxH.top() + minH.top()) / 2.0;
}
```

**核心启示**：
- **在线中位数**：面试高频题，对顶堆是唯一的 $O(\log n)$ 在线解法
- **奇偶分讨**：奇数个取大根堆顶，偶数个取两堆顶平均

### 题目 13：Dijkstra + 路径恢复（综合）

> 求单源最短路并输出最短路径。

**分析**：在 Dijkstra 基础上，用 `pre[v] = u` 记录前驱，从终点回溯到起点恢复路径。

```
数据结构：小根堆 + 前驱数组
支持操作：Dijkstra 松弛时记录前驱
关键性质：前驱数组形成最短路径树
时间复杂度：O((n+m) log n)
```

**C++ 代码**：

```cpp
int dist[N], pre[N];
bool vis[N];

void dijkstra(int s, int n) {
    memset(dist, 0x3f, sizeof dist);
    memset(pre, -1, sizeof pre);
    dist[s] = 0;
    priority_queue<pii, vector<pii>, greater<pii>> pq;
    pq.push({0, s});
    while (!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if (vis[u]) continue;
        vis[u] = true;
        for (int i = head[u]; ~i; i = ne[i]) {
            int v = e[i], w = we[i];
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                pre[v] = u;
                pq.push({dist[v], v});
            }
        }
    }
}

void printPath(int t) {
    if (t == -1) return;
    printPath(pre[t]);
    printf("%d ", t);
}
```

**核心启示**：
- **前驱数组**：路径恢复的标准方法，从终点沿 `pre` 回溯到起点
- **最短路径树**：`pre` 数组定义了一棵以起点为根的最短路径树

### 题目 14：任务调度器（LeetCode 621）

> 给定任务数组和冷却时间 $n$。相同任务之间至少间隔 $n$ 个单位时间。求完成所有任务的最短时间。

**分析**：贪心。每轮选剩余次数最多的 $n+1$ 种任务执行。用大根堆维护剩余次数。

```
数据结构：大根堆
支持操作：每轮取前 n+1 个最多的任务
关键性质：优先执行剩余次数最多的任务，最小化空闲时间
时间复杂度：O(总时间 · 26 · log 26) ≈ O(总时间)
```

**C++ 代码**：

```cpp
int leastInterval(vector<char>& tasks, int n) {
    int cnt[26] = {};
    for (char c : tasks) cnt[c - 'A']++;
    priority_queue<int> pq;
    for (int c : cnt) if (c) pq.push(c);
    int time = 0;
    while (!pq.empty()) {
        vector<int> tmp;
        int slots = n + 1;
        while (slots > 0 && !pq.empty()) {
            int top = pq.top(); pq.pop();
            if (top > 1) tmp.push_back(top - 1);
            time++; slots--;
        }
        for (int t : tmp) pq.push(t);
        if (!pq.empty()) time += slots; // 空闲时间
    }
    return time;
}
```

**核心启示**：
- **轮次贪心**：每轮 $n+1$ 个槽位，优先填剩余最多的任务
- **空闲时间**：当任务不足以填满一轮时，剩余槽位为空闲

### 题目 15：K 个最接近原点的点（LeetCode 973）

> 给定平面上 $n$ 个点，返回距离原点最近的 $k$ 个点。

**分析**：TopK 问题。维护大小为 $k$ 的大根堆（存距离最大的，堆顶是"第 k 近的"），当新点更近时替换堆顶。

```
数据结构：大小为 k 的大根堆
支持操作：新点距离 < 堆顶距离时替换
关键性质：大根堆存最近的 k 个，堆顶是第 k 近的
时间复杂度：O(n log k)
```

**C++ 代码**：

```cpp
vector<vector<int>> kClosest(vector<vector<int>>& points, int k) {
    // pq: {距离², 下标}
    priority_queue<pair<int,int>> pq;
    for (int i = 0; i < (int)points.size(); i++) {
        int d = points[i][0]*points[i][0] + points[i][1]*points[i][1];
        pq.push({d, i});
        if ((int)pq.size() > k) pq.pop();
    }
    vector<vector<int>> res;
    while (!pq.empty()) { res.push_back(points[pq.top().second]); pq.pop(); }
    return res;
}
```

**核心启示**：
- **TopK 小用大根堆**：反直觉但正确——堆中保持 $k$ 个最小的，大根堆可以 $O(1)$ 淘汰最大的
- **距离用平方**：避免浮点运算和 `sqrt`

### 🟣 竞赛级 Level 5：高难度堆应用

### 题目 16：「NOI2010」超级钢琴（洛谷 P2048）

> 长度为 $n$ 的序列，选 $k$ 个不重叠的长度 $\in [L, R]$ 的子段，使得子段和的总和最大。

**分析**：先不考虑不重叠。对每个起点 $i$，长度 $[L, R]$ 的子段和 = `pre[j] - pre[i-1]`，$j \in [i+L-1, i+R-1]$。最大 $j$ 由 ST 表 $O(1)$ 查询。将每个起点的最优子段入堆，每次取最大后将该起点的次优入堆（区间分裂）。

```
数据结构：大根堆 + ST 表（区间最大前缀和位置）
支持操作：取全局最优子段 → 区间分裂推入次优
关键性质：类似"超级多路归并"，每次取最大后分裂出新候选
时间复杂度：O((n+k) log n)
```

**C++ 代码**：

```cpp
long long pre[N];
int st[N][20]; // ST 表存 pre[] 最大值的位置

int qmax(int l, int r) { // 查询 pre[l..r] 最大值位置
    int k = __lg(r - l + 1);
    return pre[st[l][k]] >= pre[st[r-(1<<k)+1][k]] ? st[l][k] : st[r-(1<<k)+1][k];
}

struct Item {
    int i, lo, hi, best; // 起点 i, 区间 [lo,hi], 最优终点 best
    long long val;
    bool operator<(const Item& o) const { return val < o.val; }
};

long long solve(int n, int k, int L, int R) {
    // 建 ST 表（pre[] 最大值位置）
    priority_queue<Item> pq;
    for (int i = 1; i + L - 1 <= n; i++) {
        int lo = i + L - 1, hi = min(n, i + R - 1);
        int best = qmax(lo, hi);
        pq.push({i, lo, hi, best, pre[best] - pre[i - 1]});
    }
    long long ans = 0;
    for (int t = 0; t < k; t++) {
        auto [i, lo, hi, best, val] = pq.top(); pq.pop();
        ans += val;
        // 区间分裂：[lo, best-1] 和 [best+1, hi]
        if (best > lo) {
            int b = qmax(lo, best - 1);
            pq.push({i, lo, best - 1, b, pre[b] - pre[i - 1]});
        }
        if (best < hi) {
            int b = qmax(best + 1, hi);
            pq.push({i, best + 1, hi, b, pre[b] - pre[i - 1]});
        }
    }
    return ans;
}
```

**核心启示**：
- **堆 + 区间分裂**：取出最优后将区间一分为二，推入两个次优——这是"第 k 大/小"类问题的通用技巧
- **ST 表辅助**：区间最大值位置用 ST 表 $O(1)$ 查询，避免重复计算

## 五、常见技巧与思路总结

### 5.1 取负值代替小根堆

```cpp
pq.push(-x); // 入堆
int val = -pq.top(); pq.pop(); // 取出
```

### 5.2 全局偏移量

```cpp
long long delta = 0;
// 入堆: pq.push(real_val - delta)
// 出堆: real_val = pq.top() + delta
// 全局加: delta += inc
```

### 5.3 多路归并模式

堆中存 `{值, 序列编号, 指针}`，弹出后推入该序列的下一个元素。

### 5.4 TopK 问题

求前 $k$ 小 → 用大小 $k$ 的**大根堆**（淘汰大的）。
求前 $k$ 大 → 用大小 $k$ 的**小根堆**（淘汰小的）。

### 5.5 反悔贪心

堆中存"反悔代价"。贪心选择后将反悔代价入堆，后续发现更优时弹出反悔。

## 六、解题思路框架

├─ 是 → 堆

## 七、终极总结与归纳

### 7.1 二叉堆的本质

**二叉堆 = 完全二叉树 + 堆序性，用数组存储，$O(\log n)$ 插入删除、$O(1)$ 取极值。**

它是优先队列的标准实现，是贪心算法、图论算法（Dijkstra、Prim）、排序算法的核心组件。

### 7.2 核心要点

1. **上浮和下沉**是核心操作，掌握手写堆是进阶必备
2. **对顶堆**是动态中位数/第 k 大的标准方法
3. **多路归并**是堆最经典的应用模式——弹出最小后推入后继
4. **反悔贪心**通过堆维护"撤回代价"，解决看似无法贪心的问题
5. **全局偏移量**避免逐一修改堆中元素
6. **TopK 求小用大根堆，求大用小根堆**——反直觉但正确
