## 一、什么是前缀和与差分

**前缀和**（Prefix Sum）是一种预处理技巧：对数组 `a[]` 构建辅助数组 `pre[]`，使得 `pre[i] = a[1] + a[2] + ... + a[i]`。这样，任意区间 `[l, r]` 的元素之和就可以用 `pre[r] - pre[l-1]` 在 **O(1)** 时间内算出，而无需逐个遍历。

**差分**（Difference Array）是前缀和的逆运算。对数组 `a[]` 构建差分数组 `d[]`，使得 `d[i] = a[i] - a[i-1]`。当需要对区间 `[l, r]` 统一加减一个值时，只需修改 `d[l]` 和 `d[r+1]` 两个位置，最后做一次前缀和还原即可。

前缀和与差分互为逆操作，就像积分与微分的关系一样。掌握它们，就掌握了"批量查询"与"批量修改"两大利器。

```
原数组 a[]:   3   1   4   1   5   9   2   6
               ↓ 前缀和（求累加）
前缀和 pre[]: 3   4   8   9  14  23  25  31
               ↓ 差分（求相邻差）
差分   d[]:   3  -2   3  -3   4   4  -7   4
```

| 特性     | 前缀和                    | 差分                      |
|----------|---------------------------|---------------------------|
| 核心操作 | O(1) 区间求和查询         | O(1) 区间加减修改         |
| 预处理   | O(n) 建表                 | O(n) 建表                 |
| 互逆关系 | 差分数组的前缀和 = 原数组 | 前缀和数组的差分 = 原数组 |
| 高维推广 | 二维前缀和（容斥）        | 二维差分（容斥）          |
| 典型场景 | 多次查询子数组和          | 多次修改后一次性查询      |

## 二、核心思想与前置知识

### 2.1 一维前缀和的定义与模板

**定义**：`pre[i] = a[1] + a[2] + ... + a[i]`，其中 `pre[0] = 0`。

**区间求和公式**：`sum(l, r) = pre[r] - pre[l - 1]`

```
索引:    0    1    2    3    4    5
a[]:         3    1    4    1    5
pre[]:  0    3    4    8    9   14

sum(2,4) = pre[4] - pre[1] = 9 - 3 = 6
即 a[2]+a[3]+a[4] = 1+4+1 = 6  ✓
```

**模板代码**：

```cpp
const int MAXN = 1e5 + 5;
int a[MAXN], pre[MAXN];

// 构建前缀和，下标从 1 开始
void build(int n) {
    pre[0] = 0;
    for (int i = 1; i <= n; i++)
        pre[i] = pre[i - 1] + a[i];
}

// 查询 [l, r] 区间和
int query(int l, int r) {
    return pre[r] - pre[l - 1];
}
```

### 2.2 一维差分的定义与模板

**定义**：`d[i] = a[i] - a[i - 1]`，其中 `a[0] = 0`。

**区间修改操作**：对 `[l, r]` 每个元素加 `v`，只需：`d[l] += v; d[r + 1] -= v;`

**还原**：对差分数组求前缀和即得修改后的原数组。

```
原数组 a[]:     1  3  5  2  7
差分   d[]:     1  2  2 -3  5

操作：[2,4] 区间 +3
d[2] += 3 → d[2] = 5
d[5] -= 3 → d[5] = 2

还原前缀和：   1  6  8  5  10
对比原理：     1  3+3  5+3  2+3  7 = 1 6 8 5 7  ← d[5]的-3让位置5不受影响
```

**模板代码**：

```cpp
const int MAXN = 1e5 + 5;
int d[MAXN];

// 差分操作：[l, r] 区间加 v
void add(int l, int r, int v) {
    d[l] += v;
    d[r + 1] -= v;
}

// 还原：对差分数组求前缀和得到最终数组
void restore(int a[], int n) {
    for (int i = 1; i <= n; i++)
        a[i] = a[i - 1] + d[i];
}
```

### 2.3 二维前缀和的定义与模板（含容斥公式推导和图示）

**定义**：`pre[i][j]` 表示从 `(1,1)` 到 `(i,j)` 构成的子矩阵的元素之和。

**建表公式**：

```
pre[i][j] = a[i][j] + pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1]
```

**查询公式**：查询子矩阵 `(x1,y1)` 到 `(x2,y2)` 的和：

```
sum = pre[x2][y2] - pre[x1-1][y2] - pre[x2][y1-1] + pre[x1-1][y1-1]
```

**容斥原理 ASCII 图示**：

```
   y1-1   y1        y2
    |      |         |
----+------+---------+---
    |  D   |    C    |     x1-1
----+------+---------+---
    |      |/////////|     x1
    |  B   |/ query /|
    |      |/////////|     x2
----+------+---------+---

pre[x2][y2]   = A + B + C + D    （整个大矩形）
pre[x1-1][y2] =         C + D    （上方矩形）
pre[x2][y1-1] =     B     + D    （左方矩形）
pre[x1-1][y1-1] =           D    （左上角矩形）

query = (A+B+C+D) - (C+D) - (B+D) + D = A
```

其中 `A` 就是我们要查询的子矩阵区域（图中 `///` 标记区域）。

**建表的容斥示意**：

```
要求 pre[i][j]（下图中 ■ 表示 a[i][j] 这一格）：

    +----------+----+
    |          |    |
    | pre[i-1] |    |
    |   [j-1]  |    |
    +----------+----+
    |          | ■  |
    +----------+----+
      pre[i][j-1]

pre[i][j] = ■ + 上方整块 + 左方整块 - 重叠（左上角）
          = a[i][j] + pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1]
```

**模板代码**：

```cpp
const int MAXN = 1005;
int a[MAXN][MAXN], pre[MAXN][MAXN];

// 建表
void build(int n, int m) {
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            pre[i][j] = a[i][j] + pre[i - 1][j]
                       + pre[i][j - 1] - pre[i - 1][j - 1];
}

// 查询子矩阵 (x1,y1)~(x2,y2) 的和
int query(int x1, int y1, int x2, int y2) {
    return pre[x2][y2] - pre[x1 - 1][y2]
         - pre[x2][y1 - 1] + pre[x1 - 1][y1 - 1];
}
```

### 2.4 二维差分的定义与模板

**二维差分**是二维前缀和的逆操作。对子矩阵 `(x1,y1)` 到 `(x2,y2)` 统一加 `v`：

```
d[x1][y1]     += v
d[x1][y2+1]   -= v
d[x2+1][y1]   -= v
d[x2+1][y2+1] += v
```

还原时对 `d[][]` 做二维前缀和即可恢复修改后的矩阵。

**容斥示意**：

```
     y1      y2  y2+1
      |       |   |
------+-------+---+---
      | +v    | -v|      x1
      |       |   |
------+-------+---+---
      | -v    | +v|      x2+1
------+-------+---+---

对差分数组做前缀和后，恰好只有 (x1,y1)~(x2,y2) 区域被加了 v
```

**模板代码**：

```cpp
const int MAXN = 1005;
int d[MAXN][MAXN];

// 子矩阵 (x1,y1)~(x2,y2) 加 v
void add(int x1, int y1, int x2, int y2, int v) {
    d[x1][y1]     += v;
    d[x1][y2 + 1]   -= v;
    d[x2 + 1][y1]   -= v;
    d[x2 + 1][y2 + 1] += v;
}

// 还原：二维前缀和
void restore(int a[][MAXN], int n, int m) {
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++) {
            d[i][j] += d[i - 1][j] + d[i][j - 1] - d[i - 1][j - 1];
            a[i][j] = d[i][j];
        }
}
```

### 2.5 前缀和与差分的对偶关系

前缀和与差分互为逆运算，就像积分与微分的关系：

```
          差分（求差）
原数组 a[]  ──────────→  差分数组 d[]
     ↑                        │
     │                        │
     └──── 前缀和（求累加）────┘
```

| 操作         | 数学类比 | 效果               |
|--------------|----------|--------------------|
| 前缀和       | 积分 ∫   | 差分 → 原数组还原  |
| 差分         | 微分 d/dx| 原数组 → 差分分解  |
| 差分再前缀和 | ∫(d/dx)  | 恒等变换           |
| 前缀和再差分 | d/dx(∫)  | 恒等变换           |

**这意味着**：
- 需要 **多次查询区间和**？→ 一次前缀和预处理 + O(1) 查询
- 需要 **多次区间修改**？→ 转化为差分 + 一次前缀和还原
- 两者可以 **嵌套使用**：差分解决修改，前缀和解决查询

## 三、前缀和与差分的子模型与分类详解

### 3.1 一维前缀和（区间求和 O(1) 查询）

**适用信号**：题目要求多次查询子数组/子区间的元素之和。

```
状态定义：pre[i] = a[1] + a[2] + ... + a[i]
核心公式：sum(l, r) = pre[r] - pre[l-1]
时间复杂度：预处理 O(n)，单次查询 O(1)
```

```cpp
// 一维前缀和核心
for (int i = 1; i <= n; i++)
    pre[i] = pre[i - 1] + a[i];
// 查询 [l,r]
int ans = pre[r] - pre[l - 1];
```

**要点**：下标从 1 开始，`pre[0] = 0` 作为边界哨兵。

### 3.2 一维差分（区间加减 O(1) 修改）

**适用信号**：题目要求对数组进行多次区间加减操作，最后一次性输出结果。

```
状态定义：d[i] = a[i] - a[i-1]
核心操作：[l,r] 加 v → d[l] += v, d[r+1] -= v
还原方式：对 d[] 求前缀和
时间复杂度：单次修改 O(1)，还原 O(n)
```

```cpp
// 差分修改
d[l] += v;
d[r + 1] -= v;
// 还原
for (int i = 1; i <= n; i++)
    a[i] = a[i - 1] + d[i];
```

**要点**：差分数组长度要多开一位（`MAXN + 2`），防止 `r+1` 越界。

### 3.3 二维前缀和（子矩阵求和，容斥原理）

**适用信号**：在矩阵中多次查询子矩阵的元素之和。

```
建表公式：pre[i][j] = a[i][j] + pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1]
查询公式：sum(x1,y1,x2,y2) = pre[x2][y2] - pre[x1-1][y2] - pre[x2][y1-1] + pre[x1-1][y1-1]
时间复杂度：预处理 O(nm)，单次查询 O(1)
```

**容斥原理是核心**：无论建表还是查询，都要 "加两边、减重叠"。

### 3.4 二维差分（子矩阵修改）

**适用信号**：对矩阵进行多次子矩阵统一加减，最后查询结果。

```
修改公式：d[x1][y1] += v, d[x1][y2+1] -= v, d[x2+1][y1] -= v, d[x2+1][y2+1] += v
还原方式：对 d[][] 做二维前缀和
时间复杂度：单次修改 O(1)，还原 O(nm)
```

**要点**：与一维类似，四个角的加减遵循容斥原理。

### 3.5 前缀异或 / 前缀积（变体）

**前缀异或**：`pre_xor[i] = a[1] ^ a[2] ^ ... ^ a[i]`

区间异或：`xor(l, r) = pre_xor[r] ^ pre_xor[l-1]`

```cpp
for (int i = 1; i <= n; i++)
    pre_xor[i] = pre_xor[i - 1] ^ a[i];
// 查询 [l,r] 的异或和
int ans = pre_xor[r] ^ pre_xor[l - 1];
```

**前缀积**（需考虑 0 的情况）：`pre_mul[i] = a[1] * a[2] * ... * a[i]`

区间积：若无 0，`mul(l, r) = pre_mul[r] / pre_mul[l-1]`；含 0 时需要特殊处理。

### 3.6 前缀和 + 哈希/模运算（同余前缀和）

**适用信号**：求满足某种整除/余数条件的子数组个数。

**核心思路**：若 `pre[r] % k == pre[l-1] % k`，则 `sum(l, r)` 是 `k` 的倍数。

```cpp
// 统计和为 k 倍数的子数组个数
int cnt[MAXK] = {};
cnt[0] = 1;  // 空前缀
long long ans = 0;
for (int i = 1; i <= n; i++) {
    pre[i] = (pre[i - 1] + a[i] % k + k) % k;
    ans += cnt[pre[i]];
    cnt[pre[i]]++;
}
```

**原理**：同余的前缀和配对，相减后余数为 0 → 区间和整除 `k`。

### 子模型总览表

| 模型           | 预处理   | 单次操作 | 核心公式/思路                  |
|----------------|----------|----------|-------------------------------|
| 一维前缀和     | O(n)     | O(1) 查询| `pre[r] - pre[l-1]`          |
| 一维差分       | O(n)     | O(1) 修改| `d[l] += v; d[r+1] -= v`     |
| 二维前缀和     | O(nm)    | O(1) 查询| 容斥四项公式                  |
| 二维差分       | O(nm)    | O(1) 修改| 四角加减容斥                  |
| 前缀异或       | O(n)     | O(1) 查询| `pre[r] ^ pre[l-1]`          |
| 同余前缀和     | O(n)     | O(1) 统计| 哈希计数同余配对              |

### 🟢 入门级 Level 1：前缀和与差分基础

### 题目 1：最大子段和（洛谷 P1115）

> 给定一个长度为 n 的整数序列，求一个连续子段使其元素之和最大。

**分析**：经典最大子段和问题。利用前缀和思路：`sum(l,r) = pre[r] - pre[l-1]`，要使区间和最大，即对每个 `r` 找到最小的 `pre[l-1]`。遍历时维护前缀和的历史最小值即可。

```
状态定义：pre[i] = a[1] + ... + a[i]
转移方程：ans = max(pre[i] - min_pre)，其中 min_pre = min(pre[0..i-1])
边界条件：pre[0] = 0, min_pre 初始 = 0
答案位置：全局 ans
```

**C++ 代码**：

```cpp
const int MAXN = 2e5 + 5;
int a[MAXN];
long long pre[MAXN];

long long solve(int n) {
    pre[0] = 0;
    long long ans = -1e18, min_pre = 0;
    for (int i = 1; i <= n; i++) {
        pre[i] = pre[i - 1] + a[i];
        ans = max(ans, pre[i] - min_pre);
        min_pre = min(min_pre, pre[i]);
    }
    return ans;
}
```

答案位置：`solve(n)` 的返回值

**核心启示**：
- **前缀和转化**：区间和问题 → 两个前缀和的差 → 维护历史极值
- **在线维护**：不需要存储所有前缀和再回溯，边扫描边更新最小值
- **通用技巧**：`max(pre[r] - pre[l-1])` 等价于固定右端点、找最优左端点

### 题目 2：语文成绩（洛谷 P2367）

> 给定 n 个学生的初始成绩，进行 p 次操作，每次对第 l 到第 r 个学生的成绩加上一个值 v（可能为负）。求最终的最低分。

**分析**：典型差分应用。每次区间加 v 用差分 O(1) 完成，p 次操作后做一次前缀和还原，再扫描最小值。

```
核心操作：d[l] += v, d[r+1] -= v
还原方式：前缀和还原得到最终数组
答案位置：还原后数组的最小值
```

**C++ 代码**：

```cpp
const int MAXN = 5e6 + 5;
int a[MAXN];
long long d[MAXN];

long long solve(int n, int p, int ops[][3]) {
    // 初始化差分数组
    for (int i = 1; i <= n; i++)
        d[i] = a[i] - a[i - 1];
    // 执行 p 次区间修改
    for (int k = 0; k < p; k++) {
        int l = ops[k][0], r = ops[k][1], v = ops[k][2];
        d[l] += v;
        d[r + 1] -= v;
    }
    // 前缀和还原并求最小值
    long long mn = 1e18, cur = 0;
    for (int i = 1; i <= n; i++) {
        cur += d[i];
        mn = min(mn, cur);
    }
    return mn;
}
```

答案位置：`solve(n, p, ops)` 的返回值

**核心启示**：
- **差分核心场景**：多次区间修改 + 一次性查询 → 差分是最佳选择
- **差分初始化**：若原数组非零，先从原数组构建差分，再叠加操作
- **空间优化**：还原时可以边求前缀和边统计答案，不需要额外数组

### 🟡 进阶级 Level 2：前缀和与差分的典型应用

### 题目 3：Subsequences Summing to Sevens（洛谷 P3131 [USACO16JAN]）

> 给定 n 个正整数，求最长的连续子序列使其和为 7 的倍数。

**分析**：前缀和 + 模运算的经典应用。`sum(l,r)` 是 7 的倍数当且仅当 `pre[r] % 7 == pre[l-1] % 7`。记录每种余数第一次和最后一次出现的位置，取最大差值。

```
状态定义：pre[i] = (a[1] + ... + a[i]) % 7
关键性质：pre[r] ≡ pre[l-1] (mod 7) → sum(l,r) ≡ 0 (mod 7)
答案位置：同余值第一次和最后一次出现位置之差的最大值
```

**C++ 代码**：

```cpp
const int MAXN = 5e4 + 5;
int a[MAXN];
int first_pos[7], last_pos[7];

int solve(int n) {
    memset(first_pos, -1, sizeof(first_pos));
    memset(last_pos, -1, sizeof(last_pos));
    first_pos[0] = 0;  // pre[0] = 0，余数为 0 出现在位置 0
    int pre = 0, ans = 0;
    for (int i = 1; i <= n; i++) {
        pre = (pre + a[i] % 7) % 7;
        if (first_pos[pre] == -1)
            first_pos[pre] = i;
        else
            ans = max(ans, i - first_pos[pre]);
    }
    return ans;
}
```

答案位置：`solve(n)` 的返回值

**核心启示**：
- **同余前缀和**：当条件涉及整除/模时，对前缀和取模后用哈希/桶分类
- **首尾配对**：要求最长 → 记录每种余数的首次出现位置
- **边界处理**：`pre[0] = 0` 对应余数 0，位置 0 必须初始化

### 题目 4：地毯（洛谷 P3397）

> 在 n×n 的方格上铺 m 条地毯，每条覆盖一个子矩形。问每个格子被多少条地毯覆盖。

**分析**：经典二维差分问题。每铺一条地毯等价于对子矩阵做 +1 操作，所有操作结束后做二维前缀和还原。

```
核心操作：d[x1][y1]++, d[x1][y2+1]--, d[x2+1][y1]--, d[x2+1][y2+1]++
还原方式：二维前缀和还原
答案位置：还原后的 d[i][j] 即为覆盖次数
```

**C++ 代码**：

```cpp
const int MAXN = 1005;
int d[MAXN][MAXN];

void addCarpet(int x1, int y1, int x2, int y2) {
    d[x1][y1]++;
    d[x1][y2 + 1]--;
    d[x2 + 1][y1]--;
    d[x2 + 1][y2 + 1]++;
}

void restore(int n) {
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= n; j++)
            d[i][j] += d[i - 1][j] + d[i][j - 1] - d[i - 1][j - 1];
}
```

答案位置：还原后 `d[i][j]` 即每格的覆盖数

**核心启示**：

### 题目 5：领地选择（洛谷 P2004）

> 在 n×m 的矩阵中选一个 c×c 的子矩阵，使得子矩阵元素之和最大。输出左上角坐标。

**分析**：二维前缀和预处理后，枚举所有合法的 c×c 子矩阵左上角，O(1) 查询每个子矩阵的和，取最大值。

```
预处理：二维前缀和
枚举范围：左上角 (i,j) 满足 i+c-1 <= n, j+c-1 <= m
查询公式：query(i, j, i+c-1, j+c-1)
答案位置：使 query 值最大的 (i,j)
```

**C++ 代码**：

```cpp
const int MAXN = 1005;
int a[MAXN][MAXN];
long long pre[MAXN][MAXN];

void build(int n, int m) {
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            pre[i][j] = a[i][j] + pre[i - 1][j]
                       + pre[i][j - 1] - pre[i - 1][j - 1];
}

long long query(int x1, int y1, int x2, int y2) {
    return pre[x2][y2] - pre[x1 - 1][y2]
         - pre[x2][y1 - 1] + pre[x1 - 1][y1 - 1];
}

// 枚举左上角，返回最优 (ansx, ansy)
void solve(int n, int m, int c, int &ansx, int &ansy) {
    build(n, m);
    long long best = -1e18;
    for (int i = 1; i + c - 1 <= n; i++)
        for (int j = 1; j + c - 1 <= m; j++) {
            long long s = query(i, j, i + c - 1, j + c - 1);
            if (s > best) { best = s; ansx = i; ansy = j; }
        }
}
```

答案位置：`ansx, ansy`（最优子矩阵的左上角坐标）

**核心启示**：
- **二维前缀和 + 枚举**：固定子矩阵大小时，枚举左上角 + O(1) 查询
- **注意边界**：枚举范围要保证子矩阵不越界
- **最优化搜索**：前缀和将 O(n²m²) 暴力降至 O(nm)

### 题目 6：海底高铁（洛谷 P3406）

> 有 n 个城市和 n-1 段铁路（城市 i 到 i+1）。旅行经过 m 个城市（依次访问），每段铁路可以选择"单次购票"或"买票证后免费"。每段铁路有各自的单次票价 a_i、票证费 b_i 和持票后票价 c_i。求最小总花费。

**分析**：先用差分统计每段铁路被经过的次数 `cnt[i]`，然后对每段铁路贪心选择 `min(cnt[i] * a_i, b_i + cnt[i] * c_i)`。注意旅行方向可能从大到小，需要对每步的区间做正确的差分。

```
差分统计：对每步 [min(s,t), max(s,t)-1] 做 +1
贪心决策：每段铁路 cost = min(cnt * a, b + cnt * c)
答案位置：所有段 cost 之和
```

**C++ 代码**：

```cpp
const int MAXN = 1e5 + 5;
int route[MAXN];  // 旅行顺序
int d[MAXN];      // 差分数组
long long a[MAXN], b[MAXN], c[MAXN];

long long solve(int n, int m) {
    // 差分统计每段铁路经过次数
    for (int i = 1; i < m; i++) {
        int l = min(route[i], route[i + 1]);
        int r = max(route[i], route[i + 1]) - 1;
        d[l]++;
        d[r + 1]--;
    }
    // 前缀和还原 + 贪心
    long long ans = 0, cnt = 0;
    for (int i = 1; i < n; i++) {
        cnt += d[i];
        ans += min(cnt * a[i], b[i] + cnt * c[i]);
    }
    return ans;
}
```

答案位置：`solve(n, m)` 的返回值

**核心启示**：
- **差分 + 贪心**：先用差分统计频次，再用频次做决策
- **区间方向**：旅行可能反向走，差分时取 `min/max` 保证区间合法
- **独立决策**：各段铁路互不影响，各自贪心选最优方案

### 题目 7：光骓者的荣耀（洛谷 P5638）

> 有 n 个驿站排成一排，相邻驿站间有一段路程。光骓者可以使用一次传送技能，跳过连续的最多 k 段路程。求从第 1 个驿站到第 n 个驿站的最小总路程。

**分析**：总路程减去"最大的连续 k 段之和"即为答案。用前缀和预处理后，长度为 k 的滑动窗口找最大子段和：`window = pre[i] - pre[i-k]`，遍历取最大值即可。

```
状态定义：pre[i] = d[1] + d[2] + ... + d[i]（d 为段距离）
滑动窗口：window_sum = pre[i] - pre[i-k]
答案位置：pre[n-1] - max(window_sum)
```

**C++ 代码**：

```cpp
const int MAXN = 1e6 + 5;
long long d[MAXN], pre[MAXN];

long long solve(int n, int k) {
    for (int i = 1; i < n; i++)
        pre[i] = pre[i - 1] + d[i];
    long long total = pre[n - 1], mx = 0;
    for (int i = k; i < n; i++)
        mx = max(mx, pre[i] - pre[i - k]);
    return total - mx;
}
```

答案位置：`solve(n, k)` 的返回值

**核心启示**：
- **前缀和 + 滑动窗口**：固定长度的区间最大和，就是 `pre[r] - pre[r-k]` 在滑动中的最大值
- **减法思维**：最小路程 = 总路程 − 最大可跳过路程，将"最小化"转化为"最大化子问题"
- **窗口本质**：滑动窗口是前缀和差值在固定宽度上的线性扫描

### 题目 8：Karen and Coffee（Codeforces CF816B）

> 有 n 个食谱，每个食谱的适宜温度范围为 [l_i, r_i]。温度 t 是"合适的"当且仅当至少 k 个食谱包含 t。给 q 个查询 [a, b]，对每个查询输出 [a,b] 中有多少个合适的温度。

**分析**：两步走——第一步用差分标记 + 前缀和还原得到每个温度被覆盖的次数 cnt[t]；第二步将 `cnt[t] >= k` 标记为 1，再做一次前缀和，支持 O(1) 区间查询合适温度个数。

```
第一步：差分标记覆盖次数 → 前缀和还原
第二步：阈值判定 → 0/1 数组 → 再做前缀和
查询公式：ans = ok_pre[b] - ok_pre[a-1]
```

**C++ 代码**：

```cpp
const int MAXT = 200005;
int cnt[MAXT], ok_pre[MAXT];

void build(int recipes[][2], int n, int k, int T) {
    for (int i = 0; i < n; i++) {
        cnt[recipes[i][0]]++;
        cnt[recipes[i][1] + 1]--;
    }
    for (int t = 1; t <= T; t++)
        cnt[t] += cnt[t - 1];
    for (int t = 1; t <= T; t++)
        ok_pre[t] = ok_pre[t - 1] + (cnt[t] >= k ? 1 : 0);
}

int query(int a, int b) {
    return ok_pre[b] - ok_pre[a - 1];
}
```

答案位置：每次查询调用 `query(a, b)`

**核心启示**：
- **两次前缀和**：第一次还原覆盖次数，第二次支持区间查询——差分与前缀和串联使用
- **阈值转化**：连续数值域上的阈值判断，转化为 0/1 数组后再做前缀和
- **差分 + 前缀和联合**：差分解决"区间批量加"，前缀和解决"区间批量查"，两者无缝衔接

### 题目 9：最大正方形（洛谷 P1387）

> 在一个 n×m 的 01 矩阵中，找一个最大的全 1 正方形，输出边长。

**分析**：使用二维前缀和 + 二分/枚举边长。预处理前缀和后，枚举每个可能的正方形左上角和边长，用 O(1) 查询子矩阵和是否等于边长的平方（全 1）。也可直接枚举边长从大到小，找到第一个满足条件的即可。

```
预处理：二维前缀和
枚举策略：枚举每个位置为右下角，枚举边长 k
判断条件：query(i-k+1, j-k+1, i, j) == k*k
答案位置：满足条件的最大 k
```

**C++ 代码**：

```cpp
const int MAXN = 105;
int a[MAXN][MAXN];
int pre[MAXN][MAXN];

int query(int x1, int y1, int x2, int y2) {
    return pre[x2][y2] - pre[x1 - 1][y2]
         - pre[x2][y1 - 1] + pre[x1 - 1][y1 - 1];
}

int solve(int n, int m) {
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            pre[i][j] = a[i][j] + pre[i - 1][j]
                       + pre[i][j - 1] - pre[i - 1][j - 1];
    int ans = 0;
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            for (int k = ans + 1; i - k + 1 >= 1 && j - k + 1 >= 1; k++) {
                if (query(i - k + 1, j - k + 1, i, j) == k * k)
                    ans = k;
                else
                    break;
            }
    return ans;
}
```

答案位置：`solve(n, m)` 的返回值

**核心启示**：
- **前缀和判定**：子矩阵全 1 ↔ 子矩阵和 == 面积，这是前缀和的经典应用
- **枚举优化**：从当前最优解 +1 开始枚举，一旦不满足就 break
- **DP 对比**：此题更经典的做法是 DP，但前缀和提供了直观的替代思路

### 题目 10：激光炸弹（洛谷 P2280 [HNOI2003]）

> 在 5001×5001 的网格上有若干目标（坐标和价值），用一个 R×R 的正方形炸弹覆盖，求最大能摧毁的目标价值之和。

**分析**：将目标价值填入网格，做二维前缀和，然后枚举所有 R×R 的子矩阵取最大值。注意坐标从 0 开始，需要偏移到从 1 开始。

```
预处理：将目标写入网格，建二维前缀和
枚举范围：所有 R×R 子矩阵
查询公式：query(i-R+1, j-R+1, i, j)
答案位置：所有查询的最大值
```

**C++ 代码**：

```cpp
const int MAXN = 5005;
int grid[MAXN][MAXN];
int pre[MAXN][MAXN];

void build(int N) {
    for (int i = 1; i <= N; i++)
        for (int j = 1; j <= N; j++)
            pre[i][j] = grid[i][j] + pre[i - 1][j]
                       + pre[i][j - 1] - pre[i - 1][j - 1];
}

int query(int x1, int y1, int x2, int y2) {
    x1 = max(x1, 1); y1 = max(y1, 1);
    return pre[x2][y2] - pre[x1 - 1][y2]
         - pre[x2][y1 - 1] + pre[x1 - 1][y1 - 1];
}

int solve(int N, int R) {
    build(N);
    int ans = 0;
    for (int i = R; i <= N; i++)
        for (int j = R; j <= N; j++)
            ans = max(ans, query(i - R + 1, j - R + 1, i, j));
    return ans;
}
```

答案位置：`solve(N, R)` 的返回值

**核心启示**：
- **网格映射**：将离散点坐标映射到网格数组，是二维前缀和的前置步骤
- **坐标偏移**：题目坐标从 0 开始时，+1 偏移到从 1 开始更方便
- **边界限制**：R 可能大于网格大小，需要 `min(R, N)` 处理

### 题目 11：IncDec Sequence（洛谷 P4552 [Poetize6]）

> 给定一个长度为 n 的序列，每次操作可以选择区间 [l,r] 使所有元素加 1 或减 1。问：(1) 最少操作次数使序列所有元素相等；(2) 最终相等序列有多少种可能的值。

**分析**：转化为差分数组思考。所有元素相等 ↔ 差分数组 `d[2..n]` 全部变为 0（`d[1]` 是什么值不影响"全相等"）。每次操作 `[l,r]+1` 等价于 `d[l]+1, d[r+1]-1`；`[l,r]-1` 等价于 `d[l]-1, d[r+1]+1`。

设正差分之和 `pos = Σd[i] (d[i]>0, i≥2)`，负差分之和绝对值 `neg = Σ|d[i]| (d[i]<0, i≥2)`。最少操作 = `max(pos, neg)`（正负配对消除 `min(pos,neg)` 次，剩余的 `|pos-neg|` 次各自与 `d[1]` 或 `d[n+1]` 配对）。最终可能的值 = `|pos - neg| + 1`。

```
差分定义：d[i] = a[i] - a[i-1]
目标：使 d[2..n] 全为 0
最少操作：max(pos, neg)
可能值数：|pos - neg| + 1
```

**C++ 代码**：

```cpp
const int MAXN = 1e5 + 5;
long long a[MAXN];

void solve(int n, long long &ops, long long &kinds) {
    long long pos = 0, neg = 0;
    for (int i = 2; i <= n; i++) {
        long long d = a[i] - a[i - 1];
        if (d > 0) pos += d;
        else neg -= d;
    }
    ops = max(pos, neg);
    kinds = abs(pos - neg) + 1;
}
```

答案位置：`ops` 为最少操作次数，`kinds` 为可能的最终值数量

**核心启示**：
- **差分思维转化**：将"使序列相等"问题转化为"消除差分数组"
- **正负配对**：正差分与负差分可以通过一次操作同时消除
- **计数推导**：剩余的差分只能与边界配对，造成最终值的不确定性

### 题目 12：最大加权矩形（洛谷 P1719）

> 给定一个 n×n 的整数矩阵，求一个子矩阵使其元素之和最大。

**分析**：二维最大子矩阵和，经典做法是"枚举上下边界 + 压缩为一维最大子段和"。用二维前缀和可以在 O(n³) 时间内解决：枚举上边界 `x1` 和下边界 `x2`，将每列的列和压缩为一维数组，对一维数组求最大子段和。

```
枚举策略：枚举上下行边界 x1, x2
列压缩：col[j] = Σ a[x1..x2][j]（可用前缀和 O(1) 算）
一维问题：对 col[] 求最大子段和
答案位置：所有枚举的最大值
```

**C++ 代码**：

```cpp
const int MAXN = 125;
int a[MAXN][MAXN];
int pre[MAXN][MAXN];  // 列方向前缀和

int solve(int n) {
    // 列方向前缀和：pre[i][j] = a[1][j] + ... + a[i][j]
    for (int j = 1; j <= n; j++)
        for (int i = 1; i <= n; i++)
            pre[i][j] = pre[i - 1][j] + a[i][j];

    int ans = -0x3f3f3f3f;
    for (int x1 = 1; x1 <= n; x1++)
        for (int x2 = x1; x2 <= n; x2++) {
            // 压缩后一维最大子段和（Kadane 算法）
            int cur = 0;
            for (int j = 1; j <= n; j++) {
                int col = pre[x2][j] - pre[x1 - 1][j];
                cur = max(col, cur + col);
                ans = max(ans, cur);
            }
        }
    return ans;
}
```

答案位置：`solve(n)` 的返回值

**核心启示**：
- **降维思想**：二维问题 → 枚举一个维度 + 压缩为一维问题
- **前缀和加速列压缩**：`col[j] = pre[x2][j] - pre[x1-1][j]` 实现 O(1) 列和
- **Kadane + 前缀和**：两个经典算法组合，解决二维最大子矩阵
- **复杂度分析**：O(n²) 枚举行对 × O(n) Kadane = O(n³)，在 n≤120 时充分高效

### 题目 13：Load Balancing（洛谷 P3138 [USACO16FEB]）

> N 头牛在二维平面上（坐标均为偶数）。放置一条水平围栏和一条垂直围栏（坐标为奇数），将平面分成四个象限。最小化四个象限中牛数量最大值。

**分析**：坐标范围大但牛数少，先对 x、y 坐标离散化，将牛标记到压缩后的网格上。建二维前缀和后，枚举所有可能的横纵分割线位置，用 O(1) 查询四个象限各自的牛数，取四者最大值的全局最小值。

```
预处理：坐标离散化 + 网格标记 + 二维前缀和
枚举策略：枚举所有横纵分割线组合
查询公式：四象限分别用容斥计算
答案位置：四象限最大值的全局最小值
```

**C++ 代码**：

```cpp
const int MAXN = 1005;
int grid[MAXN][MAXN], pre[MAXN][MAXN];

int query(int x1, int y1, int x2, int y2) {
    if (x1 > x2 || y1 > y2) return 0;
    return pre[x2][y2] - pre[x1 - 1][y2]
         - pre[x2][y1 - 1] + pre[x1 - 1][y1 - 1];
}

int solve(int N, int cx, int cy) {
    for (int i = 1; i <= cx; i++)
        for (int j = 1; j <= cy; j++)
            pre[i][j] = grid[i][j] + pre[i - 1][j]
                       + pre[i][j - 1] - pre[i - 1][j - 1];
    int ans = N;
    for (int i = 1; i < cx; i++)
        for (int j = 1; j < cy; j++) {
            int mx = max({query(1,1,i,j), query(1,j+1,i,cy),
                          query(i+1,1,cx,j), query(i+1,j+1,cx,cy)});
            ans = min(ans, mx);
        }
    return ans;
}
```

答案位置：`solve(N, cx, cy)` 的返回值

**核心启示**：
- **离散化 + 二维前缀和**：坐标范围大但点数少时，先离散化压缩再建前缀和
- **四象限容斥**：一条横线一条纵线分割，四个区域各自用前缀和 O(1) 查询
- **枚举分割线**：离散化后候选位置大幅减少，暴力枚举复杂度可接受

### 题目 14：三步必杀（洛谷 P4231）

> 长度为 n 的数组初始全 0。进行 m 次操作，每次对区间 [l, r] 加上一个等差序列（首项 s，末项 e）。输出最终数组的异或和与最大值。

**分析**：等差序列的一阶差分是常数（公差 d），因此再做一次差分（二阶差分）后，每次操作只需修改 O(1) 个位置。设 `d = (e-s)/(r-l)`，二阶差分操作为：`c[l] += s, c[l+1] += d-s, c[r+1] -= d+e, c[r+2] += e`。最后两次前缀和还原得到原数组。

```
二阶差分操作（4 个点修改）：
  c[l] += s;  c[l+1] += d-s;  c[r+1] -= d+e;  c[r+2] += e
还原方式：两次前缀和
时间复杂度：O(m) 修改 + O(n) 还原
```

**C++ 代码**：

```cpp
const int MAXN = 1e7 + 5;
long long c[MAXN];

void addArith(int l, int r, long long s, long long e) {
    long long d = (r > l) ? (e - s) / (r - l) : 0;
    c[l] += s;
    c[l + 1] += d - s;
    c[r + 1] -= d + e;
    c[r + 2] += e;
}

void restore(long long a[], int n) {
    for (int i = 1; i <= n; i++) c[i] += c[i - 1];
    for (int i = 1; i <= n; i++) { c[i] += c[i - 1]; a[i] = c[i]; }
}
```

答案位置：还原后 `a[1..n]` 中求异或和与最大值

**核心启示**：
- **二阶差分**：等差序列的差分是常数，再差分一次就变成端点操作——高阶差分的核心思想
- **两次还原**：二阶差分需要两次前缀和才能还原到原数组，与"二阶导需要积分两次"类比
- **效率飞跃**：朴素 O(nm) → 二阶差分 O(m + n)，对大数据量（n ≤ 10⁷）至关重要

### 🔴 困难级 Level 4：高级技巧与综合

### 题目 15：上帝造题的七分钟（洛谷 P4514）

> 给定 n×m 的矩阵，初始全 0。在线执行两种操作：(1) 对子矩阵 (x1,y1)~(x2,y2) 统一加 v；(2) 查询子矩阵 (x1,y1)~(x2,y2) 的元素之和。修改与查询交替进行。

**分析**：离线可用二维差分，但在线需"改完立即查"。将二维差分信息用四棵二维树状数组（BIT）维护。对差分数组 d 求二重前缀和得 `S(x,y) = Σ d[i][j]·(x-i+1)·(y-j+1)`，展开后拆成 `Σd`, `Σi·d`, `Σj·d`, `Σij·d` 四项，各用一棵 BIT 维护。

```
维护四棵 BIT：T0(d), T1(i·d), T2(j·d), T3(i·j·d)
修改：二维差分四角操作，同步更新四棵 BIT
查询：S(x,y) = (x+1)(y+1)·T0 - (y+1)·T1 - (x+1)·T2 + T3
答案：容斥 query(x1,y1,x2,y2)
```

**C++ 代码**：

```cpp
const int MAXN = 2050;
int n, m;
long long t[4][MAXN][MAXN];

void bit_add(int k, int x, int y, long long v) {
    for (int i = x; i <= n; i += i & -i)
        for (int j = y; j <= m; j += j & -j)
            t[k][i][j] += v;
}

long long bit_sum(int k, int x, int y) {
    long long s = 0;
    for (int i = x; i > 0; i -= i & -i)
        for (int j = y; j > 0; j -= j & -j)
            s += t[k][i][j];
    return s;
}

void add(int x, int y, long long v) {
    bit_add(0,x,y,v); bit_add(1,x,y,v*x);
    bit_add(2,x,y,v*y); bit_add(3,x,y,v*x*y);
}

void rangeAdd(int x1, int y1, int x2, int y2, long long v) {
    add(x1,y1,v); add(x1,y2+1,-v);
    add(x2+1,y1,-v); add(x2+1,y2+1,v);
}

long long prefSum(int x, int y) {
    return (x+1LL)*(y+1)*bit_sum(0,x,y) - (y+1LL)*bit_sum(1,x,y)
         - (x+1LL)*bit_sum(2,x,y) + bit_sum(3,x,y);
}

long long rangeQuery(int x1, int y1, int x2, int y2) {
    return prefSum(x2,y2) - prefSum(x1-1,y2)
         - prefSum(x2,y1-1) + prefSum(x1-1,y1-1);
}
```

答案位置：`rangeQuery(x1, y1, x2, y2)` 的返回值

**核心启示**：
- **二维差分在线化**：离线差分只能"改完再查"，用 BIT 维护后可"随改随查"
- **代数拆分**：将 `(x-i+1)(y-j+1)` 展开为四项独立求和，各项可用 BIT 维护
- **四棵 BIT**：空间换时间的经典方案，每次操作 O(log n · log m)

### 题目 16：特别行动队（洛谷 P3628 [APIO2010]）

> 将 n 个士兵按顺序分成若干连续组。若一组士兵战斗力之和为 x，则该组的战斗力为 a·x² + b·x + c（其中 a < 0）。最大化所有组的战斗力之和。

**分析**：设 `pre[i]` 为前 i 个士兵的战斗力前缀和，`f[i]` 表示前 i 个士兵的最大战斗力。则 `f[i] = max{ f[j] + a·(pre[i]-pre[j])² + b·(pre[i]-pre[j]) + c }`。展开整理后，令 `Y(j) = f[j] + a·pre[j]² - b·pre[j]`，斜率优化形式为 `Y(j) = 2a·pre[i]·pre[j] + (f[i] - a·pre[i]² - b·pre[i] - c)`。因 a < 0，查询斜率递减，维护上凸包用单调队列即可 O(n) 求解。

```
DP 定义：f[i] = 前 i 人的最大总战斗力
转移方程：f[i] = max{ f[j] + a·(pre[i]-pre[j])² + b·(pre[i]-pre[j]) + c }
斜率优化：维护上凸包，查询斜率 K = 2a·pre[i] 递减
时间复杂度：O(n)
```

**C++ 代码**：

```cpp
const int MAXN = 1e6 + 5;
long long pre[MAXN], f[MAXN];
int q[MAXN];
long long a, b, c;

long long Y(int j) { return f[j] + a * pre[j] * pre[j] - b * pre[j]; }

double slope(int i, int j) {
    return (double)(Y(i) - Y(j)) / (pre[i] - pre[j]);
}

void solve(int n) {
    int h = 0, t = 0;
    q[0] = 0;
    for (int i = 1; i <= n; i++) {
        while (h < t && slope(q[h], q[h + 1]) >= 2.0 * a * pre[i])
            h++;
        int j = q[h];
        long long d = pre[i] - pre[j];
        f[i] = f[j] + a * d * d + b * d + c;
        while (h < t && slope(q[t - 1], q[t]) <= slope(q[t], i))
            t--;
        q[++t] = i;
    }
}
```

答案位置：`f[n]` 为最大总战斗力

**核心启示**：
- **前缀和 + 斜率优化**：前缀和将分组问题转化为 DP，斜率优化将 O(n²) DP 降至 O(n)
- **上凸包维护**：a < 0 使查询斜率递减，单调队列维护上凸包，队首弹出无用决策点
- **DP 提速范式**：当 DP 转移可写成 `f[i] = min/max{ Y(j) - K(i)·X(j) }` 形式时，斜率优化适用

## 五、常见技巧与思路总结

### 5.1 前缀和的"差值"思维

遇到区间和问题时，第一反应是把区间和转化为两个前缀和的差：

```
sum(l, r) = pre[r] - pre[l-1]
```

然后看题目要求什么：最大差？等差？特定模余？→ 对应不同的处理方式。

### 5.2 差分是"端点标记"

差分操作只修改区间的两个端点，相当于在端点做标记：

```
[l, r] 加 v  →  在 l 打 +v 标记，在 r+1 打 -v 标记
```

所有标记打完后，做一次前缀和扫描即可还原。这个"延迟处理"的思路和线段树的 lazy 标记异曲同工。

### 5.3 同余前缀和配对

当题目涉及"和能被 k 整除"时：

```
pre[r] ≡ pre[l-1] (mod k)  ↔  sum(l,r) ≡ 0 (mod k)
```

用桶或哈希表统计每种余数出现的次数，C(cnt, 2) 就是满足条件的区间对数。

### 5.4 二维容斥的"加减减加"口诀

无论是二维前缀和建表、查询还是差分修改，都遵循容斥原理：

```
建表：+当前格 +上方 +左方 -左上
查询：+右下 -上方 -左方 +左上
差分：+左上角 -右上角 -左下角 +右下角
```

记忆技巧：画个田字格，对角同号、相邻异号。

### 5.5 前缀和的变体扩展

| 变体       | 运算 | 区间查询公式                   | 适用场景             |
|------------|------|-------------------------------|---------------------|
| 前缀和     | +    | `pre[r] - pre[l-1]`          | 区间求和            |
| 前缀异或   | ^    | `pre[r] ^ pre[l-1]`          | 区间异或和          |
| 前缀积     | ×    | `pre[r] / pre[l-1]`          | 区间乘积（无 0）    |
| 前缀 GCD   | gcd  | 不满足逆运算，需要稀疏表/线段树| 区间 GCD            |

### 5.6 降维技巧

二维问题经常可以通过"枚举一个维度 + 压缩另一个维度"转化为一维问题：

```
二维最大子矩阵和：
  枚举上下行 O(n²) × 一维 Kadane O(m) = O(n²m)

二维区间和查询：
  直接二维前缀和 O(1) 查询
```

```
                         ┌── 查询类 ──→ 前缀和预处理

```

## 七、终极总结与归纳

### 7.1 前缀和与差分的本质

**一句话**：前缀和是"预存累计量实现快速查询"，差分是"标记端点实现快速修改"——两者互为逆运算，构成了区间操作的完整工具链。

前缀和将 **O(n) 的区间求和** 压缩为 **O(1) 的两点之差**；差分将 **O(n) 的区间修改** 压缩为 **O(1) 的两点标记**。它们的核心都是利用累加/差分的代数性质，把"区间操作"转化为"单点操作"。

在二维场景下，容斥原理是连接前缀和/差分与矩形区域的桥梁。

### 7.2 核心要点

1. **前缀和公式**：`pre[i] = pre[i-1] + a[i]`，查询 `sum(l,r) = pre[r] - pre[l-1]`
2. **差分公式**：`d[i] = a[i] - a[i-1]`，修改 `d[l] += v; d[r+1] -= v`
3. **二维容斥**：无论建表、查询还是修改，都是"对角同号、相邻异号"
4. **互逆关系**：差分数组做前缀和 = 原数组，前缀和数组做差分 = 原数组
5. **组合应用**：前缀和 + 模运算 → 同余配对；前缀和 + 极值维护 → 最大子段和
6. **降维思想**：二维问题枚举一个维度，压缩后用一维方法解决
