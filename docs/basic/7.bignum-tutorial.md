## 一、什么是高精度

在竞赛和算法题中，我们经常遇到需要处理**非常大的整数**的情况——大到 C++ 的 `long long`（最大约 9.2 × 10¹⁸）都无法存储。例如：

- 计算 100! 的精确值（有 158 位）
- 两个 10000 位数相加
- 求 2^10000 的值

这时候就需要用**数组或字符串**模拟手算过程，逐位地进行加、减、乘、除运算，这就是**高精度算法**（也叫大数运算、任意精度运算）。

高精度算法的本质是：**用数组模拟人类手工列竖式计算的过程**。就像小学时列竖式做加减乘除一样——从个位开始，逐位计算，满十进一，不够借一。只不过我们把"纸上的每一位"变成了"数组的每一个元素"。

### 高精度 vs 内置类型

| 特性 | `int` / `long long` | 高精度（数组模拟） |
|------|---------------------|-------------------|
| 位数上限 | 最多 19 位 | 理论无限（受内存限制） |
| 运算速度 | O(1)，硬件直接执行 | O(n) ~ O(n²)，逐位模拟 |
| 实现难度 | 语言内置 | 需手动实现四则运算 |
| 适用场景 | 常规计算 | 超大数运算 |
| 语言支持 | C/C++ 需手写 | Python/Java 内置大数 |

> **核心思想**：把大整数按位拆分存入数组，模拟竖式运算的进位、借位过程。

## 二、核心思想与前置知识

### 2.1 高精度的存储方式（倒序数组/vector）

高精度数的标准存储方式是**倒序存储**：将数字的**个位存在数组下标 0 的位置**，十位存在下标 1，百位存在下标 2，以此类推。

```
数字 "12345" 的倒序存储：

  原始数字:   1   2   3   4   5
              ↓   ↓   ↓   ↓   ↓
  下标:       4   3   2   1   0
  数组:     [ 5,  4,  3,  2,  1 ]
```

**为什么要倒序？**

1. **进位方便**：加法从低位到高位计算，进位自然向高下标传播
2. **扩展方便**：如果最高位产生进位，直接 `push_back` 即可
3. **对齐方便**：两个不同长度的数，低位自动对齐（都从下标 0 开始）

**字符串转 vector 的标准写法：**

```cpp
vector<int> str2vec(string &s) {
    vector<int> A;
    for (int i = s.size() - 1; i >= 0; i--)
        A.push_back(s[i] - '0');
    return A;
}
```

**vector 输出为数字字符串：**

```cpp
void print(vector<int> &A) {
    for (int i = A.size() - 1; i >= 0; i--)
        printf("%d", A[i]);
    printf("\n");
}
```

### 2.2 四则运算核心模板（加、减、乘、除 各给完整骨架代码）

以下四个模板是高精度的核心武器库，务必**理解并熟记**。

#### ▎高精度加法：A + B

```cpp
// A >= 0, B >= 0，均为倒序存储
vector<int> add(vector<int> &A, vector<int> &B) {
    vector<int> C;
    int carry = 0;
    for (int i = 0; i < (int)A.size() || i < (int)B.size() || carry; i++) {
        if (i < (int)A.size()) carry += A[i];
        if (i < (int)B.size()) carry += B[i];
        C.push_back(carry % 10);
        carry /= 10;
    }
    return C;
}
```

核心要点：循环终止条件 `i < A.size() || i < B.size() || carry`，三项缺一不可——`carry` 保证最高位进位不丢。

#### ▎高精度减法：A - B（前提 A ≥ B）

```cpp
// 调用前需确保 A >= B，否则先交换并记录负号
vector<int> sub(vector<int> &A, vector<int> &B) {
    vector<int> C;
    int borrow = 0;
    for (int i = 0; i < (int)A.size(); i++) {
        int t = A[i] - borrow;
        if (i < (int)B.size()) t -= B[i];
        C.push_back((t + 10) % 10);
        borrow = t < 0 ? 1 : 0;
    }
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

核心要点：`(t + 10) % 10` 是统一处理借位的经典技巧；最后必须去前导零。

#### ▎高精度乘法：大数 × 小数

```cpp
// A 为大数（倒序 vector），b 为普通 int
vector<int> mul(vector<int> &A, int b) {
    vector<int> C;
    int carry = 0;
    for (int i = 0; i < (int)A.size() || carry; i++) {
        if (i < (int)A.size()) carry += A[i] * b;
        C.push_back(carry % 10);
        carry /= 10;
    }
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

#### ▎高精度乘法：大数 × 大数

```cpp
// A、B 均为大数（倒序 vector）
vector<int> mul(vector<int> &A, vector<int> &B) {
    vector<int> C(A.size() + B.size(), 0);
    for (int i = 0; i < (int)A.size(); i++)
        for (int j = 0; j < (int)B.size(); j++)
            C[i + j] += A[i] * B[j];
    for (int i = 0; i + 1 < (int)C.size(); i++) {
        C[i + 1] += C[i] / 10;
        C[i] %= 10;
    }
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

核心要点：结果最多 `A.size() + B.size()` 位；先把所有乘积累加到对应位，最后统一进位。

#### ▎高精度除法：大数 ÷ 小数

```cpp
// 返回商（倒序 vector），r 为余数
vector<int> div(vector<int> &A, long long b, long long &r) {
    vector<int> C;
    r = 0;
    for (int i = (int)A.size() - 1; i >= 0; i--) {
        r = r * 10 + A[i];
        C.push_back((int)(r / b));
        r %= b;
    }
    reverse(C.begin(), C.end());
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

核心要点：除法从**高位到低位**计算，结果需要翻转；余数用 `long long` 防溢出。

### 2.3 前导零处理、负数处理、大小比较

**前导零处理**

减法和除法的结果可能出现前导零（例如 `100 - 99 = 001`），必须去除：

```cpp
while (C.size() > 1 && C.back() == 0) C.pop_back();
```

条件 `C.size() > 1` 保证结果为 0 时至少保留一位 `{0}`，不会变成空数组。

**负数处理**

高精度减法中，如果 A < B，则交换后计算，输出时加负号：

```cpp
bool neg = false;
if (!geq(A, B)) { swap(A, B); neg = true; }
vector<int> C = sub(A, B);
if (neg) printf("-");
print(C);
```

**大小比较函数**

```cpp
// 返回 true 表示 A >= B（两数均为非负倒序存储）
bool geq(vector<int> &A, vector<int> &B) {
    if (A.size() != B.size()) return A.size() > B.size();
    for (int i = A.size() - 1; i >= 0; i--)
        if (A[i] != B[i]) return A[i] > B[i];
    return true; // A == B
}
```

比较逻辑：先比位数，位数相同则从高位到低位逐位比较。

## 三、高精度的子模型与分类详解

### 3.1 高精度加法（大数 + 大数）

**适用场景**：两个数都超出 `long long` 范围，需要逐位相加。

```
信号词：大数相加、任意精度求和、位数极大
实现要点：倒序存储 → 逐位加 + 进位 → 自动扩展高位
时间复杂度：O(max(n, m))
```

加法是高精度中最基础的操作，也是许多复杂问题的基石（如阶乘之和、斐波那契大数、数楼梯）。掌握加法模板后，进制泛化也只需将 `10` 替换为对应进制基数。

### 3.2 高精度减法（大数 - 大数，含负数处理）

**适用场景**：两个大数相减，结果可能为负。

```
信号词：大数相减、精确差值
实现要点：先比较大小 → 大减小 → 逐位借位 → 去前导零 → 负号处理
时间复杂度：O(max(n, m))
注意事项：必须先判断大小关系，A < B 时交换并标记负号
```

减法的难点在于**借位传播**和**负数处理**。`(t + 10) % 10` 的技巧避免了 if-else 分支，代码更简洁。

### 3.3 高精度乘法（大数 × 大数、大数 × 小数）

**适用场景**：乘法结果位数极大，如阶乘、幂运算、组合数。

```
信号词：阶乘、连乘、幂次、乘积
大数 × 小数：O(n)，每一位乘以同一个数再进位
大数 × 大数：O(n × m)，模拟竖式乘法，C[i+j] += A[i] * B[j]
```

| 变种 | 复杂度 | 适用场景 |
|------|--------|----------|
| 大数 × 小数 | O(n) | 阶乘递推、简单倍增 |
| 大数 × 大数 | O(n²) | 通用乘法 |
| FFT/NTT 优化 | O(n log n) | 超大数乘法（竞赛进阶） |

### 3.4 高精度除法（大数 ÷ 小数、大数 ÷ 大数）

**大数 ÷ 小数**：从高位到低位模拟长除法，维护余数 `r`，复杂度 O(n)。

**大数 ÷ 大数**：实现较复杂，通常采用"试商法"——对商的每一位用二分或逐减来确定，复杂度 O(n²) 或更高。竞赛中较少直接考查大数÷大数，通常可转化为大数÷小数。

```
信号词：大数除法、整除商、取余
实现要点（大数 ÷ 小数）：
  - 从最高位开始，维护当前余数 r
  - r = r * 10 + A[i]；商的当前位 = r / b；r = r % b
  - 结果翻转 + 去前导零
```

### 3.5 高精度进制转换

**适用场景**：在非十进制下进行大数运算，或将大数在不同进制间转换。

```
信号词：B 进制加法、进制转换、非十进制大数
实现要点：
  - 将模板中的 "10" 替换为 "base"
  - 每一位的范围变为 [0, base-1]
  - 输出时 B > 10 需用字母表示（A=10, B=11, ..., Z=35）
```

```cpp
// B 进制下的高精度加法
vector<int> addBase(vector<int> &A, vector<int> &B, int base) {
    vector<int> C;
    int carry = 0;
    for (int i = 0; i < (int)A.size() || i < (int)B.size() || carry; i++) {
        if (i < (int)A.size()) carry += A[i];
        if (i < (int)B.size()) carry += B[i];
        C.push_back(carry % base);
        carry /= base;
    }
    return C;
}
```

### 3.6 高精度与其他算法结合（贪心、DP、快速幂）

这是竞赛中的高频考点：高精度不是单独出现，而是作为**工具**嵌入到其他算法框架中。

| 结合方式 | 典型题目 | 关键点 |
|----------|----------|--------|
| 贪心 + 高精度 | P1080 国王游戏 | 排序后用高精度计算前缀积和商 |
| DP + 高精度 | P1255 数楼梯 | 递推中间结果超出 long long |
| 快速幂 + 高精度 | P1045 麦森数 | 在高精度数组上实现快速幂 |
| 模拟 + 高精度 | P1009 阶乘之和 | 边算阶乘边累加 |

**核心要领**：先把算法框架写对（贪心策略、DP 转移、快速幂骨架），再把关键运算替换为高精度版本。算法逻辑不变，只是"运算单元"从 `int` 升级为 `vector<int>`。

### 子模型对比总表

| 子模型 | 时间复杂度 | 实现难度 | 核心操作 |
|--------|-----------|----------|----------|
| 加法 | O(n) | ⭐ | 逐位加 + 进位 |
| 减法 | O(n) | ⭐⭐ | 逐位减 + 借位 + 负号 |
| 乘法（大×小） | O(n) | ⭐⭐ | 逐位乘 + 进位 |
| 乘法（大×大） | O(n²) | ⭐⭐⭐ | 交叉累加 + 统一进位 |
| 除法（大÷小） | O(n) | ⭐⭐ | 长除法模拟 |
| 进制转换 | O(n) | ⭐⭐ | 替换进制基数 |
| 结合其他算法 | 视情况 | ⭐⭐⭐⭐ | 替换关键运算 |

## 四、16 道经典例题详解

### 🟢 入门级 Level 1：高精度加减法基础

### 题目 1：A+B Problem（高精）（洛谷 P1601）

> 给定两个非负整数 A、B（位数可达 500 位），求 A + B 的精确值。

**分析**：最经典的高精度入门题。将两个大数以字符串读入，倒序存入 `vector<int>`，然后逐位相加并处理进位。循环条件必须包含 `carry`，确保最高位的进位不会丢失。

```
运算类型：高精度加法
核心操作：逐位相加 + 进位传播
边界条件：最高位进位不能遗漏（循环条件包含 carry）
```

**C++ 代码**：

```cpp
vector<int> add(vector<int> &A, vector<int> &B) {
    vector<int> C;
    int carry = 0;
    for (int i = 0; i < (int)A.size() || i < (int)B.size() || carry; i++) {
        if (i < (int)A.size()) carry += A[i];
        if (i < (int)B.size()) carry += B[i];
        C.push_back(carry % 10);
        carry /= 10;
    }
    return C;
}
// 读入字符串 sa, sb → 倒序存入 A, B → C = add(A, B) → 倒序输出 C
```

答案位置：`add(A, B)` 倒序输出

**核心启示**：
- **循环终止条件是关键**：`i < A.size() || i < B.size() || carry`，三项缺一不可
- **`carry` 身兼两职**：既是进位标志，也是当前位的累加器，一个变量搞定
- **倒序存储是基石**：所有高精度运算的第一步都是倒序存储

### 题目 2：高精度减法（洛谷 P2142）

> 给定两个非负整数 A、B（位数可达 10086 位），求 A − B（结果可能为负）。

**分析**：核心分两步——①比较 A、B 的大小关系；②用大的减小的，若 A < B 则结果加负号。减法实现中，`(t + 10) % 10` 是统一处理借位的经典技巧，无需 if-else 分支。最后必须去前导零。

```
运算类型：高精度减法
核心操作：逐位相减 + 借位处理
边界条件：结果为 0 时输出 "0"（不输出 "-0"）
注意事项：必须去前导零，必须处理负数
```

**C++ 代码**：

```cpp
bool geq(vector<int> &A, vector<int> &B) {
    if (A.size() != B.size()) return A.size() > B.size();
    for (int i = A.size() - 1; i >= 0; i--)
        if (A[i] != B[i]) return A[i] > B[i];
    return true;
}

vector<int> sub(vector<int> &A, vector<int> &B) {
    vector<int> C;
    int borrow = 0;
    for (int i = 0; i < (int)A.size(); i++) {
        int t = A[i] - borrow;
        if (i < (int)B.size()) t -= B[i];
        C.push_back((t + 10) % 10);
        borrow = t < 0 ? 1 : 0;
    }
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
// 若 !geq(A, B)，先输出 "-"，再输出 sub(B, A)
```

答案位置：`geq(A, B) ? sub(A, B) : sub(B, A)`，A < B 时前面加负号

**核心启示**：
- **比较函数是减法的前置条件**：先判断谁大谁小，再用大减小
- **`(t + 10) % 10`** 是无分支借位的经典写法，务必掌握
- **去前导零要保底**：`while (size > 1 && back == 0)` 中 `size > 1` 防止结果 0 变空

### 🟡 进阶级 Level 2：乘除法与综合应用

### 题目 3：A*B Problem（洛谷 P1303）

> 给定两个非负整数 A、B（位数各可达 2000 位），求 A × B。

**分析**：大数乘大数的标准模板。模拟竖式乘法：A 的第 i 位与 B 的第 j 位的乘积，累加到结果的第 `i + j` 位上。关键优化：先把所有乘积累加完毕，最后再统一从低位到高位做进位处理。

```
运算类型：高精度乘法（大数 × 大数）
核心操作：C[i+j] += A[i] * B[j]，最后统一进位
结果长度：最多 len(A) + len(B) 位
边界条件：其中一个为 0 时结果为 0（去前导零自动处理）
```

**C++ 代码**：

```cpp
vector<int> mul(vector<int> &A, vector<int> &B) {
    vector<int> C(A.size() + B.size(), 0);
    for (int i = 0; i < (int)A.size(); i++)
        for (int j = 0; j < (int)B.size(); j++)
            C[i + j] += A[i] * B[j];
    for (int i = 0; i + 1 < (int)C.size(); i++) {
        C[i + 1] += C[i] / 10;
        C[i] %= 10;
    }
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

答案位置：`mul(A, B)` 倒序输出

**核心启示**：
- **先累加再进位**：将乘法和进位分离，代码更简洁、不易出错
- **结果长度预分配**：`A.size() + B.size()` 是安全上界，避免越界
- **统一进位方向**：从下标 0 开始，`C[i+1] += C[i] / 10`，与加法方向一致

### 题目 4：A/B Problem（洛谷 P1480）

> 给定一个不超过 5000 位的非负整数 A 和一个正整数 B（B ≤ 10⁹），求 A ÷ B 的商（整除）。

**分析**：大数除以小数，使用长除法。与加减乘不同，除法从**高位到低位**处理：维护当前余数 `r`，每次把下一位"拉下来"（`r = r * 10 + A[i]`），商为 `r / b`，更新余数 `r = r % b`。注意 B 可达 10⁹，`r * 10` 可能溢出 int，需用 `long long`。

```
运算类型：高精度除法（大数 ÷ 小数）
核心操作：从高位到低位，r = r * 10 + A[i]
注意事项：余数 r 需用 long long 防溢出
边界条件：商为 0 时输出 "0"
```

**C++ 代码**：

```cpp
vector<int> div(vector<int> &A, long long b, long long &r) {
    vector<int> C;
    r = 0;
    for (int i = (int)A.size() - 1; i >= 0; i--) {
        r = r * 10 + A[i];
        C.push_back((int)(r / b));
        r %= b;
    }
    reverse(C.begin(), C.end());
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

答案位置：`div(A, b, r)` 倒序输出（`r` 中存储余数）

**核心启示**：
- **除法方向特殊**：加减乘从低位到高位，唯独除法从高位到低位
- **余数类型要注意**：B 可达 10⁹，`r * 10` 最大约 10¹⁰，必须用 `long long`
- **结果需翻转**：因为从高位开始 push_back，结果顺序是反的

### 题目 5：阶乘之和（洛谷 P1009）

> 求 S = 1! + 2! + 3! + ⋯ + n!（n ≤ 50）的精确值。

**分析**：50! 约有 65 位数字，远超 `long long` 的 19 位上限。需要用高精度乘法（大数 × 小数）递推阶乘，同时用高精度加法逐项累加。关键优化：`i! = (i-1)! × i`，不要每次从头算。

```
运算类型：高精度乘法（大数 × 小数）+ 高精度加法
核心操作：fac = fac * i，sum = sum + fac
边界条件：fac 初始为 {1}，sum 初始为 {0}
```

```cpp
}
```

### 题目 7：[NOIP2012] 国王游戏（洛谷 P1080）

> 国王和 n 个大臣站成一排。每个人左手写数字 a、右手写数字 b。每位大臣获得的金币数 = 他前面所有人（包括国王）左手数字的乘积 ÷ 他自己右手的数字（取整除）。求一种排列方式使得获得金币最多的大臣，其金币数最小化。

**分析**：经典贪心问题。用**邻项交换法**可以证明：按 `a[i] × b[i]` 从小到大排序是最优策略。排序后，需维护"前缀乘积"——由于每个 a 最大 10000，n 最大 1000，前缀乘积可达 10000¹⁰⁰⁰，必须用高精度。需要高精度乘法（大数 × 小数）和高精度除法（大数 ÷ 小数）以及比较函数。

```
算法类型：贪心 + 高精度乘法 + 高精度除法
核心贪心：按 a[i] * b[i] 升序排列（邻项交换法证明）
核心运算：维护前缀积（大数×小数），除以 b[i]（大数÷小数），取最大值
```

**C++ 代码**：

```cpp
int n;
int a[1005], b[1005];

vector<int> prefix = {1};
vector<int> ans = {0};

void solve() {
    // 按 a[i]*b[i] 升序排序（下标 1..n 为大臣，下标 0 为国王）
    // 排序后：
    prefix = mul(prefix, a[0]); // 初始前缀积 = 国王的 a 值

        if (!geq(ans, coins)) ans = coins; // ans = max(ans, coins)
        prefix = mul(prefix, a[i]);

}
// mul(大数, int)、div(大数, long long)、geq 均使用第二节模板
```

- **贪心 + 高精度是 NOIP 经典组合**：贪心确定顺序，高精度处理大数运算
- **邻项交换法**：假设交换相邻两人，比较交换前后的最大金币数即可证明排序正确性
- **模板复用的威力**：此题同时用到乘法、除法、比较三个高精度模板

### 题目 8：税收与补贴问题（洛谷 P1023）

```
算法类型：模拟 + 精确计算
核心操作：遍历价格档位，找供需平衡点
注意事项：税收使供给曲线上移，补贴使供给曲线下移
```

**C++ 代码**：

```cpp

    int p0 = 0;
    for (int i = 0; i < n; i++)

}
```

### 题目 9：麦森数（洛谷 P1045）

> 给定正整数 P（P ≤ 3100000），求梅森数 2^P − 1 的位数以及最后 500 位数字。

**分析**：位数可以用数学公式直接计算：`digits = floor(P × log10(2)) + 1`。最后 500 位需要用**高精度快速幂**：在固定长度为 500 的数组上做快速幂运算，自动丢弃高于 500 位的部分（天然截断）。数组上的乘法是 O(500²) 的，快速幂需要 O(log P) 次乘法，总复杂度约 O(500² × log P)。

```
算法类型：快速幂 + 高精度乘法
位数公式：digits = floor(P × log10(2)) + 1
核心操作：在 500 位数组上实现平方与乘法
边界条件：最后执行 -1 操作（res[0]--）
```

**C++ 代码**：

```cpp
const int LEN = 500;
int res[LEN + 5], base_arr[LEN + 5], tmp[LEN + 5];

void bigMul(int a[], int b[]) {
    memset(tmp, 0, sizeof(tmp));
    for (int i = 0; i < LEN; i++)
        for (int j = 0; j < LEN && i + j < LEN; j++)
            tmp[i + j] += a[i] * b[j];
    for (int i = 0; i < LEN; i++) {
        if (i + 1 < LEN) tmp[i + 1] += tmp[i] / 10;
        tmp[i] %= 10;
    }
    memcpy(a, tmp, sizeof(int) * LEN);
}

void solve(int P) {
    res[0] = 1;           // 结果初始为 1
    base_arr[0] = 2;      // 底数初始为 2
    while (P) {
        if (P & 1) bigMul(res, base_arr);
        bigMul(base_arr, base_arr);
        P >>= 1;
    }
    res[0]--;             // 2^P - 1
}
// 位数 = (int)(P * log10(2.0)) + 1
// 最后 500 位 = res[499], res[498], ..., res[0]
```

答案位置：位数由公式计算；最后 500 位从 `res[499]` 到 `res[0]` 依次输出

**核心启示**：
- **快速幂不止用于取模**：在高精度数组上同样可以实现快速幂
- **固定数组长度 = 天然截断**：只需后 500 位，数组开 500，高位自动丢弃
- **数学公式省大力**：位数用 `P × log10(2)` 一行搞定，无需高精度计算

### 题目 10：B 进制星球（洛谷 P1604）

> 在 B 进制星球上（2 ≤ B ≤ 36），给定两个 B 进制非负整数，求它们的和（以 B 进制输出）。

**分析**：将标准十进制高精度加法模板中的 `% 10` 和 `/ 10` 替换为 `% base` 和 `/ base` 即可。需要注意输入输出的字符映射：当 B > 10 时，用大写字母 A~Z 表示 10~35。另外，两数均为 0 时需特判输出 "0"。

```
运算类型：高精度加法（B 进制版）
核心操作：carry 的进位基数从 10 改为 B
注意事项：字符与数值的转换（0-9 → '0'-'9'，10-35 → 'A'-'Z'）
边界条件：结果为 0 时输出 "0"
```

**C++ 代码**：

```cpp
int charToDigit(char c) {
    if (c >= '0' && c <= '9') return c - '0';
    return c - 'A' + 10;
}

char digitToChar(int d) {
    if (d < 10) return '0' + d;
    return 'A' + d - 10;
}

vector<int> addBase(vector<int> &A, vector<int> &B, int base) {
    vector<int> C;
    int carry = 0;
    for (int i = 0; i < (int)A.size() || i < (int)B.size() || carry; i++) {
        if (i < (int)A.size()) carry += A[i];
        if (i < (int)B.size()) carry += B[i];
        C.push_back(carry % base);
        carry /= base;
    }
    if (C.empty()) C.push_back(0);
    return C;
}
// 输入：B 进制字符串倒序存为 vector<int>（每位用 charToDigit 转换）
// 输出：倒序遍历 C，每位用 digitToChar 转换后输出
```

答案位置：`addBase(A, B, base)` 倒序输出，每位用 `digitToChar` 转换

**核心启示**：
- **进制泛化只需改两处**：`% 10` → `% base`、`/ 10` → `/ base`
- **字符映射必须双向**：输入 `charToDigit`，输出 `digitToChar`
- **模板的可扩展性**：一套加法框架适应所有进制，体现高精度的通用性

### 题目 11：引水入城（洛谷 P1514）

> 给定 n×m 的网格地图，每格有海拔值。水从第一行出发，只能流向海拔更低的相邻格。对每个首行城市做记忆化搜索，统计流向末行各城市的路径方案数。当网格较大时方案数远超 long long，需用高精度累加。

**分析**：水只能从高处流向低处，构成 DAG（有向无环图），路径计数用记忆化 DFS 即可。从首行每个城市出发，递归统计到达末行的路径数。由于路径数呈指数增长（500×500 网格可产生上百位的方案数），必须用高精度加法逐步累加。这是"搜索/DP + 高精度"的典型结合模式。

```
算法类型：记忆化 DFS + 高精度加法
核心操作：DAG 路径计数，dp[x][y] = Σ dp[nx][ny]
关键点：方案数溢出 long long，用 vector<int> 存储并调用 add()
时间复杂度：O(n·m)，每个格子仅计算一次
```

**C++ 代码**：

```cpp
vector<int> dp[505][505];
int h[505][505], vis[505][505], n, m;
int dx[]={0,0,1,-1}, dy[]={1,-1,0,0};

vector<int> dfs(int x, int y) {
    if (x == n) return dp[x][y] = {1};
    if (vis[x][y]) return dp[x][y];
    vis[x][y] = 1;
    dp[x][y] = {0};
    for (int d = 0; d < 4; d++) {
        int nx = x+dx[d], ny = y+dy[d];
        if (nx<1||nx>n||ny<1||ny>m) continue;
        if (h[nx][ny] < h[x][y])
            dp[x][y] = add(dp[x][y], dfs(nx, ny));
    }
    return dp[x][y];
}
// 对首行每个城市调用 dfs(1, j)，累加到总方案数
```

答案位置：累加所有 `dfs(1, j)` 的结果，倒序输出

**核心启示**：
- **搜索 + 高精度是常见组合**：DAG 路径计数呈指数增长，高精度不可避免
- **记忆化避免重复**：`vis` 标记确保每个格子只计算一次，复杂度 O(n·m)
- **模板无缝嵌入**：只需将 `+=` 替换为 `add()`，算法框架完全不变

### 题目 12：阿基里斯与乌龟（洛谷 P2388）

> 阿基里斯以速度 v₁ 追赶前方距离 d 的乌龟（速度 v₂ < v₁）。每轮阿基里斯跑到乌龟上一轮所在位置，乌龟同时前进。模拟 k 轮后求两者的精确距离差。由于距离按比例 v₂/v₁ 缩小，需高精度分数运算保持精度。

**分析**：每轮追赶后，距离变为 `d × v₂ / v₁`。经过 k 轮后距离为 `d × (v₂/v₁)^k`。浮点数在 k 较大时精度不够，需用高精度模拟分数运算：分子 = d × v₂^k，分母 = v₁^k，最终用大数÷大数求商。这是高精度在模拟/数学题中的典型应用——**用分数避免浮点误差**。

```
算法类型：高精度分数运算 + 模拟
核心操作：分子分母分别做高精度幂运算，最后大数÷大数求商
关键点：保持分数精确表示，避免浮点累积误差
依赖模板：大数×小数、大数÷大数
```

**C++ 代码**：

```cpp
vector<int> bigpow(int base, int k) {
    vector<int> res = {1};
    for (int i = 0; i < k; i++)
        res = mul(res, base);
    return res;
}

void solve(int d, int v1, int v2, int k) {
    vector<int> numer = mul(bigpow(v2, k), d); // d * v2^k
    vector<int> denom = bigpow(v1, k);          // v1^k
    vector<int> R;
    vector<int> dist = divBig(numer, denom, R); // 大数÷大数
}
// mul(大数, int) 使用 2.2 节模板
// divBig(大数, 大数) 使用题目 13 的试商法模板
```

答案位置：`dist` 为商（整数部分），`R` 为余数，倒序输出

**核心启示**：
- **分数保精度**：分子/分母分别高精度运算，彻底避免浮点累积误差
- **幂运算复用乘法模板**：`bigpow` 只是反复调用大数×小数，模板组合即可
- **模拟题常需高精度**：当迭代次数多、比值非整数时，精度问题不可忽视

### 题目 13：高精度除法——大数÷大数（洛谷 P1517）

> 给定两个正整数 A、B（位数可达数千位），求 A ÷ B 的商和余数。这是高精度四则运算中实现最复杂的操作，需要"试商法"逐位确定商的每一位。

**分析**：大数÷大数的核心是**试商法**——从商的最高位开始，每一位通过二分查找确定：维护当前余数 R（初始为 0），每次从 A 的高位取一位追加到 R 的低位，然后在 [0, 9] 内二分查找最大的 q 使得 `q × B ≤ R`，该 q 就是商的当前位。更新 `R = R - q × B` 后继续处理下一位。

```
运算类型：高精度除法（大数 ÷ 大数）
核心操作：逐位试商，每位用二分查找确定 [0,9]
时间复杂度：O(n² × log10)，n 为被除数位数
依赖模板：高精度乘法（大数×小数）、减法、比较
```

**C++ 代码**：

```cpp
vector<int> divBig(vector<int> &A, vector<int> &B, vector<int> &R) {
    vector<int> Q;
    R = {0};
    for (int i = (int)A.size()-1; i >= 0; i--) {
        R.insert(R.begin(), A[i]);
        while (R.size() > 1 && R.back() == 0) R.pop_back();
        int lo = 0, hi = 9, q = 0;
        while (lo <= hi) {
            int mid = (lo + hi) / 2;
            vector<int> t = mul(B, mid);
            if (geq(R, t)) { q = mid; lo = mid + 1; }
            else hi = mid - 1;
        }
        Q.push_back(q);
        vector<int> t = mul(B, q);
        R = sub(R, t);
    }
    reverse(Q.begin(), Q.end());
    while (Q.size() > 1 && Q.back() == 0) Q.pop_back();
    return Q;
}
```

答案位置：`Q` 为商，`R` 为余数，均为倒序 vector，倒序输出

**核心启示**：
- **试商法 = 逐位确定 + 二分**：每一位取值范围 [0,9]，二分查找 4 次即可确定
- **依赖已有模板**：`mul`（大数×小数）、`sub`、`geq` 三个基础模板协同工作
- **这是高精度四则运算的最后一块拼图**：掌握后，任意大数的四则运算均可处理

### 题目 14：上白泽慧音（洛谷 P1726）

> 当大数位数超过万位时，逐位运算的常数因子过大，需要**压位优化**。压位技术将多位十进制数字压入一个数组元素（如万进制每元素存 4 位），大幅减少数组长度和运算次数。本题因涉及极大数的反复运算，需用压位高精度加速。

**分析**：普通高精度每个数组元素存一位十进制数字，万进制（BASE=10000）则每个元素存 4 位（值域 0~9999）。数组长度缩短为 1/4，加法循环次数减少 4 倍，乘法常数降低 16 倍。关键修改点：①进制基数从 10 改为 10000；②字符串转数组时按 4 位一组切割；③输出时除最高段外，每段需用 `%04d` 补前导零。

```
算法类型：压位高精度（万进制优化）
核心操作：BASE=10000，每个数组元素存 4 位十进制
优化效果：数组长度 ÷4，加法 ×4 提速，乘法 ×16 提速
注意事项：输出时非最高段必须补前导零至 4 位
```

**C++ 代码**：

```cpp
const int BASE = 10000, DLEN = 4;

vector<int> compress(string &s) {
    vector<int> A;
    for (int i = s.size(); i > 0; i -= DLEN) {
        int st = (i - DLEN > 0) ? i - DLEN : 0;
        A.push_back(stoi(s.substr(st, i - st)));
    }
    return A;
}

vector<int> addComp(vector<int> &A, vector<int> &B) {
    vector<int> C;
    int carry = 0;
    for (int i = 0; i < (int)A.size() || i < (int)B.size() || carry; i++) {
        if (i < (int)A.size()) carry += A[i];
        if (i < (int)B.size()) carry += B[i];
        C.push_back(carry % BASE);
        carry /= BASE;
    }
    return C;
}
// 输出：最高段直接输出，其余段用 printf("%04d", C[i]) 补前导零
```

答案位置：`addComp(A, B)` 倒序输出，注意每段格式化

**核心启示**：
- **压位是工程优化的典范**：算法逻辑完全不变，只改存储粒度，性能提升数倍
- **代价是输出复杂度增加**：非最高段必须补前导零，`printf("%04d")` 不可省略
- **可进一步压至亿进制**（BASE=10⁸）：每元素存 8 位，但需注意 int 乘法溢出，改用 `long long`

### 题目 15：排列计数（洛谷 P4071）

> 求 n 个元素的排列中，恰好有 m 个元素在原位（即不动点恰好为 m 个）的排列总数。答案为 C(n, m) × D(n−m)，其中 D(k) 是 k 个元素的错排数。当 n 较大时结果远超 long long，需高精度。

**分析**：先从 n 个位置中选 m 个作为不动点（组合数 C(n,m)），剩余 n−m 个元素必须全部错排（错排数 D(k)）。错排递推公式：`D(0)=1, D(1)=0, D(k) = (k-1)·(D(k-1)+D(k-2))`。组合数用阶乘计算：`C(n,m) = n! / (m! · (n-m)!)`。所有中间量都需高精度存储，最终结果用高精度大数乘法得到。

```
算法类型：组合数学 + 高精度乘除
核心公式：ans = C(n,m) × D(n-m)
错排递推：D(k) = (k-1) × (D(k-1) + D(k-2))
依赖模板：高精度阶乘、大数÷大数、大数×大数
```

**C++ 代码**：

```cpp
vector<int> fac[1005], derang[1005];

void precompute(int n) {
    fac[0] = {1};
    for (int i = 1; i <= n; i++)
        fac[i] = mul(fac[i-1], i);
    derang[0] = {1}; derang[1] = {0};
    for (int i = 2; i <= n; i++) {
        vector<int> t = add(derang[i-1], derang[i-2]);
        derang[i] = mul(t, i - 1);
    }
}

vector<int> solve(int n, int m) {
    vector<int> R;
    vector<int> comb = divBig(fac[n], fac[m], R);
    comb = divBig(comb, fac[n-m], R);
    return mulBig(comb, derang[n-m]);
}
// D(k) = (k-1)*(D(k-1)+D(k-2))，答案 = C(n,m) * D(n-m)
```

答案位置：`solve(n, m)` 倒序输出

**核心启示**：
- **组合数学 + 高精度是竞赛高频组合**：组合数和错排数增长极快，高精度不可避免
- **错排公式要熟记**：`D(k) = (k-1)·(D(k-1)+D(k-2))`，边界 D(0)=1, D(1)=0
- **模板的威力再次体现**：阶乘用大数×小数，组合数用大数÷大数，全靠模板组装

### 题目 16：A*B Problem 升级版（洛谷 P1919）

> 给定两个位数可达 10⁶ 的非负整数 A、B，求 A × B。朴素 O(n²) 乘法超时，需用 FFT（快速傅里叶变换）将乘法优化至 O(n log n)。

**分析**：FFT 优化大数乘法的核心思想：将两个大数视为多项式的系数（每一位对应一个系数），利用 FFT 在 O(n log n) 时间内完成多项式乘法，最后对结果数组逐位进位还原为十进制大数。步骤：①将 A、B 转为复数数组；②对两个数组做 FFT 正变换；③逐点相乘；④做 FFT 逆变换；⑤取实部四舍五入并逐位进位。

```
算法类型：FFT + 高精度乘法
核心思想：大数乘法 = 多项式乘法 = FFT 点值乘法
时间复杂度：O(n log n)，n 为总位数
适用场景：位数超过 10⁴ 时，FFT 远快于 O(n²) 暴力
```

**C++ 代码**：

```cpp
typedef complex<double> cd;

void fft(vector<cd> &a, int inv) {
    int n = a.size();
    for (int i = 1, j = 0; i < n; i++) {
        int bit = n >> 1;
        for (; j & bit; bit >>= 1) j ^= bit;
        j ^= bit;
        if (i < j) swap(a[i], a[j]);
    }
    for (int len = 2; len <= n; len <<= 1) {
        double ang = 2 * acos(-1.0) / len * inv;
        cd wn(cos(ang), sin(ang));
        for (int i = 0; i < n; i += len) {
            cd w(1);
            for (int j = 0; j < len/2; j++, w *= wn) {
                cd u = a[i+j], v = a[i+j+len/2] * w;
                a[i+j] = u + v;
                a[i+j+len/2] = u - v;
            }
        }
    }
    if (inv == -1) for (auto &x : a) x /= n;
}
// 用法：A, B 转 cd 数组 → fft(fa,1), fft(fb,1) → 逐点乘
//       → fft(fa,-1) → round 取整 → 逐位进位得结果
```

答案位置：逆变换后取 `round(fa[i].real())` 逐位进位，倒序输出

**核心启示**：
- **FFT 将乘法从 O(n²) 降至 O(n log n)**：位数超过万位时，性能差距巨大
- **精度需注意**：double 的精度约 15~16 位有效数字，超大数可能需改用 NTT（数论变换）
- **这是算法竞赛中的分水岭技巧**：掌握 FFT 优化乘法，标志着进入高阶选手行列

## 五、常见技巧与思路总结

### 5.1 倒序存储是一切的基础

所有高精度运算都基于倒序存储（低位在前），这使得：
- 进位方向与下标增长方向一致
- 扩展最高位只需 `push_back`
- 两个不同长度的数自动低位对齐

**记住**：读入字符串后第一件事就是倒序存入 vector。

### 5.2 统一进位/借位的无分支写法

```cpp
// 加法进位：carry 做累加器
carry += A[i];
C.push_back(carry % 10);
carry /= 10;

// 减法借位：(t + 10) % 10 统一处理
C.push_back((t + 10) % 10);
borrow = t < 0 ? 1 : 0;
```

`(t + 10) % 10` 的妙处：当 `t >= 0` 时等于 `t`，当 `t < 0`（如 -3）时等于 `7`，恰好是借位后的正确结果。

### 5.3 前导零的安全去除

```cpp
while (C.size() > 1 && C.back() == 0) C.pop_back();
```

`C.size() > 1` 这个条件**绝不能省略**，否则结果为 0 时会把唯一的 `0` 也去掉，导致空数组。

### 5.4 乘法的"先累加再进位"策略

大数 × 大数时，先把所有 `A[i] * B[j]` 累加到 `C[i+j]`，最后再统一进位。这比边乘边进位更简洁，中间值可能较大（如 9×9 累加多次），但 `int` 完全可以容纳（2000 位数相乘时单个位置最多累加 2000 × 81 ≈ 162000，远小于 int 上限）。

### 5.5 除法的方向特殊性

加法、减法、乘法都从**低位到高位**处理，但除法必须从**高位到低位**——这是因为长除法需要先确定最高位的商。记住：除法结果需要 `reverse` 后才是标准倒序存储。

### 5.6 封装函数，像普通运算一样调用

将高精度操作封装为独立函数后，解题时可以像写普通算术一样：

```cpp
fac = mul(fac, i);       // fac *= i
sum = add(sum, fac);     // sum += fac
ans = div(prefix, b, r); // ans = prefix / b
```

这样算法逻辑一目了然，高精度的复杂性被封装在函数内部。

### 5.7 特殊值与边界检查清单

| 场景 | 需要检查 |
|------|----------|
| 加法 | 最高位进位（循环条件含 carry） |
| 减法 | A < B 时的负号；结果为 0 不输出负号 |
| 乘法 | 其中一个为 0 时结果为 0；前导零 |
| 除法 | 除数不为 0；商为 0 时正确输出 |
| 所有运算 | 前导零去除后至少保留一位 |

## 七、终极总结与归纳

### 7.1 高精度的本质

**高精度的本质是：用数组模拟人类手工竖式运算的每一步。**

它不是一种独立的"算法思想"，而是一种**实现工具**——当问题的数值范围超出内置类型时，用数组逐位存储、逐位运算。高精度的难点不在于思维深度，而在于**实现的精确性**和**模板的熟练度**。一旦掌握四大模板（加、减、乘、除），遇到任何需要大数运算的题目，都可以像搭积木一样组装出解法。

### 7.2 核心要点

1. **倒序存储**：个位在 `[0]`，高位在末尾，方便进位扩展和低位对齐
2. **四大模板**：加、减、乘、除各一个函数，独立封装、可自由组合
3. **进位/借位统一处理**：加法用 `carry`，减法用 `(t+10)%10`，无分支写法
4. **前导零必须去除**：`while (size > 1 && back == 0) pop_back()`，保留至少一位
5. **负数需特判**：减法中 A < B 时先交换再计算，输出前加负号
6. **模板可组合**：复杂问题（贪心、DP、快速幂）通过组合基础模板解决
